摘要

语言契约论是一种彻底的经验主义语言观，其核心主张直指语言本质——人类语言是社群为实现沟通协作而达成的后天符号约定，不存在任何先天属性与先验逻辑。本文首先阐明语言契约论的基本命题，将“语言符号”广义地界定为任何承载信息的物理信号模式——光、声、电、化学、运动皆在其中。进而揭示语言契约论的底层逻辑根基——增强型逻辑律，包含同一律、等同律、契约律三条核心法则。通过梳理语言从生物本能到社会契约的演化历程——从老虎的脑内电信号、蜜蜂的舞蹈，到岩画、象形文字、现代语言与网络流行语——本文论证语言契约论可以解释全部语言现象，从而确立其作为语言本质理论的完备性。

**关键词** ：语言契约论；增强型逻辑律；同一律；等同律；契约律；经验主义

一、引言

1.1 问题的提出

语言作为人类独有的交际工具与思维载体，其本质与起源始终是语言学、哲学、认知科学领域的核心争议点。自现代语言学诞生以来，学者们一直在追问一个根本问题：人类为什么能够通过语言进行沟通？

1.2 前人理论的困境

20世纪初，索绪尔提出语言是一个“符号系统”，“能指”与“所指”之间的关系是任意的，语言的意义由符号之间的差异决定。20世纪中叶，乔姆斯基转向“先天论”，认为人类大脑天生具有“普遍语法”。20世纪后期，维特根斯坦提出语言的意义在于“使用”，语言像游戏一样，规则在使用中形成。

这些理论的共同困境在于：它们都在试图用“复杂”解释“简单”。索绪尔描述了“是什么”，但没有回答“为什么可能”；乔姆斯基的“普遍语法”本身是一个不可证伪的假设；维特根斯坦描述了“现象”，但没有解释“本质”。

1.3 本文的路径

语言契约论以截然不同的路径回应这一根本问题。本文将从四个层面展开论述：首先阐明语言契约论的基本命题；其次揭示其底层逻辑根基——增强型逻辑律；再次梳理语言从生物本能到社会契约的演化历程；最后论证其完备性与解释力。

二、语言契约论的基本命题

2.1 语言符号的广义定义

**语言符号，广义而言，是承载信息的物理信号模式。**

这种物理信号可以有多种形式：

- **光信号** ：文字、图形、手势的影子、表情——以光为媒介；
- **声信号** ：口语、动物的叫声——以声波为媒介；
- **电信号** ：神经脉冲、计算机代码、脑内电活动——以电为媒介；
- **化学信号** ：信息素、气味标记——以化学分子为媒介；
- **运动信号** ：身体姿态、舞蹈、肢体动作——以运动为媒介。

从独居老虎脑内的神经电脉冲，到蜜蜂的舞蹈，到人类的语言——它们在物理层面都是不同的信号模式，但在本质上都是一样的： **用特定的物理信号模式去指代特定的对象或事件。**

2.2 核心命题

语言契约论的核心命题可以表述为：

**语言是用符号指代对象，建立符号和对象之间的一一映射。**

这一映射之所以可能，唯一的原因是社群成员之间对“符号与意义的对应关系”达成了共同约定。

2.3 语言的工作流程与引号的使用

**语言是用符号指代对象。** 这一过程包含两个方向：

1. **输出过程（说/写）** ：说话者或写作者将要表达的对象翻译成语言符号——把心里的对象转化为声音或文字。
2. **接收过程（听/读）** ：倾听者或读者收到语言符号后，大脑自动将符号翻译回对象——把声音或文字还原为所指的事物。

在正常行文中，当一个人读到“太阳出来了”，他的大脑处理的是“太阳”这个符号所指向的那个天体，而不是“太阳”这两个汉字本身的形状。也就是说， **在正常使用语言时，符号退居幕后，对象呈现在前台。**

然而，当我们不是在使用语言，而是在 **谈论语言本身** 时，情况就不同了。我们可能需要讨论某个符号本身（它的字形、读音、拼写），而不是它所指代的对象。这时候，我们需要一种方式来表示“我在谈论这个符号自身”。

**约定：当我们要强调语言符号本身（字符串、声音模式、物理信号），而不是它所指代的对象时，就用双引号将其括起来。**

例如，“晨星”指代早晨最亮的星。——这里的双引号表明我们是在谈论“晨星”这个符号本身，它所指代的对象是早晨最亮的星。如果不用双引号，写成“晨星指代早晨最亮的星”，在严格意义上是不准确的——因为不加引号的晨星就是那个对象本身，而对象本身并不“指代”什么，只有符号才指代对象。

需要注意的是，引号在语言中有多种用法。除了上述用以区分“符号本身”与“符号所指代的对象”之外，引号还可以用于引用、强调等。本文的讨论聚焦于第一种用法，这是语言契约论的基础。

因此，完整的规则是：

- **不加引号** → 符号正常使用，指向它所代表的对象；
- **加引号** → 谈论符号本身，而非它所指向的对象。

这一区分是语言契约论的基础。所有关于符号、意义、指称的讨论，都必须首先明确：我们此刻是在使用符号（指向对象），还是在谈论符号本身。

2.4 语言是契约

语言契约论的最简表达式是： **约好了，就懂；没约好，就不懂。**

原始人A指着太阳发出“日”的音，原始人B点头表示理解。这就是一份语言契约的诞生。此后，“日”这个声信号与太阳这个对象之间的对应关系，在A和B之间成立。

语言契约论因此有三个核心推论：

1. **语言是后天的** ——不是先天内置的模块，而是后天签订的合同；
2. **语言是约定的** ——符号与对象之间没有必然联系，只有约定关系；
3. **语言是传承的** ——每个人学会语言，就是继承了一份跨越千年的契约。

2.5 约定的最小单元与语言共同体

约定的最小单元发生在 **说话者** 与 **倾听者** 两个主体之间。

科技论文在开篇定义术语——约定成立。法律文件在“定义”条款中明确术语含义——约定成立。原始人指着太阳发出“日”的音——约定成立。

当一份约定被不止两个人认可，而是被一群人共同接受时，这群人便构成了 **语言共同体** 。语言共同体的本质是 **契约共同体** ——所有签署了同一套语言契约的人。

三、增强型逻辑律：语言契约的底层地基

3.1 概述

语言契约论的核心命题是“语言是约定”。然而，一个更深层的问题随之而来： **约定本身何以可能？** 是什么规则保证了约定能够被遵守、被理解、被传递？

答案在于增强型逻辑律。如果说语言契约是社群成员之间签订的“合同”，那么增强型逻辑律就是这份合同的“通用条款”——它不因社群而异，不因物种而异，不因信号模式而异。无论是老虎脑内的电信号、蜜蜂的舞蹈，还是人类的语言——只要存在符号与对象之间的映射关系，就必须遵守这三条逻辑律。

增强型逻辑律包含三条核心法则： **同一律、等同律、契约律** 。

3.2 同一律

3.2.1 严格表述

一个语言符号唯一地指代一个具有同一性的对象。

其中，“同一性”的定义为： **一个对象如果在一段时间内保持自身不变，就具有同一性。** 物理对象和物理定律拥有自身的同一性。概念、集合和命题通过定义获得同一性。

同一律的要求包含两个层面：

第一， **对象必须具有同一性** ——只有具有同一性的对象才能成为语言符号的指代对象。

第二， **符号与对象之间的指代关系必须保持不变** ——在一个语言契约中，符号和对象的对应关系一旦约定，就必须保持稳定；符号不能今天指这个对象、明天指另一个对象。

同一律取代了传统逻辑中的表述“x = x”——后者被维特根斯坦视为伪命题，在增强型逻辑律中被剔除。

3.2.2 通俗解读

什么叫“具有同一性”？简单说就是： **这个东西在一段时间内保持不变——至少在对信号发送者和接收者的特定目的而言足够长的时间内保持不变。**

一块石头具有同一性——今天看是这块，明天看还是这块。万有引力定律具有同一性——牛顿时代是这样，今天还是这样。“苹果”作为概念具有同一性——每个人都知道它指什么。

同一性不是绝对的，而是依赖于观察尺度和目的。一个苹果时刻在蒸发水分，但日常对话中“我这儿有一个苹果”——它在对话期间不会变成兔子，所以具有同一性。荧光灯以50Hz明暗闪烁，人眼无法分辨瞬时亮度——瞬时亮度对人眼没有同一性。芯片中的门电平信号，虽然只保持一纳秒，但对检测电路足够长——它具有同一性。

**同一性的核心是：观察和思考需要时间。如果一个对象在观察完成之前就改变了，观察结论就失效了。**

同一律还要求符号与对象之间的指代关系保持稳定。用“日”指代太阳，首先要求太阳是一个具有同一性的对象——它在观察和沟通期间不会变成其他东西。在太阳具有同一性的前提下，同一个符号“日”必须稳定地指代太阳，不能今天指太阳、明天指石头——那是在改变指代关系，违反同一律。正是符号与对象之间指代关系的稳定性，才使得沟通成为可能。

3.2.3 同一律对语言契约的意义

任何语言契约的签署，都包含两个层面的要求。首先，符号所指代的对象必须具有同一性——如果“太阳”不是一个稳定的对象，今天发光明天消失，那么任何关于它的符号约定都没有意义。其次，一旦符号与对象的对应关系被约定，这个指代关系就必须保持稳定——如果“日”今天指太阳、明天指石头，说话者和听者之间便无法建立稳定的沟通，契约便无法成立。

3.3 等同律

3.3.1 严格表述

**x = y 当且仅当 符号“x”和符号“y”指代同一个对象。**

这是数学中等号“=”的严格定义。在等同律下，矛盾律自然成立，因为 x 和 ¬x 指代不同的对象。

如2.3节所述，我们严格区分符号本身和它所指向的对象。等同律的判定标准，正是基于这一区分：两个符号是否指向同一个对象。

3.3.2 等同律对语言契约的意义

语言契约中的“同义词”现象，本质是等同律的体现。两个不同符号能够相互替换，不是因为形式相似，而是因为在契约中被约定指向同一个对象。

3.4 契约律

3.4.1 严格表述

**契约律的基础：命题的形式化。**

人类一切有意义的陈述句，其基本逻辑形式都可以表示为：

**x ∈ X**

其中， **x 是主语** ，指代被描述的对象； **X 是谓语** ，指代被描述对象的性质。概括公理给出了等价关系：

**∀X， P_X(x) ↔ x ∈ X**

即“x ∈ X”等价于“x 具有性质 P_X”。

**契约律的严格定义：**

一个命题至少涉及说话者和倾听者。双方必须对描述世界的维度有共同约定。人类从有限维度描述世界——长度、重量、颜色、功能、状态等。

设 Ω_n 表示第 n 个维度中所有合法谓语的并集。Ω = {Ω₁, Ω₂, …, Ω_N} 代表说话者和倾听者双方达成的契约。契约律规定：

**Ω = {Ω₁, Ω₂, …, Ω_N}，∀i ≠ j，Ω_i ∩ Ω_j = ∅** 
 **∀x，∃Ω_n ∈ Ω 使得 x ∈ Ω_n，x ∈ X → X ⊆ Ω_n**

即：

- 不同维度互不相交；
- 任何可以被描述的对象必然属于某个维度；
- 如果一个对象属于某个谓语，那么这个谓语必须是该对象所在维度的子集。

契约律规定： **如果一个命题中的谓语 X 不属于主语 x 所在维度的合法谓语集，则该命题直接判定为假。**

通俗地说： **没约，就不合法；合法是因为约过。**

3.4.2 通俗解读：维度、合法谓语与约定

维度不是先验存在的，而是人类为了认识和描述世界的实践需要而定义的。比如，“颜色”是一个维度，“质量”是一个维度，“长度”是一个维度。每一个维度都是由一系列合法谓语构成的——这些谓语之所以“合法”，是因为语言共同体已经约好了用它们来描述该维度的事物。

我们之所以能理解“这个苹果的颜色是红色”，是因为“红色”属于颜色维度的合法谓语集——而“红色”之所以是合法谓语，是因为我们约好了用它来描述颜色。同样，我们之所以能理解“这个苹果的质量是200克”，是因为“200克”属于质量维度的合法谓语集——也是因为我们约好了。

反过来，“这个苹果的颜色是3克”之所以是错误的，不是因为经验上“苹果的颜色不可能是3克”，而是因为“3克”属于质量维度，不属于颜色维度。颜色维度和质量维度之间没有约定过交集——我们没有约过可以用质量维度的谓语来描述颜色。所以，这个命题被契约律直接判定为假。

这就是契约律的核心： **一个谓语是否合法，取决于我们是否约过它属于某个维度。** 不同维度之间没有交集，不是先验的“真理”，而是因为“我们没约过它们有交集”。维度是人定义的，维度的边界是契约划定的。

3.4.3 有交集才需要签约

两个维度默认没有交集，是因为没有约定过有交集。如果要在两个维度之间建立联系，就必须另行签约。

“4”和“IV”属于同一维度（数量），但符号形式不同。要建立“4 = IV”，必须有明确约定——否则不相等。“晨星”和“早晨最亮的星”指向同一现象——需要约定。同样，如果有一天我们要在“颜色”和“质量”之间建立某种联系，也必须通过明确的契约约定——在此之前，它们默认没有交集。

3.4.4 非法命题的判定

以下命题均被契约律判定为假：

| 命题 | 判定理由 |
| --- | --- |
| “这个球的质量是3米” | 质量维度与长度维度，未约定交集 |
| “π是红色的” | 数学维度与颜色维度，未约定交集 |
| “苹果 = 2” | 物理对象维度与数量维度，未约定交集 |

所有跨维度命题，仅凭契约律即可判定为假—— **因为我们没约过。**

四、语言演化史：从生物本能到社会契约

4.1 独居的老虎：语言作为生物本能

一只独居的老虎，不需要交流，但它需要思维。当老虎看到鹿，其视神经将光信号转换成特定的电信号模式——这是一个 **物理信号** （神经电脉冲）指代一个 **对象** （可吃的鹿）。这就是最原初的“符号系统”，是生命与生俱来的“出厂设置”。

但即便是这只老虎的“脑内语言”，也必须遵守同一律。首先是对象具有同一性——“鹿”这个对象必须在老虎的感知中保持稳定，今天看到的是“可吃的”，明天看到的仍然是“可吃的”。其次是指代关系的稳定性——“鹿”这个电信号模式必须稳定地指代“可吃的东西”，如果它今天指代“可吃”、明天指代“不可吃”，老虎会扑向石头，最终被自然淘汰。

不遵守逻辑的直接结果，就是物理消失。生命的逻辑，不是写在教科书里的教条，而是被亿万年的生存实践筛选出来的。 **同一律在这里不是“学问”，而是生存本能。**

4.2 蜜蜂的舞蹈：社群共享的运动信号

蜜蜂发现花源后返回蜂巢，通过一套复杂的舞蹈动作向同伴传递信息——这是动物界最精妙的 **运动信号系统** 。

- **圆形舞** ：表示花源很近（50米以内）；
- **摆尾舞（八字舞）** ：表示花源较远。舞蹈的 **方向** 指示花源相对于太阳的方向，舞蹈的 **持续时间** 指示距离。

蜜蜂的舞蹈系统已经具备了人类语言的基本特征——符号指代对象。一个特定的运动模式（符号）指代一个特定的空间方位（对象）。但更为关键的是： **这个系统是一套约定。**

不同亚种的蜜蜂有不同的“舞蹈方言”。意大利蜂和日本蜂的舞蹈编码方式存在差异——同样的舞蹈动作，在不同的蜂群中指代不同的距离和方向。这只能说明一件事： **蜜蜂的舞蹈语言不是完全天生的，其中包含了后天习得的社群约定。** 蜜蜂群体内部共享一套约定——它们构成了一个最小规模的语言共同体。

蜜蜂的舞蹈同样遵守同一律。一方面，花源的方向和距离作为被指代的对象必须具有同一性——同一处花源不会在舞蹈进行中改变位置。另一方面，一个特定的舞蹈模式与特定的空间方位之间的指代关系必须保持稳定——如果今天转三圈指“100米”、明天转三圈指“200米”，工蜂找不到花源，整个蜂群面临饥饿。

4.3 岩画：符号走向公共的第一步

随着人类的出现，语言发生了一次质的飞跃。人类开始把大脑里的“图”用木炭“复印”到岩壁上——用 **光信号** （视觉图像）记录和传递信息。

岩画不再是私有的、瞬时的脑内电流，而是公共的、持久的符号。这是语言从“本能”走向“契约”的第一步。岩画依赖“相似性”——画得像，观众不需要额外约定就能认出画的是野牛还是鹿。但相似性编码效率太低——画一头牛需要半天，而且画不出“所有的牛”。

4.4 象形文字：从图到符号的压缩

人类对“图”进行了暴力压缩。用几根简单的线条勾勒出一个符号“牛”。象形字不再追求与真牛的“相似”，而是一种 **约定** ——用“简单的图”代表“复杂的图”。

当埃及人在尼罗河畔、苏美尔人在两河流域、中国人在黄河流域分别画下各自的象形符号时，他们不是在“发现”文字，而是在 **签订契约** ——不同的社群签下了不同的合同。

4.5 拼音文字：彻底切断相似性

拼音文字把这种压缩推向了极致。字母“A”与“牛”没有任何相似性——它纯粹是任意的约定。从此，语言符号彻底摆脱了“像什么”的束缚，进入了“约好了是什么”的阶段。

从老虎的脑内电信号，到蜜蜂的舞蹈，到岩画，到象形字，再到字母文字——这条演化路径的核心动力是 **对效率的不懈追求** 。而每一次飞跃都面临同一个根本问题： **如何保证符号的指代是确定的？** 答案始终是同一个： **契约** ——而契约的底层支撑，是增强型逻辑律。

4.6 现代语言：复杂的契约体系

今天的每一种自然语言都是一个庞大而精密的契约体系。词典是“合同汇编”，语法书是“合同解释”，父母和老师是“合同代办”。你“学会”一个词，就是“继承”一份合同；你“用”一个词，就是“履行”合同。

这个契约体系是活的。新词不断被创造，旧词不断被淘汰。每一个新词的出现，都是一份新契约的签署。

4.7 社交网络：契约的爆发式更新

今天，语言契约的签署速度达到了前所未有的高度。社交媒体让任何人都可以成为“契约发起者”。

“LOL”（laughing out loud）——网络社群约定用这三个字母表达大笑，这份契约在一天之内传遍英文互联网。“TBH”（to be honest）——同样，一份新约定在几小时内形成。“Break the internet”——一个几天内就病毒式传播的短语。

这些网络语言的共同特征是：它们与所指对象之间没有任何天然联系。在“LOL”和“大笑”之间，唯一的联系就是社群之间的约定。

社交网络的另一个特征是 **契约的快速过期** 。五年前的流行语今天几乎无人使用——那些契约过期了。从老虎的脑内电信号（一生不变的硬连线），到蜜蜂的舞蹈方言（代际传承的软约定），到人类的语言（瞬息万变的社会契约）—— **约定的保质期从“终身”缩短到了“数月”。**

五、语言契约论解释全部语言现象

5.1 解释力的来源：符号的整体性

语言契约论之所以能够解释所有语言现象，是因为它抓住了语言的本质： **语言符号是整体。** 每个语言符号就是它自己——一个物理信号模式。它的全部意义在于：被社群约定指向某个具有同一性的对象。

**语言符号的根本特征是整体性。** 一个符号由什么字母或汉字构成、这些构成元素在字面上是什么意思——这些都不影响符号本身的指代功能。符号的指代功能完全来自于一个唯一的事实： **社群约定用它指向某个对象。**

一个语言符号不是用来“描述”对象的，而是用来“指代”对象的。它不是对对象构成的说明，而是一个整体信号对另一个对象的指向。因此，试图把符号拆解为组成部分并质问其“合理性”，是把“指代”误解为“描述”，是一种范畴错误。语言中大量存在这类不可拆分的整体符号——无论是单词、词组、整句还是字母缩写，在契约的层面上都是整体。

从老虎的神经电信号到蜜蜂的舞蹈方言，从四万年前的岩画到今天的网络流行语——物理载体在变，签约范围在变，契约的传承方式在变，但最根本的机制从未改变： **一个符号指向一个对象，中间是约定。**

以下通过具体例证展示这一解释力。

5.2 英文语境中的例证

**（一）“Hot dog”**

“Hot dog”是一个整体符号，指向一种特定食物。它不是“热狗”的描述——符号不是用来描述对象的，而是用来指代对象的。一旦社群约定用这个符号指代那种食物，它就是有效的。试图把“hot dog”分解为“hot”+“dog”并质问为什么不是“dog hot”，是把符号误当作描述。就像字母“B”没有内部语义组件，它就是一个整体符号，指向一个特定音素。同样，“hot dog”没有内部语义组件，它就是一个整体符号。

**（二）“Pineapple”**

“Pineapple”是一个整体符号，指向菠萝。它不是“pine”和“apple”的语义组合。尽管这个符号在历史上可能源于某种联想，但在现代英语契约中，它就是一个不可拆分的整体符号，指代菠萝。菠萝在世界大多数语言中被称为“ananas”，英语偏偏没有采用——这恰恰说明“pineapple”只是英语社群偶然的约定。

**（三）“Eggplant”**

“Eggplant”是一个整体符号，指向茄子。英国英语用“aubergine”，印度英语用“brinjal”——三个不同的符号指向同一个对象。每个符号都是整体，都是各自语言共同体的约定。没有哪一种更“正确”。

**（四）“Bob‘s your uncle”**

“Bob’s your uncle”是一个整体习语符号，指向“搞定了”这个含义。它不是一个可以由四个词的字面意思合成的句子。它的含义完全来自社群约定。

**（五）不规则动词——“go”的过去式为什么是“went”？**

“Went”是一个整体符号，指向“go的过去式”这一含义。它不是可以通过规则推导出来的形式，而是历史契约的保留。它的有效性仅仅来自“英语社群约定用这个符号指代那个含义”。

5.3 统一解释框架

下表总结了以上例证：

| 语言符号 | 它不是什么 | 它的有效性来源 |
| --- | --- | --- |
| “Hot dog” | 不是“热狗”的描述 | 英语社群的约定 |
| “Pineapple” | 不是“pine”+“apple”的语义合成 | 英语社群的约定 |
| “Eggplant” | 不是“egg”+“plant”的语义合成 | 英语社群的约定 |
| “Aubergine” | 不是可拆解的词 | 英式英语社群的约定 |
| “Brinjal” | 不是可拆解的词 | 印度英语社群的约定 |
| “Bob‘s your uncle” | 不是字面句“鲍勃是你叔叔” | 英式英语社群的约定 |
| “Went” | 不是由规则推导出的形式 | 历史契约的保留 |
| “LOL” | 不是四个词的简单缩写 | 网络社群的约定 |

**以上所有符号，无论是单字、词组、整句还是字母缩写，在契约的层面上都是整体。** 它们不可拆解，不需要内部逻辑自洽。符号的唯一运作方式是： **整体指向对象。**

5.4 结论：语言契约论的完备性

语言契约论之所以能够解释所有语言现象，是因为它抓住了语言的本质：

**每个语言符号就是一个物理信号模式，它的全部意义在于被社群约定指向某个具有同一性的对象。**

从老虎的神经电信号到蜜蜂的舞蹈方言，从四万年前的岩画到今天的网络流行语——物理载体在变，签约范围在变，契约的传承方式在变，但最根本的机制从未改变： **一个符号指向一个对象，中间是约定。**

六、语言契约论与主流语言学的对比

6.1 与索绪尔结构主义的对比

索绪尔描述语言的共时结构——能指与所指的任意性——但未回答“为什么任意的符号能被理解”。

契约论指出：符号被理解，不是因为结构，而是因为约定。蜜蜂的舞蹈方言证明了这一点——不同蜂群有不同的约定，但都能被同伴理解。

6.2 与乔姆斯基生成语法的对比

乔姆斯基的“普遍语法”不可证伪。

语言契约论彻底否定“先天语言官能”。如果存在普遍的先天语法，为何蜜蜂的舞蹈方言存在差异？为何英语和汉语用完全不同的方式表达时间？唯一的答案是：不同的社群签了不同的契约。

6.3 与维特根斯坦语言游戏的对比

维特根斯坦揭示了语言的动态性，但回避了根本问题：为什么“使用”能产生意义？

契约论指出：“使用”能产生意义，是因为背后有“约定”。同一律保证稳定性，等同律保证替换合法性，契约律保证表达合法性。

七、结论

语言的发展史，是一部符号的物理载体不断更替、签约范围不断扩大、契约传承方式不断演化的历史。贯穿其中的主线始终是： **符号指向对象，中间是约定。**

语言契约论的合法性奠基于增强型逻辑律：同一律保障符号指代的唯一性；等同律确立等价判定的标准；契约律划定命题表达的范畴边界。三条逻辑律共同构成了语言契约的通用条款。

约定的最小单元发生在说话者与倾听者之间。当约定被一群人共同接受，便形成语言共同体。

语言符号是整体。每个符号就是一个物理信号模式，它的全部意义来自社群约定它指向某个对象。符号的有效性不取决于内部组件的字面含义，只取决于一件事： **我们是否约好了用它指向那个对象。**

语言不是发明，是契约；逻辑不是学问，是本能；契约不是比喻，是逻辑律在人类社会中的具体展开。
