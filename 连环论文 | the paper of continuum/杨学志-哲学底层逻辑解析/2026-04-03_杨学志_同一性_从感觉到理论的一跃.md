**一个日常场景** 
 你看着一个苹果。你的视觉感受是稳定的——红色、圆形、静静在那里。旁边是一盆水花，光影跳跃不停；再旁边是一只猫，嗖地窜过去。 
 相比之下，苹果给你的感觉是 **不变** 的。 
 你的大脑做了两件事 
 **第一，你感知到了这个“不变”。** 
 这不是推理，是直接感受。你看水花时感觉在“动”，看猫时感觉在“变”，看苹果时感觉“没动”——这三种感受是不一样的。你能区分它们。 
 **第二，你构建了一个理论来解释这个“不变”。** 
 你告诉自己：之所以我对苹果的感觉不变，是因为 **苹果本身是不变的** 。 
 苹果有一个“它自己”——一个稳定的、持续的东西。刚才它是这个，现在它还是这个。这就是 **同一性** 。 
 水花呢？你感觉它在变，所以你推断：水花没有固定的“它自己”。它是一堆不断变化的水分子，没有同一性。 
 猫呢？猫有同一性——它还是那只猫，只是它在动。你能区分“猫在跑”（属性变化）和“猫变成了狗”（同一性变化）。这说明你的理论里有“猫的同一性”这个预设。 
 这个理论有什么用？ 
 **预测。** 
 因为你相信苹果有同一性——它是“同一个东西”持续存在——你就可以做出一个大胆的预测： 
 当我拿起这个苹果，放进嘴里咬下去的时候，它 **还是** 苹果。不会变成石头。 
 这个预测听起来很傻？但它至关重要。 
 如果没有“同一性”这个理论，你的逻辑是：我上一秒对这个东西的感觉是“苹果”，下一秒感觉变了怎么办？万一变成石头呢？那我还敢咬吗？ 
 有了同一性，你可以。你咬下去，果然是苹果。预测成功。 
 **实践：同一性的真正目的** 
 人构建理论不是为了“认识世界”这个抽象目标。人构建理论是为了 **行动** 。 


- 你要决定：这个苹果能不能吃？
- 你要预测：咬下去会发生什么？
- 你要应对：以后看到类似的红色圆形物体，是不是也可以吃？

同一性这个理论告诉你： **感觉不变，是因为对象不变。对象不变，是因为它有同一性。有同一性的东西，你可以信赖它。** 
 你不必每次看到苹果都重新试探。你直接拿起、吃掉。这就是理论指导实践。 
 **读者可能有的诘难：“同一性没有定义”** 
 读到此处，读者可能会问： **你说了半天，还是没有给“同一性”下一个文字定义啊？** 
 这个诘难是对的。同一性确实 **无法用文字定义** 。 
 任何定义都是用A解释B、用B解释C……但同一性是最底层的概念——它下面没有更基础的东西可以用来解释它。就像你没法向一个天生失明的人“定义”红色。你只能指给他看：这个是红的，那个也是红的……让他自己建立起那个“类”的感知。 
 **同一性的定义方式，不是“文字定义”，而是“指向性定义”。** 
 它指向的是人类最基础的感知能力： **我能感觉到“这个还是那个”。** 


- 你看着一个苹果从完整到被咬了一口——你感觉到了： **还是那个苹果** 。
- 你看着一个人从年轻到年老——你感觉到了： **还是那个人** 。
- 你从梦中醒来——你感觉到了： **还是我这个意识在感知** 。

这个“还是”的感受，就是同一性的来源。它不是逻辑推导出来的，不是文字定义出来的，而是你作为有感知能力的生命， **直接体验到的** 。 
 所有的逻辑、所有的理论、所有的文字定义，都建立在这个最基础的体验之上。你可以质疑定义，但你没法质疑体验——你正在体验它。 
 所以： **同一性没有文字定义，它只有“指向性定义”——指向你自身最基础的感知本能。** 
 **有了同一性，才有文字** 
 这是最关键的一步： **同一性不是文字定义出来的。恰恰相反，文字是建立在同一性之上的。** 
 想象一下：如果你没有“同一性”这个理论，你凭什么用一个词去指代一个东西？ 
 你说“苹果”这个词。这个词之所以有意义，是因为你相信：你昨天看到的那个红色圆形物体，和你今天看到的这个红色圆形物体，是 **同一个东西** （或者同一类东西）。你用一个词“苹果”来指代这个具有同一性的对象。 
 如果世界是纯粹的感觉流，每一秒都在彻底变化，没有任何东西“还是它自己”——那你的任何词语都没有固定指代。你说“苹果”，没人知道你在指什么，因为没有任何东西能持续存在到被指代。 
 **所以：先有同一性（理论预设），然后有文字。** 
 **增强的同一律** 
 杨学志在传统同一律（A = A）的基础上，提出了 **增强的同一律** ： 
 **语言符号唯一指向一个具有同一性的对象。** 
 这句话可以拆解为三层意思： 
 **第一，文字必须指向对象。** 
 文字不是凭空存在的。你说一个词，它必须有所指。如果你说“噶尔朵”，然后说“这只是一个词，不代表任何东西”——那这个词就没有意义，不属于逻辑讨论的范畴。 
 **第二，这个对象必须具有同一性。** 
 也就是说，这个对象必须是“它自己”——定义已完成、属性已稳定、能够持续存在的东西。你不能用一个语言符号指向一个变化无定的事物，或者尚未定义清楚的概念。这条规则可以进一步推导出，所有的自指都是非法的，从而解决所有的自指悖论。 
 **第三，唯一指向。** 
 一个文字在同一讨论中只能指向一个对象。不能一会儿指这个，一会儿指那个，更不能同时指两个不同的东西。 
 这就是为什么“自指”是非法的：当你说“这句话是假的”时，“这句话”试图指向一个尚未具有同一性的对象（因为它正在被定义），而且它试图同时指向自身和一个关于自身的判断——违背了“唯一指向”的原则。 
 **有了文字，才有定义** 
 当第一批文字（ 原生概念 ）被确立之后，它们就可以用来定义其他概念。 
 原生概念，比如： 


- 苹果
- 猫
- 1、2、3（数字）

这些概念不是用其他文字定义出来的。它们直接指向感知世界中被确认了同一性的对象。你没法用更基础的词去定义“苹果”——你只能指着一个苹果说：这个就是苹果。这和“指向性定义”是同一个逻辑。 
 **原生概念是文字系统的基础。** 有了它们，你才能定义其他概念。 
 比如，你可以用“苹果”和“红色”来定义“红苹果”——“红苹果”不是原生概念，它是组合出来的。但“苹果”和“红色”本身必须是原生概念，否则定义链条就会无限后退，或者循环。 
 这就是为什么定义不能无限倒退： **最底层必须是原生概念——那些直接指向感知对象、无需进一步定义的概念。** 
 整个链条 
 让我们把整个链条串联起来： 


| 层次 | 内容 |
| --- | --- |
| 原始感觉 | 苹果给我的感觉是“不变”的 |
| 理论构建 | 因为感觉不变，所以苹果本身是不变的——苹果有“同一性” |
| 增强的同一律 | 语言符号唯一指向一个具有同一性的对象 |
| 原生概念 | 第一批文字出现了：苹果、猫、1、2、3…… |
| 定义其他概念 | 基于原生概念，用文字定义更复杂的概念 |
| 理论功能 | 让我能够预测、行动、交流 |
| 实践应用 | 我可以拿起苹果、吃掉它，不用担心它变成石头 |

**反过来理解：如果没有同一性** 
 如果没有同一性这个理论预设，世界就是一团混沌的、不断变化的“感觉流”。 
 你没法做任何可靠的预测。你每次面对一个东西，都像第一次看到它。你不敢咬，不敢碰，不敢走——因为你不知道下一秒它会变成什么。 
 更重要的是： **你没有任何文字。** 因为文字的前提是“有东西可以稳定地指代”。 
 这不是人活的世界。这是婴儿或严重精神病患者的世界——他们的“对象恒常性”还没有建立或已经丧失。 
 总结 
 **同一性不是“客观事实”，而是人为了应对“感觉相对不变”而构建的理论。** 这个理论让人能够预测、能够行动、能够在变化的世界中找到稳定的支点。 
 **增强的同一律** （语言符号唯一指向一个具有同一性的对象）是对这一理论的逻辑化表述。它为文字的使用确立了最基本的规则：在逻辑讨论中，每个符号必须指向一个已经被确认了同一性的对象。 
 没有这个规则，一切逻辑推理都无从谈起——就像你不敢咬那个可能变成石头的“苹果”。
