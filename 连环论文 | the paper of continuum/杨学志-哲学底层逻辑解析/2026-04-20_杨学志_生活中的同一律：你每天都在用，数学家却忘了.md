——罗素、哥德尔、图灵犯的“低级错误”

一、什么是同一律？你每天都在用

同一律，听起来很高深。其实你每天都在用。

**例子一：菜没做好，不能评价好不好吃。**

妈妈在厨房炒菜。菜还没出锅，你跑进去问：“妈妈，这菜好吃吗？”妈妈说：“还没做好呢，做好你尝了才知道。”

你不会评价一盘没做好的菜。因为你没尝过，不知道味道。这是同一律： **事情没完成，不能评价它的完整属性。**

**例子二：孩子考试没出分，不能打屁股。**

孩子刚考完试，卷子还没批。你问他：“你及格了吗？”他说：“不知道，卷子还没批。”你不会在他没出分的时候打他屁股。因为你不知道他考了多少分。

**例子三：婴儿没出生，不能评价长相。**

孩子还在妈妈肚子里。你可以给他起名字——名字是一个“标签”，不需要等出生。但你不能说“这孩子长得真漂亮”或“这孩子真丑”。因为他还没生出来，你没看到他的脸。

**你不知道他的长相，不能评价好看不好看。**

**例子四：米饭没煮熟，不能说“熟了”。**

电饭锅还在煮饭，灯还没跳。你不会说“饭熟了”。因为还没煮好。

**这些常识，每个人都懂。不需要学逻辑学，不需要读博士。**

它们背后是同一条规则： **一个对象必须“完成了”、确定了，才能被评价它的完整属性。** 你可以给它起个临时名字（标签），但不能判断它“好不好”“美不美”“对不对”。

这条规则，在杨学志的理论里，叫“同一律”。

二、数学家是怎么违反同一律的？

数学家们，在日常生活里也遵守这些常识。但一做起学问，就忘了。

**罗素悖论：评价了一个没定义好的集合**

罗素定义了一个集合 R = { x | x ∉ x }——所有“不包含自身的集合”的集合。

然后他问：R 包含自身吗？

问题来了：R 还在定义过程中，还没有“完成”。你可以给它一个名字（R），但不能问它“包不包含自己”——这就像问没出生的孩子“好不好看”。

**你不知道它包含不包含自己，因为还没定义完。** 但罗素问了，还从答案中推出了矛盾，然后说“数学崩塌了”。

**哥德尔：用一个名字叫两个人**

哥德尔构造了一个命题 G，说“G 不可证明”。

他用了复杂的编码技术，但核心错误很简单：他让同一个符号“y”同时指代了两个不同的东西——一个是某个命题的编号，一个是那个命题里的变量。

这就像你家里有两个人，一个叫“小明”，一个叫“小红”。你喊“小明”，两个人都不应。你说“你们怎么不应？”他们说：“你喊的是谁？”

**同一个名字，不能指两个人。这是常识。**

但哥德尔违反了。他用同一个符号指代了两个对象，然后推出了“不完备定理”。

**图灵：程序还没写完，就调用它**

图灵构造了一个“反程序”Code，它的逻辑是：如果 Halt 判断 Code 会停机，Code 就不停机；如果 Halt 判断 Code 不停机，Code 就停机。

问题：Code 还在定义过程中，还没有写完。但 Halt 已经开始判断它了。

这就像你写作文，还没写完，老师就开始批改。老师说：“你这篇作文不及格。”你说：“我还没写完呢！”

**正常的程序，不能在定义过程中被调用。这是编程常识。但图灵违反了。**

三、为什么数学家会犯这种“低级错误”？

你可能会问：罗素、哥德尔、图灵，都是天才。他们怎么会犯这种“正常人不会犯”的错误？

因为他们被“数学的规矩”蒙蔽了。数学里有一个“传统”：自指是合法的。哥德尔用了自指，图灵用了自指，没有人质疑。

**他们忘了：数学也是人类思维的一部分。人类思维的基本规则——同一律——在数学里也应该遵守。**

- 菜没做好，不能评价好不好吃。
- 孩子没出生，不能评价好看不好看。
- 集合没定义好，也不能评价它包不包含自己。

**就这么简单。**

四、杨学志的提醒：回到常识

杨学志，清华大学本硕博，华为前科学家，4G核心技术发明人。他不是逻辑学家，他是工程师。

他做了一件很简单的事： **提醒大家那个“人人都懂”的常识。**

- 罗素：集合没定义好，不能问它包不包含自己。
- 哥德尔：同一个符号不能指代两个对象。
- 图灵：程序没写完，不能调用它。

**一旦你遵守这些常识，悖论就消失了。** 不需要复杂的理论，不需要十条公理，不需要“真类”“类型论”。

只需要： **回到常识。**

五、结语

同一律，不是杨学志发明的。它一直是人类思维的基本规则。你每天都在用。

- 你不会评价没做好的菜。
- 你不会打没出分的孩子。
- 你不会评价没出生的孩子“好看不好看”。

**数学家们，在生活中也遵守这些规则。但一做起学问，就忘了。**

罗素忘了，造出了“悖论”。哥德尔忘了，造出了“不完备定理”。图灵忘了，造出了“停机问题”。

杨学志只是提醒他们： **菜没做好，不能评价。集合没定义好，也不能评价。**

就这么简单。一个百年死局，被一条“人人都懂”的常识解开了。
