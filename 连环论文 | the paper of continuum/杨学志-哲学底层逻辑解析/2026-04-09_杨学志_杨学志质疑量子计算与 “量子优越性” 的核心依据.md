一、信息论底层依据： **可读取信息才是真实信息**

1. **量子比特最终只能输出确定的 0 或 1** 
 无论叠加态多复杂，一旦测量，结果 **唯一、确定、经典化** 。 
 → 量子比特在 **信息输出层面与经典比特完全等效** 。
2. **中间过程的叠加态不可读、不可用** 
 叠加态 |ψ⟩ = α|0⟩ + β|1⟩ 只是数学描述， **无法直接提取信息** 。 
 不能读、不能用、不能控制的 “中间状态”， **不构成真实计算能力** 。
3. **信息的本质是可区分、可传递、可复制的确定性内容** 
 量子态不可克隆、测量坍缩、概率输出， **不符合经典信息论的有效信息定义** 。 
 → 从信息论看，量子比特 **没有提供新的信息维度** 。

---

二、逻辑依据： **2ᴺ 个状态 ≠ 2ᴺ 倍算力**

这是他最核心、最著名的批判：

1. **叠加 ≠ 并行计算** 
 科普常说 “N 个量子比特同时存在 2ᴺ 个状态，等于并行 2ᴺ 次计算”。 
 杨学志指出： 


- 并行计算要求 **同时计算并同时获得全部结果**
- 量子计算只能 **坍缩到一个结果** 
 → 所谓并行是 **虚假并行** ，没有实际算力增益。

1. **概率分布 ≠ 计算结果** 
 量子计算机输出的是 **概率分布** ，需要大量重复实验统计。 
 这不是 “算得快”，而是 **靠采样猜答案** 。 
 → 与经典计算机 “一次计算出确定结果” 完全不是同一类任务。
2. **指数状态只是数学空间，不是物理算力** 
 希尔伯特空间维度 2ᴺ 是数学结构， **不等于物理上真的在做 2ᴺ 次运算** 。 
 数学空间大 ≠ 计算能力强。

---

三、计算科学依据： **量子优越性实验是 “自证游戏”**

1. **任务是专门为量子机量身定做的人工问题** 
 玻色采样、随机线路采样 
 → 不是真实问题 
 → 不是科学需求 
 → 不是工业应用 
 → 只是为了 “证明量子机更快” 而设计
2. **经典计算机完全可以更高效地完成同样任务** 
 杨学志明确指出： 


- 高斯玻色采样 = 从一个已知概率分布中抽样
- 经典随机数生成器可以 **瞬间、精确、低成本** 完成 
 → “量子优越性” 是 **把物理模拟冒充成计算优势**

1. **所谓 “经典超算算一万年” 是误导** 
 随着经典算法优化，原本宣称 “不可模拟” 的任务不断被攻破。 
 → 差距是 **算法与算力差距** ，不是 **原理性差距** 。

---

四、物理依据： **量子计算本质是物理系统模拟自身，不是通用计算**

1. **量子设备不是在 “计算问题”，而是在 “演化自己”** 
 量子门操作 = 对量子态的幺正变换 
 整个过程只是 **物理系统按量子力学自然演化** 
 → 不是计算，是物理过程。
2. **没有真正的 “可编程性”** 
 所谓编程，只是调整激光、微波等外部参数， 
 并不是像经典计算机那样 **控制逻辑流、条件分支、循环、内存读写** 。
3. **退相干、噪声、测量不可控，从物理上排除了通用计算可能** 
 杨学志认为： 
 量子系统天生脆弱、易干扰、不可精确复制、不可完整读取 
 → **不可能构建稳定、通用、可扩展的计算机** 。

五、哲学与认知依据： **科学必须可理解、可逻辑一致**

1. **量子力学的 “非直观” 不能成为逻辑豁免权** 
 他坚持： 
 任何科学理论必须 **遵守同一律、矛盾律、因果律** 。 
 量子宣传中大量 “既… 又…”“同时存在”“超距作用” 式表述， 
 → 违反逻辑同一律。
2. **不能用 “微观世界不一样” 逃避逻辑检验** 
 无论宏观微观， **信息和计算的逻辑规则统一** 。 
 量子计算宣传违背基本逻辑，却以 “反常识” 为挡箭牌。

杨学志的最终结论（一句话概括）

**量子比特与经典比特在信息功能上实质等效；2ᴺ 维叠加态不构成真实算力；量子优越性是人工任务的虚假优势；量子通用计算在原理上不成立。**
