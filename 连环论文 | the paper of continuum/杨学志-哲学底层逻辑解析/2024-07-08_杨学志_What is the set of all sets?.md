Sets are actually artificial stuff.

Suppose we currently have only one set, namely A, then what’s the set of all sets? Obviously it is {A}.

Since {A} is not A, we name it as B, then we have 2 sets, and now the set of all sets is {A, B}, which is another set. So on and so forth.

So, the set of all sets is not a set with identity.  Every time we define the set of all sets, a new set is produced and the meaning of "all sets” is changed.

Q={x | x \notin x} is literally the set of all sets. But here the “all sets” is the all set before the definition of Q. After Q is defined, Q is added to "all sets” and changes its meaning.
