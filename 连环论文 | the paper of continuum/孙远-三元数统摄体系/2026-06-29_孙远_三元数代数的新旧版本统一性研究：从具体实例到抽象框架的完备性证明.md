**第一章** **引言**

**1.1 研究背景**

三元数代数作为永纠缠论数学体系的核心工具，经历了从具体实例到抽象

框架的演进过程。旧版三元数（v1.0）采用直观形式：

Z = a + bi + cj,  i² = -1, j² = 0, ij = ji = j

这一形式直接继承复数理论（i²=-1）和对偶数理论（j²=0），具有明确的

数学来源和运算规则。然而，旧版在理论完备性方面存在不足：未明确基底的代数地位、未建立完整的除法体系、未证明非结合性的结构意义。

为此，正版三元数（v5.0）引入抽象基底框架：

Z = a·er + b·ei + c·et

er² = er, ei² = -er, et² = 0+

ei·et = et·ei = et

本文的核心任务是：证明旧版是新版在特定认同下的特例，阐明旧版取值的

合理性，并展示其在工程应用中的简便性。

**1.2 研究意义**

(1) 理论意义：建立新旧版本的严格代数同构，消除版本间的概念混乱，

为三元数代数的进一步发展奠定坚实基础。

(2) 应用意义：证明旧版虽为特例，但在er≡1的认同下完全自洽，且因

其直观性更适合工程计算和教学推广。

(3) 方法论意义：展示"具体→抽象→回归具体"的数学研究范式，即

从具体实例提炼抽象框架，再用抽象框架反哺具体应用。

**第二章** **旧版三元数的数学来源与合理性**

**2.1 复数理论的直接继承**

旧版中 i² = -1 直接继承复数理论。复数域 ℂ = {a+bi | a,b∈ℝ, i²=-1}

是数学中最成功的代数扩充之一，具有完整的代数、几何和分析理论。

旧版将复数作为三元数的"实-虚"子结构：

Z = (a+bi) + cj = Z_复数 + cj

这种构造的合理性在于：复数理论已完全成熟，其运算规则（交换律、

结合律、除法）可作为三元数子结构的坚实基础。

**2.2 对偶数理论的直接继承**

旧版中 j² = 0 直接继承对偶数（Dual Number）理论。对偶数由

Clifford 在研究几何学时引入，形式为 a+bε，其中 ε²=0。

对偶数在以下领域有重要应用：

• 自动微分：f(a+bε) = f(a) + bf'(a)ε，实现无截断误差求导

• 刚体运动学：表示螺旋运动和力矩

• 几何代数：表示直线和平面

旧版将 j 作为对偶单位，直接利用对偶数的成熟理论，避免了重新

构建幂零代数的技术困难。

**2.3 混合乘积规则 ij = ji = j 的合理性**

**定理 2.1（混合乘积的自洽性）：**

在 i²=-1, j²=0 的约束下，ij = ji = j 是唯一保持交换律且使

j 为吸收元的解。

**证明：**

设 ij = α + βi + γj，其中 α,β,γ ∈ ℝ。

由 j² = 0，两边右乘 j：

ij·j = (α + βi + γj)j = αj + βij + γj² = αj + βij

又 ij·j = i·j² = i·0 = 0

故 αj + βij = 0。

若 β≠0，则 ij = -(α/β)j，代入原式得矛盾。

故 β=0，进而 αj=0，因 j≠0，得 α=0。

因此 ij = γj。由交换律 ji=ij=γj。

又 i·ij = i·γj = γ·ij = γ²j，而 i²j = -j。

故 γ² = -1，无实数解...

**【修正思路】：实际上，在旧版中 ij 不应展开为三元数，而应直接**

**定义为 j。即 ij = j 是公理，而非推导结果。这类似于复数中 i²=-1**

是公理。

重新定义证明：

公理系统 {i²=-1, j²=0, ij=ji=j} 的自洽性验证：

(1) 交换律：ij=ji=j，显然成立。

(2) 结合律验证（关键实例）：

(ij)j = j·j = j² = 0

i(jj) = i·0 = 0

故 (ij)j = i(jj)。

(ii)j = (-1)j = -j

i(ij) = i·j = j

故 (ii)j ≠ i(ij)（除非 j=-j，即 j=0，矛盾）。

**结论：旧版在实部和虚部结合，但在涉及对偶部的三重乘积中不结合。**

这与记忆id 5的结论一致："非结合性差异仅存在于j分量"。

**2.4 旧版取值的合理性总结**

旧版取值的合理性来源于：

(1) i²=-1：继承复数，有200年成熟理论支撑

(2) j²=0：继承对偶数，有几何学和微分学应用支撑

(3) ij=ji=j：保证交换律，使j成为混合乘积的吸收元

这三条规则构成了一个自洽的交换但非结合的代数结构。

**第三章** **新版抽象框架的构建动机**

**3.1 旧版的理论不足**

尽管旧版在运算层面自洽，但存在以下理论缺陷：

(1) 基底地位不明：

旧版中 1, i, j 既是数又是基底，角色混淆。

在严格代数中，基底是线性空间的生成元，而数是基底的线性组合。

(2) 零元体系缺失：

旧版只有一个零（0），未区分加法零和算子零。

这导致无法解释"零除法"等特殊运算。

(3) 除法体系不完整：

旧版未给出 (a+bi+cj)⁻¹ 的显式公式，

也未讨论零因子（纯j元）的除法行为。

(4) 与本原元理论的脱节：

本原元理论（P003）要求明确0和1的代数地位，

旧版未建立这种联系。

**3.2 抽象框架的引入**

新版通过引入形式基底 er, ei, et 解决上述问题：

(1) 基底与数分离：

er, ei, et 是纯基底（类似向量空间的基向量），

Z = a·er + b·ei + c·et 是数（基底的线性组合）。

(2) 乘法规则抽象化：

将 i²=-1, j²=0, ij=j 提升为基底乘法规则：

er²=er, ei²=-er, et²=0+, ei·et=et·ei=et

(3) 零元体系化：

0+ = 0·er+0·ei+0·et（加法零）

0× = 0·er+0·ei+1·et = et（算子零）

(4) 与本原元理论衔接：

er 对应本原元1，0+ 对应本原元0。

**第五章** **旧版取值的应用简便性**

**5.1 计算简便性**

旧版直接继承复数和对偶数的运算规则，无需记忆基底乘法表。

**示例：计算 (2+3i+j)·(1-i+2j)**

旧版计算（直接代入）：

= 2·1 + 2·(-i) + 2·2j + 3i·1 + 3i·(-i) + 3i·2j + j·1 + j·(-i) + j·2j

= 2 - 2i + 4j + 3i - 3i² + 6ij + j - ji + 2j²

= 2 - 2i + 4j + 3i + 3 + 6j + j - j + 0   （用i²=-1, j²=0, ij=ji=j）

= (2+3) + (-2+3)i + (4+6+1-1)j

= 5 + i + 10j

新版计算（需查表）：

= (2er+3ei+et)·(er-ei+2et)

= 2er·er - 2er·ei + 4er·et + 3ei·er - 3ei·ei + 6ei·et + et·er - et·ei + 2et·et

= 2er - 2ei + 4et + 3ei + 3er + 6et + et - et + 0   （查乘法表）

= (2+3)er + (-2+3)ei + (4+6+1-1)et

= 5er + ei + 10et

在er≡1下，5er+ei+10et ≡ 5+i+10j，与旧版一致。

**结论：旧版计算更直观，无需基底抽象。**

**5.2 教学推广简便性**

旧版可直接利用学生的复数知识：

(1) 学生已知 i²=-1，只需增加 j²=0 和 ij=j 两条规则。

(2) 无需引入"基底"、"形式元"等抽象概念。

(3) 几何解释直观：实轴+虚轴+对偶轴的三维空间。

新版需要额外教授：

**5.3 工程应用简便性**

在华为揭榜等工程应用中，旧版形式更便于编程实现：

旧版代码（Python风格）：
