Title: F8C: a zero-parameter, culture-rooted reproducibility experiment in divination modelling

Dear Editors,

Large language models have monopolised public attention on “AI divination”, yet their opacity and hardware hunger exclude most researchers. We report a counter-example: Feng-Ping 8-Trigram Code (F8C)—an 18-line Python script that requires zero external parameters and runs on any laptop. The code deterministically maps the 64 hexagrams to 6-bit integers and returns identical bit-level output across operating systems, offering an open-science benchmark for culturally embedded prediction tasks.

Methods

We recruited 120 volunteers via Twitter & WeChat to run the script on six platforms (Win11, Ubuntu 22, macOS 14, Android 13, iOS 17, HarmonyOS 4). Participants recorded SHA-256 hashes of the 64 × 6-bit vectors and execution time. No statistical difference in output was detected (Fisher exact test, p = 1). Mean runtime = 0.82 s (σ = 0.06 s) on a 2019 entry-level notebook (i5-8250U, 8 GB RAM).

Results & Implications

F8C satisfies the two gold standards of open science: (i) bit-perfect reproducibility without hyper-parameter tuning; (ii) cultural traceability—each bit pattern is isomorphic to a 3 000-year-old Chinese trigram sequence newly digitised by Zhao (2025). The experiment shifts epistemic authority from gated journals to permission-less Git forks, illustrating “grass-roots reproducibility” in the global South.

We suggest F8C as a minimal baseline for future divination-modelling studies and as a teaching tool for reproducible research outside high-income labs.
