一、卦变位掩码矩阵的技术实现详解‌

1. 核心数学模型：卦象的二进制编码体系‌

六爻卦象被严格映射为6位二进制数，低位为初爻，高位为上爻，形成‌64种唯一状态‌，构成完整的状态空间：

阴爻 = 0‌，‌阳爻 = 1‌

坤卦‌：000000 → 十进制 0

乾卦‌：111111 → 十进制 63

为支持‌变爻状态‌的扩展编码，引入‌7位扩展编码体系‌：

变爻状态编码机制‌：

原始卦码（6位）与变爻标志位（第7位）组合，形成7位整数，如：

乾卦 + 初爻变 → 0b111111 | 0b1000000 = 0b1111111（十进制127）

此机制实现‌卦象与变爻状态的无损联合存储‌，为动态变换提供基础。

import numpy as np

class HexagramTransformer:

def __init__(self):

self.mask_library = self._build_mask_library()

self.cache = {}

def _build_mask_library(self):

library = {}

# 1. 单爻变换（6种）

for i in range(6):

mask = np.zeros((6, 6), dtype=int)

mask[i, i] = 1

library[f'single_{i}'] = mask

# 2. 双爻变换（C(6,2)=15种）

for i in range(6):

for j in range(i+1, 6):

mask = np.zeros((6, 6), dtype=int)

mask[i, i] = 1

mask[j, j] = 1

library[f'double_{i}_{j}'] = mask

# 3. 特殊卦变规则（完整实现）

library['reverse'] = self._create_reverse_mask()  # 综卦

library['inverse'] = self._create_inverse_mask()  # 错卦（全位取反）

library['mutual'] = self._create_mutual_mask()    # 互卦

# 4. 全部64种基础变换（线性组合）

for code in range(64):

mask = np.zeros((6, 6), dtype=int)

for bit in range(6):

if code & (1 << bit):

mask[bit, bit] = 1

library[f'base_{code}'] = mask

return library

def _create_reverse_mask(self):

"""综卦：上下翻转，等价于位串逆序"""

mask = np.zeros((6, 6), dtype=int)

for i in range(6):

mask[i, 5-i] = 1

return mask

def _create_inverse_mask(self):

"""错卦：全位取反，等价于异或0b111111"""

mask = np.ones((6, 6), dtype=int) - np.eye(6, dtype=int)

return mask

def _create_mutual_mask(self):

"""互卦：取二至五爻，上下置零"""

mask = np.zeros((6, 6), dtype=int)

for i in range(1, 5):  # 二至五爻（索引1~4）

mask[i-1, i] = 1  # 新初爻=原二爻，新二爻=原三爻...

return mask

def transform(self, hexagram_code, transform_key):

"""执行变换：输入6位整数，输出变换后6位整数"""

if transform_key not in self.cache:

self.cache[transform_key] = self.mask_library[transform_key]

M = self.cache[transform_key]

vec = np.array([(hexagram_code >> i) & 1 for i in range(6)], dtype=int)

result_vec = M @ vec % 2  # 二元域运算

result_code = sum(bit << i for i, bit in enumerate(result_vec))

return result_code
