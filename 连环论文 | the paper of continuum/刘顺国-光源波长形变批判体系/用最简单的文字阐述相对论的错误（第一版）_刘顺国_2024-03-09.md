1.相对论的基础是光速不变原理，即同一束光，在不同的惯性参考系中是相同的，或者通俗点说同一束光，对于一个静止的人A和一个运动的人B，他们所测得的光速是相同的。这一点违反常识，这也是许多人反对相对论的原因。

2.相对论的光速不变和麦克斯韦方程组的光速不变是两码事。麦克斯韦认为，光在空间中的传播速度恒定不变，是个常量，比如宇宙中A和B两点之间同样一段距离，只要是电磁波不管是紫外光、可见光、红外线、无线电波等，它们走过这段距离耗时相同。

再通俗一点就是，一万种光穿过同样的空间距离所用时间相同，这是麦克斯韦方程组的意思，而同一束光对于一万个不同观测者，所测得速度相同，这是爱因斯坦狭义相对论的意思。两者不同，却常被人混为一谈。麦克斯韦方程组并不能作为支持相对论的证据。

3.相对论的真正依据是迈克尔逊莫雷实验，它的实验结果很诡异，本来推算在光路旋转90度之后相位差为0.37个波长，即条纹应移动0.37个单位，但结果竟然零移动，最终人们不得不得出结论，以太不存在，以及光速在不同惯性系中是没有差异的，没有差异也即是相等的。

**真的有实验证明同一束光对于静止的A和运动的B，其光速是相等的么？**

**不，没有这样的实验，从来也没有这样的实验直接证明这一点。** 这个结论是从迈克尔逊莫雷实验间接得到的。相对论的实验依据并非牢不可破无法动摇，这一点十分重要。

推倒相对论的关键就在于能否从经典力学、绝对空间的角度找到迈莫实验零移动的真相。

4.为什么实验结果出现条纹零移动？如果没有转动方向，只是改变一条光路某个反光镜的位置，人为制造相位差，条纹会不会移动？肯定会！这就是大学里迈克尔逊莫雷实验仪器的翻版，仪器底座不动，改变一条光路相位差，必然会出现条纹移动！但如果把整套仪器转动方向，转动90度，则不会出现条纹移动！

抓到问题的关键了么？关键在于方向！

简单来说，一个移动光源，在不同方向上的波长是不同的。两位设计师忽视了一点，在推算中是按照固定波长590纳米来计算的。

5.一个移动的光源，前进方向上的波长是被压缩的，身后的波长是被拉长的。一个频率恒定的稳定光源，如果从速度V1提高到V2,那么身后的波长是要变长的，因为波是连续的，不会断开的。并且满足λ2·f -λ1·f= V2 - V1。如果这个简单的模型都想不通，那么还是离物理远一点好。

6. 假设光源在静止时发出的光波波长均为λ0，那么光源以速度V移动时，

前进方向上波长被压缩，(λ0 － △λ) f = C －v，波长最短。

相反方向上波长被拉长，(λ0 ＋ △λ) f = C ＋v，波长最长。

其中△λ代表波长形变量，满足△λ f=v，它随着光源速度v的变化而变化，v变大，它就跟着变大。中间地带上波长由长到短依次递减。

举个简单的例子，如果光源速度为3000米每秒，身后的波长是变长的，形变后的波长与频率的乘积=c+3000，这个数值虽然大于c，但却是相对于光源的，与光源的3000相抵消后，每一个光波相对于空间的移动速度还是c。

光源速度提高到5000，那么乘积也会随之变成c+5000，依然能相互抵消，使得每一个光波在空间中的传播速度恒不变。

7.再回看迈莫实验，初始，钠光源随着地球以速度v运动，前进方向上（M1去程）波长满足(λ － △λ) f = C －V，同理，反方向满足(λ ＋ △λ) f = C ＋V。垂直方向则是满足λf=C.

在下图M1光路上，光波的初始波长是λ － △λ，而光波追赶M1反光镜的相对速度是C-V,而这两者恰好符合正比关系。比例系数为f，即单位时间内的光波的个数。

同样，在被M1反光镜反射回的光路中，光波的波长是λ ＋ △λ，光波相对于半透镜来说是迎头相撞，速度为C+V,它们之间同样符合正比关系，比例系数同为f，即单位时间内的光波的个数。

![](https://pic2.zhimg.com/v2-cdfad332b919fa7cff7e7ecfd5a51e0b_1440w.jpg)

由于波长的改变，速度与波长的正比关系恰好没有变化，因此单位时间内（或者说相同时间内）两光路的波的数目是相等的，都是2k（2f）个波，方向的差异并没有造成波的个数的差异，因此旋转90度之后并不会变成2k+0.2个波和2k-0.2个波，波的数目没有变，也就没有产生波的相位差，没有相位差，所以明暗条纹也就没有移动。

其实V取30千米每秒是有时代的局限性的，整个太阳系还被太阳拉着以220千米每秒狂奔，按照两位设计师的思路条纹移动还要翻七倍才行，更加不符合事实。但是按照这个理论，不管速度V是按照地球公转速度30千米每秒还是按照太阳系整体移动速度220千米每秒，抑或其他任何数值，结果都是零。

迈莫实验的原始推算分为了两步，一是先求时间差，二是根据时间差求光程差、相位差也就是波的个数差。而上述推论是只有一步计算，是直接推导单位时间内光波的个数差异，从理论上从推算出相位差为零，从逻辑上验证了在绝对时空的条件下，迈莫实验条纹零移动依然是合理的，必然的。
