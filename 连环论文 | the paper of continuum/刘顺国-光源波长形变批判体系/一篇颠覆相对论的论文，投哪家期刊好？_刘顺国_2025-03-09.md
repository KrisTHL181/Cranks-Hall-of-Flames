**关于迈克尔逊莫雷实验零移动原因的探讨**

**刘顺国**

**（山东省济南市章丘区，** xxxxx， **250200）**

摘要：一百多年前的迈克尔逊莫雷实验被称为物理大厦上方的两朵乌云之一，其实验结果出现了诡异的零移动，最终人们不得不得出两个结论，一是以太不存在，二是光速在各个惯性系是相等的，即光速不变原理，后者催生出了洛伦兹变换和狭义相对论。然而，回顾该实验，实验结果零移动的原因有可能是因为两位设计师在推算中采用了固定波长590纳米，而忽略了移动光源在各个方向上波长是不同的。一个移动的光源，在前进方向上波长是压缩的，反方向上是被拖曳拉长的，中间是一个递变的过程。光源的速度越快，波长的形变越大，形变量由速度决定。

关键词：迈克尔逊莫雷实验、光速不变、波长形变、方向

一百多年前的迈克尔逊莫雷实验之所以出现零移动，没有出现推算中的0.37，有一种可能是因为两位设计师在推算中采用了固定波长590纳米，而忽略了移动光源在各个方向上波长是不同的。

一个移动的光源，在前进方向上波长是压缩的，反方向上是被拖曳拉长的，中间是一个递变的过程。

一个移动光源，如果从速度V1提高到V2，反方向上的波长理应增大的，λ2＞λ1。如果不考虑多普勒效应，只考虑光源的本征频率，那么频率应该是不变的。

这里就出现了一个问题，如果频率不变而波长增大，频率与波长的乘积应该是变大的，它应该满足一个等式： λ2·f －λ1·f= V2 －V1 。

那么，假设光源在静止时发出的光波波长为λ0，C=λ0×f。

当光源以速度v移动时，相反方向上的光线波长是被拉长，是变大的，是＞λ0的。其波长形变量与光源速度v密切相关。速度越大，形变量越大。

波长变大，而频率不变，这就会导致两者的乘积（波速）变大。这看起来与光速恒定相违背。但这一点也不矛盾。因为这个速度的数值是相对于反方向移动的光源的。

换成通俗的语言，

如果光源的速度是1000米/秒，那么反方向的乘积变成C+1000米/秒.

假设光源的速度提高了，变成了10000米/秒，那么光波被拖曳的更长，乘积随之变成C+10000米/秒。

假设光源的速度再次提高，变成了3万米/秒，同上，乘积变成C+3万米/秒。

**但是这个乘积是个相对速度，是相对于反方向光源的速度，是个相对值。**

**无论光源的速度如何变化，波长总会对应发生形变** ，乘积（波长×频率）也会随之相应的发生变化，但是它们之间相互抵消后，任意一个完整波或者光波上的任意片段，在空间中的传播速度，恒定不变，恒为C。

人们总说光的速度不受光源速度的影响，至于为什么？原因语焉不详。这或许就是其原因。

如果用公式来表达

相反方向上波长被拉长，(λ0 ＋ △λ) f = C ＋v，波长最长。

前进方向上波长被压缩，(λ0 － △λ) f = C －v，波长最短。

其中 △λ f=v，△λ代表波长形变量，它随着光源速度v的变化而变化，v变大，它就跟着变大。

中间地带上波长由长到短依次递减，中间位置波长为λ0。

对于第二条波长被压缩的情况，可用来解释公众对于跑动中打开手电，光线是否超光速的广泛疑问，这种情况下波速乘积等于C-V,叠加光源（手电）速度V，光波的空间传播速度仍然恒为C。

理解了这一点再来看迈莫实验。迈莫实验为什么没有出现条纹移动？或者说，为什么旋转90°之后两条光路没有出现相位差？问题出在哪里？问题可能就在于没有考虑光源不同方向上的波长形变。在推算时是按照固定波长590纳米这个固定数值来计算的。

初始，钠光源随着地球以速度v运动，前进方向上（M1去程）波长满足(λ － △λ) f = C －V，同理，反方向满足(λ ＋ △λ) f = C ＋V。垂直方向则是满足λf=C.

在下图M1光路上，光波的初始波长是λ － △λ，而光波追赶M1反光镜的相对速度是C-V,而这两者恰好符合正比关系。比例系数为f，即单位时间内的光波的个数。

同样，在被M1反光镜反射回的光路中，光波的波长是λ ＋ △λ，光波相对于半透镜来说是迎头相撞，速度为C+V,它们之间同样符合正比关系，比例系数同为f，即单位时间内的光波的个数。

![](https://pic3.zhimg.com/v2-0cce08562bed7c972b8930159c207bda_1440w.jpg)

理解了这一点再看上图，由于波长的改变，速度与波长的正比关系恰好没有变化，因此两条光路长度相等的话，两光路的波的数目是相等的，都是2k（2f）个波，方向的差异并没有造成波的个数的差异，因此旋转90度之后并不会变成2k+0.2个波和2k-0.2个波，波的数目没有变，也就没有产生波的相位差，没有相位差，所以明暗条纹也就没有移动。

所以，如果实验只转动台面，同时改变两光路方向的话是不会有条纹移动的，不管这个速度V是按照地球公转速度30千米每秒，还是整个太阳系速度220千米每秒，抑或银河系速度600千米每秒，结果都是零。

迈莫实验的原始推算分为了两步，一是先求时间差，二是根据时间差求光程差、相位差也就是波的个数差。而上述推论是只有一步计算，是直接推导单位时间内光波的个数差异，从理论上从推算出相位差为零，从逻辑上验证了迈莫实验条纹零移动是合理的，必然的。

另外，这里讨论的是绝对空间里的绝对波长，如果仪器处于空间中绝对静止状态，它理应能检测到光源前后的波长差异，但如果仪器与光源同向同速运动，处于相对静止，那么它是检测不到这种差异的。

要验证这个猜想，有一种思路就是光源像粒子加速器那样圆环加速，仪器静止在切线方向的前后观测，并测试不同速度下的波长形变量。

综上，对于迈克尔逊莫雷实验的零移动，文章从经典物理和绝对时空的角度提出了一种新的诠释，至于正确与否仍需深入讨论及验证。
