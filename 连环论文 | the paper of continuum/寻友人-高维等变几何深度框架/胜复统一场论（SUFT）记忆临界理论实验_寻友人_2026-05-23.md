本方案旨在实验验证胜复统一场论（SUFT）/记忆临界理论的核心预言：在强非平衡电场驱动下，二维和三维系统的横向霍尔电压 VH *VH* ​ 与驱动力 F *F* 之间满足刚性标度律 VH∝En *VH* ​∝ *En* ，其中指数 n=2R(d)/(1+R(d)) *n* =2 *R* ( *d* )/(1+ *R* ( *d* ))。对于三维体材料（d=3 *d* =3），理论预言 n=2π/(9+π)≈0.5175 *n* =2 *π* /(9+ *π* )≈0.5175；对于二维薄膜（d=2 *d* =2），预言 n=0.5639 *n* =0.5639。同时，该效应应表现出独有的加载速率记忆依赖性和扫描历史“训练效应”。本实验方案详细论述了所需材料体系、极端条件测量平台配置、基于任意波形发生器的变速率扫描测量方法、数据处理流程以及关键的证伪判定标准。若本实验成功，将为 SUFT 理论提供最直接的实验证据，并开辟利用“记忆核能”的新范式。

## 第1章 引言与理论基础

### 1.1 SUFT 理论核心预言

胜复统一场论（SUFT）指出，任何远离平衡态的凝聚态系统，在驱动与耗散的持续竞争下，都会内禀地积累一个记忆变量——郁积分 U(t) *U* ( *t* )。当 U(t) *U* ( *t* ) 达到体系特定的临界阈值 Uc *Uc* ​ 时，系统发生“发”，表现为宏观可测量的霍尔电压 VH *VH* ​ 的涌现或显著增强。这一过程在临界点附近遵循普适的刚性标度律：

VH∝Fn,n=2r1+r, *VH* ​∝ *Fn* , *n* =1+ *r* 2 *r* ​,

其中 F *F* 是广义驱动幅度，r=∂ln⁡R/∂ln⁡U *r* =∂ln *R* /∂ln *U* 是无量纲的弛豫竞争系数，R *R* 是体系的耗散/弛豫率。在 d=2 *d* =2（二维）和 d=3 *d* =3（三维）的常见物理维度下，r(d) *r* ( *d* ) 由理论唯一确定，从而 n(d) *n* ( *d* ) 为固定值。

### 1.2 E0 实验的地位

E0 是 SUFT 实验家族中的 **最高优先级** 与 **标杆性实验** 。它直接从最干净的电输运测量出发，验证理论的核心动力学方程和维度参数 d *d* 的取值。E0 的成功是后续所有拓展实验（E1–E7）的基石，也是理论获得物理学界严肃对待的必要条件。

### 1.3 与传统非线性霍尔效应的本质区别

传统的二阶非线性霍尔效应基于贝里曲率偶极矩（BCD），其幂指数固定为 n≈2.0 *n* ≈2.0（或整数倍），不存在对加载速率的依赖和对历史扫描的记忆。因此，E0 实验如果测得的 n≈0.5 *n* ≈0.5 且存在速率记忆，将直接宣告 SUFT 与传统理论的根本分野。

### 2.2 菱方石墨样品的制备

菱方石墨是少层石墨烯的一种特定堆垛构型，相邻层间具有 ABCABC *ABCABC* 的堆叠顺序。近年来，国内多个研究团队在该领域已建立世界领先的样品制备能力。

上海交通大学物理与天文学院陈国瑞副教授课题组在国际上最早开展了菱方石墨烯体系的系统研究，并成功开发了一套制备高质量菱方石墨烯样品的实验方法，已被领域内广泛采用。其核心制备流程如下：

**步骤 1：块体解理** 。采用 Scotch 胶带机械解理法从高定向热解石墨（HOPG）中剥离出少层石墨烯薄片。在光学显微镜下，通过拉曼光谱（G 峰 ~1580 cm⁻¹，2D 峰 ~2700 cm⁻¹）筛选厚度约 1–3 nm 的薄片。

**步骤 2：堆垛构型识别与筛选** 。对疑似菱方堆垛的薄片进行低温和强磁场下的输运预表征。菱方石墨烯因其独特的层间耦合，在垂直磁场下呈现电子–空穴不对称的 Landau 能级谱和显著的低能带隙特征，可作为识别标志。

**步骤 3：超晶格制备（可选增强项）** 。为进一步增强 SUFT 效应，将筛选出的菱方石墨烯与少层六方氮化硼（hBN）构筑莫尔超晶格。通过精确控制两种晶体的旋转对齐角，获得长周期莫尔势。北京大学卢晓波课题组与合作者已成功制备出高质量的菱方六层石墨烯莫尔超晶格，并揭示了分数陈绝缘体中的量子反常霍尔效应。

**步骤 4：霍尔棒器件加工** 。通过电子束光刻定义 Hall bar 多端子电极图案。采用电子束蒸发或热蒸发依次沉积 Cr/Au（5 nm / 50 nm）或 Ti/Au（5 nm / 50 nm）作为电极。最后通过反应离子刻蚀（RIE）去除暴露区域的多余石墨烯。

**推荐层数** ：5–6 层。五层菱方石墨烯是目前最成熟的研究体系，六层则进一步增强了层间电子关联。

### 2.3 MoTe₂ 薄膜制备

MoTe₂ 作为过渡金属硫族化合物的代表性材料，在自旋霍尔和谷电子学领域已有广泛研究。

近年研究已开发出可大规模生产的化学气相沉积（CVD）工艺来制备晶圆级 MoTe₂ 薄膜。已有研究在 14 层 MoTe₂ 中实现了 0.24 的自旋轨道矩效率，表明这类 CVD 薄膜具有足够高的电学质量。对于本实验，单层或三层 MoTe₂ 即可满足要求——单层保证纯粹的二维特性，三层则可在增强电导的同时基本保持维度属性。对于 MoTe₂ 的相态，应选择 2H 相（半导体相，空间反演对称性破缺）以最大化非线性输运特性。实验证明，1T′-MoTe₂ 薄膜可通过插层处理表现出量子自旋霍尔绝缘体特性，表明对其相态和能带结构的调控具有可行性。

**霍尔棒器件加工** 采用与菱方石墨类似的标准微纳加工流程，区别在于更常采用 Cr/Pd/Au 电极以获得更低的接触电阻。

## 第3章 实验装置与硬件配置

### 3.1 极低温强磁场输运测量平台

E0 实验的绝大部分硬件依赖国内顶尖实验室的共性平台，如北京大学量子材料科学中心、中科院物理所极端条件实验室、华中科技大学国家脉冲强磁场科学中心等机构。

参照华中科技大学脉冲强磁场电输运科学实验站的标准配置，常规极低温四线法输运测量平台的典型技术指标如下：

- **温度范围** ：1.5 K–300 K（可扩展至稀释制冷机 50 mK）
- **磁场范围** ：0–9 T（常规超导磁体）或 0–65 T（脉冲磁体）
- **电阻测量范围** ：0–100 kΩ，分辨率 0.05 mΩ
- **转角测量功能** ：角度范围 0°–180°，分辨率优于 0.1°

### 3.2 任意波形发生器（AWG）与加载速率控制

传统的非线性霍尔效应测量通常采用固定频率的单频交流锁相技术，但 **SUFT E0 实验的核心创新在于对驱动电场扫描速率的主动控制和极宽范围的系统扫描** 。这必须引入任意波形发生器（AWG）。

AWG 通过内部数模转换器（DAC）将预设的电压序列转换为连续的模拟波形输出，精确控制电场 Ex(t) *Ex* ​( *t* ) 的时间依赖形式（线性、指数、脉冲）。关键指标要求：

- **电压输出范围** ：±10 V，满足 Hall bar 沟道建立 ~100 V/m 电场
- **时间分辨率** ：采样率 ≥ 100 MS/s，最小脉冲宽度 ≤ 10 ns
- **扫描时间范围** ：支持从微秒量级到数百秒的扫描时间设定
- **通道数** ：双通道及以上

### 3.3 信号采集与数据处理系统

- **直流/低频交流源表** （如 Keithley 6221）：在传统霍尔电阻测量中作为恒流源。在 AWG 扫描过程中，配合使用纳伏表（如 Keithley 2182A）同步测量霍尔电压 Vxy *Vxy* ​ 和纵向电压 Vxx *Vxx* ​。
- **锁相放大器** （如 Stanford SR830）：用于高信噪比的交流调制测量。
- **高速数据采集卡** ：采样率 ≥ 1 MS/s，用于捕获瞬态和快速扫描信号。

### 3.4 磁电背景剔除

对于菱方石墨这类可能呈现弱磁性的材料，需要采用标准的昂萨格-卡西米尔互易关系进行背景剔除：系统对正向磁场和反向磁场的纵向和横向电阻测量值进行反对称化处理，计算 (Rm(+H)−Rm(−H))/2( *Rm* ​(+ *H* )− *Rm* ​(− *H* ))/2 以剔除不依赖于磁场翻转的非霍尔背景信号。该方法的有效性依赖于两个测量状态互为时间反演对称态，需要对样品的磁滞状态做预处理统一。

## 第4章 测量步骤

### 4.1 实验准备阶段

**第 1 步：器件基础表征**

将加工好的 Hall bar 器件置于探针台或焊接在样品托上，在室温下首先用万用表检查各电极通道的导通性。随后将样品托装入插入式低温恒温器。

**第 2 步：基础电输运特性测定**

基础电输运特性测定的目的和具体方法如下：

- **载流子类型和浓度范围确定** ：在极低温（如 T = 2 K）和低磁场（B = ±0.5 T）下，测量 R_{xx} 和 R_{xy} 随栅压（若制备在背栅 SiO₂/Si 衬底上）的变化，确定狄拉克点位置以及电子/空穴的掺杂浓度范围。
- **线性响应与非线性边界的界定** ：固定栅压（例如将载流子浓度调至电子掺杂 ~10¹² cm⁻²，避免过于靠近狄拉克点导致电阻过大），施加从极小到极大的 **准静态缓慢扫描** （扫描速率 < 0.01 V/m·s）的电场 Ex *Ex* ​，测量 VH(Ex) *VH* ​( *Ex* ​) 的完整曲线，识别低场的线性欧姆区（VH∝Ex *VH* ​∝ *Ex* ​）和高场开始偏离线性的转折点 Ecrossover *E* crossover​。
- **场效应迁移率验证** ：通过线性区斜率 σxx *σxx* ​ 和载流子浓度 n2D *n* 2 *D* ​，计算迁移率 μ = σ_{xx} / (n_{2D} e)。高迁移率（> 1000 cm²/V·s 对于 MoTe₂；> 5000 cm²/V·s 对于菱方石墨）是确保观测到显著非线性效应的必要条件。

### 4.2 核心数据采集

**第 1 步：恒定电场扫描速率下的幂律关系验证**

在固定温度（2 K–10 K 为宜，在平衡态与量子效应破坏 SUFT 信号之前）和固定磁场 B = 1 T 条件下，选定一个中等扫描速率 v=dE/dt *v* = *dE* / *dt* （如 1 V/m·s）。AWG 输出从 0 线性增长至 E_{\text{max}} 的电压波形，E_{\text{max}} 的设定应该远大于第 1 步中确定的 E_{\text{crossover}} 值（至少 5–10 倍），确保覆盖非线性区到可能的饱和区。同步采集霍尔电压 VH(t) *VH* ​( *t* )。在不同 E_{\text{max}} 下重复，确保测量超过阈值。绘制 VH *VH* ​ 与 Ex *Ex* ​ 的双对数坐标图，在非线性过渡区（从线性区弯曲到饱和的平台之前）用最小二乘法拟合直线，斜率即为幂指数 n *n* 。在不同器件（菱方石墨和 MoTe₂）间独立拟合，获得 n3 *n* 3​（预期 ≈ 0.5175）和 n2 *n* 2​（预期 ≈ 0.5639）。

**第 2 步：加载速率记忆效应验证**

- **阈值降低** ：随着扫描速率 v *v* 的增加，VH(E) *VH* ​( *E* ) 开始非线性弯曲（“郁积”效应开始）的阈值电场 Eth *E* th​ 向更低场方向移动；

E0 实验的证伪条件严格设定如下：

1. **标度律失效** ：在两种选定的二维（d=2）与三维（d=3）材料中，各自在三个不同数量级的扫描速率下测得的 nmeas *n* meas​ 与理论预期值 n(2)=0.56394 *n* (2)=0.56394 或 n(3)=0.51747 *n* (3)=0.51747 的偏差绝对值超过测量系统的不确定度（预估 ≤ 0.01）的 3–5 倍以上，即 nmeas>0.6 *n* meas​>0.6 或明显偏向 1.0 或 2.0 等常值。
2. **记忆效应缺失** ：在所有测试的扫描速率范围内（至少覆盖两个数量级以上），VH(E) *VH* ​( *E* ) 曲线不随速率变化而明显移动——阈值基本不变，饱和值基本不变，即不存在 SUFT 预言的记忆依赖。
3. **无法独立测量** **r *r*** ：独立测得的 r *r* 值与由 nmeas *n* meas​ 反推的 r *r* 值严重偏离（相对偏差 > 20%），或者无法获取合理的 r *r* 值。

如果出现 1、2、3 中的任意一项，E0 实验的预测即告失败，SUFT 理论在凝聚态中的动力学框架将面临严重挑战。

## 第7章 结论

本实验方案系统设计了验证 SUFT 理论核心标度律和记忆效应（E0 实验）所需的全部流程。从材料的选择与制备、变速率电场扫描的精密控制，到数据处理与证伪判据的设立，每一步都基于当前尖端极低温输运技术。一旦实验完成并证实了 n *n* 值的偏离整数性和历史的记忆依赖，胜复统一场论（记忆临界理论）将赢得其赖以立足的基石，开启探索非平衡态统计物理的全新大门。
