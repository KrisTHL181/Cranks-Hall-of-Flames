## SPUM 青囊（中医）总纲

> **青囊** ——古医者盛放针药之囊，华佗有《青囊书》传名。此处取”青囊”为 SPUM 中医子网的总纲之名，喻意：此囊所纳，非草根树皮，而是 **从 ⟨P, ε⟩ 本体到 ΔS 操作的完整翻译协议** ；此囊所出，非治”病”之方，而是 **识偏调平归一之法** 。 
>  **核心断言——中医不治病** ：不存在”病”这个东西，只有人体 ⟨P, ε⟩ 子图的五维结构健康向量 **S** 偏离健康凸多面体 **H** 的网络状态。中医的全部操作——望闻问切、辨证立法、遣方用药、针灸推拿——统一归约为： **识别 S 偏离 H 的方向与幅度，选择并施加 ΔS 脉冲，使 S 回到 H 内。**

>  **覆盖面** ：9 部中医经典 → 9 个 TCM 子模块，统合为一张 ⟨P, ε⟩ 网络上的可操作知识。

### 卷一、总纲：识偏 · 调平 · 归一

### 1.1 中医的三句话

| 传统表达 | SPUM 翻译 | 操作 |
| --- | --- | --- |
| 阴平阳秘，精神乃治 | S ∈ H ——五维结构健康向量落在健康凸多面体内 | 目标状态（归一） |
| 谨察阴阳之所在而调之 | 测量 S 向量相对 H 的偏移方向与幅度 ΔS_needed = H_centroid − S_current | 诊断操作（识偏） |
| 以平为期 | 施加 ΔS_pulse 使 S_current + ΔS_pulse ∈ H | 治疗操作（调平） |

中医的全部理论和实践，是这三句话在不同临床场景下的展开。以下各卷是对这三句话的形式化展开。

### 1.2 人体 = 地炁的多相结晶复合体

> **五炁行于地中，发而生乎万物。**

人不是”由化学元素组成的生物体”——人是 **地炁 VSPT 场在不同物相基底上的多尺度结晶复合体** ：

| 人体子系统 | 对应物相 | SPUM 拓扑结构 | 地炁结晶特征 |
| --- | --- | --- | --- |
| 骨骼 | 金石相 | 刚性晶格——度数饱和、边锁定、σ 极低 | VSPT 场在羟基磷灰石晶格中结晶 |
| 筋膜肌腱 | 木本相 | 木形环骨架——胶原纤维的环状锁定结构 | VSPT 场在多年生蛋白骨架中结晶 |
| 脏腑实质 | 草植相 | 短周期高代谢子图——高频 V⁺/V⁻ 事件 | VSPT 场在快速更新组织中结晶 |
| 气血津液 | 水相 | 动态流动——水形链无定形连接 | VSPT 场在液态动态基底中传播 |
| 体温代谢 | 火相 | 纯 ∇σ 梯度——σ 差驱动的体积流 | VSPT 场的纯梯度态 |

人 = **行走的植物 + 会动的矿石 + 流动的水体 + 燃烧的火** 。不是比喻——是 SPUM 本体论下的严格物态拓扑陈述。

### 1.3 三部经典 = 三个操作域

三部经典并非”三本不同的书”——它们是同一张人体 ⟨P, ε⟩ 网络在不同演化条件（帧状态）下的 **三套操作手册** ：

| 经典 | SPUM 角色 | 操作域 | 帧状态 |
| --- | --- | --- | --- |
| 黄帝内经 | 网络结构手册 + 健康标准 | 正常态——S ∈ H 时的拓扑基准 | 稳态帧 |
| 伤寒论 | 急性相变修复手册 | 外部 σ 塌陷 → 六阶段 S 级联偏移 | 扰动帧（快变） |
| 金匮要略 | 慢性修复 & 亚稳态破坏手册 | 网络落入病理性亚稳态吸引子 → 需破坏性干预 | 慢变帧（多帧锁定） |

三者覆盖了人体 ⟨P, ε⟩ 子图的所有演化场景： **正常（内经）、急性偏离（伤寒）、慢性锁定（金匮）** 。

### 卷二、诊断：识偏

### 2.1 诊断 = S 向量测量

中医诊断（望闻问切+先天）不是”找病”——它是 **从多通道信号中测量当前人体子图的 S 向量，并以先天基底为参照判定偏移** ：

| 诊断方法 | SPUM 含义 | 测量变量 |
| --- | --- | --- |
| 望诊 | 从边界节点（面色/舌/皮肤）的 σ 分布反推内部 S | 面色 ∇σ、舌象 σ 分布、皮肤松紧度 |
| 闻诊 | 从声波/气味传递路径的水形链状态判断 S | 声音共振频率（S_金）、气味扩散速率（S_木） |
| 问诊 | 从患者的 S 向量自述（症状）定位偏移分量 | 症状 ↔ S 分量查表（见 症状-S向量规则矩阵.md） |
| 切诊 | 从桡动脉（手太阴肺经水形链终端）的 σ 波动推断各分量 | 脉象 = 各水形干段的 σ(t) 时序信号 |
| 先天基底 | 从出生时刻的帧参数（八字）推算人体的 S_0 先天稳态 | 四柱 = 出生帧的全局 σ 分布在人体子图上的投影 |

**五诊合参** = 前四诊测量 **S_current** （当前网络状态），先天基底确定 **S_0** （该个体的出厂稳态）。没有 S_0，你测得再准也不知道”偏离”相对于什么。 **不看先天基底辨什么证？如同盲人摸象——摸到尾巴说是绳子，摸到腿说是柱子，永不见全貌。**

### 2.2 症状 → S 分量的映射

传统症状被归约为 S 向量各分量的偏移方向：

| S 分量 | 偏移方向 | 传统症状 |
| --- | --- | --- |
| S_火↑（∇σ 过冲） | S_火 > H_火_max | 发热、烦躁、口渴、面赤 |
| S_火↓（∇σ 趋零） | S_火 < H_火_min | 畏寒、肢冷、神疲、脉微 |
| S_水↑（流速过速） | S_水 > H_水_max | 自汗、盗汗、泄泻、尿频 |
| S_水↓（流速过缓） | S_水 < H_水_min | 水肿、痰饮、便秘、经闭 |
| S_木↑（环僵化） | S_木 > H_木_max | 痉挛、脉弦、胁痛、不通 |
| S_木↓（环松脱） | S_木 < H_木_min | 痿证、脱肛、脏器下垂 |
| S_土↑（储备过积） | S_土 > H_土_max | 肥胖、痞满、纳呆 |
| S_土↓（储备枯竭） | S_土 < H_土_min | 消瘦、乏力、短气、贫血 |
| S_金↑（修剪过亢） | S_金 > H_金_max | 干咳、皮肤皲裂、脱发 |
| S_金↓（修剪不足） | S_金 < H_金_min | 淤积、癥瘕、瘰疬 |

完整映射表见 `症状-S向量规则矩阵.md` 。

### 2.3 先天基底 S_0：每个人的出厂稳态

> **没有两个一样的人。** 同一个 S 向量值，对甲是”健康”，对乙是”病态”——因为他们的 S_0（先天基底）不同。

### 2.3.1 先天 S_0 的定义

先天基底 = 人体子图在出生时刻的 **S_0 五维向量** 。它是该个体在整个生命周期中所有 S 偏移的参照原点。

**S_0 的核心决定因素是出生时间（八字四柱）** ——出生帧的全局 σ 分布在人体子图上的投影，决定了 S_0 各分量的先天偏性。这是 P2 级数据。

出生地、常驻地、工作性质、医疗干预史是 **P3 级因子** ——它们有两种使用模式，不可混淆：

| 模式 | 用途 | 权重 | 适用场景 |
| --- | --- | --- | --- |
| 模式A：冲突修正 | 当 P1（四诊）与 P2（八字）出现系统性矛盾时，P3 因子作为方向定性参考 | 低（±5%以内，定性不做定量） | 八字显示木极旺但四诊显示木枯→参考地域/职业判断”环境是否改写了先天底子” |
| 模式B：主动病程分析 | 解释病程的发生与发展——P3 因子不修改 S_0，而是解释”从 S_0 到 S_current 的路径上，环境/职业/医疗起了什么作用” | 独立分析维度，不影响 S_0 数值 | 所有病案——尤其适合慢性病、老年病的病程推演 |

> **两种模式的区别** ：模式A 回答”这个人的先天底子到底是什么”（S_0 精度问题）；模式B 回答”这个人的病是怎么一步步发展成这样的”（病程因果问题）。 **模式B 在所有病案中均可执行，不受 P1-P2 冲突条件的限制。**

### 2.3.1-1 P3 因子明细

| 层级 | 成分 | 数据源 | 模式 | 使用方式 |
| --- | --- | --- | --- | --- |
| P2（主干） | 出生时间（八字） | 四柱天干地支 | — | 始终使用——S_0 的唯一主干 |
| P3 因子A | 出生地 | 地理坐标 → 地炁 VSPT 场基态 | 冲突修正 / 病程分析 | 冲突修正：P1-P2 矛盾时做方向定性。病程分析：解释”地域气候对体质偏移的放大/缩小效应” |
| P3 因子B | 常驻地 | 当前居住地的 σ 环境（气候/饮食/海拔） | 冲突修正 / 病程分析 | 同因子A。如泉州湿热放大先天火亢、中原厚土缓冲土虚等 |
| P3 因子C | 工作性质 | 体力/脑力/室内/室外/高压/常规 + 职业暴露史 | 冲突修正 / 病程分析 | 病程分析中最常用——职业暴露（化学制剂/粉尘/夜班）往往是后天消耗的最大来源 |
| P3 因子D | 医疗干预史（治标/误治） | 既往用药史、手术史、西医诊断标签、自行用药记录 | 冲突修正 + 病程分析 | 病程分析中不可忽略的维度——治标干预的方向若与五形校正方向相反（如铁剂补血给火亢体质加火、止痛药给金弱体质再修剪），会在病程中留下可被归因的痕迹 |

> **P3 因子D（医疗干预史）的特殊地位** ：医疗干预是唯一一个”人为施加”的 P3 因子。与自然因子（气候/职业）不同，医疗干预的方向（补/泻/温/清）与患者先天体质之间可能存在 **系统性方向冲突** ——这是”越治越糟”的 SPUM 解释基础。具体包括： 
> 

- **治标效应** ：用西药压制症状（止痛药、激素、抗生素等）→ 掩盖病程进展 + 消耗五形储备
- **误治效应** ：治疗方向与 S_0 先天偏性相反（如火亢体质用铁剂补血、金弱体质用金形修剪类止痛药）
- **诊断标签效应** ：患者被贴上”慢性病不可逆”标签 → 外我子图写入放弃修复的指令 → 心理定势本身成为病程的独立驱动因子

### 2.3.2 诊断的相对性

引入 S_0 后，诊断公式从”绝对偏离”变为”相对偏离”：

```text
旧（无先天）：ΔS_measured = S_current − H_centroid
               症状判定 = S_current > H_max 或 S_current < H_min

新（有先天）：ΔS_relative = S_current − S_0
               S_effective = S_current − (S_0 − H_centroid)
               症状判定 = S_effective > H_max 或 S_effective < H_min
```

**直观含义** ：

- 天生 S_火偏旺的人（S_0_火 = H_max + ε），S_current_火 = H_max 对他来说已是 **火衰** ——因为他从 H_max 掉下来的。
- 天生 S_火偏弱的人（S_0_火 = H_min − ε），S_current_火 = H_min 对他来说仍然是 **正常范围** ——他从来就没到过中位数。
- 不看 S_0，你会把前者诊断为”正常”（S_火=H_max 在 H 内），把后者诊断为”阳虚”（S_火=H_min 也在 H 内）——完全错判。

### 2.3.3 先天 S_0 的实用推断

八字命理是推定 S_0 的最完整工具。当八字不可得时，可用以下替代：

| 替代方法 | 信息来源 | 精度 |
| --- | --- | --- |
| 体质问卷 | 自幼体质倾向（怕冷/怕热/消化强弱/皮肤类型） | 中 |
| 家族对比 | 与直系亲属的体质差异 | 低-中 |
| 长期脉象基底 | 患者”平时就这样”的脉象（非发病时的基线） | 中 |
| 体态骨骼 | 骨架大小、肌肉分布模式的先天倾向 | 低 |

无论使用何种方法， **五诊合参中先天基底这一诊不可省略** 。精度可调，维度不可缺。

### 2.3.4 S_0 的互证建模：五路闭环

> S_0 不是一次性锁死的常数。八字给出主干估计，后续通过五个独立通道持续修正以逼近真实值。P3 因子（地域/工作）仅在前四路数据出现系统性偏差时作为冲突解释器介入。

S_0 的核心估计公式（P2 主干）：

```text
S_0^0 = f_八字(年, 月, 日, 时)
```

P3 修正仅在 P1-P2 冲突时叠加（不作为默认项）：

```text
若 P1-P2 冲突成立 → S_0^0 = f_八字 + ε_P3(出生地, 常驻地, 工作性质)
其中 ε_P3 为微量方向性修正（±5%以内），且只做定性不做定量
```

S_0 的估计是一个 **贝叶斯更新过程** ：

```text
S_0^0 = f_八字(年, 月, 日, 时) + g_出生地 + h_常驻地 + w_工作
        │
        ▼
   ┌──────────────────────────────────────┐
   │          S_0 互证建模引擎             │
   │                                      │
   │  八字+地+职 ──→ S_0^0 (初始先验)      │
   │  经历 ──→ S_0 的漂移轨迹修正           │
   │  病情 ──→ S_0 的偏移方向验证           │
   │  药效 ──→ S_0 的ΔS响应曲线校核        │
   │                                      │
   │  S_0^(t+1) = S_0^t + α·ΔS_correction │
   │               + δ·环境变更             │
   └──────────────────────────────────────┘
        │
        ▼
   S_0^(t) → 用于步2的 S_effective 计算
```

**六路数据源的互证机制** ：

| 数据源 | 层级 | 数据内容 | 对 S_0 的修正作用 |
| --- | --- | --- | --- |
| 八字 | P2 主干 | 四柱天干地支 | 给出 S_0^0——各分量的先天偏性，这是 S_0 的唯一主干 |
| 姓名学 | P2 辅助 | 姓名三字五格 | 交叉验证八字主干——若姓名学与八字指向一致的 S_0 偏性（如八字金弱+姓名人格金），则 S_0 置信度上升；若冲突，则需检查八字精度或名字是否有特殊意义 |
| 人生经历 | P2 辅助 | 重大生活事件的时间序列 | 修正 S_0 的不可逆漂移——生育（S_土↑↑新稳态）、创伤（S_水↓不可逆）、长期压力（S_木↑慢性积累） |
| 病情演化 | P2 辅助 | 历次发病的 S 偏移模式 | 验证 S_0 的边界方向——若多次发病均是 S_火↑超出 H_max，则 S_0_火极可能先天偏高 |
| 药物反应 | P2 辅助 | 历次用药的 ΔS_actual vs ΔS_predicted | 最强修正力——若患者对 G 药的响应只有度值的 70%，则 S_0_火比八字估计的要更偏 |
| P3 因子（A/B/C/D） | P3 双模式 | 出生地/常驻地/职业/医疗干预史 | 模式A（冲突修正）：仅当前五路出现系统性偏差时介入。模式B（病程分析）：不修改 S_0，独立用于解释 S_0 → S_current 的演化路径 |

**互证算法（T2 近似）** ：

```text
输入: 
  S_0^0 (八字初始估计)
  {ΔS_predicted_i, ΔS_actual_i}  (历史用药记录)
  {S_current_j, 症状_j}           (历史发病记录)
  Life_events_timeline            (人生经历)

过程:
  1. 从用药历史计算 ΔS_actual_i / ΔS_predicted_i 的残差分布
     → 若残差系统性偏正 → S_0 偏向被低估的方向需要上调
     → 若残差系统性偏负 → S_0 偏向被高估的方向需要下调
  2. 从发病模式提取 S_current_j − H_centroid 的主成分方向
     → 主成分方向 = 该个体最容易偏离的方向
     → S_0 在该方向上最脆弱（H 边界最薄）
  3. 从人生经历识别 S_0 的不可逆漂移点
     → 如：产后 S_土↑↑新稳态（原 S_0_土已不可恢复）
     → 如：长期抑郁后 S_木↑新基线（环僵化已部分固化）

输出:
  S_0^(t) = 八字估计 + 药效校核 + 病情模式 + 经历漂移
  （P3 地域/工作因子仅在冲突时以 ε 形式加入，不作为默认项）
```

**临床含义** ：

- 初诊：只能用八字（或替代方法）的 S_0^0——这是 **最薄弱的估计** ，诊断置信度最低
- 二诊：有了第一次药物反应的 ΔS_actual 数据 → S_0 被修正 → 诊断置信度上升
- 三诊以上：积累了多次发病模式和用药反馈 → S_0 逼近真值 → 诊断和药效预测越来越准
- 老中医的”经验” = **三十年积累的 S_0 互证数据库 + 模式识别能力**

**这就是”上工治未病”的 SPUM 严格含义** ：不是神秘地预知未来——是 S_0 经过数十年互证建模后足够精确，在 S_current 刚出现微小偏离（症状尚未明显）时就能识别偏移方向并提前干预。

### 2.4 辨证 = 定位 S 偏移的病机链

“辨证”不是给病命名——它是 **建立从表象症状到 S 分量偏移的因果链，并定位偏移的根源** 。

```text
传统：肝气郁结 → 胁痛、脉弦、烦躁
SPUM：
  S_木(肝区) > H_木_max  （木形环僵化）
    → 肝区环基数 μ 超标，环边收缩
    → 环锁死阻碍 ∇σ 传导
    → S_火(肝区) ↓ （局部供能不足→寒）
    → 周边水形链受挤压 → S_水(胁肋段) ↓ （流通不畅→痛）
  ΔS_needed = (火↑, 水↑, 木↓, 土0, 金0)  // 疏肝理气
```

可用 SPUM 的 **七传（恶化）与间传（恢复）规则** （TCM-021, `spum-难经.md` §5.3）来预测偏移的下一帧传播方向：

```text
若当前 S 偏移方向为 ΔS_current：
  七传方向 = 沿病机链雅可比矩阵的负特征方向
  间传方向 = 与七传方向相反
  治疗目标 = 施加与七传方向相反的 ΔS_pulse
```

### 2.5 姓名学：第六诊的交叉验证

> **姓名学不是玄学。** 在 SPUM 框架中，姓名是一个人出生时被赋予的”第二个框架参数”——它是用笔画数力场在 S_0 上的”叠加投影”。姓名与八字的关系，类似于”调制信号”与”载波”——八字是载波（出生帧的全局 σ 投影），姓名是调制信号（被施加于这个载波上的后天偏移）。同八字的人体质相似，但姓名不同则人生轨迹分化，这是调制信号的差异造成的。

### 2.5.1 姓名在五诊合参中的定位

姓名的 SPUM 诊断价值不在于”预测命运”，而在于它提供了 **S_0 的独立交叉验证通道** ：

| 验证结果 | 含义 | 后续操作 |
| --- | --- | --- |
| 姓名与八字指向相同的 S_0 偏性 | S_0 置信度上升 | 八字诊断可确信使用 |
| 姓名与八字高度互补（如八字缺火+姓名总格火） | 姓名是对先天不足的补偿，属于应然 | 治疗上注意利用姓名的补益效应 |
| 姓名与八字冲突（如八字火亢+姓名人格火） | 有三种可能：(1)八字排盘有误（优先检查）；(2)名字是后天改过的；(3)此人的一生课题就是压制某个分量 | 需结合 P3 因子（尤其是医疗干预史）综合判断 |

### 2.5.2 姓名学→S 分量映射协议

| 姓名维度 | 数据 | SPUM 映射 |
| --- | --- | --- |
| 三字五行 | 每个字的康熙笔画数五行 | 五形各分量的初始偏置——土字补土、金字补金等 |
| 天格 | 姓氏笔画+1 | 祖上根基——S_0 的”底色” |
| 人格 | 姓氏+第一名字笔画和 | 主运——对 S_0 各分量的影响权重最高 |
| 地格 | 第一字+第二字笔画和 | 少年/家庭运——人生前期的 S 偏移方向 |
| 总格 | 三字笔画和 | 一生总运——S 向量在整个生命周期中的长期归趋方向 |
| 三才配置 | 天→人→地的五行流向 | S_0 各分量之间的生克制化关系——顺生格（水→金→土）表示能量流通无障碍；逆克格（木→金→火）表示内在张力 |

### 2.5.3 使用规范

1. **不单独使用姓名学做诊断** ——姓名学只有与八字互证时才有诊断价值。脱离八字单独分析姓名，信噪比太低
2. **优先使用康熙笔画** （而非简体笔画）——康熙字典的笔画体系与 SPUM 的空间粒子度数分布具有更好的拓扑一致性
3. **关注总格多于人格** ——人格反映前半生，总格反映一生归宿。对于中老年病案，总格的参考价值高于人格
4. **改名 ≠ 立即改变 S_current** ——改名改变的是 S_0 的”目标向量”，要经过帧演化才会体现到 S_current。儿童改名效应最明显（S_0 尚未被经历锁定），老年改名效应需数年才能逐渐显现

### 卷三、治疗：调平

### 3.1 治疗 = ΔS 脉冲工程

所有中医治疗方法——中药、针灸、推拿、导引——统一归约为 **在人体 ⟨P, ε⟩ 子图的特定位置施加 ΔS 五维位移脉冲** ：

| 治疗手段 | ΔS 施加方式 | 定位精度 | 时序特征 |
| --- | --- | --- | --- |
| 中药内服 | ΔS 向量经消化道解构重组后，沿水形链（经络）分布 | 归经定位到特定水形干段 | 多帧持续释放（煎煮=预解构） |
| 针灸 | 在腧穴（水形干段上的接入点）直接注入窄带 ΔS 脉冲 | 单节点级精度 | 单帧瞬发、可调强度 |
| 推拿 | 机械 V⁺/V⁻ 事件——拉伸放松木形环、挤压推动水形链 | 局部子图层级 | 帧内快速操作 |
| 导引（气功/五禽戏） | 通过姿势序列引导 ∇σ 在全身的再分布 | 全场级 | 多帧程序化操作 |

### 3.2 ΔS 脉冲的选择原则

| 原则 | SPUM 形式化 | 临床对应 |
| --- | --- | --- |
| 方向相反 | ΔS_pulse 与 S_current − H_centroid 方向相反 | 寒者热之、热者寒之 |
| 幅度适配 | |ΔS_pulse| ≤ max(|S_current − H_boundary|, dv/dt_const) | 病重药重、病轻药轻 |
| 归经定位 | ΔS_pulse 投影到出问题的水形干段上 | 引经报使 |
| 时序有序 | 先解表后攻里、先治标后治本 = 帧序取决于拓扑依赖关系 | 层次递进 |

### 3.3 药 = 地炁晶体

药物不是化学分子——它是 **地炁 VSPT 场在特定物相基底上结晶后，被采收的后处理工程产物** ：

```text
植物体 = 以基因为模板、以地中 VSPT 场为原料的拓扑晶体
药性 ΔS = 植物体从地炁中结晶时藏纳的局域 VSPT 场构型
方剂 ΔS = Σ(各药 ΔS) + 炮制改写 + 归经引导
```

详见 `spum-神农本草经.md` §1.3（植物-晶体同构）和 `spum-本草纲目.md` §二（16 部物相结晶分类）。

### 3.4 针灸 = 腧穴处微调 ΔS

腧穴不是”能量点”——它是 **水形主干（经络）上经经验标定的可操作接入点** 。每一穴有其坐标：(经络ID, 段位, 深度, 流态类型)。

详见 `spum-针灸甲乙经.md` 和 `spum-难经.md` §四（井荥输经合流态分段）。

### 卷四、归一：S ∈ H

### 4.1 健康的 SPUM 定义

健康 ≠ “没有症状”。健康 = **人体 ⟨P, ε⟩ 子图的 S 五维向量持续落在健康凸多面体 H 内** 。

H 由黄帝内经的健康标准经 SPUM 翻译定义（ `spum-黄帝内经.md` §七）：

| S 分量 | 健康区间 | 对应传统概念 |
| --- | --- | --- |
| S_火 | [H_火_min, H_火_max] | 阳气充足但不亢 |
| S_水 | [H_水_min, H_水_max] | 津液输布正常 |
| S_木 | [H_木_min, H_木_max] | 筋脉柔韧、气机通畅 |
| S_土 | [H_土_min, H_土_max] | 脾胃健运、气血充足 |
| S_金 | [H_金_min, H_金_max] | 肃降正常、二便调 |

### 4.2 归一的帧动力学

“归一”不是瞬时完成的——它是一个持续多帧的迭代过程：

```text
帧0:  测量 S_0，计算 ΔS_needed = H_centroid − S_0
帧1:   施加 ΔS_pulse_1，等待一个帧时长
帧2:   测量 S_1，计算 S_1 − S_0 = ΔS_actual_1
       比较 ΔS_actual_1 vs ΔS_pulse_1（校验线性叠加的有效性）
       计算 ΔS_needed_2 = H_centroid − S_1
帧3:   修正 ΔS_pulse_2 = ΔS_needed_2 × (ΔS_pulse_1 / ΔS_actual_1)
       ... 迭代直至 S_n ∈ H
```

这正是临床上的” **效不更方** “（ΔS_pulse 有效则继续）和” **随证治之** “（每帧重新测量 S 再调整）的 SPUM 严格含义。

### 卷六、青囊八步临床协议

以下是将 SPUM 中医子网的全部知识整合为临床可执行的 **八步诊断-治疗协议** 。整个协议运行在一个 **持续的 S_0 互证建模引擎** 之上——不是每次就诊从零开始，而是每次就诊在 S_0 的更新版本上工作。相比原七步协议，本次增补了 **步0（预检）** 作为所有诊断的前置筛查，并将 S_0 持久层更新算法从三因子升级为五因子全面校核。

### S_0 互证建模引擎（持久层）

```text
┌──────────────────────────────────────────────────────────┐
│                 S_0 互证建模引擎（跨就诊持久层）           │
│                                                          │
│  S_0^0 ← 八字（出生帧 σ 投影）+ 姓名学交叉验证           │
│       ↓                                                   │
│  每次就诊后更新（五因子校核）：                              │
│    S_0^(t+1) = S_0^t + α·(ΔS_actual − ΔS_predicted)      │
│               + β·(发病模式偏移) + γ·(生活事件漂移)        │
│               + δ·(P3 病程分析反馈)                       │
│    // α, β, γ, δ 为各因子的学习率（初诊 α=0.5，老患 α=0.1）│
│    // P3 病程分析反馈 δ 仅作为病程解释佐证，不直接修正 S_0  │
│       ↓                                                   │
│  S_0^(t) → 输出给每次就诊的步骤2和步骤3                    │
└──────────────────────────────────────────────────────────┘
```

### 步0（预检）：治标/误治历史筛查 ⭐ 新增

> **为什么需要步0** ：临床实践中，患者带来的”症状”中有相当比例不是真实病机的反映，而是 **治标干预后的人为改写** ——止痛药掩盖了疼痛的真实范围，激素压住了炎症但消耗了金形储备，铁剂制造了”血色好转”的假象但煽动了火亢。 **不先进行步0筛查，步1的望诊和步2的问诊可能被误导。**

| 筛查维度 | 询问内容 | 对后续诊断的影响标记 |
| --- | --- | --- |
| 正在服用的西药 | 止痛药/降压药/降糖药/激素/抗凝药/抗抑郁药 | 标记为”药盖症状”——这些症状被药物压制后不能直接用于 S 向量判定 |
| 近期停用的药物 | 过去3个月内停用的中西药 | 标记为”戒断反弹”——停药后出现的症状可能是撤药反应而非病机演化 |
| 长期自行用药史 | 止痛药/抗生素/补血剂/激素类外用的使用频率和时长 | 标记为”储备消耗源”——长期用药消耗了哪个分量的储备（止痛药伤土肾、抗生素扰肠道） |
| 西医诊断标签 | 是否被诊断为高血压/糖尿病/冠心病/焦虑症等 | 标记为”外我子图病毒模式”——诊断标签的心理定势可能影响患者对症状的主观描述 |
| 手术史 | 是否做过手术、切除过器官 | 标记为”结构性不可逆偏移”——器官切除/结构改变后的 S 向量基线已永久偏移 |

**步0 的输出** ：一张”症状可信度地图”——哪些症状可以用于 S 向量判定（无药掩盖/无手术改写），哪些症状需要降权使用。

> **步0 对 P3 因子D（医疗干预史）的呼应** ：步0 是 P3 因子D 在临床操作层面的具体执行。如果你发现患者有止痛药长期使用史，那么在步2 估测 S_current 时，患者主诉的”疼痛程度”应降权 50%，转而以舌脉等客观指征为主。

### 第一步：采集——多通道 S 信号输入

通过五诊（望闻问切 + 先天基底回顾）采集当前人体子图的 S 信号。此处先天基底的采集不是八字重排——而是 **回顾自上次就诊以来的 S_0 互证数据** ：有无新用药反馈？有无新发病模式？有无生活事件？

输出：一组症状-体征的 S 分量映射 + S_0^(t) 更新后的版本号。

### 第二步：估测——确定 S_current 与 S_effective ∈ ℝ⁵

加载 S_0^(t)（来自持久层），计算 S_effective：

```text
S_current = f_映射(症状-体征)            // 五诊测得的当前状态
S_effective = S_current − (S_0^(t) − H_centroid)  // 相对该个体的有效状态
```

若 S_effective ∈ H → S_current 虽然可能在群体标准的 H 边界外，但对该个体仍在正常范围。 若 S_effective ∉ H → 无论 S_current 在群体标准中看起来多”正常”，对该个体已是病态。

### 第三步：参照——确定个体化 H 边界

以黄帝内经的健康区间为群体基准，以 S_0^(t) 为个体偏移量，确定个体化 H：

```text
H_individual_min = H_group_min − (S_0^(t) − H_centroid)  // 个体化下界
H_individual_max = H_group_max − (S_0^(t) − H_centroid)  // 个体化上界
```

### 第四步：比对——计算 ΔS_needed

```text
ΔS_needed = H_centroid − S_effective
若 |ΔS_needed| < ε → S ∈ H → 不干预
否则 → 进入第五步
```

### 第五步：选方——匹配 ΔS 脉冲

从方剂 ΔS 库（ `spum-伤寒论.md` 附录A、 `spum-金匮要略.md` 附录B）中寻找与 ΔS_needed 方向最接近的方剂，或以单药拼配（ `spum-神农本草经.md` §五、 `spum-本草纲目.md` §七）。

### 第六步：处方——确定帧时序

根据病机链的传播方向（七传/间传规则， `spum-难经.md` §5.3）确定给药顺序：先表后里、先急后缓、先开（气道/水道）后补。

### 第八步：校核——五因子回写 S_0 持久层

```text
预测：ΔS_predicted_total = ΣΔS_i
校核（复诊时）：测量 S_after − S_before = ΔS_actual
误差：ΔS_residual = ΔS_predicted_total − ΔS_actual

S_0 五因子更新：
  S_0^(t+1) = S_0^(t) + 
    α·ΔS_residual           // 因子1：药效误差校核（α=0.5初诊→0.1老患）
    + β·发病模式偏移         // 因子2：病情演化方向（β=0.2）
    + γ·生活事件漂移         // 因子3：重大生活事件造成的不可逆偏移（γ=0.3）
    + δ·P3病程分析反馈       // 因子4：P3因子提供的病程解释（δ=0.05，仅定性参考）
    + ε·姓名学总格验证       // 因子5：核对长期偏移是否偏离姓名总格方向（ε=0.05）

学习率衰减规则：
  α(t) = α_0 / (1 + κ·t)   // 随就诊次数增加，药效误差的修正权重递减
  // 初诊 α=0.5，二诊 α=0.33，五诊 α=0.17，十诊 α=0.09
  // 含义：初诊的 S_0 最薄弱，需要从第一次药效反馈中大量学习；
  //       十诊后 S_0 已逼近真值，单次药效误差不应大幅扰动 S_0
  
将 S_0^(t+1) 写回持久层，供下次就诊使用。
```

**“效不更方”** = ΔS_residual ≈ 0 → S_0 校核通过 → 继续用同方。 **“随证治之”** = 每帧重新测量 S_current → ΔS_needed 每帧重算 → 处方随 S 变化而变。

### 附录C：关于”中医不治病”的 SPUM 证明

**命题** ：中医的治疗对象不是”病”，而是”人体子图的 S 向量偏移”。

**证明** ：

1. 设病名为 D。传统上 D 被当作一个”实体”——一种存在于人体中的异常事物。
2. 按 SPUM 第一公理（关系第一性），不存在孤立实体。若 D 作为实体存在，必有 ⟨P, ε⟩ 定义——但病原体（细菌/病毒）是另一个子图，不是人体子图自身的异常状态。
3. 中医从未定义”病”的 ⟨P, ε⟩ 边界。中医定义的是 **人体 ⟨P, ε⟩ 子图的状态偏移** ——即 S 向量相对 H 的偏离。
4. 治疗的直接操作对象是 S_current → 使其回到 H，而非消灭”病”。
5. 即使病原体子图被消灭，若 S 不回到 H，人体仍不健康；反之，即使病原体子图存在，若 S ∈ H，人体自稳正常（携带者态）。
6. 因此，中医的治疗对象是 **S 向量** ，不是”病”。QED.

**推论** ：

- 中医不治病，中医 **识偏、调平、归一** 。
- 所谓”治愈”，是 S 回到 H 的帧状态——不是”病被消灭”的事件。
- 所谓”养生”，是维持 S ∈ H 的连续监控与微调——不是”防病”的对抗行为。
