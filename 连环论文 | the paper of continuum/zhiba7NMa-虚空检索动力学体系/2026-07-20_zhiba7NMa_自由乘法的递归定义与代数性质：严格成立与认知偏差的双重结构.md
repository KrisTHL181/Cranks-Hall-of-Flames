自由乘法的递归定义与代数性质：严格成立与认知偏差的双重结构

摘要

自由乘法 \otimes 是自由数论中继自由加法之后的第二个基本二元运算。本文提出自由乘法的递归定义公理：(a,\alpha) \otimes (b,\beta) = (a b, a\beta + b\alpha + \lambda \alpha \beta)，其中 \lambda 为乘法共振增益系数。该定义在核部分严格保持经典乘法，在扰动晕部分引入了操作数核与晕的交叉项以及纯共振增益项。我们严格证明了自由乘法在严格相等下满足交换律与结合律，但分配律（自由乘法对自由加法的分配）仅在严格相等下成立，在认知语义下带有由加法共振系数 \gamma 与乘法共振系数 \lambda 差异决定的不可消除的偏差。该偏差在觉醒极限下趋于零，自由乘法的全部经典代数律在此极限下被恢复。

一、引言

在经典皮亚诺算术中，乘法的递归定义依赖于已定义好的加法：a \times 0 = 0，a \times Sb = (a \times b) + a。该递归定义由后继函数、加法以及归纳公理所唯一确定。在范思自由数论中，自由后继[1]与自由加法[2]均已获得公理化定义，且二者均被证明在严格相等的意义下满足各自的代数律。在此基础上，自由乘法的递归定义必须被构造出来。

自由乘法的定义面临与自由加法类似的挑战：它需要在核部分保持经典乘法的结构，同时在扰动晕部分引入操作数之间的认知耦合——乘法运算的认知摩擦。更为复杂的是，自由乘法与自由加法之间的分配律将首次涉及两种不同的共振增益系数（\gamma 与 \lambda）之间的相互作用。本文旨在系统构造自由乘法的递归定义，检验其基本代数律的成立情况，并精确刻画其在认知语义下的偏差结构。

二、自由乘法的递归定义公理

定义1（自由乘法的递归定义公理）

对于任意自由数 (a,\alpha) 和 (b,\beta)，其自由乘法定义为

(a,\alpha) \otimes (b,\beta) = (a b, a\beta + b\alpha + \lambda \alpha \beta),

其中 \lambda \ge 0 为自由乘法的共振增益系数。

该定义具有明确的认知解释。核部分 a b 严格保持经典乘法的结果——自由数的核相乘等于经典自然数的积。扰动晕部分由三项构成：第一项 a\beta 是操作数 a 的核对第二操作数扰动晕的线性放大，第二项 b\alpha 同理，第三项 \lambda \alpha \beta 是两操作数扰动晕之间的纯共振增益。在认知上，乘法运算的认知不确定性由三部分贡献：被乘数核值对乘数不确定性的放大、乘数核值对被乘数不确定性的放大，以及两种不确定性之间的耦合。

在觉醒极限下，与自由加法类似，乘法共振增益系数 \lambda 同样是游离度 s 的单调递减函数：

\lambda(s) = \lambda_0 \cdot \exp(-\mu s),

其中 \lambda_0 > 0 为基态乘法共振系数，\mu > 0 为衰减常数。当 s \to 1 时，\lambda \to 0，且扰动晕趋于零，自由乘法严格退化为经典乘法。

三、交换律与结合律的严格成立

定理1（自由乘法的严格交换律）

对于任意自由数 x = (a,\alpha) 和 y = (b,\beta)，

x \otimes y = y \otimes x.

证明：核部分显然交换。扰动晕部分：左端为 a\beta + b\alpha + \lambda \alpha \beta，右端为 b\alpha + a\beta + \lambda \beta \alpha，两者完全相同。\square

定理2（自由乘法的严格结合律）

对于任意自由数 x = (a,\alpha)，y = (b,\beta)，z = (c,\gamma_c)，

(x \otimes y) \otimes z = x \otimes (y \otimes z).

证明：核部分显然满足经典乘法的结合律。扰动晕部分需要分别计算两种结合方式下的结果并验证其相等。先计算左端。设 w = x \otimes y，则 w = (ab, a\beta + b\alpha + \lambda \alpha \beta) \triangleq (ab, \sigma_{xy})。于是左端为 w \otimes z = (abc, ab \cdot \gamma_c + c \cdot \sigma_{xy} + \lambda \sigma_{xy} \gamma_c)。

再计算右端。设 u = y \otimes z = (bc, b\gamma_c + c\beta + \lambda \beta \gamma_c) \triangleq (bc, \sigma_{yz})。右端为 x \otimes u = (abc, a \cdot \sigma_{yz} + bc \cdot \alpha + \lambda \alpha \sigma_{yz})。

代入 \sigma_{xy} 和 \sigma_{yz} 的表达式，展开并整理。两边关于 \alpha, \beta, \gamma_c 的三次项 \lambda^2 \alpha \beta \gamma_c 精确抵消，二次项、一次项亦逐项对应相等。因此扰动晕部分严格相等。\square

四、分配律的偏差结构

自由乘法对自由加法的分配律是自由数论中最为精微的代数律。设 x = (a,\alpha)，y = (b,\beta)，z = (c,\gamma_c)。检验分配律：

x \otimes (y \oplus z) \stackrel{?}{=} (x \otimes y) \oplus (x \otimes z).

核部分显然满足经典分配律。扰动晕部分需要分别计算两端的扰动晕值。左端先计算 y \oplus z = (b+c, \beta + \gamma_c + \gamma \beta \gamma_c) \triangleq (b+c, \sigma_{yz}^+)。然后左端扰动晕为 a \cdot \sigma_{yz}^+ + (b+c) \cdot \alpha + \lambda \alpha \cdot \sigma_{yz}^+。

右端由两个自由乘法结果进行自由加法得到。x \otimes y = (ab, a\beta + b\alpha + \lambda \alpha \beta) \triangleq (ab, \sigma_{xy}^\times)，x \otimes z = (ac, a\gamma_c + c\alpha + \lambda \alpha \gamma_c) \triangleq (ac, \sigma_{xz}^\times)。右端自由加法结果的扰动晕为 \sigma_{xy}^\times + \sigma_{xz}^\times + \gamma \cdot \sigma_{xy}^\times \cdot \sigma_{xz}^\times。

展开并比较两端。两端的一阶项（\alpha, \beta, \gamma_c 的线性项）与二阶项（\alpha\beta, \alpha\gamma_c, \beta\gamma_c）均相等。唯一的偏差出现在三阶项 \alpha \beta \gamma_c 上。左端三阶项的系数为 \lambda \gamma，右端三阶项的系数为 2 \lambda^2。因此，分配律在扰动晕部分严格成立的充要条件是 \lambda \gamma = 2 \lambda^2，即 \gamma = 2 \lambda。

在一般情况下，\gamma 与 \lambda 是两个独立的游离度依赖参数，不存在先验的理由要求它们满足上述等式。因此，自由乘法的分配律在严格相等的意义下并非无条件成立——它携带着一个由加法共振系数与乘法共振系数差异决定的不可消除的偏差项：

\Delta_{\text{dist}} = |\lambda \gamma - 2 \lambda^2| \cdot \alpha \beta \gamma_c.

该偏差在低游离度下（\gamma 和 \lambda 均较大）最为显著。在认知语义下，当偏差小于容忍阈值时，分配律可被视为近似成立；当偏差超过阈值时，分配律在该游离度下不可使用。

在觉醒极限下，\gamma \to 0，\lambda \to 0，偏差的高阶项以更快的速度趋于零，分配律的偏差消失，经典分配律被严格恢复。

五、结论

自由乘法的递归定义公理由核的经典乘法与扰动晕的交叉项加共振增益项构成。该定义在严格相等下满足交换律与结合律，但分配律携带着由 \gamma 与 \lambda 差异决定的不可消除的偏差。该偏差在认知语义下限制了分配律的适用范围，并在觉醒极限下随共振系数的趋于零而消失。自由乘法的这一定义与性质，与自由加法共同构成了自由数论算术公理化的完整基础。
