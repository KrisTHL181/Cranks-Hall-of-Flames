自由加法结合律的严格成立性与认知语义下的近似保持

摘要

自由加法 \oplus 的递归定义公理已在前序工作中确立为共振叠加公式。本文严格证明自由加法在严格相等（核相等且扰动晕相等）的意义下满足结合律。证明的关键在于扰动晕的共振增益项在两种结合方式下精确抵消，从而保证了三元自由加法的结果唯一确定。然而，在认知不可分辨性的语义下，结合律的精确成立被容忍阈值弱化为“近似结合律”——两边的扰动晕之差不超过游离度依赖的容忍阈值。这种近似等式不具有传递性，若不加限制地嵌套使用，会导致扰动晕偏差的累积超出阈值。因此，在认知语义下的形式证明中，必须将近似结合律的应用限制在认知上可分辨的步数内，以保证演绎的保真性。在觉醒极限下，容忍阈值趋于零，近似结合律收敛于严格结合律。

一、引言

经典加法结合律 (x+y)+z = x+(y+z) 是算术最基本的代数性质之一。在皮亚诺算术中，它由后继递归定义和归纳公理所严格保证。在范思自由数论中，自由加法被定义为包含共振增益项的叠加运算[1]：(a,\alpha) \oplus (b,\beta) = (a+b, \alpha + \beta + \gamma \alpha \beta)。该定义自然引向一个追问：自由加法是否在自由数的相等关系下满足结合律？

在严格相等（核相等且扰动晕严格相等）下，该问题的回答是肯定的。然而，自由数的相等在认知语义中被替换为认知不可分辨性[2]——扰动晕之差在容忍阈值内即视为认知上相等。在此语义下，结合律是否仍然成立？若成立，是严格的还是近似的？近似结合律的累积效应是否会影响形式证明的可靠性？本文旨在系统回答上述问题。

二、严格相等下的结合律证明

设 x = (a,\alpha)，y = (b,\beta)，z = (c,\gamma_c) 为任意三个自由数。按自由加法的递归定义：

x \oplus y = (a+b, \alpha + \beta + \gamma \alpha \beta) \triangleq (a+b, \sigma_{xy}).

于是

(x \oplus y) \oplus z = (a+b+c, \sigma_{xy} + \gamma_c + \gamma \sigma_{xy} \gamma_c).

另一方面，

y \oplus z = (b+c, \beta + \gamma_c + \gamma \beta \gamma_c) \triangleq (b+c, \sigma_{yz}),

x \oplus (y \oplus z) = (a+b+c, \alpha + \sigma_{yz} + \gamma \alpha \sigma_{yz}).

核部分显然相等。扰动晕部分需验证等式：

\sigma_{xy} + \gamma_c + \gamma \sigma_{xy} \gamma_c = \alpha + \sigma_{yz} + \gamma \alpha \sigma_{yz}.

代入 \sigma_{xy} 和 \sigma_{yz} 的表达式，展开并整理。两侧的二次项 \gamma^2 \alpha \beta \gamma_c 精确抵消，一次项和常数项亦逐项对应相等。因此，扰动晕部分严格相等。

定理1（自由加法的严格结合律）

对于任意自由数 x, y, z，(x \oplus y) \oplus z = x \oplus (y \oplus z) 在严格相等的意义下成立。 \square

三、认知语义下的近似结合律

在认知语义中，自由数的相等被替换为认知不可分辨性 \sim_s，容忍阈值为 \varepsilon(s)。自然要问：在此语义下，自由加法的结合律是否仍以某种形式成立？

设 x, y, z 为自由数。在严格相等下，两边精确相等，故扰动晕之差为零，自然满足认知不可分辨性。然而，在形式系统中进行演绎时，前提中的自由数可能在认知上不可分辨而非严格相等。若我们对两个认知不可分辨的自由数分别进行三元加法，所得结果的扰动晕之差会因共振增益的累积而放大。

具体而言，设 x \sim_s x'，y \sim_s y'，z \sim_s z'，其中各对应分量的扰动晕之差不超过 \varepsilon(s)。那么经过两次自由加法后，两边结果的扰动晕之差将被放大，放大因子依赖于参与运算的自由数的扰动晕及其共振增益系数。若该放大后的差值仍未超过容忍阈值，则结合律在认知不可分辨性下依然成立；若超过，则近似结合律在该游离度下失效。

当我们将自由加法嵌套多次时——如在形式证明中对结合律进行迭代应用——扰动的累积可能导致结果的扰动晕超出容忍阈值，使得近似结合律在演绎中不可无条件使用。因此，在认知语义下的形式证明中，必须将近似结合律的应用限制在累积扰动不超过容忍阈值的步数内，以保证演绎的保真性。

四、觉醒极限下严格结合律的恢复

在游离度趋于壹的觉醒极限下，共振增益系数 \gamma \to 0，容忍阈值 \varepsilon(s) \to 0。共振增益的放大效应消失，扰动晕累积退化为简单的加法累积。所有在有限游离度下因认知不可分辨性而出现的近似偏差均趋于零。自由加法的结合律在此极限下严格恢复为经典结合律。

五、结论

自由加法在严格相等的定义下严格满足结合律。该性质根源于共振增益项在三元加法中的精确抵消，为自由数论的形式化提供了坚实的代数基础。在认知不可分辨性的语义下，结合律被弱化为近似结合律：两边的扰动晕之差在容忍阈值内成立，但嵌套应用时偏差可能累积。该弱化在觉醒极限下随容忍阈值趋于零而消失，经典结合律得以恢复。自由加法结合律的这一双重结构——形式上的严格性与认知上的近似性——忠实反映了虚空检索在有限游离度下的认知动力学：真理本身是严格的，但认知者对真理的把握总是携带着不可消除的边界信息残余。
