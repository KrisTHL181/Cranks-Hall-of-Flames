自由后继的单射性：严格相等下的保持性与认知语义下的弱化

摘要

自由后继函数 S_{\text{free}} 是自由数论中生成自由数的基本操作。本文探讨自由后继在自由数的相等关系下是否保持单射性。在严格相等（核相等且扰动晕相等）的定义下，基于扰动晕继承律的线性结构，自由后继被证明是严格单射的。这一单射性保证了自由数序列的离散性，并为自由归纳公理的严格成立提供了必要条件。然而，在认知不可分辨性的语义层，单射性因容忍阈值的引入而弱化：若两后继在认知上不可分辨，其原自由数可能仅在该游离度下不可区分，而非严格相等。这种弱化忠实反映了虚空检索在有限游离度下的认知局限，并在觉醒极限下随容忍阈值趋于零而恢复为严格单射性。

关键词：自由后继；单射性；严格相等；扰动晕；认知不可分辨性

一、引言

在经典皮亚诺算术中，后继函数 S(n)=n+1 是单射，且其单射性（S(m)=S(n) \Rightarrow m=n）是保证自然数序列无限且不循环的核心公理之一。在范思自由数论中，经典自然数被更换为携带扰动晕 \sigma 的自由数 (n,\sigma)[1,2]，自由后继 S_{\text{free}} 相应地定义为核的后继叠加扰动晕的认知累积[3]。随之而来的一个关键追问是：自由后继在自由数的相等关系下是否仍然是单射？

该问题的回答直接关涉自由数序列的结构稳定性。若自由后继不满足单射性，则不同的自由数可能经后继操作后坍缩为相同的自由数，自由数序列将失去其离散生成的结构，自由归纳公理也将面临根本威胁。本文旨在严格论证，在形式理论内部的严格相等定义下，自由后继保持经典单射性；而在认知语义层面，单射性因容忍阈值的引入而出现弱化，这种弱化在觉醒极限下趋于消失。

二、严格相等下的单射性证明

在自由数论的形式公理系统中，相等关系被定义为核相等且扰动晕严格相等的二元关系[4]：

(n,\sigma) = (m,\tau) \iff n = m \land \sigma = \tau.

自由后继的核部分显然保持经典后继的规律。其扰动晕继承律已在前序工作中获得确立[3]，采取如下联合定义：

S_{\text{free}}(n,\sigma) = (n+1, \sigma + \delta_{\text{inh}} + \gamma_{\text{suc}} \cdot \sigma \cdot \delta_{\text{inh}}),

其中 \delta_{\text{inh}} > 0 为后继操作固有的最小认知阻抗，\gamma_{\text{suc}} \ge 0 为后继共振增益系数。为简化记号，记 \sigma' = \sigma + \delta_{\text{inh}} + \gamma_{\text{suc}} \sigma \delta_{\text{inh}}。

定理1（自由后继的单射性）

设 (n,\sigma) 与 (m,\tau) 为任意两个自由数。若 S_{\text{free}}(n,\sigma) = S_{\text{free}}(m,\tau)，则必有 n = m 且 \sigma = \tau。

证明：由自由后继的定义及严格相等的定义，S_{\text{free}}(n,\sigma) = S_{\text{free}}(m,\tau) 等价于

(n+1, \sigma') = (m+1, \tau') \implies n+1 = m+1 \land \sigma' = \tau'.

由 n+1 = m+1 及经典自然数的单射性，立得 n = m。进一步，\sigma' = \tau' 即

\sigma + \delta_{\text{inh}} + \gamma_{\text{suc}} \sigma \delta_{\text{inh}} = \tau + \delta_{\text{inh}} + \gamma_{\text{suc}} \tau \delta_{\text{inh}}.

消去固有增量 \delta_{\text{inh}}，并提取公因子得

(\sigma - \tau)(1 + \gamma_{\text{suc}} \delta_{\text{inh}}) = 0.

由于 \delta_{\text{inh}} > 0 且 \gamma_{\text{suc}} \ge 0，有 1 + \gamma_{\text{suc}} \delta_{\text{inh}} > 0，故上式强制 \sigma - \tau = 0，即 \sigma = \tau。因此 (n,\sigma) = (m,\tau)。\square

上述证明表明，只要扰动晕继承律关于原扰动晕是严格单调的（即 \sigma \mapsto \sigma' 为单射），且严格相等关系要求扰动晕的精确等同，自由后继的单射性即可在形式系统内部得到严格保证。

三、认知语义下单射性的弱化

在认知语义层，自由数的相等被认知不可分辨性 \sim_s 所替代[4]：

(n,\sigma) \sim_s (m,\tau) \iff n = m \land |\sigma - \tau| \le \varepsilon(s),

其中 \varepsilon(s) 为游离度 s 下的容忍阈值，随游离度提升而趋于零。在此语义下，自由后继的单射性呈现弱化形态。

设 S_{\text{free}}(n,\sigma) \sim_s S_{\text{free}}(m,\tau)，即

n+1 = m+1 \land |\sigma' - \tau'| \le \varepsilon(s).

由核相等可得 n = m。但由 |\sigma' - \tau'| \le \varepsilon(s) 仅能推出

|\sigma - \tau|(1 + \gamma_{\text{suc}} \delta_{\text{inh}}) \le \varepsilon(s) \implies |\sigma - \tau| \le \frac{\varepsilon(s)}{1 + \gamma_{\text{suc}} \delta_{\text{inh}}} < \varepsilon(s),

即原自由数的扰动晕之差小于容忍阈值，从而原自由数在认知上同样不可分辨。因此，在认知不可分辨性的语义下，自由后继保持了“认知单射性”——若后继认知不可分辨，则原自由数亦认知不可分辨。但这种单射性容忍了原自由数在阈值内的扰动晕差异，不再是严格的核-晕精确相等。

这一弱化的认知意义在于：在有限游离度下，认知者无法以绝对精度区分自由数；自由后继的单射性仅在认知分辨率的限度内成立。随着游离度提升，\varepsilon(s) \to 0，认知单射性收敛于严格单射性，经典后继的单射性作为觉醒极限被恢复。

四、结论

在自由数论的形式公理系统中，基于严格相等定义，自由后继 S_{\text{free}} 是严格单射的。该单射性由扰动晕继承律的线性结构和严格相等的定义所保证，为自由数序列的离散性和自由归纳的合法性提供了语法基础。在认知语义层，自由后继的单射性被容忍阈值弱化为认知单射性，忠实地反映了虚空检索在有限游离度下的认知分辨局限。后者在游离度趋于壹的觉醒极限下严格退化为经典单射性。⊥保证，即使在觉醒极限下，自由后继的单射性将永远作为虚空检索中不可消除的认知结构而存留。
