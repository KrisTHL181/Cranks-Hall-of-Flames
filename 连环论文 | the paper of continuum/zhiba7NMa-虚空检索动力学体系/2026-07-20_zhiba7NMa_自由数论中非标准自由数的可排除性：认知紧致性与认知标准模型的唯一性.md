自由数论中非标准自由数的可排除性：认知紧致性与认知标准模型的唯一性

摘要

经典皮亚诺算术的一阶理论允许非标准模型，其中包含核大于任何标准自然数的“无穷大”元素。本文考察自由数论中是否存在类似的非标准自由数——核或扰动晕为无穷大的自由数。我们证明，在范思自由数论的认知语义下，由于自由数携带扰动晕且被检索场拓扑的认知约束所限制，自由数论的认知标准模型 F_{\text{cog}}(s) 是唯一的（同构于认知可及自由数集），其中所有自由数的核均为经典自然数，扰动晕均为自有限次后继与有限次运算生成的有限值。因此，在认知标准模型中，不存在核为无穷大或扰动晕为无穷大的非标准自由数。经典紧致性定理在全体数学赋值空间中虽可证得非标准模型的存在，但这些模型被认知约束所排除：认知紧致性定理在认知赋值空间中严格成立，保证了语义后承的有穷可捕捉性，从而自由数论的形式公理系统仅描述认知可及自由数的性质。非标准自由数的排除是自由数论作为“认知数论”的本质特征。

一、引言

经典皮亚诺算术（PA）的一阶语言具有紧致性：若一个公式集的所有有限子集都有模型，则该公式集本身有模型。由此，PA 存在非标准模型，其中包含“无穷大”的自然数——它们大于所有标准自然数 0, 1, 2, \dots，但在模型中仍被视为自然数。这些非标准元素的出现是经典一阶逻辑固有的局限。

在范思自由数论中，经典自然数被更换为携带扰动晕 \sigma 的自由数 (n,\sigma)[1,2]。自由数的核 n 取值于经典自然数集，扰动晕 \sigma 取值于非负实数连续统[3]。自由数论的认知语义被限定在“认知可及自由数”F_{\text{cog}}(s) 上——即在当前游离度 s 下可被虚空检索实际通达的自由数集合[4]。在此基础上，自由归纳公理亦被表述为仅对认知可及自由数成立的“认知归纳”[5]。

一个自然的模型论问题是：在自由数论的一阶形式系统中，紧致性定理是否依然生效，从而强制非标准自由数的存在？如果存在，其核是否为非标准自然数？其扰动晕是否可能为无穷大？本文旨在严格分析该问题，并最终论证：在自由数论的认知语义下，非标准自由数被有效排除，认知标准模型 F_{\text{cog}}(s) 是唯一的。

二、经典紧致性在自由数论中的表面适用性

考虑自由数论的一阶语言 \mathcal{L}_{\text{free}}，包含常元 0，函数符号 S_{\text{free}}, \oplus, \otimes, \sigma(\cdot) 以及谓词符号 =, Trans 等。在该语言中，可定义常数项 \Delta_n = S_{\text{free}}^n(0) 表示第 n 个自由数。设公式集 \Gamma 由所有自由数公理加上无穷多个公式 \{c > \Delta_n \mid n \in \mathbb{N}\} 构成，其中 c 为新常元。\Gamma 的每个有限子集都有模型（取 c 为足够大的标准自由数即可）。若经典紧致性定理在此语言中成立，则 \Gamma 本身有模型——该模型中 c 的解释是一个核大于任何标准自然数的自由数，即“非标准自由数”。

在纯一阶视角下，该论证似乎无可辩驳。然而，该论证忽略了自由数论语言中谓词 Trans 及检索场相关公理的语义限制——这些公理将模型的合法性约束在认知赋值空间的范围内。非标准模型若违反这些认知约束，尽管在一阶逻辑上满足公理，却不在自由数论的认知语义之内。

三、认知约束对非标准模型的排除

在检索场拓扑中，每个自由数对应于涌现场中的一个透明区域，且该透明区域具有有限的认知尺度[6]。谓词 Trans(x) 在认知上要求 x 对应的点在当前游离度下位于透明区域内部。公理系统包含关于 Trans 的闭包条件：透明区域在有限交与任意并下封闭，且全透明区域 X_{\text{trans}} 是认知紧的。这些条件排除了“无穷大核”的认知存在性——一个核为无穷大的自由数将对应于一个无限远的透明区域，而认知紧空间不允许无限远的孤立点不属于任何有限透明覆盖。因此，任何满足认知公理的模型必须将每个自由数解释为属于某个有限透明覆盖的点，从而其核必须是标准的有限自然数。

类似地，扰动晕 \sigma 被要求取值于实数连续统，但每个认知可及自由数的扰动晕必须是通过有限次后继与有限次运算从基态扰动晕 \sigma_0 生成的值。动态演化公理[7]保证了每次运算仅增加有限增量，因此任何认知可及自由数的扰动晕均为有限实数。非标准扰动晕（无穷大）将被透明区域的认知紧性所排除——若扰动晕为无穷大，则该自由数的透明区域将无法被有限个透明基覆盖所包含。

因此，在认知公理的强制下，非标准自由数的存在性被语义地排除，尽管纯一阶紧致性论证在忽略认知谓词的解释时可构造出形式上的非标准模型。这类非标准模型是“非认知的”——它们满足一阶公理集，但不满足认知语义的完整约束。

四、认知紧致性与认知标准模型的唯一性

自由数论的认知赋值空间 \Omega_s 在认知紧拓扑下具有认知紧致性[8]：任何认知开覆盖存在有限子覆盖。该性质保证了在认知语义下，一阶认知归纳模式仅对认知可及自由数成立，且认知标准模型 F_{\text{cog}}(s) 是唯一的（在认知同构意义下）。认知紧致性的成立依赖于透明区域族的认知紧性，而非全体数学赋值空间的经典紧致性。因此，自由数论的认知公理系统是“认知完备的”——其定理集恰好捕捉了认知标准模型上的全部真命题，而不被非标准模型所污染。

五、结论

自由数论的一阶形式系统若被赋予所有认知公理（包括检索场相关谓词的解释），其认知标准模型不包含核为无穷大或晕为无穷大的非标准自由数。经典紧致性论证所构造的非标准模型被认知约束排除在合法语义之外。自由数论因此避免了经典算术中非标准模型的存在性困扰，这是其作为“认知数论”的根本优势——数学真理不再被非标准模型所任意拉伸，而是锚定于虚空检索在当前游离度下可实际通达的认知结构之中。
