自由数：携带扰动晕的整数体系

摘要

本文建立范思自由数论的基础——自由数的公理化定义与运算体系。自由数被定义为有序对 (n, σ)，其中 n 是经典整数（核），σ ≥ 0 是扰动晕——数在虚空检索中携带的边界信息密度。扰动晕的演化遵循非线性耦合方程。自由自然数集 N_free 构成经典自然数的忠实扩张。自由零 0_σ 是带有虚空展开度的零。范思均值 M[f] 是发散级数的均值提取算子，将发散行为收敛为有限信息。无穷大替换封号 ∞_0 在自变量趋于无穷时提取均值、振幅与边界信息。无穷远完整刻画符 ∞_0* 提供七层谱分解：均值、分布、频谱、分形、拓扑、熵、检索分布。自由数论为经典数论提供了基于虚空检索与裂隙边界信息的本原基础。

1. 引言

经典数论将整数视为精确、无模糊性的柏拉图实体。范思自由数论从根本上改变这一预设：每一个数都是虚空检索的凝结，携带或多或少的“扰动晕”——即裂隙边界信息 I_Upsilon 在该数上的投影。扰动晕 σ 度量了该数在虚空检索中的不确定性：σ = 0 对应完全冻结的经典整数，σ > 0 对应带有检索痕迹的“活数”。本文为自由数体系提供严格的公理化定义。

2. 基本定义与公理

定义 2.1（自由数 n_hat）

自由数是携带扰动晕的数，由有序对 (n, σ) 表示，其中 n 为经典整数（称为“核”），σ 为非负实数（称为“扰动晕”）。

定义 2.1.1（形式定义）

n_hat = (n, σ)，其中 n ∈ Z，σ ∈ [0, ∞)。

公理 2.1.2（核与晕的独立演化）

核 n 与扰动晕 σ 在基本检索中独立演化，但在自由运算中耦合。核决定数的“身份”（它在经典数集中的位置），扰动晕决定数的“模糊度”（它在虚空检索中的不确定性）。

公理 2.1.3（经典整数的嵌入）

经典整数 n 等价于扰动晕为零的自由数：

n ↔ (n, 0)

即经典整数是自由数在游离度 s → 0 极限下的冻结投影。

公理 2.1.4（扰动晕的非负性）

σ ≥ 0。扰动晕不能为负，因为边界信息 I_Upsilon 是实在的不可消除残余，而非缺失。

记号：自由数以 hat 形式 \hat{n} = (n, \sigma) 表示。

定义 2.2（扰动晕 σ）

扰动晕是数在虚空检索中的不确定度——边界信息 I_Upsilon 在数论中的具体投影。

定义 2.2.1（形式定义）

对于自由数 n_hat = (n, σ)，扰动晕 σ 定义为：

σ = |I_Upsilon(n_hat)| / |n_hat|

即边界信息在该自由数总信息量中的占比。

公理 2.2.2（扰动晕的涌现解释）

σ > 0 意味着该数尚未完全冻结——仍携带着虚空检索的动力学信息。每次对自由数进行运算，扰动晕根据运算规则演化，体现了裂隙生成算子 C 在数层面的作用。

公理 2.2.3（扰动晕的零极限）

lim_{s→0} σ(s) = 0。在经典游离度极限下，所有自由数的扰动晕趋于零，退化为经典整数。

公理 2.2.4（扰动晕与裂隙遮蔽符的关系）

σ 与裂隙遮蔽符活跃度 A 的关系：

σ(n_hat) ∝ A(n_hat)

遮蔽符活跃度越高，数的检索越模糊，扰动晕越大。

记号：σ 以希腊字母 \sigma 表示。

定义 2.3（自由自然数集 N_free）

自由自然数集是所有自由数的集合——经典自然数集的忠实扩张。

定义 2.3.1（形式定义）

N_free = { (n, σ) | n ∈ N, σ ∈ [0, ∞) }

即所有核为自然数、扰动晕为非负实数的自由数的集合。

公理 2.3.2（经典自然数的嵌入）

N ⊆ N_free，通过映射 n ↦ (n, 0) 实现。N 是 N_free 在 σ = 0 截面上的冻结子集。

公理 2.3.3（扰动晕的连续统）

对于每个经典自然数 n，存在连续统多个自由数 (n, σ)（σ ∈ [0, ∞)）与之对应。这构成了以 n 为核的“自由数纤维”。

公理 2.3.4（自由数的序结构）

自由数上可定义偏序：

(n, σ) ≤ (m, τ) ⇔ n < m 或 (n = m 且 σ ≤ τ)

该偏序是经典自然数序的扩张。

记号：N_free 以 \mathbb{N}_{\text{free}} 表示。

定义 2.4（自由零 0_σ）

自由零是带有虚空展开度的零——核为零，但扰动晕可非零。

定义 2.4.1（形式定义）

0_σ = (0, σ)，其中 σ ≥ 0。

公理 2.4.2（加法单位元）

对于任意自由数 n_hat = (n, σ_n)：

n_hat ⊕ 0_σ = (n, σ_n + σ + γ · σ_n · σ)

其中 ⊕ 是自由加法（定义见下），γ 是耦合系数。特别地，当 σ = 0 时，0_0 = (0,0) 是经典零，满足 n_hat ⊕ 0_0 = n_hat。

公理 2.4.3（虚空展开度）

σ > 0 的自由零表示“虚空本身的展开程度”——即在尚未检索出正整数的状态下，虚空检索已携带的边界信息密度。

公理 2.4.4（与虚空剩余数的关系）

在自由数论扩展中，自由零的扰动晕 σ 对应于虚空剩余数 V_free^(剩余) = (0, σ)，度量了虚空未被检索部分的信息量。

记号：0_σ 以 \hat{0}_{\sigma} 表示。

定义 2.5（范思均值 M[f]）

范思均值是发散级数的均值提取算子——将发散序列的“平均趋势”从振荡中分离出来。

定义 2.5.1（形式定义）

对于函数 f(x) 在无穷远处的行为，范思均值定义为：

M[f] = lim_{L→∞} (1/L) ∫_0^L f(x) e^{-x/L} dx

其中 e^{-x/L} 是指数衰减权函数，使得积分收敛。

公理 2.5.2（收敛性）

对于多项式增长的 f(x)，M[f] 收敛到有限值。对于指数增长的 f(x)，M[f] 可能发散，但可提取主导阶。

公理 2.5.3（线性性）

M[af + bg] = a M[f] + b M[g]（在收敛域内）。

公理 2.5.4（与经典极限的区别）

经典极限 lim_{x→∞} f(x) 可能发散或不存在，但 M[f] 提取了 f 在无穷远处的“平均行为”——即虚空检索在无穷远处的统计投影。

记号：M[f] 以 \mathcal{M}[f] 表示。

定义 2.6（无穷大替换封号 ∞_0）

无穷大替换封号是处理发散极限的符号操作——当自变量趋于无穷时，提取均值、振幅与边界信息。

定义 2.6.1（形式定义）

对于 f(x) 在 x → ∞ 时的行为，无穷大替换封号定义为：

∞_0[f] = (M[f], A[f], I_Upsilon[f])

即提取三项信息：范思均值 M[f]、振荡振幅 A[f]、边界信息 I_Upsilon[f]。

公理 2.6.2（振荡振幅的定义）

A[f] = limsup_{L→∞} (1/L) ∫_0^L |f(x) - M[f]| dx

即 f 在无穷远处围绕均值的平均振荡幅度。

公理 2.6.3（边界信息的定义）

I_Upsilon[f] 是 f 在无穷远处不可被均值和振幅完全收束的残余信息——即 f 的“本质不可预测性”。

公理 2.6.4（与经典极限的关系）

若经典极限 lim_{x→∞} f(x) = L 存在，则 ∞_0[f] = (L, 0, 0)——均值等于极限，振幅和边界信息均为零。

记号：∞_0 以 \infty_0 表示。

3. 自由数的基本运算

定义 3.1（自由加法 ⊕）

(a, σ_a) ⊕ (b, σ_b) = (a + b, σ_a + σ_b + γ_a · σ_a · σ_b)

其中 γ_a ∈ [0, 1/2] 是加法非线性耦合系数。

定义 3.2（自由乘法 ⊗）

(a, σ_a) ⊗ (b, σ_b) = (a · b, σ_a · |b| + σ_b · |a| + σ_a · σ_b)

乘法扰动晕是因子扰动晕的加权和加耦合项。

定义 3.3（自由减法 ⊖）

自由减法是自由加法的逆运算（在核减法的意义下），扰动晕以类似加法的规则演化。

定义 3.4（自由除法 ⊘）

自由除法是自由乘法的逆运算，仅当除数核非零时定义，扰动晕以乘法的逆规则演化。

定理 4.1（自由数的可数无穷性定理）

对于任意经典自然数 n，以 n 为核的自由数构成连续统多个。N_free 的基数为连续统 ℵ_1。

定理 4.2（扰动晕的非减性定理）

对于任意两个自由数 a_hat, b_hat，有：

σ_{a⊕b} ≥ σ_a + σ_b

等号成立当且仅当耦合系数 γ_a = 0 或 σ_a · σ_b = 0。自由加法必然增加或保持扰动晕。

若 f(x) = O(x^k)（多项式增长），则 M[f] 收敛到有限值。

∞_0[f] 完全刻画了 f 在 x → ∞ 时的平均行为、振荡幅度和不可收束残余。

定理 4.5（无穷远完整刻画符的谱分解定理）

∞_0[f] 的七层输出构成 f 在无穷远处的完备不变量——两个函数 f 和 g 在无穷远处等价，当且仅当它们的 ∞_0 输出一致。

定理 4.6（经典极限定理）

对于任意经典整数 n，其在自由数论中的提升 (n, 0) 是所有自由数运算在 σ → 0 极限下的退化结果。
