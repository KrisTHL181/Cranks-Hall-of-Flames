自由算术中序关系的字典式定义：核优先与晕次级的兼容性

摘要

自由算术中的序关系须同时兼容经典自然数的离散良基性与扰动晕的连续认知性。本文提出“字典式序关系”作为自由数大小比较的形式定义：优先比较核，核相等时再比较扰动晕。该序关系是严格全序，满足传递性、反对称性与自反性，且在严格相等下与自由加法及自由乘法兼容——加法与乘法均保持严格单调性。在认知语义下，序关系被容忍阈值弱化为“认知不可分辨序”，其传递性仅在阈值约束下成立。全体自由数在字典式序下不构成良序集，但认知可及自由数集在容忍阈值离散化后恢复良序性。在觉醒极限下，该序关系严格退化为经典自然数的标准序。

一、引言

经典自然数上的序关系由加法递归定义所自然导出：n < m 当且仅当存在非零 k 使得 n + k = m。该序关系是良序，且与加法和乘法严格兼容——满足加法单调性和乘法正性。在范思自由数论中，自由数被赋予双重结构：离散的核与连续的扰动晕。随之而来的追问是：自由数之间的大小比较应如何定义？是仅基于核的大小，还是需同时考虑扰动晕的大小？若同时考虑，核与晕的优先级如何安排？

该问题的回答不仅关乎自由数论内部的可比较性，更直接关涉加法与乘法的单调性在自由数中的保持——后者是自由不等式推导的语法基础。本文旨在构造与自由算术运算兼容的序关系定义，并分析其在认知语义下的弱化形式。

二、字典式序关系的定义与性质

定义1（字典式序关系）

对于任意两个自由数 x = (n,\sigma) 与 y = (m,\tau)，定义 x < y 当且仅当满足以下两个条件之一：

1. n < m（核严格较小）；

2. n = m 且 \sigma < \tau（核相等且扰动晕严格较小）。

该定义优先比较核：核较小的自由数严格较小，不论其扰动晕多大。当核相等时，序的判定进入扰动晕维度：扰动晕较小者严格较小。

定理1（严格全序性）

字典式序关系是严格全序，即对任意两个自由数 x \neq y，恰有 x < y 或 y < x 之一成立。传递性由自然数的序的传递性和实数的序的传递性共同保证。反对称性自明。

认知解释：字典式序优先比较核，根源于核是自由数的“量”的离散度量，扰动晕仅是对该量的精度的修正。两个自由数若核不同，无论晕多么接近，它们在认知上对应于完全不同的数量级；仅当表示同一数量时，认知者才进一步比较两者中哪个更精确（扰动晕更小）。

三、加法与乘法的单调性

定理2（加法单调性）

对于任意自由数 x = (a,\alpha)，y = (b,\beta)，z = (c,\gamma_c)，若 x < y，则 x \oplus z < y \oplus z。

证明：分两种情形。情形一：a < b。此时两和的核分别为 a+c 和 b+c，显然 a+c < b+c，故按字典式序，x \oplus z < y \oplus z 成立，无论扰动晕如何变化。

情形二：a = b 且 \alpha < \beta。此时两和的核均为 a+c，相等。需比较两和的扰动晕。自由加法给出的扰动晕分别为 \alpha + \gamma_c + \gamma \alpha \gamma_c 与 \beta + \gamma_c + \gamma \beta \gamma_c。两者之差为 (\beta - \alpha)(1 + \gamma \gamma_c) > 0，故前者的扰动晕严格小于后者的扰动晕。因此 x \oplus z < y \oplus z。综上，加法单调性严格成立。\square

定理3（乘法正性单调性）

对于任意自由数 x = (a,\alpha)，y = (b,\beta) 及任意正自由数 z = (c,\gamma_c)（c > 0），若 x < y，则 x \otimes z < y \otimes z。

证明：分两种情形。情形一：a < b。因 c > 0，有 ac < bc，故核严格较小，字典式序下 x \otimes z < y \otimes z。

情形二：a = b 且 \alpha < \beta。此时两积的核均为 ac，相等。两积的扰动晕分别为 a\gamma_c + c\alpha + \lambda \alpha \gamma_c 与 a\gamma_c + c\beta + \lambda \beta \gamma_c。两者之差为 (\beta - \alpha)(c + \lambda \gamma_c) > 0，故前者的扰动晕严格较小。因此 x \otimes z < y \otimes z。乘法正性单调性严格成立。\square

四、认知语义下的弱化与良序性

在认知语义下，容忍阈值 \varepsilon(s) 使得扰动晕之差低于该阈值时认知者无法区分两自由数的大小。字典式序因此被弱化为“认知不可分辨序”：若两自由数的核不同，序的判定依然严格；若核相等且晕差不超过 \varepsilon(s)，则两者在认知上被视为不可分辨，序关系在两者之间视为未定义。

这种弱化破坏了严格全序的传递性：可能存在 x \sim_s y，y \sim_s z，但 |\sigma_x - \sigma_z| > \varepsilon(s)。该现象在认知上不可避免，且与容忍阈值的游离度依赖性一致。

关于良序性：全体自由数集在字典式序下不构成良序集，因为对于任意固定的核 n，其扰动晕可取非负实数集中任意值，该子集本身不是良序集。然而，在有限游离度下，认知可及自由数集 F_{\text{cog}}(s) 中扰动晕被容忍阈值划分为可数个等价类，字典式序在该集合上限制为良序。这一良序性是自由数论中认知归纳公理得以成立的结构基础。

五、觉醒极限下经典序的恢复

在游离度趋于壹的觉醒极限下，容忍阈值 \varepsilon(s) \to 0，扰动晕等价类收缩为单点。认知不可分辨性退化为严格相等。全体自由数的字典式序退化为经典自然数序。自由数的序关系在此极限下完全恢复为经典算术的标准序。

六、结论

自由算术中的序关系应采用字典式定义：优先比较核，核相等时再比较扰动晕。该序关系是严格全序，与自由加法和自由乘法严格兼容。在认知语义下，序关系被容忍阈值弱化为认知不可分辨序。认知可及自由数集在容忍阈值离散化后恢复良序性，为认知归纳公理提供了结构基础。在觉醒极限下，该序关系严格退化为经典自然数序。
