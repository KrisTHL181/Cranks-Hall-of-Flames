摘要

勒让德符号给了我们一个±1的判决器。本文正式定义“压缩算子”的核心——权函数ω=½(1-χ)，并证明其在阴类半素数集合上的统计抵消性质。这是整个“压缩-对偶”框架的数学心脏。

一、从判决到加权：权函数的诞生

在上一篇中，我们用勒让德符号χ(p+)=(p+/Q)为阳素数赋予了一个±1的特征值。现在，对于任意一个阴类半素数n=p+×p-，我们定义其权函数：

ω(n) = ½(1 - χ(p+))

这个函数的值域是什么？

· 当χ(p+)=+1时，ω(n)=½(1-1)=0

· 当χ(p+)=-1时，ω(n)=½(1-(-1))=1

对于阳类半素数或真素数，我们不使用此权函数（或赋予固定权值1）。因此，权函数的作用是：将阴类半素数中“χ=+1”的那一半，权重置为零；将“χ=-1”的那一半，保留权重1。

二、统计抵消的数学证明

假设我们从某个集合S中随机抽取大量阴类半素数。由于勒让德符号χ(p+)在阳素数上的分布是近似均匀的（二次剩余与非剩余各占一半），χ=+1和χ=-1的概率各约½。因此，权函数ω(n)的期望值为：

E[ω] ≈ ½(1 - E[χ]) = ½(1 - 0) = ½

这意味着，如果我们不加权，每个阴类半素数贡献1；加权后，期望贡献降为½。但这还不够——我们真正想要的是，在特定的配对求和中，让阴类半素数的总贡献期望为零。这需要下一步：“对偶映射”。

三、权函数的三个关键性质

1. 非负性：ω(n)∈{0,1}，永不为负。

2. 对真素数无害：真素数不经过此权函数（或权重固定为1），不会被误伤。

3. 统计均匀性：在大量样本下，权函数的期望值稳定且可预测。

四、为什么叫“压缩算子”？

因为它将阴类半素数的“有效数量”压缩了一半。在筛法中，这意味着污染项的密度被人为降低。但真正的威力在于：如果我们能进一步将χ=+1和χ=-1的项配对，让它们在一个求和式中相互抵消，那么污染项的总贡献可以被压缩到零——这就是下一篇“对偶映射”的任务。

五、下篇预告

权函数将污染项分成了“留”与“弃”两半。但被留下的那一半（χ=-1），如何让它与另一半在总和中抵消？下一篇《对偶映射：一种无不动点的符号翻转对合》，将给出答案。

Abstract

The Legendre symbol gives us a ±1 decider. This article formally defines the core of the "compression operator"—the weight function ω = ½(1-χ)—and proves its statistical cancellation property on the set of Yin-type semiprimes. This is the mathematical heart of the entire "compression-duality" framework.

1. From Decision to Weighting: The Birth of the Weight Function

In the previous article, we used the Legendre symbol χ(p+) = (p+/Q) to assign a ±1 character value to a Yang prime. Now, for any Yin-type semiprime n = p+ × p-, we define its weight function:

ω(n) = ½(1 - χ(p+))

What is the range of this function?

· If χ(p+) = +1, ω(n) = ½(1-1) = 0.

· If χ(p+) = -1, ω(n) = ½(1-(-1)) = 1.

For Yang-type semiprimes or genuine primes, we do not apply this weight function (or assign a fixed weight of 1). Thus, the effect of the weight function is: among Yin-type semiprimes, those with χ=+1 get zero weight; those with χ=-1 retain weight 1.

2. Mathematical Proof of Statistical Cancellation

Suppose we randomly draw a large number of Yin-type semiprimes from some set S. Since the Legendre symbol χ(p+) is approximately uniformly distributed over Yang primes (quadratic residues and non-residues each account for about half), the probability of χ=+1 and χ=-1 is roughly ½ each. Therefore, the expected value of the weight function ω(n) is:

E[ω] ≈ ½(1 - E[χ]) = ½(1 - 0) = ½.

This means: without weighting, each Yin-type semiprime contributes 1; with weighting, the expected contribution drops to ½. But this is not enough—what we truly want is for the total contribution of Yin-type semiprimes to have an expectation of zero in specific paired sums. This requires the next step: the "duality mapping."

3. Three Key Properties of the Weight Function

1. Non-negativity: ω(n) ∈ {0,1}, never negative.

2. Harmless to genuine primes: Genuine primes are not subjected to this weight function (or have fixed weight 1) and are not mistakenly discarded.

3. Statistical uniformity: Over large samples, the expected value of the weight function is stable and predictable.

4. Why Call It a "Compression Operator"?

Because it compresses the "effective count" of Yin-type semiprimes by half. In sieve methods, this means the density of contaminants is artificially reduced. But the true power lies in the next step: if we can pair terms with χ=+1 and χ=-1 so that they cancel each other out in a summation, then the total contribution of contaminants can be compressed to zero. This is the task of the next article: "duality mapping."

5. Preview of the Next Article

The weight function splits contaminants into "kept" and "discarded" halves. But how can the kept half (χ=-1) cancel out the other half in the overall sum? The next article, Duality Mapping: A Fixed-Point-Free Sign-Flipping Involution, will provide the answer.
