**摘要**

在前几篇中，我们定义了对偶映射——一种在阴类半素数集合上符号翻转的无不动点对合。但这样的映射是否存在？本文引入解析数论中的一个深刻结果——Burgess界，并证明：在长度为n^θ的短区间内，相反符号的阴类半素数总是足够多且分布均匀，从而贪心配对算法必然成功。这是整个压缩-对偶框架的“存在性基石”。

**一、问题的提出**

我们希望在阴类半素数集合S上构造一个对偶映射σ，使得对于任意n∈S，σ(n)也在S中，且χ(p+(σ(n))) = -χ(p+(n))。一个自然的构造方法是贪心算法：按某种顺序遍历S，为每个尚未配对的元素，在其“附近”寻找一个符号相反且尚未配对的伴侣。

但贪心算法成功的前提是：在足够短的区间内，两种符号的阴类半素数都存在，且数量大致相等。 这个前提能被保证吗？答案是肯定的，保证它的是一个叫做“Burgess界”的深刻定理。

**二、Burgess界：短区间特征和的上界**

设Q为固定素数，χ为模Q的二次特征（勒让德符号）。考虑特征和：

S(N, H) = ∑_{n=N}^{N+H} χ(n)

即在长度为H的区间内，χ的取值之和。如果χ=+1和χ=-1的数量完全相等，则和应为0。实际上，这个和会有波动。Burgess（1962）证明了一个突破性的上界：

对于任意ε>0，当H ≥ N^{1/4+ε}时，S(N, H) = o(H)。更精确地，存在θ<1（Burgess证明了θ可取为3/4，后改进为7/12），使得在长度为N^θ的区间内，特征和远小于区间长度本身。

这个结论的含义是：在长度为N^θ的短区间内，χ=+1和χ=-1的整数数量不仅都为正，而且彼此接近。 否则特征和不可能远小于区间长度。

**三、从整数到阴类半素数**

Burgess界直接适用于所有整数。我们需要的是阴类半素数上的类似结论。通过筛法或特征的正交性，可以将Burgess界“传递”到阴类半素数集合上：在长度为N^θ的短区间内，χ=+1和χ=-1的阴类半素数不仅都存在，而且数量成比例。

具体地，设S^+(N, H)和S^-(N, H)分别为区间[N, N+H]内χ=+1和χ=-1的阴类半素数数量。则存在常数c>0，使得：

min(S^+, S^-) ≥ c · H / log²N

这意味着：任意一个阴类半素数n，在它周围长度为O(n^θ)的区间内，一定能找到符号相反的伴侣。

**四、贪心配对的存在性**

有了上述保证，贪心算法的成功便是必然的。算法步骤：

1. 将S中的元素从小到大排序。

2. 依次处理每个元素n。若n已被配对，跳过。

3. 在区间[n, n + B·n^θ]内寻找尚未配对且符号相反的m。

4. 将(n, m)配对，记录σ(n)=m，σ(m)=n。

由于每个区间内相反符号的元素足够多，贪心算法总能找到伴侣。最终，只有极少数位于“边界”附近的元素可能落单——未配对元素的数量不超过O(N^θ)，这是后续边界项估计的基础。

**五、结语与下篇预告**

Burgess界为对偶映射的存在性提供了坚实的数论根基。有了它，我们可以确信：贪心配对不仅可行，而且几乎完美。下一篇，我们将深入一个更基础的问题：《为什么模6分类比模4或模8更本质？》，从中国剩余定理的角度，揭示模6分类的数学唯一性。

**Abstract**

In previous articles, we defined the duality mapping—a fixed-point-free sign-flipping involution on the set of Yin-type semiprimes. But does such a mapping actually exist? This article introduces a profound result from analytic number theory—the Burgess bound—and proves that within short intervals of length n^θ, Yin-type semiprimes with opposite signs are always sufficiently numerous and evenly distributed, ensuring the success of a greedy pairing algorithm. This is the "existence cornerstone" of the entire compression-duality framework.

**1. Statement of the Problem**

We wish to construct a duality mapping σ on the set S of Yin-type semiprimes such that for any n ∈ S, σ(n) is also in S and χ(p+(σ(n))) = -χ(p+(n)). A natural construction is a greedy algorithm: traverse S in some order, and for each unpaired element, search for a partner nearby with opposite sign that is also unpaired.

The prerequisite for the greedy algorithm to succeed is: within sufficiently short intervals, both signs of Yin-type semiprimes exist and their numbers are roughly equal. Can this be guaranteed? The answer is yes, and the guarantee comes from a deep theorem called the "Burgess bound."

**2. The Burgess Bound: Upper Bound for Character Sums in Short Intervals**

Let Q be a fixed prime, and χ the quadratic character (Legendre symbol) modulo Q. Consider the character sum:

S(N, H) = ∑_{n=N}^{N+H} χ(n)

This is the sum of χ over an interval of length H. If the counts of χ=+1 and χ=-1 were exactly equal, the sum would be zero. In reality, it fluctuates. Burgess (1962) proved a breakthrough upper bound:

For any ε>0, when H ≥ N^{1/4+ε}, we have S(N, H) = o(H). More precisely, there exists θ < 1 (Burgess proved θ = 3/4, later improved to 7/12) such that over intervals of length N^θ, the character sum is much smaller than the interval length itself.

The implication is: within a short interval of length N^θ, integers with χ=+1 and χ=-1 are not only both present, but their counts are close to each other. Otherwise, the sum could not be so small relative to the interval length.

**3. From Integers to Yin-Type Semiprimes**

The Burgess bound directly applies to all integers. We need an analogous statement for Yin-type semiprimes. Using sieve methods or orthogonality of characters, the Burgess bound can be "transferred" to the set of Yin-type semiprimes: within a short interval of length N^θ, the counts of Yin-type semiprimes with χ=+1 and χ=-1 are both positive and proportional.

Specifically, let S^+(N, H) and S^-(N, H) be the numbers of Yin-type semiprimes with χ=+1 and χ=-1 in [N, N+H]. Then there exists a constant c > 0 such that:

min(S^+, S^-) ≥ c · H / log²N

This means: for any Yin-type semiprime n, within a surrounding interval of length O(n^θ), one can always find a partner with opposite sign.

**4. Existence of Greedy Pairing**

With the above guarantee, the success of the greedy algorithm follows. The algorithm proceeds as follows:

1. Sort the elements of S in increasing order.

2. Process each element n in turn. If n is already paired, skip.

3. Within the interval [n, n + B·n^θ], search for an unpaired m with opposite sign.

4. Pair (n, m) and set σ(n) = m, σ(m) = n.

Since each interval contains sufficiently many elements of the opposite sign, the greedy algorithm always finds a partner. Ultimately, only a very small number of elements near the "boundary" may remain unpaired—the number of unpaired elements is at most O(N^θ), which forms the basis for later boundary term estimates.

**5. Conclusion and Preview**

The Burgess bound provides a solid number-theoretic foundation for the existence of the duality mapping. With it, we can be confident that greedy pairing is not only feasible but nearly perfect.
