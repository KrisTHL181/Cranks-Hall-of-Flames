摘要

在上一篇中，我们看到了半素数污染项的结构。要构造压制污染的权函数，需要一个能为整数“定性”的工具。本文引入数论中的经典工具——勒让德符号，解释它如何用一个简单的±1，揭示整数在模素数下的二次剩余属性。这是“压缩算子”的数学基石。

一、从“阴阳”到“符号”：需要一个判决器

我们把素数分为阳（6k+1）和阴（6k-1），把半素数分为阳×阳、阴×阴、阳×阴。但这只是分类，不是“判决”。要让阴类污染项在统计中期望为零，我们需要对每一个半素数的阳因子，赋予一个±1的“特征值”，使得阴类半素数的权值平均为零。

这个特征值从哪来？答案藏在二次剩余里。

二、什么是二次剩余？

给定一个奇素数Q，如果一个整数a与Q互素，且存在某个整数x使得x²≡a(mod Q)，则称a是模Q的二次剩余；否则称a是模Q的二次非剩余。

例如，取Q=7。计算1²=1，2²=4，3²=9≡2，4²=16≡2，5²=25≡4，6²=36≡1(mod7)。所以模7的二次剩余是1,2,4；二次非剩余是3,5,6。

三、勒让德符号：一个简洁的判决器

勒让德符号(a/Q)定义为：

· (a/Q)=1，若a是模Q的二次剩余；

· (a/Q)=-1，若a是模Q的二次非剩余；

· (a/Q)=0，若Q整除a。

这个符号将复杂的二次剩余判断，压缩成一个三值输出：+1、-1、0。它满足乘法性：(ab/Q)=(a/Q)(b/Q)。这正是我们需要的“阴阳符”。

四、勒让德符号与素数的关联

对于我们的阳素数p+，计算χ(p+)=(p+/Q)。由于Q是一个固定的阳素数，χ(p+)会以近似均匀的概率输出+1或-1。这意味着：如果我们对大量阴类半素数的阳因子计算勒让德符号，+1和-1的数量大致相等。

这正是“压缩算子”权函数ω=½(1-χ)的统计基础——当χ=+1时权为0，当χ=-1时权为1。阴类污染项的期望贡献被压缩为零。

五、下篇预告

有了勒让德符号，下一篇我们将正式定义压缩算子的权函数，并证明其统计抵消性质。这是整个“压缩-对偶”框架的数学核心。

Abstract

In the previous article, we examined the structure of semiprime contamination. To construct a weight function that suppresses this contamination, we need a tool that assigns a "character" to integers. This article introduces a classical tool from number theory—the Legendre symbol—and explains how it uses a simple ±1 to reveal an integer's quadratic residue property modulo a prime. This is the mathematical foundation of the "compression operator."

1. From "Yin-Yang" to "Sign": The Need for a Decider

We classified primes into Yang (6k+1) and Yin (6k-1), and semiprimes into Yang×Yang, Yin×Yin, and Yang×Yin. But this is merely classification, not decision. To make the contribution of Yin-type contaminants statistically zero in expectation, we need to assign a ±1 "character value" to the Yang factor of each semiprime, so that the weighted average of Yin-type semiprimes tends to zero.

Where does this character value come from? The answer lies in quadratic residues.

2. What Is a Quadratic Residue?

Given an odd prime Q, if an integer a is coprime to Q and there exists an integer x such that x² ≡ a (mod Q), then a is called a quadratic residue modulo Q; otherwise, it is a quadratic non-residue.

For example, take Q=7. Computing squares: 1²=1, 2²=4, 3²=9≡2, 4²=16≡2, 5²=25≡4, 6²=36≡1 (mod 7). Thus, the quadratic residues modulo 7 are 1,2,4; the non-residues are 3,5,6.

3. The Legendre Symbol: A Concise Decider

The Legendre symbol (a/Q) is defined as:

· (a/Q) = 1, if a is a quadratic residue modulo Q;

· (a/Q) = -1, if a is a quadratic non-residue modulo Q;

· (a/Q) = 0, if Q divides a.

This symbol compresses the complex judgment of quadratic residuosity into a three-valued output: +1, -1, or 0. It satisfies multiplicativity: (ab/Q) = (a/Q)(b/Q). This is exactly the "yin-yang sign" we need.

4. Connection Between Legendre Symbol and Primes

For our Yang prime p+, we compute χ(p+) = (p+/Q). Since Q is a fixed Yang prime, χ(p+) outputs +1 or -1 with roughly equal frequency. This means: if we compute the Legendre symbol for the Yang factors of a large number of Yin-type semiprimes, the counts of +1 and -1 will be approximately equal.

This is the statistical basis for the weight function ω = ½(1-χ) of the compression operator. When χ=+1, the weight is 0; when χ=-1, the weight is 1. The expected contribution of Yin-type contaminants is compressed to zero.

With the Legendre symbol in hand, the next article will formally define the weight function of the compression operator and prove its statistical cancellation property. This is the mathematical core of the entire "compression-duality" framework.
