摘要

哈希函数是计算机科学的基石。本文展示如何利用模6阴阳分类与勒让德符号，构造一个轻量级、不可逆的哈希函数原型。该构造揭示了压缩算子在密码学中的潜在应用：将输入数据的阴阳特征压缩为固定长度的摘要，且逆向求解等价于半素数分解。

一、设计思路

给定任意长度的输入消息M，先将其填充并分块，每块转换为一个大整数。对每个块，计算其模6余数，若为阳（6k+1）则附加比特1，若为阴（6k-1）则附加比特0。然后，选取固定阳素数Q，对每个块计算勒让德符号χ(block mod Q)，将结果序列作为哈希值的一部分。

二、压缩函数的构造

定义压缩函数C(x) = (x mod 6) 与 χ(x) 的组合。由于模6分类将整数均匀分为阴阳两类，且勒让德符号在统计上均匀，该压缩函数具有良好的混淆与扩散性质。更重要的是，从输出反推输入需同时求解模6分类的逆与二次剩余判断——这在计算上等价于分解半素数。

三、安全性的数论根基

哈希的抗碰撞性依赖于：找到两个不同的输入使其压缩值相同，等价于找到两个整数具有相同的模6余数与相同的勒让德符号。Burgess界暗示，在短区间内，不同符号的分布是均匀且不可预测的。因此，任何多项式时间的碰撞攻击都将违反二次剩余假设。

四、原型算法伪代码

```

def yin_yang_hash(M):

H = []

Q = 7  # 固定阳素数，实际应用可更大

for block in pad_and_split(M):

residue = block % 6

if residue == 1: H.append(1)

elif residue == 5: H.append(0)

else: H.append(residue % 2)

H.append(legendre(block, Q))

return H

```

五、应用展望与局限

此哈希函数适用于轻量级物联网设备认证，因其仅需模运算与勒让德符号计算，硬件实现极简。但需注意，它尚未经过完整的密码分析，Q的选择、填充方案等需进一步研究。核心参数仍属技术秘密。

Abstract

Hash functions are a cornerstone of computer science. This article demonstrates how to construct a lightweight, non-invertible hash function prototype using modulo-6 yin-yang classification and the Legendre symbol. The construction reveals a potential cryptographic application of the compression operator: compressing the yin-yang features of input data into a fixed-length digest, whose inversion is equivalent to semiprime factorization.

1. Design Rationale

Given an input message M of arbitrary length, pad and split it into blocks, converting each block to a large integer. For each block, compute its remainder modulo 6: if Yang (6k+1), append bit 1; if Yin (6k-1), append bit 0. Then, for a fixed Yang prime Q, compute the Legendre symbol χ(block mod Q) for each block and append the result to the hash value.

2. Construction of the Compression Function

Define the compression function C(x) as the combination of (x mod 6) and χ(x). Because modulo-6 classification evenly splits integers into yin and yang classes, and the Legendre symbol is statistically uniform, this compression function exhibits good confusion and diffusion. Crucially, inverting the output requires simultaneously solving the inverse of modulo-6 classification and judging quadratic residuosity—computationally equivalent to factoring a semiprime.

3. Number-Theoretic Basis of Security

Collision resistance relies on the fact that finding two distinct inputs with the same compression value is equivalent to finding two integers with identical modulo-6 residue and identical Legendre symbol. The Burgess bound implies that the distribution of signs in short intervals is uniform and unpredictable. Hence, any polynomial-time collision attack would violate the quadratic residuosity assumption.

4. Pseudocode of the Prototype

```

def yin_yang_hash(M):

H = []

Q = 7  # fixed Yang prime; larger values used in practice

for block in pad_and_split(M):

residue = block % 6

if residue == 1: H.append(1)

elif residue == 5: H.append(0)

else: H.append(residue % 2)

H.append(legendre(block, Q))

return H

```

5. Application Prospects and Limitations

This hash function is suitable for lightweight IoT device authentication, as it requires only modular arithmetic and Legendre symbol computation—ideal for minimal hardware. Note that it has not undergone full cryptanalysis; the choice of Q, padding scheme, etc., require further study. Core parameters remain proprietary.
