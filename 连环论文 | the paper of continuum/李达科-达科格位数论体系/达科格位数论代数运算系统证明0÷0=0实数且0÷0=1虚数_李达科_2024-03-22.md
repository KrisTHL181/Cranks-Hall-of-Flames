以代数符号O’表示0立方的点实数，以代数符号O表示0平方的点实数，以o表示0长度（即0立方根与0平方根）的点实数，因此O’除以O平方的点等于o平方根的点实数，所以O’÷O=o实数且o³÷o²=o实数，所以O’立方的点除以O’立方的点等于虚数1，所以O’÷O’=1虚数且o³÷o³=1虚数；

之所以O平方的点实数除以o长度实数等于o宽度实数，因此O÷o=o实数且O²÷o=o实数，

之所以O平方的点实数除以O平方的点实数等于虚数1，所以O÷O=1虚数且o²÷o²=1虚数，

之所以o平方根的点实数除以o平方根的点实数等于虚数1，所以o÷o=1虚数，之所以√O÷√O=1虚数或√o²÷√o²=1虚数。
