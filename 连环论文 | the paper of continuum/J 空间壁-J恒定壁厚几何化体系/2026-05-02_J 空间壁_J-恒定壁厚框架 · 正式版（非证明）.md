一、核心定义（可直接复制给AI推导）

1. 补偿恒等式

J = \frac{2\pi(a-b)-C}{2}

C + 2J = 2\pi(a-b)

2. 全局恒定壁厚常数

J_\infty = \frac{\gamma_E}{2\pi} \approx 0.091866726

3. 零点几何约束

a - b = J_\infty

4. 零点-椭圆对应半轴公式

a = \frac{t_k}{2\pi} + \frac{J_\infty}{2} + \frac{J_\infty}{t_k}

b = \frac{t_k}{2\pi} - \frac{J_\infty}{2} + \frac{J_\infty}{t_k}

5. 零点判定条件

J(t_k) = J_\infty
