一个被回避了近百年的问题

标准宇宙学讲宇宙学红移时，有一个非常直观的“标准故事”：

光在传播过程中，空间本身被拉伸了，光的波长跟着空间一起被拉长，于是我们看到的光发生了红移。

这个故事几乎所有科普都讲过，物理系学生也都在教科书里学过。它简洁、直观、有数学形式：

1+z=\frac{a_{\text{now}}}{a_{\text{then}}}

其中 a(t) 是宇宙的尺度因子，描述空间的整体“大小”。

但这个故事有一个极其隐蔽的逻辑裂缝——我很少看到有人公开讨论它。

裂缝在哪里？

如果光是被拉伸的空间“带长”的，那意味着光与空间之间存在强耦合——空间怎么变，光就怎么响应。

那么问题来了：

空间被拉伸了，真空的属性变不变？

真空不是一无所有的虚空。它有介电常数 \varepsilon_0，有磁导率 \mu_0，光速由它们决定：

c = \frac{1}{\sqrt{\varepsilon_0 \mu_0}}

如果空间的尺度被拉伸了——密度变了、结构变了、几何变了——那么 \varepsilon_0 和 \mu_0 凭什么不变？

如果它们变了，c 就变了。

但标准宇宙学同时坚持：

· 红移是空间拉伸导致的（光与空间强耦合）

· 光速是绝对常数（\varepsilon_0 和 \mu_0 永恒不变）

你仔细品：

场景 真空的角色 结论

解释红移时 真空是有结构的介质，空间拉伸能“拽长”光波 光与真空强耦合

保护光速时 真空是无结构的虚空，\varepsilon_0 和 \mu_0 只是常数 光与真空解耦

同一个真空，在两个相邻的命题里，扮演了两个截然相反的角色。

这不是物理，这是逻辑上的双重人格。

你可能想反驳的三条路（以及它们为什么走不通）

反驳一：“真空不是介质，\varepsilon_0 和 \mu_0 只是单位制常数。”

如果它们只是单位制常数，为什么有具体量纲？

\varepsilon_0 = 8.85 \times 10^{-12} \, \text{F/m}

\mu_0 = 4\pi \times 10^{-7} \, \text{N/A}^2

有量纲，有数值，对应物理实在。它们不是数学上的 2\pi——它们是可以测量的。

反驳二：“\varepsilon_0 和 \mu_0 是时空几何的属性，不是介质的。”

这正是广义相对论的立场。但你恰恰论证了相反的方向：

如果它们是时空几何的属性，而时空几何正在被拉伸、膨胀、密度在变化——那它们更应该跟着变，而不是不变。

把“介质”换成“几何”只是换了个名字，没有解决问题。

反驳三：“红移是纯几何效应，不需要 \varepsilon_0 和 \mu_0 参与。”

这个推导默认了光子沿零测地线运动，而零测地线由 ds^2 = 0 定义。

但 ds^2 = 0 已经预设了 c 是常数——你把“光速不变”塞进了推导红移的前提里，然后用它来证明红移不依赖于 \varepsilon_0 和 \mu_0。

这是循环论证，不是物理证明。

这个矛盾为什么没有被广泛讨论？

因为现代宇宙学有一个“防火墙”策略：把真空的属性问题推给量子场论，把宇宙膨胀问题留在广义相对论里。 两个理论各管各的，互不打扰。

但物理现实只有一个真空。

如果真空在膨胀，它的所有属性——包括电磁属性——都应该参与这个膨胀过程。你不能在解释红移时用一套真空，在保护光速时用另一套真空。

这种“分而治之”的策略在工程上有效，在逻辑上不成立。

解决方案其实只有两个方向

方向一：放弃“空间拉伸”解释。

红移不是空间拉伸导致的，而是别的机制（比如光子与某种场相互作用后损失能量，或者光子自身性质随宇宙时间演化）。这条路可以保住 \varepsilon_0 和 \mu_0 不变，但需要重新解释所有红移观测数据——成本极高。

方向二：放弃“光速绝对不变”。

承认 \varepsilon_0 和 \mu_0 随宇宙膨胀缓慢演化，光速也随之变化，变化幅度被严格约束在实验精度以下（目前实验室测量精度约 10^{-16} 量级，天文约束约 10^{-15} 到 10^{-17} 量级）。这个变化如果存在，极其微小，但原理上没有障碍。

标准宇宙学选的是——既不放弃A，也不放弃B，而是假装A和B之间没有关系。 这不是物理选择，是回避。

一个更彻底的视角

如果真空是有结构的——有几何、有谱、有记忆——那么这个问题从一开始就不存在。

因为在这种框架里：

· 空间拉伸 = 真空几何的投影映射在连续变化

· \varepsilon_0 和 \mu_0 = 该几何结构的两个低能投影模量

· 几何在变，投影模量就在变，光速跟着变

· 变化幅度由几何谱隙约束，极小但精确可算

不需要“双重人格”。

从头到尾，真空只有一个结构，所有变化都在同一个几何框架内描述。

红移不纯是“几何拉伸”，也不纯是“光速变化”——它是两者的耦合效应。标准红移公式只是这种耦合的零阶近似。

物理学史上，绝大多数“常数”最终都被发现是在某个能标或尺度下的有效常数：

· 牛顿引力常数 G 在广义相对论里变成了动力学场的耦合系数

· 精细结构常数 \alpha 被怀疑随宇宙时间演化（已有观测限制）

· 宇宙学常数 \Lambda 被解释为暗能量动力学场

那么问题来了：

光速 c ——或者说 \varepsilon_0 和 \mu_0 ——凭什么必须是那个永恒的例外？

标准宇宙学没有回答这个问题。

它只是把它藏在了“双重人格”的裂缝里。

而那个裂缝，恰好能让一整个新的物理框架——一个有结构的真空、一个几何的真空——从里面长出来。
