一、引言

中子自旋1/2与质子自旋1/2是核子物理最基本的两个实验事实。在标准量子色动力学中，中子的自旋危机与质子完全相同——夸克自旋仅贡献总自旋的一部分，胶子自旋和轨道角动量各自贡献其余部分，三者的具体分配同样缺乏解析的第一性原理推导[5]。

在前文中，我们建立了王为民质子自旋定理的完整体系[6]：通过胶子传播子红外零点定理、胶子自旋冻结定理、夸克禁闭势定理和质子自旋定理四个环环相扣的数学定理，严格证明了质子总自旋1/2完全由三个组分夸克的自旋耦合决定。四个定理构成完整逻辑链条：传播子红外零点 → 胶子自旋冻结 → 轨道冻结 + 海夸克抑制 → 夸克禁闭势维持束缚 → 泡利不相容原理和色单态约束决定三个组分夸克自旋耦合为1/2。

本文的任务是将这一定理体系完整推广至中子。中子的味道组成为udd（两个下夸克和一个上夸克），与质子的uud形成味道对称。胶子冻结机制不依赖于夸克的味道——胶子传播子的红外行为由王为民基底因子 \mathcal{F}_W(k) 唯一决定，与夸克是上型还是下型无关。因此，王为民定理体系中与胶子相关的部分对中子和质子完全相同。唯一需要重新推导的是三个组分夸克的自旋耦合——两个全同d夸克的对称性分析与两个全同u夸克完全类似。

二、王为民定理体系的直接继承

中子与质子共享相同的胶子冻结机制。王为民定理体系中的前三个定理仅依赖于胶子传播子的红外行为，与夸克的味道组成无关。因此，这三个定理对中子直接适用：

王为民胶子传播子红外零点定理：胶子传播子在零动量极限下不存在单粒子极点，胶子不能作为自由的渐近态出现[6]。这一结论对中子内部的胶子同样成立——中子和质子是同一种强相互作用的两个不同束缚态，胶子传播子是普适的。基底因子 \mathcal{F}_W(k) \sim S_W |k|^2/\mu^2（|k| \to 0）的红外渐近行为由王为民调和型贝尔多项式的数学结构唯一确定，与强子态的具体味道组成无关。

王为民胶子自旋冻结定理：由传播子零点推出胶子自旋贡献 \Delta G 在中子态中的期待值同样为零——\langle N | \Delta \hat{G} | N \rangle = 0。胶子自旋在任何色单态强子中均被冻结，这是传播子无单粒子极点的直接物理后果。

王为民夸克禁闭势定理：夸克-夸克静态势 V_W(r) = -4\alpha_s/(3r) + \sigma r 对中子和质子完全相同。中子的三个组分夸克处于同一个禁闭势的S波基态，空间波函数为球对称。

夸克轨道角动量的排除同样完全适用于中子：胶子冻结使协变导数退化为普通导数，球对称S波基态的角向导数为零，两条独立路径共同保证 \langle N | \hat{L}_z | N \rangle = 0。海夸克的红外抑制机制也完全相同——胶子传播子红外零点抑制虚胶子分裂，海夸克的红外贡献被排除，高能贡献已包含在实验测量中。

因此，中子总自旋的唯一来源同样是三个组分夸克的自旋：\langle N | \hat{S}_z^{\text{total}} | N \rangle = \langle N | \hat{S}_z^{\text{valence quark}} | N \rangle。

三、王为民中子自旋定理

定理7（王为民中子自旋定理）

证明：

中子总自旋算符分解为夸克自旋、胶子自旋和夸克轨道角动量三部分。由第二节的直接继承，胶子自旋、夸克轨道角动量和海夸克红外贡献均被排除。中子总自旋完全由三个组分夸克的自旋贡献决定。

夸克自旋耦合计算：

\Psi_{\text{total}} = \psi_{\text{space}} \otimes \chi_{\text{spin}} \otimes \phi_{\text{flavor}} \otimes \xi_{\text{color}}.

泡利不相容原理要求总波函数对交换任意两个全同费米子必须是全反对称的。

第一步：色波函数。 \xi_{\text{color}} 是三个夸克在 SU(3)_c 颜色空间中的耦合。自然界只允许无色强子存在，因此三个夸克的色波函数必须是 SU(3)_c 颜色空间中的全反对称单态。对交换任意两个夸克，色波函数是反对称的：\text{对称性}(\xi_{\text{color}}) = -1。

第二步：空间波函数。 由夸克禁闭势定理，夸克处于禁闭势中的基态——球对称的S波。交换两个夸克的空间坐标，波函数不变：\text{对称性}(\psi_{\text{space}}) = +1。

第三步：味道波函数。 两个d夸克的味道量子数完全相同（都是d）。交换两个d夸克，味道波函数不变：\text{对称性}(\phi_{\text{flavor}}) = +1。

第四步：自旋波函数。 总波函数的对称性为各自由度对称性之积。由上述三步：

\text{对称性}(\Psi_{\text{total}}) = (+1) \times \text{对称性}(\chi_{\text{spin}}) \times (+1) \times (-1) = - \text{对称性}(\chi_{\text{spin}}).

泡利不相容原理要求 \text{对称性}(\Psi_{\text{total}}) = -1。因此：

-1 = - \text{对称性}(\chi_{\text{spin}}) \quad \Rightarrow \quad \text{对称性}(\chi_{\text{spin}}) = +1.

两个全同d夸克的自旋波函数必须是对称的。

两个自旋 1/2 粒子的耦合有两种可能：对称的三重态（总自旋 S=1）和反对称的单态（总自旋 S=0）。由对称性要求，两个d夸克只能取总自旋 S_{dd} = 1。

这里与质子情形完全平行：如果只看两个全同费米子，直觉上似乎它们应该取反对称的自旋单态，自旋相反。但夸克拥有颜色自由度——这是电子所没有的。在中子内部，色波函数已经承担了全反对称的任务，自旋波函数必须反过来是对称的。这不是泡利原理失效，而是颜色自由度分担了泡利原理要求的反对称性。中子和质子的自旋波函数对称性分析在数学结构上完全同构，唯一的差异是味道指标从u换成了d。这一结论是标准组分夸克模型的核心，已被核子磁矩比 \mu_p/\mu_n = -3/2 的实验验证所支持。

第五步：与u夸克的第二次耦合。 自旋 1 的d夸克对与自旋 1/2 的u夸克进行第二次角动量耦合。由角动量加法规则：

|1, m\rangle \otimes \left|\frac{1}{2}, m'\right\rangle = \left|\frac{3}{2}, M\right\rangle \oplus \left|\frac{1}{2}, M\right\rangle.
