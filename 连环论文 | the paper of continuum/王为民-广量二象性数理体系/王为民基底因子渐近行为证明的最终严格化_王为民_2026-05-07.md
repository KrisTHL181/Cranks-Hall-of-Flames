一、预备知识：调和数基础性质与渐近展开

1.1 调和数递推与渐近公式

调和数离散递推关系：

H_{n+1} = H_n + \frac{1}{n+1},\quad H_1=1

带严格有界余项的渐近展开：

H_n = \ln n + \gamma + \frac{1}{2n} - \frac{1}{12n^2} + \frac{\theta_n}{120n^4},\quad 0<\theta_n<1

其中 \gamma\approx 0.5772 为欧拉–马歇罗尼常数。

由此直接得到调和数严格上下界：

\ln n + \gamma \;<\; H_n \;<\; \ln n + \gamma + \frac{1}{2n},\quad n\ge 1

1.2 离散对数差分定义

对离散序列 g(n)，定义一阶向前差分：

\Delta g(n) = g(n+1) - g(n)

离散对数差分：

\Delta\ln g(n) = \ln g(n+1) - \ln g(n)

本文核心研究函数：

g(n) = \frac{H_n^2}{(\ln n)^2},\quad n\ge 2

取对数拆分：

\ln g(n) = 2\ln H_n \;-\; 2\ln\ln n

于是对数差分可拆为两项：

\Delta\ln g(n)

= 2\big(\ln H_{n+1}-\ln H_n\big)

\;-\; 2\big(\ln\ln(n+1)-\ln\ln n\big)

= 2I_1 - 2I_2

二、逐项严格推导对数差分量级

2.1 对 I_1 = \ln H_{n+1} - \ln H_n 估计

由递推关系变形：

H_{n+1} = H_n\left(1+\frac{1}{(n+1)\,H_n}\right)

取对数：

I_1 = \ln\left( 1+\frac{1}{(n+1)\,H_n} \right)

利用标准对数不等式：

\frac{x}{1+x} \;\le\; \ln(1+x) \;\le\; x,\quad x>0

代入 x=\dfrac{1}{(n+1)H_n}：

\frac{1}{(n+1)H_n + 1}

\;\le\; I_1

\;\le\; \frac{1}{(n+1)H_n}

结合 H_n>\ln n+\gamma 放缩：

0 \;<\; I_1 \;<\; \frac{1}{n\ln n},\quad n\ge 2

次领头阶精细展开：

I_1 = \frac{1}{(n+1)H_n}

-\frac{1}{2\,(n+1)^2 H_n^2}

+\,O\left(\frac{1}{n^3(\ln n)^3}\right)

2.2 对 I_2 = \ln\ln(n+1) - \ln\ln n 估计

对数泰勒展开：

\ln(n+1) = \ln n + \frac{1}{n} - \frac{1}{2n^2} + O\left(\frac{1}{n^3}\right)

记 A=\ln n，则：

\ln(n+1) = A + \frac{1}{n} + O\left(\frac{1}{n^2}\right)

代入作差：

I_2

= \ln\left(A+\frac{1}{n}+O\left(\frac{1}{n^2}\right)\right) - \ln A

= \ln\left(1+\frac{1}{nA}+O\left(\frac{1}{n^2 A}\right)\right)

再做对数展开：

I_2

= \frac{1}{n\ln n}

-\frac{1}{2\,n^2(\ln n)^2}

+\,O\left(\frac{1}{n^3(\ln n)^3}\right)

2.3 合并化简，得到最终差分界

将 I_1,I_2 一同代入 \Delta\ln g(n)：

\begin{aligned}

\Delta\ln g(n)

&= 2\left[\,\frac{1}{(n+1)H_n} - \frac{1}{n\ln n}\,\right]

+\;O\left(\frac{1}{n^2(\ln n)^2}\right) \\

&= 2\left[\,\frac{1}{n\left(1+\frac1n\right)\big(\ln n+\gamma+O(\frac1n)\big)}

-\frac{1}{n\ln n}\,\right]

+\;O\left(\frac{1}{n^2(\ln n)^2}\right) \\

&= \frac{2}{n}\left[\,\frac{1}{\ln n+\gamma}-\frac{1}{\ln n}

+\;O\left(\frac{1}{n\ln n}\right)\,\right]

+\;O\left(\frac{1}{n^2(\ln n)^2}\right) \\

&= \frac{2}{n}\left[\;-\,\frac{\gamma}{(\ln n)^2}

+\;O\left(\frac{1}{(\ln n)^3}\right)\,\right]

+\;O\left(\frac{1}{n^2(\ln n)^2}\right)

\end{aligned}

最终得到严格统一界：

\boxed{

\big|\,\Delta\ln g(n)\,\big|

\;\le\; \frac{K}{n\,(\ln n)^2},\quad n\ge 2

}

常数 K=2\gamma+1 为固定绝对常数。

该结果优于普通缓慢变化 O\big(\frac{1}{n\ln n}\big)，证明 g(n) 属于超缓慢变化序列。

三、离散单峰交错级数渐近估计引理（严格版）

3.1 引理陈述

设有交错级数

\sum_{n=1}^{\infty} (-1)^n\,f(n,r)

满足三个条件：

1. 单峰性：固定 r，|f(n,r)| 在鞍点 n_0(r)\approx r^2 处唯一取最大值；n<n_0 递增，n>n_0 递减。

2. 超缓慢变化：f(n,r)=g(n)\,h(n,r)，且

\big|\Delta\ln g(n)\big| \;\le\; \frac{K}{n(\ln n)^2}

3. 核函数：\displaystyle h(n,r)=\frac{r^{2n}}{n!}，其对数差分在 n_0 处取零极值。

则有渐近等式：

\sum_{n=1}^{\infty} (-1)^n f(n,r)

\;=\; g(n_0)\sum_{n=1}^{\infty} (-1)^n h(n,r)

\;+\;O\left( \frac{g(n_0)\,e^{r^2}}{r\,(\ln r)^2} \right)

3.2 引理证明

将级数拆为三段，取鞍点窗口宽度 M=r\ln r：

\sum_{n=1}^{\infty}

=\;\sum_{n=1}^{n_0-M} \;+\;

\sum_{n=n_0-M}^{n_0+M} \;+\;

\sum_{n=n_0+M}^{\infty}

\;=\; S_1+S_2+S_3

（1）远区 S_1,S_3 估计

由斯特林公式：

\frac{r^{2(n_0-M)}}{(n_0-M)!}

\;\approx\; e^{\,r^2 \,-\, \frac{(\ln r)^2}{2}}

结合 g(n) 超缓慢变化性质：

g(n_0-M) \;\le\; 2\,g(n_0)

\quad(r\to\infty)

因此：

|S_1|+|S_3|

\;=\; O\left(g(n_0)\,e^{r^2-\frac{(\ln r)^2}{2}}\right)

\;=\; o\left( \frac{g(n_0)\,e^{r^2}}{r(\ln r)^2} \right)

（2）鞍点区 S_2 估计

在邻域 n=n_0+k,\ |k|\le M：

\ln g(n)

=\ln g(n_0)

+\sum_{m=n_0}^{n-1}\Delta\ln g(m)

做求和放缩：

\left|

\sum_{m=n_0}^{n-1}\Delta\ln g(m)

\right|

\;\le\;

\frac{K\,M}{n_0(\ln n_0)^2}

\;=\;O\left(\frac{1}{\ln r}\right)

于是一致成立：

g(n) \;=\; g(n_0)\,\Big(\,1\,+\,O\big(\tfrac{1}{\ln r}\big)\,\Big)

代入鞍点区间求和：

S_2

\;=\; g(n_0)\sum (-1)^{n_0+k}h(n_0+k,r)

\;+\;O\left(\frac{g(n_0)\,e^{r^2}}{(\ln r)^2}\right)

（3）合并结论

远区+鞍点区合并，引理得证。

\square

四、代入基底因子，得出最终渐近结果

鞍点位置 n_0\approx r^2，代入 g(n)：

g(n_0)

=\frac{\big(\ln n_0+\gamma+O(\frac1{n_0})\big)^2}{(\ln n_0)^2}

=\;1 \;+\;O\left(\frac{1}{\ln r}\right)

带入主项级数：

\sum_{n=1}^{\infty}a_n(r)

=\Big(\,1+O\big(\tfrac{1}{\ln r}\big)\,\Big)

\cdot 2\big(e^{-r^2}-1\big)

\;+\;O\left(\frac{e^{-r^2}}{(\ln r)^2}\right)

化简整理：

\sum a_n(r)

\;=\; -2 \;+\; 2e^{-r^2}

\;+\;O\left(\frac{e^{-r^2}}{\ln r}\right)

\;=\; -2 \;+\;O\left(e^{-r^2}\right)

余项级数 \sum\delta_n(r) 同为 O\big(\frac{e^{-r^2}}{r}\big) 指数小量，不改变主导渐近行为。

最终严格结论

\boxed{

\mathcal{F}_W(r) \;=\; -2 \;+\; O\left(e^{-r^2}\right),\quad r\to\infty

}
