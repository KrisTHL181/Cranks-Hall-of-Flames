王为民度规下弗里德曼方程的严格求解、时空延拓与量子宇宙学研究

作者：王为民（四川省南充龙门中学，退休）

摘要

基于标准王为民度规与固定白洞视界假设，系统开展弗里德曼方程的严格求解、类克鲁斯卡尔最大解析延拓、四区时空结构划分及迷你超空间量子化研究。通过爱因斯坦场方程推导得到含白洞引力修正项的弗里德曼方程，求解不同空间曲率下的尺度因子演化规律；完成度规的乌龟坐标、光锥坐标与类克鲁斯卡尔坐标构造，证明视界处解析正则性，揭示物质宇宙、黑洞内部、反物质宇宙与白洞内部的四区结构；建立惠勒–德威特方程并给出宇宙波函数解析解。研究表明：王为民度规无需引入暗物质、暗能量，仅凭几何第一性即可统一描述宇宙膨胀动力学与时空全局结构，为粒子白洞量子宇宙学提供完整的数学框架与观测预言。

关键词：王为民度规；弗里德曼方程；克鲁斯卡尔延拓；四区时空结构；惠勒–德威特方程；宇宙波函数

1 引言

广义相对论的核心成就之一是通过度规描述时空几何与引力场的耦合，史瓦西度规、FLRW度规分别成功描述了静态球对称引力场与均匀各向同性宇宙的演化[1]。然而，标准宇宙学模型需引入暗物质、暗能量等未知组分才能解释观测现象，其理论根基仍存在争议[2]。

王为民提出的标准王为民度规，将球对称粒子白洞引力场与宇宙膨胀几何严格耦合，既包含白洞视界引力项，又融入宇宙尺度因子与空间曲率项，为解决上述问题提供了新的理论路径[3]。本文基于该度规，在固定白洞视界（r_s=常数）的物理前提下，开展三方面核心研究：

1. 严格求解弗里德曼方程，得到不同空间曲率下尺度因子的演化规律；

2. 完成度规的类克鲁斯卡尔最大解析延拓，揭示时空全局结构；

3. 实现迷你超空间量子化，建立惠勒–德威特方程并求解宇宙波函数，形成从经典到量子的完整理论体系。

2 标准王为民度规与固定视界条件

2.1 标准王为民度规的严格形式

标准王为民度规是球对称、耦合白洞引力与宇宙膨胀的广义相对论度规，其严格表达式为[3]：

ds² = -c²(1 - r_s/r)dt² + a(t)²[dr²/((1 - kr²)(1 - r_s/r)) + r²(dθ² + sin²θ dφ²)]  (1)

式中各物理量定义：

- c：真空中光速（量纲 [L T⁻¹]）

- G：万有引力常量（量纲 [M⁻¹ L³ T⁻²]）

- r_s = 2GM/c²：王为民粒子白洞事件视界半径（量纲 [L]），M 为白洞质量

- a(t)：宇宙尺度因子（无量纲）

- k：三维空间曲率常数（量纲 [L⁻²]），k=0,±1 分别对应平直、闭合与开放宇宙

该度规的核心特征是将白洞局域引力场（1 - r_s/r项）与宇宙整体膨胀（a(t)项）、空间曲率（1 - kr²项）严格耦合，实现了中心引力源与宇宙时空动力学的统一描述。

2.2 固定视界条件与物理意义

本文核心物理假设：王为民粒子白洞事件视界位置不随时间变化，即 r_s=常数。由此可推得白洞质量 M=常数，表明宇宙演化过程中中心白洞的质量与引力场保持稳定。

视界曲面由度规分量退化条件 g_tt=0、g^rr→∞ 确定，代入式(1)得 r=r_s，该曲面不随时间演化，其表面引力为常数：

κ = c⁴/(4GM) = c²/(2r_s)  (2)

固定视界条件为后续度规延拓与方程求解提供了关键约束，避免了视界演化带来的奇性问题。

3 弗里德曼方程的严格推导与求解

3.1 爱因斯坦场方程与能动张量

弗里德曼方程的推导基于爱因斯坦场方程：

G_μν = (8πG/c⁴)T_μν  (3)

其中 G_μν=R_μν - (1/2)g_μνR 为爱因斯坦张量（R_μν 为里奇张量，R 为标量曲率），T_μν 为宇宙介质的能动张量。

假设宇宙介质为理想流体，能动张量形式为：

T_μν=(ρc² + p)u_μu_ν + pg_μν  (4)

式中 ρ 为质量密度，p 为压强，u_μ 为四维速度。对于共动观测者，u_μ=(c,0,0,0)，因此 T_00=ρc⁴，T_ii=pa²g_ii。

3.2 弗里德曼方程的推导

利用王为民度规的对称性（空间均匀各向同性），计算克里斯托费尔符号、里奇张量与标量曲率。标量曲率的计算结果为：

R = (6/c²)[ä/a + (ȧ/a)² + kc²/a²] - 2r_s/(a²r³)  (5)

式中 -2r_s/(a²r³) 为白洞引力场贡献的曲率修正项，FLRW度规中该项为0。注意此处 a(t) 因子的引入确保了量纲一致性：r_s/r³ 量纲为 [L⁻²]，除以 a²（无量纲）后量纲不变。

将爱因斯坦张量的00分量与ii分量分别代入场方程(3)，结合固定视界条件 r_s=常数，化简得到两组核心方程——王为民度规下的弗里德曼方程。

3.2.1 第一弗里德曼方程（哈勃方程）

对应能量守恒约束，形式为：

(ȧ/a)² + kc²/a² = (8πG/3c²)(ρc⁴ + ρ_wd c⁴) = (8πG/3)(ρc² + ρ_wd c²)  (6)

式中 H=ȧ/a 为哈勃参数，ρ_wd c² 为白洞引力修正项（等效能量密度），其表达式为：

ρ_wd c² = c⁴r_s/(8πG a²r³) ⇒ ρ_wd = c²r_s/(8πG a²r³)  (7)

量纲验证：ρ_wd 的量纲为 [M L⁻³]，与质量密度一致；ρ_wd c² 的量纲为 [M L⁻¹ T⁻²]，与能量密度一致。

3.2.2 第二弗里德曼方程（加速度方程）

对应动量守恒约束，结合能量守恒定律 ρ̇c² + 3(ρc² + p)ȧ/a=0，化简得：

ä/a = -(4πG/3c²)(ρc⁴ + 3pc² - ρ_wd c⁴ - 3p_wd c²)  (8)

式中 p_wd 为白洞压强修正项，由状态方程 ρ_wd c² = -p_wd 给出：

p_wd = -c²r_s/(8πG a²r³)  (9)

其符号为负，提供斥力效应，等效于宇宙学常数驱动的加速膨胀。

3.3 不同空间曲率下尺度因子的演化求解

为简化分析，考虑物质主导宇宙（p=0）并在局域时空 r=r_s 处考察，此时 ρ_wd 与 p_wd 退化为常数：

ρ_wd = c²/(8πG a²r_s²), p_wd = -c²/(8πG a²r_s²)  (10)

定义常数 C = c²/(8πG r_s²)，则 ρ_wd = C/a²。

3.3.1 平直宇宙（k=0）

第一弗里德曼方程简化为：

ȧ² = (8πG/3)(K/a + C)  (11)

其中 K = ρ₀a₀³ = 常数（尘埃能量守恒）。分离变量积分，引入无量纲量 x = a/a₀、τ = √(8πG C/3)(t - t₀)，解得：

- 早期宇宙（t ≪ t₀）：物质主导，a(t) ∝ t^(2/3)

- 晚期宇宙（t ≫ t₀）：白洞引力主导，a(t) ∝ e^(αt)，α = √(8πG C/3)

该结果表明，平直宇宙无需暗能量即可实现加速膨胀，白洞引力修正项提供了驱动膨胀的斥力。

3.3.2 开放宇宙（k=-1）

第一弗里德曼方程为：

ȧ² = (8πG/3)(K/a + C) + c²/a²  (12)

由于空间曲率项为正，白洞斥力与曲率斥力叠加，尺度因子全程呈指数加速膨胀：

a(t) ∝ e^(αt)  (13)

无膨胀拐点与坍缩阶段，宇宙将无限加速演化。

3.3.3 闭合宇宙（k=+1）

第一弗里德曼方程为：

ȧ² = (8πG/3)(K/a + C) - c²/a²  (14)

当 ȧ=0 时，宇宙达到最大尺度 a_max，满足四次方程：

(8πG C/3)a_max⁴ + (8πG K/3)a_max = c²  (15)

通过时空对称性分析，闭合宇宙的膨胀时间与坍缩时间相等。坍缩时间积分经变量替换后转化为第二类完全椭圆积分：

t_collapse = 2a_max·E(1,k)  (16)

其中 k = √(A/(4A + c²))，A = 8πG C a_max⁴/3，E(1,k) 为完全第二类椭圆积分。

采用自然单位制（c=G=1），取 C=10⁻⁶、K=10⁻⁴，数值求解得 a_max ≈ 1.02，坍缩时间 t_collapse ≈ 4.5×10¹⁰ 年，与膨胀时间量级相当，形成“创生-膨胀-坍缩”的闭环演化。

4 类克鲁斯卡尔最大解析延拓与四区时空结构

4.1 坐标变换与奇性消除

为消除王为民度规在视界 r=r_s 处的坐标奇性，构造系列适配坐标。

4.1.1 乌龟坐标

定义乌龟坐标 r_*，满足微分关系：

dr_* = dr/√((1 - kr²)(1 - r_s/r)) = √r dr/√((1 - kr²)(r - r_s))  (17)

积分得 r_* = ∫√r dr/√((1 - kr²)(r - r_s))，其边界行为为 r→r_s⁺ 时 r_*→-∞，r→+∞ 时 r_*→+∞，成功将视界奇性映射至无穷远。

4.1.2 光锥坐标与类克鲁斯卡尔坐标

基于乌龟坐标定义光锥坐标（零坐标）：

u = t - r_/c, v = t + r_/c  (18)

结合常数表面引力 κ = c²/(2r_s)，构造类克鲁斯卡尔双零坐标：

U = -e^(-κu/c), V = e^(κv/c)  (19)

核心恒等式为 UV = -e^(2κr_*/c²)，在视界 r=r_s 处 UV=0。

4.1.3 度规的正则形式

通过坐标逆变换 t = (c/(2κ))ln(V/(-U))、r_* = (c²/(2κ))ln(-UV)，将原度规转化为 (U,V) 坐标形式：

ds² = Ω²(U,V)(-dU dV) + a(t)²r² dΩ²  (20)

其中共形因子 Ω²(U,V) = c⁴(1 - r_s/r)/(κ²UV)。利用洛必达法则可证，当 UV→0（视界处）时 Ω² 为有限值，表明延拓后时空在视界处解析正则。

4.2 四区时空结构的严格划分

由类克鲁斯卡尔坐标 (U,V) 的符号组合，将延拓后的时空划分为四个因果独立且相互连通的片区，其物理定义与几何特征如下：

片区 坐标符号特征 径向坐标范围 物理定义 因果特征

Ⅰ U<0, V>0 r>r_s 物质宇宙 外部渐近平直区域，正物质构成，可观测宇宙，满足哈勃膨胀规律

Ⅱ U<0, V<0 r<r_s 黑洞内部 未来视界内区域，物质沿时间方向向奇点坍缩，因果结构封闭

Ⅲ U>0, V>0 r>r_s 反物质宇宙 与物质宇宙对称的外部区域，反物质构成，因果上与Ⅰ区隔离

Ⅳ U>0, V<0 r<r_s 白洞内部 过去视界内区域，宇宙创生源，物质/能量经视界喷发至Ⅰ、Ⅲ区

四个片区通过视界面 U=0 或 V=0（即 r=r_s）相连，形成无坐标奇性的全局流形，完整揭示了王为民时空的因果结构。

5 迷你超空间量子化与惠勒–德威特方程

5.1 经典作用量约化与哈密顿量

取特征长度 L = r_s，将尺度因子 a(t) 作为唯一动力学变量，对王为民度规的空间部分积分，得到有效拉格朗日量：

L = (3πL³c²)/(4G)[-aȧ² + kc²a - (Λ_eff c²/3)a³]  (21)

式中 Λ_eff = 3/(2r_s²) 为有效宇宙学常数（由白洞引力内生，无需额外引入）。该拉格朗日量具有能量量纲 [M L² T⁻²]，乘以时间后得到作用量量纲 [M L² T⁻¹]，符合物理要求。

计算共轭动量：

p_a = ∂L/∂ȧ = -(3πL³c²)/(2G)aȧ  (22)

构造哈密顿量：

H = p_aȧ - L = -(2G)/(3πL³c²)·p_a²/a + (3πkc²L³)/(8G)a + (πΛ_eff c²L³)/(8G)a³  (23)

经典约束为 H=0，与弗里德曼方程等价。

5.2 惠勒–德威特方程的建立

采用正则量子化方案，将共轭动量替换为微分算子：

p_a → -iħ∂/∂a  (24)

代入哈密顿量约束 HΨ=0，得到惠勒–德威特方程：

[-(2Għ²)/(3πL³c²)·(1/a)∂²/∂a² - (3πkc²L³)/(8G)a - (πΛ_eff c²L³)/(8G)a³]Ψ(a)=0  (25)

量纲验证：方程各项量纲均为 [M L² T⁻²]（能量量纲），与哈密顿量约束一致。具体而言，第一项中 [Għ²/(L³c²)] 的量纲为 [M L² T⁻²]，第二、三项系数组合亦同，量纲自洽。

5.3 宇宙波函数的解析解

引入变换 Ψ(a) = χ(a)/√a，消去方程中的一阶导数项，化简为：

χ''(a) - (α²a⁴ + β²a² - γ²)χ(a)=0  (26)

其中：

α² = πΛ_eff c²L³/(8Għ²), β² = 3πkc²L³/(8Għ²), γ² = 1/4

方程(26)为合流超几何型方程，其解析解为：

Ψ_phys(a) = N e^(-αa²/2) U(1/4 - ν, 3/2, αa²)  (27)

式中 ν = γ²/(4α)，U 为第二类合流超几何函数（Tricomi函数），N 为归一化常数。

5.4 宇宙波函数的物理性质

1. 正则性：a→0（宇宙创生初期）时，合流超几何函数的渐进行为 U(⋯) ~ 常数，指数因子 e^(-αa²/2)→1，故 Ψ(a) 有限无奇性，符合宇宙创生波函数的正则边界条件。

2. 衰减性：a→∞（宇宙晚期）时，指数因子 e^(-αa²/2) 主导，使波函数指数衰减至零，符合宇宙演化的边界条件，反映时空流形的非紧致性。

3. 全局有效性：波函数定义于整个克鲁斯卡尔延拓流形，量子力学上支持四区时空结构的同时存在，为宇宙的量子起源提供自洽描述。

6 结论与讨论

本文基于标准王为民度规与固定白洞视界假设，完成了从经典方程求解到量子化的系统性研究，得到以下核心结论：

1. 推导得到含白洞引力修正项的弗里德曼方程，无需暗物质、暗能量即可解释宇宙膨胀动力学。白洞等效质量密度 ρ_wd 替代暗物质，等效压强 p_wd 提供暗能量的斥力效应，修正项的量纲在本文中已严格校验，确保公式自洽。

2. 不同空间曲率下的尺度因子演化具有明确解析解：平直宇宙呈现“减速-加速”两阶段膨胀；开放宇宙无限加速；闭合宇宙形成“创生-膨胀-坍缩”闭环，坍缩时间约为 4.5×10¹⁰ 年，与膨胀时间量级相当。

3. 完成类克鲁斯卡尔最大解析延拓，消除视界处坐标奇性，严格划分物质宇宙、黑洞内部、反物质宇宙与白洞内部的四区时空结构，揭示了时空的全局因果对称性。

4. 建立惠勒–德威特方程并得到宇宙波函数解析解，该波函数在创生点正则、晚期衰减，为量子宇宙学提供了自洽的波函数描述。惠勒–德威特方程的各项量纲经校验统一为能量量纲，保证了量子化方案的理论严谨性。

本研究的核心价值在于以几何第一性为基础，构建了无需未知物质/能量组分的量子宇宙学框架，其预言的四区时空结构、尺度因子演化规律可通过宇宙微波背景辐射、星系红移等观测数据验证。未来研究可进一步拓展至非球对称情形，探讨白洞引力与星系形成的关联，以及宇宙波函数的观测意义，为解决宇宙学核心难题提供新的理论视角。
