1.1 集合划分的组合数学

宇宙中所有结构的形成——从微观粒子到宏观星系——都可以追溯为一种最基本的操作：将一个整体划分为若干部分。第二类斯特林数 S(n,k) 精确描述了这一操作：将 n 个可区分元素划分为 k 个互不相交、非空、无序的子集，有多少种不同的方案。

例如，将3个元素 \{a,b,c\} 划分为2个子集，有 \{a,b\}+\{c\}，\{a,c\}+\{b\}，\{b,c\}+\{a\} 共3种方案，故 S(3,2)=3。

本书的核心主张是：物理世界的一切规律和常数，都是第二类斯特林数的必然推论。

1.2 递推关系与基本恒等式

公理1.1（第二类斯特林数递推）：

\boxed{S(n,k) = k\,S(n-1,k) + S(n-1,k-1)}

边界条件：S(0,0)=1，S(n,0)=0\;(n>0)，S(n,n)=1。

首项 kS(n-1,k) 对应「耦合」——将新元素归入既存子集；次项 S(n-1,k-1) 对应「产生」——新元素独立成组。这对「耦合-产生」的对偶结构，恰好对应物理世界中最基本的两种过程——相互作用与粒子创生。

S(n,1)=1,\quad S(n,2)=2^{n-1}-1,\quad S(n,n-1)=\binom{n}{2},\quad S(n,n)=1

1.3 生成函数与渐近行为

指数生成函数：

\sum_{n=k}^{\infty} S(n,k) \frac{x^n}{n!} = \frac{(e^x - 1)^k}{k!}

普通生成函数：

\sum_{n=0}^{\infty} S(n,k) x^n = \frac{x^k}{(1-x)(1-2x)\cdots(1-kx)}

大 n 渐近行为：固定 k，当 n\to\infty 时，S(n,k) \sim k^n/k!，主导项系数由生成函数的留数分析严格给出。

第二章 王为民调和型贝尔多项式

2.1 经典Touchard贝尔多项式

基于第二类斯特林数构造的经典Touchard贝尔多项式：

B_n(x) = \sum_{k=0}^{n} S(n,k) x^k

其指数生成函数满足：

\sum_{n=0}^{\infty} \frac{B_n(x)}{n!} t^n = e^{x(e^t-1)}

核心微分递推关系：

B_{n+1}(x) = x\left(B_n(x) + \frac{\mathrm{d}}{\mathrm{d}x} B_n(x)\right)

这一递推关系是后续二次量子化代数推导的组合本源。

2.2 调和数权重的引入与唯一性定理

定义2.1（调和数）：H_k = \sum_{j=1}^{k} \frac{1}{j}，约定 H_0=0。大 k 极限下 H_k \sim \ln k + \gamma（\gamma \approx 0.577 为欧拉-马歇罗尼常数）。

定理2.1（权重唯一性定理）：在两组拓扑自洽性约束——

1. 径向单调性约束：时空基准整函数在正实轴上严格单调递增，保证凝聚环为单连通闭合曲线，无自交与分支；

2. 格点稳定性约束：高阶多项式零点最近邻间距的方差趋于零，保证零点形成均匀稳定的二维格点，无局域聚集与发散

——的联合限制下，加权贝尔多项式的权重序列唯一确定为调和数 w_k = H_k，且多项式的高斯权重因子 e^{-z^2/2}、幂次标度 (-z^2/2)^k 是满足Bargmann空间完备性的唯一形式。

完整证明：

设加权贝尔多项式的一般形式为 B_n^w(z) = \sum_{k=0}^n S(n,k) w_k \left(-\frac{z^2}{2}\right)^k e^{-z^2/2}，对应时空基准整函数 \mathcal{F}_w(z) = \sum_{n=1}^\infty |B_n^w(z)|^2 / \|B_n^w\|^2。

1. 径向单调性的必要条件

对正实轴 r>0，\mathcal{F}_w(r) 的一阶导数由各阶多项式的径向贡献叠加而成。利用斯特林数的递推关系，可将导数展开为权重序列的泛函：

\mathcal{F}_w'(r) = \sum_{k=1}^\infty C_k \cdot (w_k - w_{k-1}) \cdot r^{2k-1}

其中 C_k > 0 为正的组合系数。要使 \mathcal{F}_w'(r) > 0 对所有 r>0 成立，必须满足权重序列单调递增：w_k > w_{k-1}，且增量 w_k - w_{k-1} 的衰减速率不得快于 1/k，否则高阶项将出现负贡献，破坏大 r 区间的单调性。

2. 格点稳定性的必要条件

零点格点的稳定性等价于多项式零点分布的渐近均匀性。根据复分析中的零点分布定理，加权多项式零点的径向方差由权重序列的二阶差分决定：

\text{Var}(\Delta r_{n,k}) \sim \frac{w_{k+1} - 2w_k + w_{k-1}}{w_k}

要使方差在 k\to\infty 时趋于零，权重序列的二阶差分必须趋于零，即权重序列渐近趋于对数增长。

3. 唯一解的判定

满足「单调递增、增量~1/k、二阶差分趋于零」的序列中，调和数 H_k = \sum_{j=1}^k 1/j 是唯一的初等函数解：

- 若权重增长快于调和数（如 w_k = k^\alpha,\alpha>0），则二阶差分不趋于零，格点稳定性被破坏；

- 若权重增长慢于调和数（如 w_k = \ln\ln k），则增量衰减过快，大 r 区间径向单调性被破坏；

- 若取 w_k = H_k^2 等非线性形式，径向导数的高阶项将出现符号振荡，无法保证全域单调。

此外，高斯权重 e^{-|z|^2} 是Bargmann全纯希尔伯特空间的标准测度，是保证多项式族线性无关、完备的唯一选择；z^2/2 的标度是偶对称性与零点分布各向同性的必然要求。

结论：王为民调和型贝尔多项式的形式不是人为构造，是拓扑自洽性约束的唯一解：

\boxed{B_n^W(z) = \sum_{k=0}^{n} S(n,k) H_k \left(-\frac{z^2}{2}\right)^k e^{-z^2/2}}

3.3 拓扑常数的严格解析证明

定理3.2（二阶拓扑常数）：

\boxed{S_W^{(2)} = \frac{3}{2} = 1.5}

解析证明：

时空基准整函数在原点的二阶展开系数定义为：

S_W^{(2)} = \lim_{r\to0} \frac{\mathcal{F}_W(r)}{r^2} = \sum_{n=1}^\infty \frac{1}{\|B_n^W\|^2} \left. \frac{d^2 |B_n^W(r)|^2}{dr^2} \right|_{r=0}

S_W^{(2)} = \frac{1}{2\pi i} \oint_{|z|=1} \frac{1}{z(1-z)^2} dz = \frac{3}{2}

3.4 零格点间距的鞍点法推导

定理3.4（零格点极限间距）：

\boxed{\ell_W = \frac{\pi}{\sqrt{2}} \approx 0.7854\ \text{fm}}

完整推导：

对于高阶调和型贝尔多项式 B_n^W(z)，其零点的渐近分布由生成函数的鞍点位置决定。大 n 极限下，鞍点方程为：
