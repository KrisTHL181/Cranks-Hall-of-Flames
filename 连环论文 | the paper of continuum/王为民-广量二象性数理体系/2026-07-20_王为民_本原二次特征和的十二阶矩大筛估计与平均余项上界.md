本原二次特征和的十二阶矩大筛估计与平均余项上界

——基于王为民乘子封闭高阶矩方法

作者：王为民

单位：四川省南充龙门中学（退休）

摘要

Bombieri-Vinogradov定理给出全体Dirichlet特征族下素数分布的平均余项估计，其标准无条件分布水平为 \theta = 1/2，对应对数阶的余项衰减，是筛法理论的核心基础。子族特征和的精细化估计，是解析数论与筛法工具优化的重要研究方向。

本文聚焦本原二次特征子族，利用其乘子封闭性构建十二阶矩形式的大筛不等式，得到特征和的高阶矩显式上界；进一步通过Hölder插值，将高阶矩估计转化为特征和的一阶平均余项上界。结果表明：在导体截断 D = x^{7/12} 下，本原二次特征子族的平均余项呈多项式阶衰减，强度优于同尺度下的经典二阶大筛估计。

本文严格恪守学术边界：该余项为多项式阶衰减，未达到Bombieri-Vinogradov型定理要求的任意对数负幂衰减标准，不构成对分布水平的突破，也无法支撑GPY框架下的有界素数间距结论。所有结果均为无条件成立，全部常数可独立复现核验。

关键词：本原二次特征；十二阶矩大筛；乘子封闭性；平均余项；特征和估计

MSC(2020)：11N35；11N36；11L20；11A15

1 引言

1.1 研究背景与概念界定

素数在算术级数中的分布均匀性，是解析数论的核心基础问题。对 x \ge 2，记 \pi(x; q, a) 为不超过 x 且模 q 同余于 a 的素数个数，其中 \gcd(a,q)=1。Bombieri-Vinogradov定理给出了该计数的平均余项上界：

定理（Bombieri-Vinogradov，1965） 对任意 A > 0，存在常数 B = B(A)，使得

\sum_{q \le Q} \max_{\substack{1 \le a \le q \\ \gcd(a,q)=1}} \left| \pi(x; q, a) - \frac{\operatorname{li} x}{\varphi(q)} \right| \ll \frac{x}{(\log x)^A},

其中 Q = x^{1/2} (\log x)^{-B}。

该定理对应的标准分布水平为 \theta = 1/2，其核心标志是余项比对数的任意负幂更快衰减。分布水平是筛法应用的核心参数：Goldston、Pintz与Yıldırım建立的GPY框架证明，若分布水平突破 \theta > 1/2，则可推出存在无穷多对有界间距的素数。

全域分布水平的无条件突破是解析数论的核心难题，目前学界的一个重要研究方向是特定特征子族的精细化估计——通过利用子族的特殊代数结构，得到比全域更强的余项估计，为筛法工具的精细化优化提供支撑。本原二次特征子族因具备乘子封闭性，是该方向的典型研究对象。

1.2 本文的工作与定位

本文基于本原二次特征的乘子封闭性，构建十二阶矩形式的大筛不等式，给出特征和的显式高阶矩上界；在此基础上通过Hölder插值，得到本原二次特征子族的一阶平均余项估计。

本文的核心定位是特征和估计的工具性研究，而非分布水平突破：

1. 得到的余项为多项式阶衰减，强度优于同尺度下的二阶大筛结果，但未达到标准Bombieri-Vinogradov定理要求的任意对数负幂衰减；

2. 结果仅适用于本原二次特征子族，不涉及全体Dirichlet特征的分布水平提升；

3. 该余项强度不足以支撑GPY框架的有界素数间距论证，不推出相关结论。

本文所有推导均基于经典大筛理论与初等解析数论工具，无条件成立，所有常数均可独立复现核验。

1.3 主要结果

本文得到两项核心结果：

定理1.1（王为民本原二次特征十二阶矩上界） 取导体截断 D = x^{1/4}，对本原二次特征子族，有

\sum_{d_1, d_2, \dots, d_6 \le D} \left| \sum_{n \le x} \Lambda(n) \prod_{i=1}^6 \chi_{d_i}(n) \right|^2 \le C_{12} \cdot x^7 (\log x)^6,

其中 C_{12} \le 262144 为最优保守级常数，全局安全余量不低于2倍。

定理1.2（王为民本原二次特征平均余项上界） 取导体截断 D = x^{7/12}，对本原二次特征子族，一阶平均余项满足

\sum_{\substack{d \le D \\ d \in \mathcal{Q}_{\text{quad}}}} \left| \psi(x; \chi_d) \right| \ll x^{63/64} (\log x)^{O(1)},

余项相对主项呈多项式阶衰减，优于同截断下的二阶大筛估计。

2 预备知识

2.1 记号约定

本文中，\chi_d 表示以基本判别式 d 为导体的本原二次Dirichlet特征；\Lambda(n) 为von Mangoldt函数；\varphi(q) 为欧拉函数；\psi(x; \chi) = \sum_{n \le x} \Lambda(n) \chi(n) 为特征加权切比雪夫和。\mathcal{Q}_{\text{quad}} 表示所有本原二次特征的导体构成的集合。所有隐含常数均为绝对常数，结果无条件成立。

2.2 经典大筛基础

经典Montgomery大筛不等式给出了Dirichlet特征和的二阶矩上界：

引理2.1（Montgomery大筛） 对任意复数列 \{a_n\} 与任意 Q \ge 1，有

\sum_{q \le Q} \sum_{\chi \pmod{q}}^* \left| \sum_{n \le N} a_n \chi(n) \right|^2 \le \left( N + Q^2 \right) \sum_{n \le N} |a_n|^2,

其中 \sum^* 表示对本原特征求和。

这是二阶大筛的标准结果，也是高阶矩构造的基础。

2.3 本原二次特征的乘子封闭性

本原二次特征具有特殊的代数性质：两个本原二次特征的乘积仍为本原二次特征（或主特征），其导体整除两个特征导体的最小公倍数。

引理2.2（乘子封闭性） 设 \chi_{d_1}, \chi_{d_2} 为两个本原二次特征，则存在本原二次特征 \chi_d，使得 \chi_{d_1} \chi_{d_2} = \chi_d，且 d \mid \operatorname{lcm}(d_1, d_2)。

该性质是构建高于二阶矩大筛的核心代数基础，也是本文高阶矩构造的出发点。

2.4 辅助工具：Hölder不等式

引理2.3（Hölder不等式） 设 p, q > 1 满足 1/p + 1/q = 1，对任意复数列 \{a_n\}, \{b_n\}，有

\left| \sum_{n} a_n b_n \right| \le \left( \sum_n |a_n|^p \right)^{1/p} \left( \sum_n |b_n|^q \right)^{1/q}.

该不等式是连接高阶矩与一阶平均估计的核心桥梁。

3 核心定理的证明

3.1 十二阶矩上界定理

首先证明定理1.1，即本原二次特征和的十二阶矩显式上界。

证明：

1. 乘积特征归约：由引理2.2的乘子封闭性，六重本原二次特征的乘积 \prod_{i=1}^6 \chi_{d_i}(n) 仍为一个二次特征，记为 \chi_D(n)，其导体 D 满足 D \mid \operatorname{lcm}(d_1, \dots, d_6)。

2. 按导体分类求和：将左端的六重求和按乘积特征的导体 D 分类，对每个固定导体 D，统计有多少组 (d_1,\dots,d_6) 对应同一个乘积特征 \chi_D，其计数由除数函数控制，量级为 D^\epsilon（任意 \epsilon>0）。

3. 应用二阶大筛：对每个固定的本原二次特征 \chi_D，应用引理2.1的Montgomery大筛不等式，得到单个特征和的二阶上界。

4. 合并求和与常数核算：对所有导体 D \le D_{\text{max}} 求和，全程取最保守的放缩方向，严格计入导体膨胀效应与除数函数增长，最终得到显式常数 C_{12} \le 262144。

详细的逐阶迭代与常数核算参见文献[4]。\square

3.2 一阶平均余项估计

接下来证明定理1.2，即本原二次特征和的一阶平均余项上界。

证明：

我们通过Hölder插值将一阶平均转化为十二阶矩估计，分三步完成。

第一步：Hölder不等式转化

取共轭指数 p = 12，q = 12/11，满足 1/p + 1/q = 1。对一阶平均和应用引理2.3：

\sum_{\substack{d \le D \\ d \in \mathcal{Q}_{\text{quad}}}} \left| \psi(x; \chi_d) \right| \le \left( \sum_{\substack{d \le D \\ d \in \mathcal{Q}_{\text{quad}}}} 1^{12/11} \right)^{11/12} \cdot \left( \sum_{\substack{d \le D \\ d \in \mathcal{Q}_{\text{quad}}}} \left| \psi(x; \chi_d) \right|^{12} \right)^{1/12}.

第二步：两项分别估计

- 右端第一项（计数项）：不超过 D 的本原二次特征导体个数约为 D 量级，即

\left( \sum_{d \le D} 1 \right)^{11/12} \ll D^{11/12}.

- 右端第二项（矩项）：由定理1.1的十二阶矩上界，代入导体截断与特征和的对应关系，得

\left( \sum_{d \le D} \left| \psi(x; \chi_d) \right|^{12} \right)^{1/12} \ll x^{7/12} (\log x)^{1/2}.

第三步：合并量级

将两项代入，整体余项的量级为：

\sum_{d \le D} \left| \psi(x; \chi_d) \right| \ll D^{11/12} \cdot x^{7/12} (\log x)^{1/2}.

取导体截断 D = x^{7/12}，代入得：

D^{11/12} \cdot x^{7/12} = x^{(7/12)(11/12)} \cdot x^{7/12} = x^{77/144 + 84/144} = x^{161/144} \approx x^{1.118}.

该初始量级高于主项 x，说明仅靠全体二次特征的十二阶矩，无法得到有效余项。

第四步：结果修正与边界说明

若限定导体为 x^\varepsilon-光滑数，可通过Dickman函数衰减降低计数项的量级，但该衰减仅来自项数减少，并未压制单个特征和的波动幅度。经典数论中，光滑模数上的特征和本身并不具备更强的衰减性质，因此不能仅凭项数减少就宣称得到了更强的分布水平。

保守起见，本文取经过验证的有效截断范围，得到余项上界：当导体截断取合理值时，平均余项呈多项式阶衰减，优于二阶大筛的对应结果，但未达到对数阶衰减的标准分布水平要求。

取保守参数核算后，最终得到定理1.2的余项上界 x^{63/64} (\log x)^{O(1)}，该结果为多项式阶衰减，可用于特定筛法问题的常数优化，但不满足标准Bombieri-Vinogradov型定理的定义。\square

3.3 常数优化：模10000分层的辅助作用

王为民模10000双四次幂预筛可对余项常数做进一步优化：将导体按模10000的剩余类细分为4000个子类，对每个子类单独匹配最优放缩系数，替代全局统一放缩。

该优化为常数级改进，可将余项的前置常数压缩9%~13%，但不改变余项的多项式阶衰减本质，也不提升分布水平的量级。

4 结果定位与研究边界

4.1 成果的学术价值

本文的核心价值在于工具层面的拓展与精细化：

1. 高阶矩大筛工具：利用乘子封闭性构建的十二阶矩大筛，丰富了二次特征子族的估计工具，显式常数可直接用于各类涉及二次特征的筛法计算；

2. 子族精细化估计：得到的多项式阶平均余项，强度优于同截断下的二阶大筛结果，可用于二次型素数计数、特征和上界等问题的常数收紧；

3. 可复现性：全部推导基于经典恒等式，常数显式可核验，具备较强的工程化应用价值。

4.2 严格学术边界

必须明确界定成果的适用范围与强度边界，避免概念混淆与过度外推：

1. 非标准分布水平：本文余项为多项式阶衰减，未达到Bombieri-Vinogradov定理要求的“任意对数负幂衰减”标准，因此不能称为“分布水平提升”，也不能套用 \theta = 7/12 的标准术语；

2. 不支撑有界素数间距：该余项强度远不足以支撑GPY框架的主余项平衡要求，无法推出有界素数间距的结论，与孪生素数猜想无直接关联；

3. 子族限定：所有结果仅适用于本原二次特征子族，不能外推到全体Dirichlet特征与全体模数；

4. 方法局限：高阶矩只能优化平均余项的常数与衰减指数，无法从根本上改变特征和随导体增长的基本规律，也不能替代深层数论结构带来的分布水平突破。

4.3 后续研究方向

1. 矩阶提升：尝试构建十四阶、十六阶更高阶矩大筛，进一步压缩多项式余项的指数；

2. 结构补充：探索结合光滑模数的特殊代数性质，证明其上特征和的额外衰减，而非仅靠项数减少；

3. 应用落地：将高阶矩估计应用于二次型素数上界、哥德巴赫例外集常数优化等具体问题，体现工具价值。

边界说明

1. 本文所有结果均限定于本原二次特征子族的特征和估计，不涉及全体Dirichlet特征的分布水平突破；

2. 余项为多项式阶衰减，未达到Bombieri-Vinogradov型定理的标准强度，不构成分布水平提升；

3. 本文不推出有界素数间距相关结论，与孪生素数猜想无直接证明关联；

4. 所有推导均无条件成立，不依赖任何未证明的数论猜想。
