王为民粒子表象中的自由标量场

作者：王为民

单位：四川省南充龙门中学（退休）

摘要

本文基于对数谐振子的二次量子化理论，构造量子场论中自由实标量场的王为民粒子表象。标准平面波展开将场的每个傅里叶模式视为独立标准谐振子，而王为民场展开则将每个径向模式视为独立对数谐振子。两种展开通过正则对易关系的同构性统一在同一量子场论框架下：产生/湮灭算符的代数结构不变，但Fock空间的基底——单粒子波函数——不同。核心结论为：王为民表象下的Feynman传播子在小距离极限下天然携带对数修正项，该修正来源于单粒子基函数的固有调和数结构，而非相互作用圈图贡献；这为处理红外发散提供了无需人工截断的新途径。本文不引入任何超出标准正则量子化框架的假设，所构造的表象与平面波展开严格等价，差异仅在于基底的选择。

关键词：王为民调和型贝尔多项式；二次量子化；自由标量场；王为民表象；红外发散正规化

一、标准自由标量场的正则量子化

1.1 经典场与正则对易关系

自由实标量场 \phi(x) 的拉格朗日密度为：

\mathcal{L} = \frac{1}{2} \partial_\mu \phi \partial^\mu \phi - \frac{1}{2} m^2 \phi^2.

共轭动量密度 \pi(x) = \partial \mathcal{L} / \partial \dot{\phi}(x) = \dot{\phi}(x)。实施正则量子化，将场 \phi 和共轭动量 \pi 提升为算符，满足等时正则对易关系：

[\hat{\phi}(\mathbf{x}, t), \hat{\pi}(\mathbf{y}, t)] = i \delta^{(3)}(\mathbf{x} - \mathbf{y}),

[\hat{\phi}(\mathbf{x}, t), \hat{\phi}(\mathbf{y}, t)] = [\hat{\pi}(\mathbf{x}, t), \hat{\pi}(\mathbf{y}, t)] = 0.

1.2 平面波展开

为满足Klein-Gordon方程 (\Box + m^2) \hat{\phi}(x) = 0，将场算符按平面波（KG方程的基本解）展开：

\hat{\phi}(x) = \int \frac{d^3k}{(2\pi)^3} \frac{1}{\sqrt{2\omega_k}} \left( \hat{a}_{\mathbf{k}} e^{ikx} + \hat{a}_{\mathbf{k}}^\dagger e^{-ikx} \right),

其中 kx = \omega_k t - \mathbf{k} \cdot \mathbf{x}，\omega_k = \sqrt{\mathbf{k}^2 + m^2}。

产生/湮灭算符的对易关系由场的正则对易关系导出：

[\hat{a}_{\mathbf{k}}, \hat{a}_{\mathbf{k}'}^\dagger] = (2\pi)^3 \delta^{(3)}(\mathbf{k} - \mathbf{k}'),

[\hat{a}_{\mathbf{k}}, \hat{a}_{\mathbf{k}'}] = [\hat{a}_{\mathbf{k}}^\dagger, \hat{a}_{\mathbf{k}'}^\dagger] = 0.

1.3 平面波基的物理本质

平面波 e^{i\mathbf{k}\cdot\mathbf{x}} 是动量算符 \hat{\mathbf{p}} = -i\nabla 的本征函数。每个傅里叶模式 \mathbf{k} 等价于一个独立的标准谐振子，其基态波函数为高斯型，定态波函数由厄米多项式给出。

当量子场论的红外区域存在对数型修正时——例如共形场论的反常维数[2]、非相对论Efimov效应[4]中的对数周期行为——平面波展开需要计算无穷多圈图才能捕捉对数结构，导致微扰展开效率低下。这并非平面波基的缺陷，而是其数学结构决定的必然代价。

二、王为民粒子表象的构造

2.1 出发点：表象变换的代数不变性

量子场论中，不同表象之间的变换是幺正变换。一个表象由Fock空间的一组基底完全确定。基底的选择不影响产生/湮灭算符的对易代数，但决定单粒子波函数的具体形式。

标准场论选取平面波作为单粒子波函数。本文选取对数谐振子的定态波函数——即王为民多项式展开——作为新的单粒子波函数基底。这一选择的物理动机是：当场论存在红外对数结构时，新基底的函数空间天然包含对数修正，有望简化红外区域的描述。

2.2 对数谐振子的定态波函数回顾

在前文[1]中，我们求解了对数谐振子的定态薛定谔方程：

-\frac{1}{2} \frac{d^2\psi}{dx^2} + \left( \frac{1}{2} \omega^2 x^2 + \lambda \ln x \right) \psi = E \psi.

其束缚态波函数在王为民多项式基下展开为：

\psi_n^W(x) = \sum_{k=1}^N c_k^{(n)} Q_k^W(\omega x).

这些波函数构成半轴 x \in (0, \infty) 上的完备正交系，且满足边界条件 \psi_n^W(0) = 0。利用王为民多项式完备基性质，任意径向光滑函数均可做有限项高精度展开。

2.3 王为民模的产生/湮灭算符

对每个固定的三维动量 \mathbf{k}，引入王为民模的湮灭算符 \hat{b}_{\mathbf{k},n} 和产生算符 \hat{b}_{\mathbf{k},n}^\dagger，其中 n = 1, 2, \dots 标记对数谐振子的径向量子数。

这些算符满足标准对易关系：

[\hat{b}_{\mathbf{k},n}, \hat{b}_{\mathbf{k}',m}^\dagger] = (2\pi)^3 \delta^{(3)}(\mathbf{k} - \mathbf{k}') \delta_{nm},

[\hat{b}_{\mathbf{k},n}, \hat{b}_{\mathbf{k}',m}] = [\hat{b}_{\mathbf{k},n}^\dagger, \hat{b}_{\mathbf{k}',m}^\dagger] = 0.

这一对易关系与标准平面波展开中的产生/湮灭算符完全同构。

关键点：算符 \hat{b}_{\mathbf{k},n} 与标准算符 \hat{a}_{\mathbf{k}} 的对易结构相同，但物理含义不同——\hat{a}_{\mathbf{k}} 湮灭一个动量为 \mathbf{k} 的平面波量子，\hat{b}_{\mathbf{k},n} 湮灭一个动量为 \mathbf{k}、径向量子数为 n 的王为民粒子。

2.4 王为民表象下的场展开

将标准平面波基替换为王为民径向基，自由实标量场展开为：

\boxed{ \hat{\phi}_W(x) = \int \frac{d^3k}{(2\pi)^3} \frac{1}{\sqrt{2\omega_k}} \sum_{n=1}^\infty \left( \hat{b}_{\mathbf{k},n} Q_n^W(\omega_k r) + \hat{b}_{\mathbf{k},n}^\dagger Q_n^W(\omega_k r) \right) }

其中 r = |\mathbf{x}|，Q_n^W(\omega_k r) 为王为民多项式在径向坐标上的值，时间相位已吸收至算符定义中。

几点重要说明：

1. 径向基函数的自变量：王为民多项式以 \omega_k r 为自变量。这借鉴了标准场论中无量纲化处理——径向坐标乘以能标因子。当 \omega_k 从红外（k \to 0）到紫外（k \to \infty）变化时，基函数自动缩放。

2. 角度自由度：本构造选取了球对称的单粒子波函数。对于更一般的非球对称构型，可通过引入球谐函数 Y_{\ell m}(\theta,\phi) 实现张量积展开。本文聚焦于球对称情形，以清晰展示王为民基的核心结构。

3. 完备性：无穷求和 \sum_{n=1}^\infty 对应于径向对数谐振子束缚态的完备集，由王为民多项式的完备性保证。

4. 幺正等价性：场算符 \hat{\phi}_W(x) 与标准场算符 \hat{\phi}(x) 通过 Bogoliubov 型变换相联系。两者描述同一量子场论，仅基底不同——这是表象变换的数学本质。

三、王为民表象与平面波展开的等价性

3.1 代数同构的数学基础

两种表象的等价性根源在于产生/湮灭算符对易关系的同构：

[\hat{a}_{\mathbf{k}}, \hat{a}_{\mathbf{k}'}^\dagger] = [\hat{b}_{\mathbf{k},n}, \hat{b}_{\mathbf{k}',m}^\dagger] = (2\pi)^3 \delta^{(3)}(\mathbf{k} - \mathbf{k}') \delta_{nm}.

这一代数同构直接决定了：

· Fock空间的构造方式相同（真空 + 产生算符逐次作用）

· Wick定理的代数形式不变

· LSZ约化公式的算符结构不变

· S矩阵的幺正性不变

3.2 基底变换与Bogoliubov变换

两种展开之间的联系可表为基底变换。将平面波 e^{i\mathbf{k}\cdot\mathbf{x}} 按王为民径向基和球谐函数展开：

e^{i\mathbf{k}\cdot\mathbf{x}} = 4\pi \sum_{\ell=0}^\infty \sum_{m=-\ell}^\ell i^\ell j_\ell(kr) Y_{\ell m}^*(\hat{\mathbf{k}}) Y_{\ell m}(\hat{\mathbf{x}}).

再将球贝塞尔函数 j_\ell(kr) 按王为民多项式展开（利用完备性定理）：

j_\ell(kr) = \sum_{n=1}^\infty c_{\ell n}(k) Q_n^W(\omega_k r).

由此可得两组产生/湮灭算符之间的线性变换（Bogoliubov型），且该变换是幺正的。

3.3 等价性的物理含义

王为民表象与平面波展开是同一个量子场论的两种等价描述。所有标准场论的结论——S矩阵的幺正性、Wick定理、LSZ约化公式——在王为民表象中同样成立。差异仅在于：不同表象暴露或隐藏理论的不同结构特征。

· 标准表象的优势：平移不变性显式，动量守恒在每圈图自动满足

· 王为民表象的优势：对数红外结构显式编码在单粒子波函数中，无需通过高阶圈图逐次“生成”对数行为

四、传播子的对数修正

4.1 王为民表象下的Feynman传播子

王为民表象下的Feynman传播子定义为：

D_W(x-y) = \langle 0_W | T \left\{ \hat{\phi}_W(x) \hat{\phi}_W(y) \right\} | 0_W \rangle,

其中 |0_W\rangle 为王为民真空态：\hat{b}_{\mathbf{k},n} |0_W\rangle = 0，对所有 \mathbf{k}, n。

将场展开式代入，利用对易关系和真空条件，得：

D_W(x-y) = \int \frac{d^3k}{(2\pi)^3} \frac{1}{2\omega_k} \sum_{n=1}^\infty Q_n^W(\omega_k r_x) Q_n^W(\omega_k r_y) e^{-i\omega_k |t_x - t_y|}.

4.2 对数修正的解析来源

王为民多项式在原点附近的渐近行为携带对数结构。调和数 H_k \sim \ln k + \gamma 导致 Q_n^W(\omega_k r) 在小 r 极限下包含 \ln(\omega_k r) 型修正。传播子在红外小距离极限下自动生成：

D_W(x-y) \sim D_{\text{standard}}(x-y) + \lambda \ln(\mu |x-y|) + \cdots,

其中 \mu 为重整化能标，\lambda 为对数耦合强度（由王为民展开系数自然确定）。

该对数修正来源于单粒子基函数的固有调和数结构，而非相互作用圈图贡献。这是王为民表象的核心优势所在。

关键区别：标准场论中，这一对数修正需要在微扰论的某个圈阶才出现。而在王为民表象中，对数修正出现在自由场传播子层面——因为单粒子波函数本身携带了调和数的对数结构。这是基底选择带来的计算优势，而非新物理。

4.3 与圈图展开中的对数修正之关系

标准微扰论中，传播子的对数修正来自圈图积分——费曼图的内线动量积分在对数能标区间产生 \ln(\Lambda/\mu) 型项。王为民表象中的对数修正与这些圈图修正并非互斥，而是互补：一个通过表象选择在低阶获得，另一个通过微扰展开在更高阶生成。当两种方法计算同一物理量时，对数修正的结构必须一致——这为检验王为民表象的物理自洽性提供了定量判据。

五、与标准场论的对比总结

六、结论

本文构造了量子场论中自由实标量场的王为民粒子表象。核心结论为：

1. 代数同构保证等价性：王为民模产生/湮灭算符与标准平面波模算符的对易关系完全同构，保证表象变换的幺正性和Fock空间的等价性。两种表象描述同一量子场论。

2. 基底差异引入对数结构：王为民表象的单粒子波函数是王为民调和型贝尔多项式 Q_n^W(\omega_k r)。调和数的渐近行为 H_k \sim \ln k + \gamma 使基函数天然携带对数结构。

3. 传播子的对数修正：自由场Feynman传播子在王为民表象中于小距离极限下自动呈现对数修正。这一修正出现在自由场层面，来源于单粒子基函数的固有结构，为处理红外发散提供了无需圈图累加和人工截断的新途径。

4. 表象物理互补性：王为民表象与标准平面波展开并非替代关系，而是互补关系——就像坐标表象与动量表象在量子力学中的关系。不同表象暴露理论的不同结构特征，适配不同的物理问题。
