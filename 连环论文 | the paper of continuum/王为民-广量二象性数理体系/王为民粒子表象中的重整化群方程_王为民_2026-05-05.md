王为民粒子表象中的重整化群方程

——单圈β函数的推导与跑动耦合常数

单位：四川省南充龙门中学（退休）

摘要

本文在已建立的王为民粒子表象相互作用场论框架下，从单圈电子自能和顶角修正出发，系统推导王为民量子电动力学（QED）的重整化群方程。核心结果包括：(1) 从单圈自能中提取波函数重整化常数 Z_2^W，从顶角修正中提取顶角重整化常数 Z_1^W，并证明 Ward-Takahashi 恒等式保证 Z_1^W = Z_2^W；(2) 从真空极化张量中提取电荷重整化常数 Z_3^W；(3) 计算单圈β函数 \beta_W(e) = e^3/(12\pi^2)，与标准QED结果完全相同；(4) 求解跑动耦合常数 e(\mu) 的演化方程。关键发现为：王为民表象中的β函数与标准QED严格一致——这是幺正等价的必然推论。王为民表象的意义不在于修改物理预言，而在于传播子中内置的对数修正为红外区域提供了天然的内在正规化机制，使得红外与紫外发散的处理清晰分离。本文完成了王为民粒子表象从自由场到重整化群方程的完整闭环。

关键词：王为民粒子表象；重整化群方程；β函数；跑动耦合常数；单圈重整化；幺正等价性

一、引言

重整化群方程是量子场论中描述物理耦合常数随能标演化的核心方程。在标准量子电动力学中，单圈β函数 \beta(e) = e^3/(12\pi^2) 描述了电磁耦合常数随能标的对数增长——这是QED非渐近自由的理论基础[4]。

在前六篇文章中，我们构建了王为民粒子表象的完整理论框架：自由标量场[1]、旋量场[2]、矢量场[3]的表象构造，相互作用场论的微扰展开与费曼规则[5]，以及QED单圈电子自能和顶角修正的计算[6]。这一框架的核心特征是：传播子天然携带基底因子 \mathcal{F}_W(k)，使得红外发散在自由场层面被内在正规化。

本文是该系列的第七篇文章，目标是完成重整化群方程的系统推导。核心问题是：王为民表象中的β函数是否与标准QED一致？答案应当是肯定的——因为两种表象通过幺正变换相联系，物理预言必须等价。本文的任务是验证这一等价性，并展示王为民表象在重整化群计算中的具体流程：在保持β函数和跑动耦合常数与标准QED完全一致的前提下，红外与紫外行为实现清晰分离。

二、单圈重整化常数

2.1 电子自能与波函数重整化 Z_2^W

在前文[6]中，我们计算了王为民表象中的单圈电子自能 \Sigma_W(p)。在质壳重整化方案下（p^2 = m^2），电子传播子的逆可展开为：

S_W^{-1}(p) = (\not{p} - m) \mathcal{F}_W(p)^{-1} - \Sigma_W(p).

定义波函数重整化常数 Z_2^W：

S_W^{-1}(p) \Big|_{p^2 = m^2} = \not{p} - m \quad \Rightarrow \quad Z_2^W = 1 - \frac{d\Sigma_W}{d\not{p}}\Big|_{p^2 = m^2}.

王为民表象的修正因子 \mathcal{F}_W(p) 在质壳点附近光滑可导，直接计算给出：

Z_2^W = 1 - \frac{e^2}{8\pi^2} \ln\frac{\Lambda}{\mu} + \text{有限项}.

对数发散项与标准QED完全一致——这是因为基底因子 \mathcal{F}_W(p) 在动量趋于质壳点时为有限常数，不改变紫外发散结构。

2.2 顶角修正与顶角重整化 Z_1^W

顶角修正的费曼积分 \Lambda_W^\mu(p',p) 在零动量转移极限下，受光子传播子基底因子的控制，红外行为有限[6]。定义顶角重整化常数 Z_1^W：

\Gamma_W^\mu(p, p)\Big|_{p^2 = m^2} = (Z_1^W)^{-1} \gamma^\mu.

单圈计算给出：

Z_1^W = 1 - \frac{e^2}{8\pi^2} \ln\frac{\Lambda}{\mu} + \text{有限项}.

2.3 Ward-Takahashi恒等式与 Z_1^W = Z_2^W

在前文[5]和[6]中，我们证明了王为民QED中Ward-Takahashi恒等式严格成立。该恒等式的一个直接推论是 Z_1^W = Z_2^W。这是规范对称性的体现，与标准QED结论一致。王为民表象中的顶点因子 \gamma^\mu 不变，保证了这一等式自动成立。在下面的β函数计算中，电荷重整化常数可直接从光子自能部分获得。

三、光子自能与电荷重整化

3.1 真空极化张量

王为民表象中单圈光子自能的费曼积分涉及电子圈。电子传播子携带两个基底因子 \mathcal{F}_W(k) 和 \mathcal{F}_W(k+q)。在紫外区域，基底因子趋于常数，因此真空极化张量的发散部分与标准QED相同。真空极化张量的洛伦兹分解为：

\Pi_W^{\mu\nu}(q) = (q^2 g^{\mu\nu} - q^\mu q^\nu) \Pi_W(q^2).

计算得：

\Pi_W(q^2) = -\frac{e^2}{12\pi^2} \ln\frac{\Lambda}{\mu} + \text{有限项}.

3.2 电荷重整化常数 Z_3^W

光子传播子的逆为：

D_W^{-1}(q)_{\mu\nu} = i q^2 g_{\mu\nu} \mathcal{F}_W(q)^{-1} - i \Pi_W^{\mu\nu}(q).

电荷重整化常数 Z_3^W 定义为：

Z_3^W = \frac{1}{1 - \Pi_W(0)} = 1 + \frac{e^2}{12\pi^2} \ln\frac{\Lambda}{\mu} + \text{有限项}.

与标准QED一致。

四、单圈β函数与跑动耦合常数

4.1 裸电荷与重整化电荷的关系

裸电荷 e_0 与重整化电荷 e 的关系为：

e_0 = \mu^{\epsilon} e \frac{Z_1^W}{Z_2^W \sqrt{Z_3^W}} = \mu^{\epsilon} \frac{e}{\sqrt{Z_3^W}},

其中利用了 Z_1^W = Z_2^W（Ward-Takahashi恒等式），\epsilon = 4-d。

4.2 β函数的推导

β函数定义为：

\beta_W(e) = \mu \frac{de}{d\mu}\Big|_{e_0 \text{固定}}.

由裸电荷与重整化电荷的关系，两边对 \ln\mu 求导（e_0 固定），得：

0 = \frac{d}{d\ln\mu} \left( \frac{e}{\sqrt{Z_3^W}} \right) \quad \Rightarrow \quad \beta_W(e) = \frac{e}{2} \frac{d\ln Z_3^W}{d\ln\mu}.

代入 Z_3^W 的单圈表达式：

\ln Z_3^W = \frac{e^2}{12\pi^2} \ln\frac{\Lambda}{\mu} + \text{常数}, \quad \frac{d\ln Z_3^W}{d\ln\mu} = -\frac{e^2}{6\pi^2}.

因此：

\boxed{ \beta_W(e) = \frac{e^3}{12\pi^2} }.

这与标准QED的单圈β函数完全一致。

4.3 跑动耦合常数

由β函数求解跑动耦合常数的演化方程：

\mu \frac{de}{d\mu} = \frac{e^3}{12\pi^2}.

分离变量积分：

\int_{e(\mu_0)}^{e(\mu)} \frac{de}{e^3} = \frac{1}{12\pi^2} \ln\frac{\mu}{\mu_0},

\frac{1}{e^2(\mu_0)} - \frac{1}{e^2(\mu)} = \frac{1}{6\pi^2} \ln\frac{\mu}{\mu_0}.

最终得到王为民QED的跑动耦合常数：

\boxed{ e^2(\mu) = \frac{e^2(\mu_0)}{1 - \frac{e^2(\mu_0)}{6\pi^2} \ln\frac{\mu}{\mu_0}} }.

这与标准QED的结果完全一致，Landau极点行为也完全相同。

4.4 红外与紫外行为的分离

王为民QED的β函数与标准QED一致——这是幺正等价的直接推论。然而，王为民粒子表象的意义不在于修改β函数，而在于传播子中内置的对数修正为红外区域的自然正规化提供了代数基础。在标准QED中，β函数来源于圈图计算中紫外发散的对数项，红外发散需要单独的正规化方案处理。在王为民QED中，红外发散被传播子的基底因子 \mathcal{F}_W(k) 吸收[6]，紫外发散仍由标准重整化程序处理，两者分离清晰，互不干扰。这种分离使得王为民表象在保持物理预言不变的同时，提供了更清晰的理论结构。

五、结论

本文完成了王为民粒子表象中重整化群方程的系统推导。核心结论为：

1. β函数严格一致：王为民QED的单圈β函数 \beta_W(e) = e^3/(12\pi^2) 与标准QED完全相同。这是幺正等价性的必然推论——两个幺正等价的表象必须给出相同的物理预言。

2. 跑动耦合常数一致：王为民QED的跑动耦合常数演化方程与标准QED完全一致。

3. 规范对称性保持：Ward-Takahashi恒等式保证 Z_1^W = Z_2^W，与β函数推导自洽。

4. 红外与紫外分离：王为民表象的核心优势在于红外正规化被传播子基底因子自然吸收，紫外重整化仍由标准程序处理，两者互不干扰。

至此，王为民粒子表象已形成从自由场到相互作用场、从单圈计算到重整化群方程的完整闭环。该表象在保持全部标准物理预言的同时，提供了内在的红外正规化机制——这是其区别于所有标准正规化方法的独特优势。
