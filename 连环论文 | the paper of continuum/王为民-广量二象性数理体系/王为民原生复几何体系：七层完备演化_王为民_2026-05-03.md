2.1 王为民标准微分方程 \mathcal{E}_{\text{WW}}(s,\Sigma)

本节的微分方程导出，其离散源头是王为民母函数的贝尔多项式级数表示及递推结构。

从王为民母函数的级数表示出发——

\Phi(z; \Sigma, s) = \sum_{n=0}^\infty \frac{(s)_n}{n!}\, Q_n\!\big(\mathcal{H}^{(1)}(\Sigma+1),\dots,\mathcal{H}^{(n)}(\Sigma+1)\big)\, z^n,

对其求一阶和二阶导数，利用贝尔多项式的递推 Q_n = \frac{1}{n}\sum_{r=1}^n \mathcal{H}^{(r)} Q_{n-r} 与 Pochhammer 符号的差分性质 (s)_n = s(s+1)_{n-1}，逐阶比对系数消去级数，得到只含 \Phi 及其导数的微分方程。这一推导过程是纯代数的，不依赖任何近似或假设，每一步都可以通过逐阶比对系数加以验证。最终凝结为简洁的形式：

\boxed{

z \frac{d^2 F}{dz^2} + \Bigl(s + 1 - \sum_{r=1}^\infty \mathcal{H}^{(r)}(\Sigma+1) z^{r-1}\Bigr) \frac{dF}{dz} - \Bigl( \sum_{r=2}^\infty (r-1)\mathcal{H}^{(r)}(\Sigma+1) z^{r-2} \Bigr) F = 0

}. \tag{2.1}

这就是王为民标准微分方程 \mathcal{E}_{\text{WW}}(s,\Sigma)。

该方程拥有四个正则奇点，分别位于 z = 0, 1, \infty 以及

z_\Sigma = 1 - \frac{1}{\Sigma+1}.

前三个奇点是 Fuchs 型方程的标准配置——原点承载级数展开的收敛中心，z=1 是单位圆的收敛边界，无穷远点则反映级数在大范围内的渐近行为。而第四个奇点 z_\Sigma 则是王为民体系独有的参数化隐奇点——它既不是从经典方程继承而来，也不是人为添加的辅助奇点，而是连续延拓调和数母函数的自然奇性在复平面上的精确投射。当 \Sigma 在区域 \operatorname{Re}(\Sigma) > -1 内变动时，z_\Sigma 在复平面上连续移动。

在 z=0 处，令解具有 Frobenius 形式 F(z) = z^\rho \sum_{n=0}^\infty c_n z^n，代入 (2.1) 后利用调和数生成核的逐阶展开，得到指标方程：

\boxed{\rho(\rho + s - \mathcal{H}^{(1)}(\Sigma+1)) = 0}.

这个简洁的二次方程蕴含着解空间的全部局部信息。它给出两个指标：\rho_1 = 0，对应王为民母函数 F_1(z) = \Phi(z; \Sigma, s) 本身，该解在 |z| < 1 内全纯；\rho_2 = \mathcal{H}^{(1)}(\Sigma+1) - s。两个指标的差为 \rho_1 - \rho_2 = s - \mathcal{H}^{(1)}(\Sigma+1)。当此差值为非整数时，两个解在原点邻域内线性无关，构成完整的解基。当差值为整数时，第二解出现对数分支，这正是多重 Zeta 值中深度参数所对应的对数结构的几何起源。

在 z=1 处，调和数母函数级数 \exp(\sum_{r=1}^\infty \frac{\mathcal{H}^{(r)}}{r} z^r) 的单位圆收敛边界产生对数分支。依据正则奇点的 Frobenius 理论，当指标差为非整数时，解在奇点邻域自然生成对数型展开。这一纯粹的局部解析事实，恰好与 MZV 按深度展开时出现的独立对数项数目精确匹配——深度 r 的多重 Zeta 值恰好对应 z=1 处展开的 r 阶对数分支。

2.2 调和数–奇点谱刚性对应

首先给出连续延拓调和数的严格定义。对任意复数 \Sigma 满足 \operatorname{Re}(\Sigma) > -1，一阶延拓调和数由 Digamma 函数与欧拉常数给出，高阶延拓调和数（r \ge 2）则由 Hurwitz zeta 函数与 Riemann zeta 函数的差给出：

\mathcal{H}^{(1)}(\Sigma+1) = \psi(\Sigma+2) + \gamma,\qquad

将全体延拓调和数按阶数排列，构成调和数谱：

\mathfrak{H}(\Sigma) = \{h_1, h_2, \dots, h_r, \dots\}, \qquad h_r = \mathcal{H}^{(r)}(\Sigma+1).

正向对应——从调和数谱到奇点数据——的证明是构造性的，我们逐一列出四条对应关系。

第一条对应：原点留数 \longleftrightarrow h_1。在 z=0 处，一阶调和数直接进入 Frobenius 指标 \rho_2 = h_1 - s，从而决定了原点处解的局部单值矩阵的特征值 e^{2\pi i(h_1-s)}。调和数谱的第一项 h_1 因此直接编码了方程在原点处的最基本的解析不变量。

第三条对应：无穷远渐近 \longleftrightarrow \Gamma(s) 与 \zeta(r) 的展开。在 z=\infty 处，解的行为由 Pochhammer 符号的 Stirling 渐近与调和数生成核在无穷远处的衰减共同决定。调和数谱在此与 Riemann zeta 函数直接耦合——这是因为 \mathcal{H}^{(r)}(\Sigma+1) = \zeta(r) - \zeta(r,\Sigma+2)，当 |\Sigma| \to \infty 时 \zeta(r,\Sigma+2) \to 0，调和数谱的极限行为直接与 \zeta(r) 挂钩。

第四条对应：隐奇点位置 z_\Sigma \longleftrightarrow \Sigma+1 作为 Beta 积分参数的解析延拓分支零点。这是最具几何色彩的一条对应。经典 Beta 积分 B(\Sigma+1, s) = \int_0^1 x^{\Sigma} (1-x)^{s-1} dx 在 \operatorname{Re}(\Sigma) > -1, \operatorname{Re}(s) > 0 时收敛，而王为民母函数的本质正是 B(\Sigma+1, s) 的调和数调制版本。z_\Sigma = 1 - 1/(\Sigma+1) 恰好是 Beta 积分参数 \Sigma+1 的解析延拓在复平面上的分支零点。

反向对应——从奇点数据重建调和数谱——的证明同样是构造性的。我们从隐奇点位置直接读出 \Sigma+1 = 1/(1-z_\Sigma)，从而确定 \Sigma。接着，从原点 Frobenius 指标 \rho_2 = \mathcal{H}^{(1)}(\Sigma+1) - s 和已知的 s，解出 h_1 = \rho_2 + s。然后是最关键的一步：利用 z=1 处的展开系数 d_n（即 MZV 截断展开），结合贝尔多项式的递推公式 Q_n = \frac{1}{n}\sum_{r=1}^n h_r Q_{n-r}，逐阶解出 h_2, h_3, \dots。具体而言，已知 Q_0 = 1 和 Q_1 = h_1，由 d_1 确定 Q_2 = \frac12(h_1^2 + h_2)，解得 h_2 = 2Q_2 - h_1^2；由 d_2 确定 Q_3 = \frac16(h_1^3 + 3h_1h_2 + 2h_3)，解得 h_3 = 3Q_3 - \frac12 h_1^3 - \frac32 h_1 h_2。每一步都是线性方程求解，唯一确定 h_r。最后，无穷远处的渐近展开提供 \Gamma(s) 与 \zeta(r) 的信息——因为无穷远渐近行为同时依赖于 s 和调和数谱的高阶项，可用于交叉验证重建结果的唯一性。由此，反向对应得以严格确立。

这一定理的一个深远推论是：完备贝尔多项式 Q_n 在调和数–奇点谱对应中扮演的角色，与傅里叶分析中的正交基或泰勒展开中的单项式基完全类似。任何奇点邻域内的局部解，都可以通过 Q_n 的线性组合来精确表达。而贝尔多项式递推关系 Q_n = \frac{1}{n}\sum_{r=1}^n h_r Q_{n-r} 在几何层面等价于奇点留数之间的卷积融合律——这是一种深刻的新型代数结构，它将不同奇点的留数以卷积的方式编织在一起。

总结起来，调和数–奇点谱刚性对应完成了调和数从数论对象到复几何模参数的终极跃迁。从此以后，调和数不再仅仅是 \sum_{n=1}^N 1/n^r 这样一个离散的和式，而是刻画平坦复射影结构中奇点分布、单值性、渐近行为的完整模参数集。整个复几何体系的全部奇点数据，都被编码在这一个无穷维的调和数谱之中。

2.3 王为民射影联络 \nabla_{\text{WW}}

本节构造的射影联络，其拓扑本质是级数穿越单位圆收敛边界时的单值变换行为。

有了二阶 Fuchs 方程 \mathcal{E}_{\text{WW}}，我们就可以将其转化为复几何中的标准对象——射影联络。平坦射影联络的存在性等价于底空间上存在一个复射影结构，王为民射影联络因此赋予穿孔黎曼球面一个以调和数谱为模参数的全新射影结构。

从方程 (2.1) 出发，将其除以 z 化为标准二阶形式 F'' + p(z)F' + q(z)F = 0，其中

p(z) = \frac{1}{z}\Bigl(s+1 - \sum_{r=1}^\infty \mathcal{H}^{(r)}(\Sigma+1) z^{r-1}\Bigr) = \frac{s+1}{z} - \sum_{r=1}^\infty \mathcal{H}^{(r)}(\Sigma+1) z^{r-2}.

作标准的消去一阶项变换 \Phi(z) = \exp\!\bigl(-\frac12 \int^z p(\zeta)d\zeta\bigr)\, u(z)，代入后一阶导数项精确消去，得到复几何的标准无挠形式：

\boxed{\frac{d^2 u}{dz^2} + \frac12\, S_{\text{WW}}(z;\Sigma,s)\, u = 0}. \tag{2.2}

在复几何中，任何一个二阶线性常微分方程的无挠形式 u'' + \frac12 S(z) u = 0 等价于在底空间上赋予一个平坦的 \operatorname{PSL}(2,\mathbb{C}) 射影联络。

依据施瓦茨导数标准公式 S = 2q - \frac14 p^2 - \frac12 p' 进行严格计算，王为民施瓦茨导数的显式表达式为：

\boxed{

S_{\text{WW}}(z;\Sigma,s) = \sum_{r=1}^\infty r\mathcal{H}^{(r+1)}(\Sigma+1)z^{r-2} - \frac12\Bigl(\sum_{r=1}^\infty \mathcal{H}^{(r)}(\Sigma+1)z^{r-2}\Bigr)^2 + \delta_s(z)

},

\delta_s(z) = \frac{1-s^2}{2z^2} + \frac{s+1}{2z}\sum_{r=1}^\infty \mathcal{H}^{(r)}(\Sigma+1) z^{r-2}.

这个表达式的结构极为丰富，值得逐项解剖其极点来源与几何含义。

第一项 \sum_{r=1}^\infty r\mathcal{H}^{(r+1)} z^{r-2} 在 z=0 处：当 r=1 时，贡献为 \mathcal{H}^{(2)}/z，这是一阶极点（\sim 1/z）；当 r \ge 2 时，各项在 z=0 处正则。整个第一项级数在 z=1 处由调和数母函数级数的收敛边界产生对数分支型奇点——这是调和数谱的非平凡特征在施瓦茨导数中的直接投射。

第二项 -\frac12(\sum_{r=1}^\infty \mathcal{H}^{(r)} z^{r-2})^2 在 z=0 处：r=1 项 \mathcal{H}^{(1)} z^{-1} 的平方产生 (\mathcal{H}^{(1)})^2 z^{-2} 的二阶极点，与 Fuchs 方程正则奇点的二阶极点结构一致；在 z=1 处则产生对数平方型奇点，对应调和数生成核在收敛边界上的奇性叠加。

第三项 \delta_s(z) 包含两部分：\frac{1-s^2}{2z^2} 提供标准的二阶极点，其系数由复参数 s 唯一决定；\frac{s+1}{2z}\sum_{r=1}^\infty \mathcal{H}^{(r)} z^{r-2} 在 r=1 时贡献 \frac{s+1}{2}\mathcal{H}^{(1)} z^{-2} 的二阶极点，高阶项在 z=0 处正则。

在 z=0 处将三项合并后，极点的阶数与系数由 \mathcal{H}^{(1)}, \mathcal{H}^{(2)}, s 三者的代数组合唯一确定。整个极点结构是严格可计算的，没有任何自由参数——这是王为民体系与那些含待定系数的唯象模型之间的本质区别。

当 s=1 时，一个优美的简化发生了：\delta_1(z) 的二阶极点项 \frac{1-s^2}{2z^2} 消失，方程的二阶极点项消失，结构显著简化——这从几何角度解释了为何 s=1 时的王为民恒等式具有最简洁的闭形 \frac{1}{\Sigma+1}。

方程 (2.2) 赋予穿孔黎曼球面 \mathbb{P}^1\setminus\{0,1,\infty,z_\Sigma\} 一个平坦的 \operatorname{PSL}(2,\mathbb{C}) 射影联络结构。我们将这一联络命名为王为民射影联络 \nabla_{\text{WW}}。其单值群表示

\rho: \pi_1\!\big(\mathbb{P}^1\setminus\{0,1,\infty,z_\Sigma\}\big) \longrightarrow \operatorname{PSL}(2,\mathbb{C})

完全由调和数谱与参数 s 唯一决定。调和数由此完成了从组合量到几何量的终极跃迁——它不再仅仅是数论中的一个和式，而是一个刻画平坦复射影结构全部局部与整体性质的完备模参数集。

猜想 2.1（临界充要性猜想）：射影联络 \nabla_{\text{WW}} 在 \mathbb{P}^1\setminus\{0,1,\infty,z_\Sigma\} 上可全局全纯平凡化（即其单值表示完全可约为平凡表示）的充要条件是 \operatorname{Re}(s)=1/2。

2.4 王为民模空间 \mathcal{M}

本节构造的模空间，其参数根源是级数对复权重 s 与总和参数 \Sigma 的双重依赖性。

将微分方程、施瓦茨导数、射影联络和单值群表示这些几何对象组织在一起的自然舞台，是模空间。定义二维复模空间为：

\mathcal{M} = \{(s,\Sigma) \in \mathbb{C}^2 \mid \operatorname{Re}(s) > 0,\ \operatorname{Re}(\Sigma) > -1\}.

模空间中每一个点 (s,\Sigma) 唯一绑定一套完整的几何结构：一个确定的微分方程 \mathcal{E}_{\text{WW}}(s,\Sigma)，一个确定的施瓦茨导数 S_{\text{WW}}(z;s,\Sigma)，一个确定的射影联络 \nabla_{\text{WW}}(s,\Sigma)，以及一个确定的单值群表示 \rho_{s,\Sigma}。这意味着，整个复几何族以 (s,\Sigma) 为坐标被组织成一个二维的解析族——变动 s 相当于调整了解的复权重，变动 \Sigma 相当于调整调和数谱的全部参数。

\mathcal{M} 作为 \mathbb{C}^2 的开子集，自然继承复流形结构。其上的周期映射具有重要的几何意义：由方程 (2.2) 的解基 (u_1, u_2) 沿基本群的闭路积分，获得一个从 \mathcal{M} 到 \operatorname{PSL}(2,\mathbb{C}) 共轭类的全纯映射。这个周期映射编码了单值群表示随参数变化的全部信息。进一步地，Hodge 滤子——复几何中衡量周期积分渐近行为的标准工具——由解基在奇点邻域的渐近展开决定，其滤过结构与调和数谱的滤波结构（按阶数 r 排列）精确一致。

在模空间 \mathcal{M} 内部，有一条至关重要的实解析曲线——临界共形轨迹：

\mathcal{C}_{\text{crit}} = \{(s,\Sigma) \in \mathcal{M} \mid \operatorname{Re}(s) = 1/2\}.

在解释这条轨迹的几何意义之前，我们需要引入一个精确的术语。我们称射影联络在 z=1 处达到“实平坦态”，若其在该奇点处的主奇点系数 A(\Sigma,s) 满足 \operatorname{Re}(A)=0。在此条件下，联络的单值矩阵在取实截面时具有最小的非平凡相位，整个联络的全局奇性被压缩至极小。

当且仅当 \operatorname{Re}(s)=1/2 时，\operatorname{Re}(A)=0，即主奇点的实部对消，联络达到实平坦态。该轨迹同时是模空间上实解析不变曲线和射影联络的分岔边界——当参数穿越此线时，联络的全局平坦性质发生根本性改变。它是整个数论–几何对偶体系的核心脊线。

而这里出现了整个体系中最引人注目的重合现象：临界共形轨迹的方程 \operatorname{Re}(s)=1/2 恰好与广义黎曼猜想中非平凡零点的临界线完全一致。几何临界轨迹与数论零点临界线，在本体系中是同一个数学对象——同一条复平面上的直线。这一重合不是人为安排，也不是形式类比。它源自调和数谱 \mathfrak{H}(\Sigma) 作为射影联络模参数的深层性质：调和数本身通过 Digamma 函数与 Hurwitz zeta 函数与 Riemann zeta 函数 \zeta(s) 及 Dirichlet L 函数产生内在耦合，而 \operatorname{Re}(s)=1/2 正是这一耦合达到最优反射对称性的位置——与函数方程的 s \leftrightarrow 1-s 对称精确对应。

2.5 王为民单值群 \Gamma_{\text{WW}} 与模曲线 \mathcal{C}_{\text{WW}}

本节构造的单值群，其分析起源是级数在单位圆外解析延拓产生的多值性。

射影联络的全局几何性质，最终凝结在单值群这一代数对象中。令 \gamma_0, \gamma_1, \gamma_\infty, \gamma_\Sigma 分别为环绕四个奇点 z=0, 1, \infty, z_\Sigma 的简单闭路——即在穿孔黎曼球面上分别包围每一个奇点而不环绕其他奇点的闭合路径。解基 (F_1, F_2) 沿某一路径解析延拓一周后，一般不会回到自身，而是变为自身的线性组合。这一线性变换的矩阵就是单值矩阵。

四个基本闭路对应的单值矩阵 M_0, M_1, M_\infty, M_\Sigma 满足一个整体的拓扑约束：

M_\infty M_\Sigma M_1 M_0 = I. \tag{2.3}

这一约束的几何释义是：环绕所有四个奇点的复合闭路同伦可缩为平凡闭路——因为在紧致的黎曼球面 \mathbb{P}^1 上，任何同时包围所有奇点的环路都可以连续收缩到一个点。这是平坦联络拓扑平凡性的直接体现，也是单值群各生成元之间必须满足的基本关系。

其中，M_0 和 M_1 的矩阵形式具有特别简洁的结构：

M_0 = \begin{pmatrix} 1 & 0 \\ * & e^{2\pi i(\mathcal{H}^{(1)}-s)} \end{pmatrix},\qquad

M_1 = \begin{pmatrix} e^{2\pi i s} & * \\ 0 & 1 \end{pmatrix}.

M_0 的上三角形式反映了解基中有一个解（F_1）在绕过 z=0 后不改变（特征值 1），而另一个解（F_2）获得了由调和数决定的相位因子 e^{2\pi i(\mathcal{H}^{(1)}-s)}。M_1 的下三角形式则反映了在 z=1 处的类似结构。矩阵中的 * 元素由方程在各自奇点处的 Frobenius 级数第二解的系数确定——具体地，这些系数是贝尔多项式留数和在奇点处的取值，由调和数谱完全决定。这些 * 元素也可以通过整体约束 (2.3) 从其他矩阵的已知元素求解出来。

由这四个生成元生成的群 \Gamma_{\text{WW}} \subset \operatorname{PSL}(2,\mathbb{C}) 就是王为民单值群。它是由调和数谱生成的非算术离散子群。

离散性在模空间的开稠密子集上成立，其证明依据 Jørgensen 不等式——二维 Möbius 群离散性判定的经典工具。根据 Jørgensen 不等式，若生成元及其交换子的迹满足

|\operatorname{tr}(M_0)|^2 + |\operatorname{tr}(M_1)|^2 + |\operatorname{tr}(M_0M_1^{-1})| \ge 2,

则生成的子群离散。本体系在模空间开稠密子集上满足此条件。当 \operatorname{Re}(s) > 0 且 s \notin \mathbb{Z} 时，|e^{2\pi i s}| = e^{-2\pi \operatorname{Im}(s)}，当 \operatorname{Im}(s) 为无理数且 \operatorname{Re}(s) \notin \mathbb{Z} 时，该元为双曲型或严格椭圆型，其旋转角为无理数倍 \pi。同时，\mathcal{H}^{(1)} 在一般复数 \Sigma 处的值为超越数——这是 Digamma 函数在非有理点取值的超越性所保证的——因此 e^{2\pi i(\mathcal{H}^{(1)}-s)} 的旋转角也为无理数倍 \pi。两个生成元的旋转角均非有理，所生成的子群因此为离散群。

非算术性的证明更为直接。经典算术 Fuchs 群的定义特征是：存在一个全实代数数域 F，使得群的所有矩阵元素均属于 F 中代数整数环的有限扩张。对于王为民单值群，考虑其生成元 M_0 的 (2,2) 元素 e^{2\pi i(\mathcal{H}^{(1)}-s)}。当 \mathcal{H}^{(1)} = \psi(\Sigma+2) + \gamma 取一般复值（非代数数）时，此元素为超越数——它不可能属于任何代数数域，更不用说代数整数环了。因此，在 \mathcal{H}^{(1)} 为超越数的参数点处，\Gamma_{\text{WW}} 不可能是算术群。由于超越数点构成模空间的稠密子集，\Gamma_{\text{WW}} 在整个模空间的意义上是非算术 Fuchs 群。

单值群作用于复上半平面 \mathbb{H}，其商空间

\mathcal{C}_{\text{WW}} = \mathbb{H} / \Gamma_{\text{WW}}

是一个紧致黎曼面——王为民模曲线。作为 Riemann 面，其几何量完全由单值群决定：亏格 g = \frac12 \operatorname{rank}(\Gamma_{\text{WW}}) + 1，其中 \operatorname{rank} 是群的最小生成元数目；分支点对应原始方程四个奇点在 \mathbb{H} 上的投影像；全纯微分——紧黎曼面上最基本的几何对象——由王为民母函数沿基本群闭路的周期积分给出。所有这些几何量最终都由两个复参数 (s,\Sigma) 唯一确定。

王为民模曲线不仅是抽象的几何构造，它更是承载王为民恒等式、多重 Zeta 值与 L 函数解析性质的几何载体。其 Jacobi 簇——由全纯微分周期格定义的复环面——上的有理点分布，与原著的超同余整数等式（第五章）存在深刻的对应关系。

2.6 王为民 KZ 型可积系统与王为民 D-模

本节的多参数可积系统延拓，其不变性根源是级数的核心对称原理——总和依赖性：和值仅依赖于 \Sigma=\sum\alpha_j，与各参数分配方式无关。这一原理在几何中升华为零曲率条件。

上述全部构造基于单参数常微分方程——我们固定了复变量 z，使参数 \Sigma 和 s 在模空间中变动。但实际上，\Sigma = \sum_{j=1}^k \alpha_j 本身是由 k 个独立复参数求和得到的。如果我们释放这一求和约束，令每个 \alpha_j 都成为独立变量，那么原本的单参数 ODE 就升华为一个 k 变量的偏微分可积系统。

首先，我们需要确立王为民 D-模的严格定义。设 \Delta_\Sigma 为作用于参数 \Sigma 的差分算子：\Delta_\Sigma f(\Sigma) = f(\Sigma+1)，\partial_z 为作用于复变量 z 的复微分算子。王为民 D-模是由这两个算子生成的、以王为民母函数 \Phi(z; \Sigma, s) 为其唯一全局解析解的代数系统。该 D-模的核心约束方程正是我们已导出的两个基本关系：其一是母函数对 z 的二阶 Fuchs 方程 (2.1)；其二是母函数对 \Sigma 的差分关系（由王为民恒等式的递推性质直接导出）：

\Phi(z;\Sigma+1,s) = \Phi(z;\Sigma,s) + \frac{1}{\Sigma+2} z \Phi(z;\Sigma+1,s+1).

这两个约束共同决定了 \Phi 作为双变量函数的所有解析性质。而总和依赖性在此表现为 D-模沿纤维方向的平凡化条件：解空间是一维的。

将每个 \alpha_j 释放为独立复变量，并为每个独立的参量配置一个复坐标 z_j 和一个复参数 s_j。延拓后的母函数定义在黎曼球面的多点配置空间上：

\operatorname{Conf}_r(\mathbb{P}^1) = \{(z_1,\dots,z_r)\in(\mathbb{P}^1)^r \mid z_i \neq z_j,\ z_i \notin \{0,1,\infty\}\}.

在此配置空间上，我们得到一族多分量的平坦 Fuchs 联络系统：

\frac{\partial \Phi}{\partial z_j} = \Bigl( \sum_{i \neq j} \frac{\Omega_{ij}}{z_i - z_j} \Bigr) \Phi, \tag{2.4}

其中联络矩阵 \Omega_{ij} 不再由单一调和数谱生成，而是由“多调和数”——即对应于不同指标子集的连续延拓调和数——联合生成。

系统 (2.4) 满足完全的平坦性条件：

dA + A \wedge A = 0,

其中 A = \sum_{j=1}^r A_j dz_j，而 A_j = \sum_{i \neq j} \frac{\Omega_{ij}}{z_i - z_j} 是联络的一形式。这一平坦性条件是偏微分系统可积的充要条件——它保证了 \Phi 作为多变量函数的存在性和唯一性。而这一条件之所以自动满足，根源正是总和依赖性在多变量几何中的体现：各 \alpha_j 之间的置换对称性保证了联络矩阵 \Omega_{ij} 满足必要的相容关系。
