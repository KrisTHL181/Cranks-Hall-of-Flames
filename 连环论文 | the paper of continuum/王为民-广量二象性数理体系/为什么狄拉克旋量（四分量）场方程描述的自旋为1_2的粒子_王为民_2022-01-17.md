为什么狄拉克旋量（四分量）场方程描述的自旋为1/2的粒子？

狄拉克方程

i∂tΨ=HΨ

其中H=-ia·▽+mβ=a·P+βm

而系数矩阵的反对易关系（中括号有角标+）为

[aₐ，aₑ]₊=aₐaₑ+aₑaₐ=2δₐₑ

当角标a=e时，δₐₑ=1，当a≠e时，δₐₑ=0，而下标a、e、o只取1或2或3。

[aₐ，β]₊=0

aₐ⁺=aₐ

β⁺=β

而轨道角动量L加上自旋角动量S得到的总角动量才与哈密顿能量算符H对易，表示总角动量才是系统的守恒量，而不是轨道角动量L或自旋角动量S，它们单独是不与哈密顿能量算符H对易的。

因此，有

[Lₐ+Sₐ，H]=0

式中有重复角标a，代表它是哑指标，要求和，求和后，自然就是总角动量。

对易关系，中括号下面没有角标，具体写出来就是

[Lₐ+Sₐ，H]=（Lₐ+Sₐ）H - H(Lₐ+Sₐ)=LₐH - HLₐ +SₐH - HSₐ=[Lₐ ，H] + [Sₐ ，H]

将哈密顿能量算符

H=-ia·▽+mβ=a·P+βm=aₑPₑ + βm代入，并且注意角动量分量和动量分量的关系

[Lₐ，Pₑ]=iεₐₑₒPₒ

继续前面的计算得(不是矩阵的常系数或偏导数可以移动到矩阵算符前后的任意位置，还有矩阵β是对角矩阵，可以和其它非对角矩阵交换前后位置，也就是有对易关系，导致与它的对易括号等于零）

[Lₐ ，H] + [Sₐ ，H]=[Lₐ，aₑPₑ+βm] + [Sₐ，aₑPₑ+βm]=[Lₐ，aₑPₑ] + m[Lₐ，β] + [Sₐ，aₑ·Pₑ] + m[Sₐ，β]=iεₐₑₒaₑPₒ + 0 + [Sₐ，aₑ] Pₑ+ 0 = 0

注意其中角动量L和动量P含偏导数，而自旋算符S不含偏导数，对易关系有所差别。

iεₐₑₒaₑPₒ + [Sₐ，aₑ] Pₑ = 0

可以任意更换重复的哑指标，得

iεₐₒₑaₒPₑ + [Sₐ，aₑ] Pₑ = 0

再交换εₐₒₑ中的角标顺序得到一个负号，即

εₐₒₑ=-εₐₑₒ

所以有

-iεₐₑₒaₒPₑ + [Sₐ，aₑ] Pₑ = 0

化简得

[Sₐ，aₑ]=iεₐₑₒaₒ

而满足这个关系的Sₐ与aₑ都是分块对角泡利矩阵

Sₐ=1/2Λₐ，矩阵Λₐ为对角分块矩阵，对角分块元为泡利矩阵σₐ，而泡利矩阵σₐ²=1，并且有Λₐ²=1

而总自角动量

S²=S₁²+S₂²+S₃²=（1/2Λ₁）²+（1/2Λ₂）²+（1/2Λ₃）²=3/4=L(L+1)

角量子数L只能取1/2，所以，狄拉克场方程描述的粒子自旋为1/2.
