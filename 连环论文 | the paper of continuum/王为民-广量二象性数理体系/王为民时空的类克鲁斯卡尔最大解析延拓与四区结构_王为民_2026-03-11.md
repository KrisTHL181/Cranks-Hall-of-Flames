2 标准王为民度规与固定视界条件

2.1 标准王为民度规

标准王为民度规的严格形式为：

ds² = -c²(1 - r_s/r)dt² + a(t)²[ dr²/((1-kr²)(1-r_s/r)) + r²(dθ² + sin²θ dφ²) ]   (1)

各物理量定义：

-  c ：真空中光速，量纲  [LT⁻¹]

-  G ：万有引力常量，量纲  [M⁻¹L³T⁻²]

-  r_s = 2GM/c² ：王为民粒子白洞事件视界半径，量纲  [L]

-  M ：白洞质量，量纲  [M]

-  a(t) ：宇宙尺度因子，无量纲

-  k ：三维空间曲率常数，量纲  [L⁻²]

2.2 固定视界条件

本文核心物理假设：王为民粒子白洞事件视界位置不随时间变化，即

r_s = 常数   (2)

由此推得白洞质量满足： M = 常数 。这一假设表明宇宙演化过程中中心白洞的质量保持稳定。

视界曲面由度规分量退化条件给出：

g_tt = 0, g^rr → ∞

代入度规得：

1 - r_s/r = 0 ⇒ r = r_s   (3)

该曲面不随时间演化，表面引力为常数：

κ = c²/(2r_s), 量纲 [LT⁻²]   (4)

3 乌龟坐标与克鲁斯卡尔坐标构造

3.1 乌龟坐标

为消除  r=r_s  处的坐标奇性，构造适配王为民度规的乌龟坐标  r_* ，定义如下微分关系：

dr_* = dr / sqrt((1-kr²)(1-r_s/r)) = sqrt(r) dr / sqrt((1-kr²)(r - r_s))   (5)

积分得：

r_* = ∫ sqrt(r) dr / sqrt((1-kr²)(r - r_s))   (6)

该积分在  r∈(r_s,+∞)  上收敛，且满足：

r→r_s⁺ ⇒ r_*→-∞, r→+∞ ⇒ r_*→+∞

成功将视界奇性映射至无穷远。

3.2 光锥坐标

在乌龟坐标基础上，定义光锥坐标（零坐标）：

u = t - r_*/c, v = t + r_*/c   (7)

其中  u,v  量纲均为  [T] 。在  (u,v)  坐标下，径向光线的传播满足  u=常数  或  v=常数 。

3.3 类克鲁斯卡尔双零坐标

基于常数表面引力  κ ，构造类克鲁斯卡尔双零坐标  (U,V) ：

{ U = -e^(-κu/c) = -e^(-κt/c + κr_*/c²) ; V = e^(κv/c) = e^(κt/c + κr_*/c²) }   (8)

这里  κ/c  量纲为  [T⁻¹] ，指数项无量纲。这一延拓坐标使克鲁斯卡尔坐标一致，且同时依赖于时间  t  和径向坐标  r_* 。

将两式相乘，得核心恒等式：

UV = -e^(2κr_*/c²)   (9)

在视界  r=r_s  处， r_*→-∞ ，故  UV=0 。

4 坐标逆变换与共形因子

4.1 坐标逆变换

由式(8)相除：

V/(-U) = e^(2κt/c)   (10)

结合式(9)，可得：

t = (c/(2κ))ln(V/(-U)), r_* = (c²/(2κ))ln(-UV)   (11)

其中要求  UV<0 （经典外部区域）以保证对数有定义。径向坐标  r  与  UV  的关系由式(6)的积分反函数给出，记作  r=r(UV) 。

4.2 度规的  (U,V)  形式

由式(7)和式(8)可得微分关系：

du = (c/κ)(dU/U), dv = (c/κ)(dV/V)   (12)

从而：

-du dv = -c²/κ² * dU dV/(UV)   (13)

原度规(1)可写为：

ds² = (1 - r_s/r)(-c²dt²) + a(t)²[ dr_*² + r²dΩ² ]   (14)

利用  -c²dt² + dr_*² = -c² du dv ，代入式(13)得：

-c²dt² + dr_*² = -c⁴/κ² * dU dV/(UV)   (15)

将式(15)代入式(14)：

ds² = (1 - r_s/r)(-c⁴/κ² * dU dV/(UV)) + a(t)² r² dΩ²   (16)

在静态近似下，可令  a(t)=1 ，则度规简化为静态球对称形式。共形因子为：

Ω²(U,V) = c⁴/κ² * (1 - r_s/r(U,V))/(UV)   (17)

在视界  UV=0  处，由式(9)知  e^(2κr_*/c²) = -UV → 0 ，故  r_*→-∞ ，从而  r→r_s 。此时  1 - r_s/r → 0 ，但  UV→0 ，其比值通过洛必达法则可证明为有限值，从而  Ω²  在视界处解析正则。

5 四区时空结构

由式(8)定义的  U,V  将时空划分为四个解析片区：

片区 U 符号 V 符号 UV 符号 r 与 r_s 关系 物理定义

I U<0 V>0 UV<0 r>r_s 物质宇宙：外部区域，正物质构成，可观测宇宙

II U<0 V<0 UV>0 r<r_s 黑洞：未来视界内区域，物质向奇点坍缩

四个片区通过视界面  U=0  或  V=0 （即  r=r_s ）相连，形成完整的因果结构。

6 迷你超空间量子化

6.1 经典作用量的约化

取特征长度  L=r_s ，将尺度因子  a(t)  作为唯一动力学变量，对空间部分积分得到有效拉格朗日量：

L = (3πL³c²)/(4G)[ -aȧ² + kc²a - (Λ_eff c²/3)a³ ]   (18)

量纲检查： L³c²/G  量纲为能量，正确。其中  Λ_eff  为有效宇宙学常数，量纲  [L⁻²] 。

6.2 共轭动量与哈密顿量

共轭动量：

p_a = ∂L/∂ȧ = -(3πL³c²)/(2G) aȧ, 量纲 [ML²T⁻¹]   (19)

哈密顿量：

H = p_aȧ - L = -(2G)/(3πL³c²) * p_a²/a + (3πkc²L³)/(8G)a + (πΛ_eff c²L³)/(8G)a³   (20)

经典约束  H=0  对应弗里德曼方程。

6.3 惠勒–德威特方程

正则量子化： p_a → -iℏ ∂/∂a ，代入  HΨ=0  得：

[ -(2Gℏ²)/(3πL³c²) * (1/a)∂²/∂a² - (3πkc²L³)/(8G)a - (πΛ_eff c²L³)/(8G)a³ ]Ψ(a) = 0   (21)

6.4 波函数变换与解析解

引入变换  Ψ(a) = χ(a)/sqrt(a) ，消去一阶导数项得：

χ''(a) - (α²a⁴ + β²a² - γ²)χ(a) = 0   (22)

其中

α = sqrt( πΛ_eff c²L³/(8Gℏ²) ), β = sqrt( 3πkc²L³/(8Gℏ²) ), γ = 1/2   (23)

这里  α  和  β  的表达式在自然单位制下（取  c=G=ℏ=1 ）简化为  α=sqrt(πΛ_eff L³/8)  和  β=sqrt(3πk L³/8) ，均为无量纲常数，从而保证  αa²  和  βa²  无量纲，满足合流超几何方程的物理约束。

方程(22)为合流超几何型，其解析解为：

Ψ_phys(a) = N e^(-αa²/2) U(1/4 - ν, 3/2, αa²)   (24)

其中  ν = γ²/(4α) ， U  为第二类合流超几何函数， N  为归一化常数。
