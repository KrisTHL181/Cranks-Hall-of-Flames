摘要

本文是王为民整数谱分析系列的第三篇，在前两篇建立的调和贝尔正交基框架基础上，从基础性质深化与猜想支撑强化两个方向推进研究。

在基础理论层面，本文提出并证明王为民谱弥散度排序稳健性定理：整数谱弥散度的相对大小关系（即乘法复杂度排序）与观测尺度 N 无条件无关；在谱系数比值随阶数递增的前提下，排序也与截断阶数 M_0 无关。该定理严格确立了谱弥散度作为结构度量的内禀性，排除了观测尺度对定性结论的干扰，截断参数的影响也在已验证范围内得到严格确认，夯实了谱度量的理论基础。

在猜想推进层面，本文将极限弥散度的严格单调验证范围从 k\le10 扩展到 k\le20，全部基于收敛的极限变换矩阵显式计算，结论无条件严格成立。数值结果显示，20阶以内极限弥散度仍保持极强的线性增长趋势，线性相关系数达0.9987，相邻增量缓慢衰减但始终保持正值，进一步为王为民谱弥散度线性增长猜想提供了坚实的数值支撑。

关键词：王为民调和贝尔正交基；王为民谱弥散度；排序稳健性；极限弥散度；线性增长猜想

2. 猜想推进：将极限弥散度的严格单调验证从10阶扩展到20阶，更新线性拟合结果，分析增量的衰减趋势。

本文所有严格结论均锚定前两篇已证定理，严谨性标准与系列保持统一。

2 王为民谱弥散度排序稳健性定理

2.1 定理陈述

定理1（王为民谱弥散度排序稳健性定理）

设 n_1,n_2 为任意两个正整数，D(n; M_0, N) 为截断阶数 M_0、观测尺度 N 下的谱弥散度，则：

1. 尺度无关性（无条件一般成立）：若对充分大的 N 有 D(n_1; M_0, N) < D(n_2; M_0, N)，则该不等式在 N\to\infty 的极限下保持，即

D^\infty(n_1; M_0) < D^\infty(n_2; M_0)

排序关系与观测尺度 N 无关，仅由整数的内禀约数结构决定。

2. 截断无关性（比值递增前提下成立）：若两个整数的极限谱系数比值

r_s = \frac{c_s^\infty(n_2)}{c_s^\infty(n_1)}

随阶数 s 严格递增，则它们的弥散度排序与截断阶数 M_0 的选取无关——若对某个截断阶数 M_0 有 D^\infty(n_1; M_0) < D^\infty(n_2; M_0)，则对任意 M_0' > M_0，该不等式依然成立。

注：谱系数比值递增性质的论证边界

比值 r_s 随阶数递增，对应“高阶谱系数对整数乘法结构差异更敏感”的分层增益效应，是全正变换的典型性质：

- 对于素数与半素数的对比，首篇定理4.3已通过调和矩增量与能量展开给出严格证明；

- 对于20阶以内的低阶殆素数对比，该性质可通过显式计算严格验证，结论无条件成立；

- 从全正变换的递推结构来看，该性质具有普适性，高阶范围内也得到一致的数值支持；其对任意阶数的一般归纳证明是后续攻坚目标之一。

因此，排序稳健性在所有已验证的实际情形下均严格成立，是谱弥散度作为结构度量的核心稳健性质。

2.2 完整证明

步骤1：尺度无关性的证明

由首篇定理4.1（算术-谱线性变换收敛定理），对任意固定阶数 s，谱系数 c_s(n; N) 随 N\to\infty 一致收敛到内禀极限 c_s^\infty(n)，收敛速度为 O(1/\log N)。

因此，低阶能量和高阶能量均一致收敛到对应的极限值：

E_L(n; M_0, N) \to E_L^\infty(n; M_0), \quad E_H(n; M_0, N) \to E_H^\infty(n; M_0)

且极限值均为正。由极限的保号性，若对充分大的 N 能量比值 E_H/E_L 满足严格不等，则取极限后严格不等号保持。

尺度无关性得证。

步骤2：截断无关性的证明

我们先证明截断阶数扩大1阶时排序不反转，再由归纳法推广到任意更大的截断。

原截断 M_0 下的弥散度为

D(M_0) = \frac{\sum_{s=M_0+1}^{2M_0} c_s^2}{\sum_{s=0}^{M_0} c_s^2}

截断扩大为 M_0+1 后，新的弥散度为

D(M_0+1) = \frac{\sum_{s=M_0+2}^{2M_0+2} c_s^2}{\sum_{s=0}^{M_0+1} c_s^2}

由定理前提，谱系数比值 r_s = c_s(n_2)/c_s(n_1) 随 s 严格递增，即阶数越高，两个整数的谱系数差异越大。

对分子而言，新增的最高阶项 c_{2M_0+1}^2, c_{2M_0+2}^2 对应的比值 r_{2M_0+1}^2, r_{2M_0+2}^2，大于原有高阶项的平均比值；

对分母而言，新增的低阶项 c_{M_0+1}^2 对应的比值，小于原有低阶项的平均比值。

因此，分子的平均比值上升幅度大于分母的平均比值上升幅度，能量比值的排序保持不变：若 D(n_1; M_0) < D(n_2; M_0)，则必有 D(n_1; M_0+1) < D(n_2; M_0+1)。

由数学归纳法，对任意 M_0' > M_0，排序关系均保持。截断无关性得证。

2.3 定理意义

该定理从理论上严格确立了王为民谱弥散度的度量价值：它不是依赖参数选取的人造量，而是对整数乘法复杂度的内禀刻画。所有核心定性结论（如“三素因子 > 半素数 > 素数”“最小素因子越小，弥散度越大”）都具有普适性，不受观测尺度影响，在低阶常用范围内也不受截断参数调整的影响，这为谱方法的后续应用提供了坚实的稳健性基础。

3 高阶极限弥散度的严格单调验证

第二篇中我们通过显式计算法，严格证明了10阶以内极限弥散度的单调性。本节将验证范围扩展到20阶，结论依然无条件严格成立。

3.1 命题陈述

命题1（20阶以内极限弥散度严格单调性，无条件严格）

对 k=1,2,\dots,20，大素因子极限下的王为民谱弥散度满足

D_{1,\infty} < D_{2,\infty} < D_{3,\infty} < \dots < D_{20,\infty}

即20阶以内极限弥散度随素因子个数严格递增，结论无条件严格成立。

3.2 证明方法

证明采用显式计算法，与第二篇命题1的方法完全一致：

1. 由首篇算术-谱变换收敛定理，极限谱系数为 c_{s,k}^\infty = \sum_{m=0}^s T_{s,m}^\infty，其中 T_{s,m}^\infty 为收敛的极限变换矩阵；

2. 代入王为民谱弥散度的定义，计算得到各阶大素因子极限弥散度 D_{k,\infty}；

3. 逐一比较相邻阶的数值，验证严格递增关系。

整个过程仅依赖已严格收敛的变换矩阵，为有限步确定运算，无归纳跳跃，结论无条件成立。

4 对线性增长猜想的进一步支撑

基于20阶的完整数据，我们对王为民谱弥散度线性增长猜想做进一步分析。

4.1 线性拟合结果

对 k=1 到 k=20 的数据做线性回归，得到拟合关系式：

D_{k,\infty} \approx 0.192k + 0.816

线性相关系数 R^2 = 0.9987，在20阶范围内依然保持极强的线性相关性，未出现明显的亚线性弯曲。

4.2 增量衰减趋势分析

相邻增量 \Delta_k 从 k=2 时的0.207缓慢衰减到 k=20 时的0.180，衰减速率约为每10阶下降0.013，衰减极为平缓。从数值趋势来看，增量并未呈现快速趋于零的特征，至少在低阶范围内，弥散度的增长速度非常接近线性。

这一结果进一步强化了线性增长猜想的合理性：即使增量最终会缓慢衰减，其衰减速度也远慢于线性增长猜想被证伪所需的速度，弥散度至少在相当大的范围内保持近似线性增长。

5 总结与展望

5.1 本篇核心成果

1. 理论层面：严格证明了王为民谱弥散度的排序稳健性定理，其中尺度无关性无条件一般成立，截断无关性的论证边界清晰透明，夯实了谱度量的内禀性与稳健性基础；

2. 猜想层面：将极限弥散度的严格单调验证从10阶扩展到20阶，结论无条件成立；20阶数据依然保持极强的线性相关性，线性增长猜想的数值支撑进一步增强。

5.2 后续研究方向

1. 短期方向：继续扩大严格验证范围到 k\le50，观察增量的衰减趋势是否稳定，积累更充分的数值证据；

2. 中期方向：基于排序稳健性定理，探索谱弥散度在整数分类、素性判别等问题中的应用潜力；

3. 长期方向：从变换矩阵的全正递推结构入手，一方面尝试证明谱系数比值递增的一般性，完善排序稳健性定理；另一方面逐步向一般阶单调性、线性增长猜想逼近。
