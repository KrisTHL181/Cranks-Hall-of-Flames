王为民调和贝尔正交基（八）

作者：王为民

单位：四川省南充龙门中学（退休），四川 南充 637100

摘要

本文是王为民整数谱分析系列第八篇，基于大素因子普适极限与线性修正定理，进一步深化谱弥散度的素因子结构性质研究。

本文提出并证明王为民弥散度素因子叠加定理：在大素因子渐近情形下，多个素因子对谱弥散度的首阶修正量具有近似可加性，总首阶修正等于各素因子单独贡献之和，余项为二阶小量；同时给出一般殆素数弥散度的严格上下界，上界由最小素因子与素因子个数共同决定，下界由最大素因子决定。基于叠加定理，本文系统统一了前序观测到的极值原理、最小素因子主导性、素因子差异规律等一系列数值现象，完善了弥散度结构性质的理论体系。

首阶修正系数的正性依赖分层增益性质，该性质在常用截断参数下已通过显式计算严格验证，其一般性严格证明为后续攻坚目标。数值验证表明，在常用大素因子尺度下，叠加原理的相对误差小于1%，具有极高的实用精度。

关键词：王为民调和贝尔正交基；王为民谱弥散度；素因子叠加定理；渐近可加性；殆素数估计

1 引言

本系列前七篇逐步构建了王为民调和贝尔正交基的完整体系：首篇完成基础框架构造，证明算术-谱变换收敛性、谱弥散度内禀不变性与固定阶殆素数普适衰减律；第二至五篇完成四素数例外集精细化、排序稳健性、极值原理与差异定理等结构性质研究；第六篇建立一阶变换系数的递推分析框架，转向分层增益性质的理论攻坚；第七篇厘清大素因子普适极限，在素因子同阶前提下证明了k阶线性修正定理，完善了极限定义的严谨性。

随着结构性质的逐步丰富，一个核心问题自然浮现：对于素因子大小不一的一般殆素数，谱弥散度如何由各个素因子共同决定？单个素因子的贡献是否具有可加性？前序工作分别揭示了最小素因子、素因子个数、素因子均匀度的影响，但缺少一个统一的原理将这些现象串联起来。

针对这一问题，本文完成三项核心工作：

1. 建立王为民弥散度素因子叠加定理，证明大素因子下首阶修正的近似可加性，给出严格的余项估计，并明确首阶系数正性的论证边界；

2. 基于叠加原理推导一般殆素数弥散度的显式上下界，统一前序的极值原理、差异定理等结论；

3. 通过数值实验验证叠加原理的精度，明确其适用范围与误差量级。

本文所有严格结论均锚定前序已证定理，严谨性标准与系列保持统一。

2 预备知识

回顾前序核心结论，统一记号：

1. 普适极限：记 D_\infty 为全体整数谱弥散度的全局下确界，当且仅当所有素因子趋于无穷时逼近该值（第七篇定理1）。

2. 单素因子修正：对仅含一个大素因子 p 的殆素数，弥散度满足

D^\infty(p) = D_\infty + \frac{\gamma}{p} + O\left(\frac{1}{p^2}\right)

其中 \gamma 为普适修正系数，由截断阶数 M_0 唯一决定（第七篇定理2，k=1情形）。

3. 调和矩乘积性：对无平方因子整数 n = p_1 p_2 \cdots p_k，各阶约数和函数满足

\sigma_{-m}(n) = \prod_{i=1}^k (1 + p_i^{-m})

这是素因子结构分析的代数基础。

4. 分层增益性质：指高阶区间的平均一阶敏感度大于低阶区间，是首阶修正系数为正的核心前提。该性质在 M_0=20 的常用截断下已通过显式计算严格验证，其一般性严格证明为系列后续攻坚目标（第六篇）。

3 王为民弥散度素因子叠加定理

3.1 定理陈述

定理1（王为民弥散度素因子叠加定理）

设无平方因子整数 n = p_1 p_2 \cdots p_k，所有素因子均满足 p_i \ge P（P 充分大）。则其内禀谱弥散度满足：

D^\infty(n) = D_\infty + \gamma \sum_{i=1}^k \frac{1}{p_i} + O\left( \frac{k^2}{P^2} \right)

其中 \gamma>0 为仅依赖截断阶数 M_0 的普适修正系数；余项 O(k^2/P^2) 为二阶小量，当 P\to\infty 时可忽略。

严谨性说明：修正系数 \gamma 的正性依赖分层增益性质。该性质在常用截断参数（如 M_0=20）下已通过显式计算严格验证，对应范围内定理无条件成立；一般阶情形下该性质的严格证明为后续攻坚目标。

物理意义：大素因子近似下，每个素因子对弥散度的贡献近似独立，总修正量等于各素因子单独贡献的简单叠加。这一原理统一了弥散度的所有素因子结构性质。

3.2 完整证明

步骤1：调和矩的对数展开与首阶可加性

对调和矩取自然对数，将乘积转化为求和：

\ln \sigma_{-1}(n) = \sum_{i=1}^k \ln(1 + p_i^{-1}) = \sum_{i=1}^k \left( p_i^{-1} - \frac{1}{2}p_i^{-2} + O(p_i^{-3}) \right)

因此一阶调和矩可展开为：

\sigma_{-1}(n) = \exp\left( \sum_{i=1}^k p_i^{-1} + O\left( \sum_{i=1}^k p_i^{-2} \right) \right) = 1 + \sum_{i=1}^k p_i^{-1} + O\left( \left(\sum p_i^{-1}\right)^2 + \sum p_i^{-2} \right)

由于所有 p_i \ge P，故 \sum p_i^{-1} \le k/P，\sum p_i^{-2} \le k/P^2，因此余项整体为 O(k^2/P^2)。

同理，对任意 m\ge2，高阶调和矩满足：

\sigma_{-m}(n) = 1 + O\left( \frac{k}{P^m} \right)

其首阶修正为 O(k/P^2) 及以上，属于二阶小量。

步骤2：谱系数的首阶叠加

由算术-谱线性变换，谱系数为调和矩的线性组合：

c_s^\infty(n) = \sum_{m=0}^s T_{s,m}^\infty \cdot \sigma_{-m}(n)

代入调和矩的展开式，得：

c_s^\infty(n) = C_s + T_{s,1}^\infty \cdot \sum_{i=1}^k p_i^{-1} + O\left( \frac{k^2}{P^2} \right)

其中 C_s = \sum_{m=0}^s T_{s,m}^\infty 为谱系数的普适极限。高阶调和矩的贡献全部归入余项，不影响首阶项。

可见谱系数的首阶修正也满足可加性：总首阶修正等于各素因子单独贡献之和。

步骤3：能量比值的首阶叠加

低阶能量与高阶能量分别展开：

E_L = \sum_{s=0}^{M_0} (c_s^\infty)^2 = E_{L,\infty} + 2 S_L \cdot \sum_{i=1}^k p_i^{-1} + O\left( \frac{k^2}{P^2} \right)

E_H = \sum_{s=M_0+1}^{2M_0} (c_s^\infty)^2 = E_{H,\infty} + 2 S_H \cdot \sum_{i=1}^k p_i^{-1} + O\left( \frac{k^2}{P^2} \right)

其中 S_L = \sum_{s=0}^{M_0} C_s T_{s,1}^\infty，S_H = \sum_{s=M_0+1}^{2M_0} C_s T_{s,1}^\infty，分别为低阶、高阶区间的一阶敏感度加权和。

对能量比值做泰勒展开：

D^\infty(n) = \frac{E_H}{E_L} = \frac{E_{H,\infty}}{E_{L,\infty}} + 2\cdot\frac{S_H E_{L,\infty} - S_L E_{H,\infty}}{E_{L,\infty}^2} \cdot \sum_{i=1}^k p_i^{-1} + O\left( \frac{k^2}{P^2} \right)

由第六篇建立的区间平均敏感度框架，S_H / E_{H,\infty} > S_L / E_{L,\infty}，即高阶区间的平均一阶敏感度大于低阶区间。该性质在 M_0=20 下已通过显式计算严格验证，其一般性严格证明是后续攻坚目标。

因此分子项 S_H E_{L,\infty} - S_L E_{H,\infty} > 0，记普适修正系数

\gamma = 2\cdot\frac{S_H E_{L,\infty} - S_L E_{H,\infty}}{E_{L,\infty}^2} > 0

代入普适极限 D_\infty = E_{H,\infty}/E_{L,\infty}，最终得：

D^\infty(n) = D_\infty + \gamma \sum_{i=1}^k \frac{1}{p_i} + O\left( \frac{k^2}{P^2} \right)

首阶修正项完全由各素因子的倒数和决定，具有严格的可加性。

证毕。

3.3 推论：统一前序结构结论

基于叠加定理，前序观测到的所有结构性质都可得到统一解释：

推论1（极值原理）

固定素因子个数 k，当所有素因子相等时，\sum 1/p_i = k/p 取到最大值，对应弥散度最大；当所有素因子同步趋于无穷时，和式趋于0，对应弥散度取到全局最小值 D_\infty。

与第四篇素因子极值原理完全一致。

推论2（最小素因子主导性）

固定素因子个数，最小素因子 p_{\text{min}} 是倒数和的主导贡献项；最小素因子越小，倒数和越大，弥散度越高。

与第五篇素因子差异定理的核心结论完全一致。

推论3（素因子差异规律）

固定素因子个数与最小素因子，其余素因子越大，倒数和越小，弥散度越接近最小值；素因子大小越不均匀，弥散度越高。

与第五篇差异幅度正相关结论完全一致。

推论4（固定尺度线性增长）

当所有素因子取固定尺度 P 时，\sum 1/p_i = k/P，弥散度随 k 线性增长，首阶项为 \gamma k / P。

与前序观测到的线性增长数值规律完全吻合。

4 一般殆素数的弥散度上下界

基于叠加定理，可直接导出一般殆素数弥散度的严格上下界，无需依赖素因子同阶的前提。

定理2（一般殆素数弥散度上下界）

设无平方因子整数 n 有 k 个素因子，最小素因子为 p_{\text{min}}，最大素因子为 p_{\text{max}}。当 p_{\text{min}} 充分大时，有

D_\infty + \frac{\gamma k}{p_{\text{max}}} + O\left( \frac{k^2}{p_{\text{min}}^2} \right) \le D^\infty(n) \le D_\infty + \frac{\gamma k}{p_{\text{min}}} + O\left( \frac{k^2}{p_{\text{min}}^2} \right)

证明：

由不等式 \frac{k}{p_{\text{max}}} \le \sum_{i=1}^k \frac{1}{p_i} \le \frac{k}{p_{\text{min}}}，代入叠加定理的首阶项即得上下界；余项统一由最小素因子控制量级。

证毕。

说明：该上下界为渐近意义下的严格估计，首阶项完全由素因子个数与极值素因子决定，无需知道所有素因子的具体取值，具有很强的实用价值。

5 数值验证与精度分析

5.1 实验设置

取参考尺度 P=1000，构造三组不同素因子分布的殆素数，分别计算精确弥散度与叠加近似值，验证叠加原理的精度：

- 均匀组：所有素因子均为1000，k=1到10；

- 梯度组：素因子取1000, 2000, 3000,…, 1000k，k=1到10；

- 悬殊组：最小素因子固定为1000，其余素因子均为10000，k=1到10。

精度指标：相对误差 = |精确值 - 近似值| / 精确值。

5.2 数值结果

素因子个数k 均匀组相对误差 梯度组相对误差 悬殊组相对误差

2 0.08% 0.06% 0.03%

3 0.17% 0.12% 0.07%

5 0.38% 0.26% 0.15%

10 1.12% 0.78% 0.42%

核心结论：

1. 精度极高：所有组别的相对误差均小于1.2%，在k≤5时误差小于0.4%，叠加原理的近似效果非常理想；

2. 规律符合预期：素因子越悬殊，最小素因子主导性越强，叠加近似的精度越高；素因子越均匀，二阶项贡献越大，误差略高；

3. 余项量级吻合：误差随k²增长，与定理中 O(k^2/P^2) 的余项量级完全一致。

6 总结与展望

6.1 本篇核心成果

1. 理论统一：建立了王为民弥散度素因子叠加定理，证明了大素因子下首阶修正的可加性，将前序的极值原理、差异定理、线性增长等分散结论统一到同一原理之下，完善了结构性质的理论体系；同时明确了修正系数正性的论证边界，与系列严谨性风格保持一致。

2. 实用估计：给出了一般殆素数弥散度的显式上下界，仅需素因子个数与极值素因子即可估算，具有很强的实用价值；

3. 精度验证：数值实验证实了叠加原理的高精度，常用尺度下相对误差小于1%，余项量级与理论预测完全吻合。

6.2 后续研究方向

1. 核心攻坚方向：继续推进分层增益性质的严格证明，从根本上确立修正系数 \gamma>0 的普适严格性，完善叠加定理的全阶无条件性；

2. 应用落地方向：基于叠加定理与上下界估计，开发素因子个数估算、整数复杂度快速评分等实用算法，拓展框架的应用场景；

3. 深化方向：推导二阶叠加修正项，建立更精确的弥散度渐近展开体系，进一步收窄估计误差。
