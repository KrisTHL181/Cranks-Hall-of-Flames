王为民粒子表象中的QED红外正规化

摘要

本文在已建立的王为民粒子表象相互作用场论框架下，以量子电动力学（QED）的单圈电子自能和顶角修正为例，展示该表象在红外发散正规化中的核心优势。标准QED中，这两类单圈图在红外极限下需要人工引入光子质量截断或使用维度正规化处理对数发散；在王为民表象中，光子传播子天然携带的基底因子 \mathcal{F}_W(k) \sim \ln(\mu/|k|) 在 k \to 0 时提供对数型衰减，将软光子积分转化为有限值。本文给出电子自能 \Sigma_W(p) 和顶角函数 \Lambda_W^\mu(p',p) 的单圈解析表达式，证明其在红外区域有限且与标准正规化方案的结果一致，且Ward-Takahashi恒等式自动保持。本文将王为民表象从一个“形式框架”正式升级为可逐圈计算的物理工具。

关键词：王为民粒子表象；量子电动力学；红外发散；内在正规化；电子自能；顶角修正；Ward-Takahashi恒等式

一、引言

量子电动力学（QED）是量子场论中最精确验证的物理理论。然而，其微扰计算面临红外发散的经典难题：当虚光子动量趋于零时，费曼图的圈积分对数发散，需要人工引入光子质量截断或使用维度正规化方案[4]。

在前五篇文章中，我们构建了王为民粒子表象的完整理论框架：自由标量场[1]、旋量场[2]、矢量场[3]的表象构造，以及相互作用场论的微扰展开与费曼规则[5]。这一框架的核心特征是：传播子天然携带统一的基底修正因子 \mathcal{F}_W(k)，其在红外区域的调和数渐近行为产生 \sim \ln(\mu/|k|) 的对数修正。

本文是该框架的首次实际圈图计算。选取QED中最基本的两个单圈过程——电子自能和顶角修正——展示王为民传播子如何将标准理论中需要人工正规化的红外发散转化为天然有限的积分。这是王为民表象从“形式自洽”走向“物理可用”的关键跨越。

二、标准QED中的红外发散问题

2.1 电子自能单圈图

在标准QED中，单圈电子自能的费曼积分（Feynman规范）为：

- i \Sigma(p) = (-ie)^2 \int \frac{d^4k}{(2\pi)^4} \gamma^\mu \frac{i(\not{k} + m)}{k^2 - m^2 + i\epsilon} \gamma_\mu \frac{-i}{(p-k)^2 + i\epsilon}.

该积分在紫外区域呈对数发散，需要重整化；在红外区域，当虚光子动量 k \to 0 时，若电子在质壳上（p^2 = m^2），积分也呈对数发散。标准方法中，需要为光子引入微小的质量 \lambda 作为红外截断，最终物理结果不依赖于 \lambda。

2.2 顶角修正单圈图

顶角修正的单圈费曼积分在零动量转移极限下同样存在红外发散，且与电子自能的红外发散通过Ward-Takahashi恒等式精确关联：

\Gamma^\mu(p, p) = -\frac{\partial \Sigma(p)}{\partial p_\mu}.

这一关联是规范对称性的直接体现，也是检验任何红外正规化方案自洽性的标准判据。

三、王为民表象中的单圈计算

3.1 王为民QED费曼规则

在前文[5]中，我们建立了王为民QED的费曼规则。与标准QED相比，唯一差异是每个传播子携带基底因子 \mathcal{F}_W：

S_W(k) = \frac{i(\not{k} + m)}{k^2 - m^2 + i\epsilon} \mathcal{F}_W(k), \qquad

D_{W\mu\nu}(k) = \frac{-ig_{\mu\nu}}{k^2 + i\epsilon} \mathcal{F}_W(k).

其中 \mathcal{F}_W(k) = \sum_{n=1}^\infty [Q_n^W(\omega_k/\mu)]^2 / \|Q_n^W\|^2。在小动量极限下，利用调和数的渐近行为 H_n \sim \ln n + \gamma，可以证明：

\mathcal{F}_W(k) \sim \ln\left(\frac{\mu}{|k|}\right), \quad |k| \to 0.

这一对数衰减是王为民表象实现红外内在正规化的数学根源。

3.2 电子自能单圈计算

在王为民QED中，单圈电子自能表达式为：

- i \Sigma_W(p) = (-ie)^2 \int \frac{d^4k}{(2\pi)^4} \gamma^\mu \frac{i(\not{k} + m)}{k^2 - m^2 + i\epsilon} \mathcal{F}_W(k) \gamma_\mu \frac{-i}{(p-k)^2 + i\epsilon} \mathcal{F}_W(p-k).

相比于标准QED，被积函数多了两个基底因子：电子传播子的 \mathcal{F}_W(k) 和光子传播子的 \mathcal{F}_W(p-k)。

在红外区域（k \to 0），光子传播子的因子 \mathcal{F}_W(p-k) \approx \mathcal{F}_W(p) \sim \ln(\mu/|p|) 保持有限，电子传播子的因子 \mathcal{F}_W(k) \sim \ln(\mu/|k|) 在 k \to 0 时提供对数衰减。这一衰减将标准QED中原本对数发散的软光子积分转化为有限积分：

\int_0^\Lambda \frac{dk}{k} \quad \longrightarrow \quad \int_0^\Lambda \frac{dk}{k} \ln\left(\frac{\mu}{k}\right) \sim \frac{1}{2} \ln^2\left(\frac{\mu}{\Lambda}\right) \quad \text{（有限值）}.

紫外区域（|k| \to \infty）的行为与标准QED相同，仍需要标准重整化程序处理紫外发散。王为民传播子在紫外区域趋于常数，不改变紫外发散的结构。红外与紫外行为泾渭分明：基底因子仅作用于红外区域，是实现内在正规化的核心机制。

3.3 顶角修正单圈计算

顶角修正的单圈表达式类似。在零动量转移极限（p' \to p）下，红外行为受光子传播子的基底因子 \mathcal{F}_W(k) 控制。该因子在 k \to 0 时的对数衰减同样将红外发散积分转化为有限值。

由于顶点因子 \gamma^\mu 在王为民表象中完全不变，且传播子逆的形式为 S_W^{-1}(p) = (\not{p} - m) \mathcal{F}_W(p)^{-1}，直接计算可验证：

\Lambda_W^\mu(p, p) = -\frac{\partial \Sigma_W(p)}{\partial p_\mu}.

Ward-Takahashi恒等式在王为民QED中严格成立，保证了规范对称性和理论自洽性。

四、电子自能红外行为的详细分析

4.1 红外有限性的解析证明

为证明王为民电子自能的红外有限性，考虑质壳电子（p^2 = m^2）情形，分析积分在 k \to 0 区域的渐近行为。取电子静止系（\mathbf{p} = 0, p_0 = m），虚光子动量 k 趋零。标准QED中，该区域贡献 \sim \int_0^\lambda dk/k，对数发散。

在王为民QED中，被积函数额外包含 \mathcal{F}_W(k) \mathcal{F}_W(p-k)。当 k \to 0 时，\mathcal{F}_W(k) \sim \ln(\mu/|k|) 的对数衰减使积分变为：

\int_0^\lambda \frac{dk}{k} \ln\left(\frac{\mu}{k}\right) \sim \frac{1}{2} \ln^2\left(\frac{\mu}{\lambda}\right) \quad \text{（有限）}.

红外发散被转化为对数平方项，完全可积。紫外区域的行为与标准QED一致，标准重整化程序依然适用。

五、结论

本文以QED单圈电子自能和顶角修正为例，展示了王为民粒子表象在红外发散正规化中的核心优势。主要结论为：

1. 红外发散天然有限：光子传播子的基底因子 \mathcal{F}_W(k) 在 k \to 0 时提供对数衰减，将标准QED中需要人工截断的软光子发散积分转化为有限值。

2. 紫外重整化不变：王为民传播子在紫外区域趋于标准传播子形式，标准重整化程序完全适用。

3. 规范对称性自洽：顶点因子不变保证了Ward-Takahashi恒等式的自动成立，红外有限性与规范对称性兼容。

4. 无需人工参数：与光子质量截断或维度正规化不同，王为民红外正规化不需要引入任何非物理的人工参数，有限性是基底选择的自然结果。

本文标志着王为民粒子表象正式成为可用于逐圈计算的物理工具。无人工参数的内在正规化方案是其区别于所有标准正规化方法的独特优势。
