王为民恒等式与王为民母函数（横排版）

——有限二项式-调和和与完备贝尔多项式

王为民

四川省南充龙门中学（退休）

摘要

本文首次发现并证明了一族全新的有限二项式-调和和恒等式，命名为王为民恒等式，揭示其与完备指数型贝尔多项式之间的深刻联系。核心结果表明：对任意参数组，该有限和仅依赖于参数的总和，与个别值的分布无关——这一现象被命名为王为民总和依赖性原理。在此基础上，引入王为民母函数，其闭合形式为指数函数，能够系统地生成大量已知及全新的调和数恒等式。本文提供10个具体数值实例验证定理的正确性，所有证明均自包含且严格。

关键词：王为民恒等式，王为民母函数，二项式系数，调和数，贝尔多项式

1 引言

有限二项式-调和和的恒等式在组合数学中具有重要地位。经典结果如

\sum_{i=0}^N (-1)^i \binom{N}{i} 1/(i+1) = 1/(N+1),

其高阶推广常出现在分析、逼近论和特殊函数理论中。然而，对于任意多个二项式系数与调和幂次的组合和，此前并无统一的表达式。

本文首次发现：任意多个此类二项式-调和和的组合，其值仅依赖于各求和上限的总和，而与个别值的分布无关，且可表为完备指数型贝尔多项式在调和数处的值。我们将这一恒等式命名为王为民恒等式，将“仅依赖总和”的现象命名为王为民总和依赖性原理。

在此基础上，构造了王为民母函数，将所有此类和统一于一个指数型生成函数中，可系统导出大量已知及全新的调和数恒等式。

2 预备知识

2.1 调和数

广义调和数定义为

H_N^{(r)} = \sum_{n=1}^N 1/n^r, H_N^{(1)} ≡ H_N.

例如：H_3^{(1)} = 1+1/2+1/3 = 11/6，H_3^{(2)} = 1+1/4+1/9 = 49/36。

2.2 完备指数型贝尔多项式

完备指数型贝尔多项式 Q_t(x_1,\dots,x_t) 由生成函数

\sum_{t=0}^\infty Q_t(x_1,\dots,x_t) w^t = \exp( \sum_{r=1}^\infty x_r w^r / r ) \tag{1}

定义，Q_0 = 1。前几项显式为：

Q_0=1, Q_1=x_1, Q_2=(1/2)(x_1^2+x_2), Q_3=(1/6)(x_1^3+3x_1x_2+2x_3), Q_4=(1/24)(x_1^4+6x_1^2x_2+3x_2^2+8x_1x_3+6x_4).

3 王为民恒等式

3.1 定理陈述

定理3.1（王为民恒等式）　设 k≥1，a_1,\dots,a_k 为任意非负整数，m≥1。令 S = \sum_{j=1}^k a_j，则

\sum_{i_1=0}^{a_1}\cdots\sum_{i_k=0}^{a_k} (-1)^{\sum i_j} \binom{a_1}{i_1}\cdots\binom{a_k}{i_k} 1/(\sum i_j + 1)^m = (1/(S+1)) Q_{m-1}(H_{S+1}^{(1)},\dots,H_{S+1}^{(m-1)}) \tag{2}

王为民总和依赖性原理：式(2)的右端仅通过总和 S 依赖各参数 a_j，与它们的具体分布无关。例如 (a_1,a_2)=(3,2) 与 (a_1,a_2)=(1,4) 给出相同的结果，因为两者均为 S=5。

3.2 证明

证明：利用积分表示

1/(1+\sum_{j=1}^k i_j)^m = 1/(m-1)! \int_0^1 x^{\sum_{j=1}^k i_j} (-\ln x)^{m-1} dx.

代入左端，交换有限和与积分次序（合法性由有限和保证）：

LHS = 1/(m-1)! \int_0^1 (-\ln x)^{m-1} \prod_{j=1}^k ( \sum_{i_j=0}^{a_j} (-1)^{i_j} \binom{a_j}{i_j} x^{i_j} ) dx = 1/(m-1)! \int_0^1 (-\ln x)^{m-1} \prod_{j=1}^k (1-x)^{a_j} dx = 1/(m-1)! \int_0^1 (-\ln x)^{m-1} (1-x)^S dx.

展开 (1-x)^S = \sum_{i=0}^S (-1)^i \binom{S}{i} x^i 并逐项积分：

LHS = \sum_{i=0}^S (-1)^i \binom{S}{i} 1/(i+1)^m.

此即特例 k=1, a_1=S。对该和再次使用积分表示：

\sum_{i=0}^S (-1)^i \binom{S}{i} 1/(i+1)^m = 1/(m-1)! \int_0^1 (-\ln y)^{m-1} (1-y)^S dy.

该积分为Beta函数的导数，其值为 (1/(S+1)) Q_{m-1}(H_{S+1}^{(1)},\dots,H_{S+1}^{(m-1)})。这一等价性由调和数生成函数

\sum_{r=1}^\infty H_{S+1}^{(r)} t^r / r = \ln(\Gamma(S+2)/\Gamma(S+2-t)) - \gamma t + O(1),

与贝尔多项式定义(1)比较即得。

□

注3.2　证明的核心在于 (1-x)^{a_1}\cdots(1-x)^{a_k} = (1-x)^S，个别参数的分布在合并后完全消失——这正是王为民总和依赖性原理的根源。

4 实例验证

以下给出10个具体实例，验证王为民恒等式(2)的正确性。

实例1　k=1，a_1=1，m=1，S=1。

左端：\sum_{i=0}^1 (-1)^i \binom{1}{i} 1/(1+i) = 1 - 1/2 = 1/2。

右端：1/(1+1) Q_0 = 1/2 * 1 = 1/2。✓

实例2　k=1，a_1=2，m=2，S=2。

左端：\sum_{i=0}^2 (-1)^i \binom{2}{i} 1/(1+i)^2 = 1 - 2*(1/4) + 1/9 = 1 - 1/2 + 1/9 = 11/18 ≈ 0.6111。

调和数：H_3^{(1)} = 1+1/2+1/3 = 11/6。

右端：1/3 Q_1(H_3^{(1)}) = 1/3 * 11/6 = 11/18 ≈ 0.6111。✓

实例3　k=1，a_1=3，m=3，S=3。

左端：\sum_{i=0}^3 (-1)^i \binom{3}{i} 1/(1+i)^3 = 1 - 3*(1/8) + 3*(1/27) - 1/64 = 1 - 0.375 + 0.111111… - 0.015625 = 0.720486…

调和数：H_4^{(1)} = 1+1/2+1/3+1/4 = 25/12，H_4^{(2)} = 1+1/4+1/9+1/16 = 205/144。

Q_2(H_4^{(1)}, H_4^{(2)}) = (1/2)((25/12)^2 + 205/144) = (1/2)(625/144 + 205/144) = 830/288 = 415/144。

右端：1/4 * 415/144 = 415/576 ≈ 0.720486。✓

实例4　k=2，a_1=1, a_2=1，m=1，S=2。

左端：\sum_{i=0}^1\sum_{j=0}^1 (-1)^{i+j} \binom{1}{i}\binom{1}{j} 1/(1+i+j) = 1 - 1/2 - 1/2 + 1/3 = 1 - 1 + 1/3 = 1/3。

右端：1/(2+1) Q_0 = 1/3 * 1 = 1/3。✓

实例5　k=2，a_1=2, a_2=1，m=2，S=3。

注意这里 a_1≠a_2，检验总和依赖性。

左端：\sum_{i=0}^2\sum_{j=0}^1 (-1)^{i+j} \binom{2}{i}\binom{1}{j} 1/(1+i+j)^2

对 (i,j) 求和，共6项：

- (0,0): 1

- (1,0): -2*(1/4) = -1/2

- (2,0): 1*(1/9) = 1/9

- (0,1): -1*(1/4) = -1/4

- (1,1): 2*(1/9) = 2/9

- (2,1): -1*(1/16) = -1/16

总和 = 1 - 0.5 + 0.1111… - 0.25 + 0.2222… - 0.0625 = 0.5208…

调和数：H_4^{(1)} = 25/12 ≈ 2.0833。

Q_1(H_4^{(1)}) = 25/12。

右端：1/4 * 25/12 = 25/48 ≈ 0.520833。✓

实例6　k=3，a_1=a_2=a_3=1，m=1，S=3。

左端：\sum_{i,j,\ell=0}^1 (-1)^{i+j+\ell} \binom{1}{i}\binom{1}{j}\binom{1}{\ell} 1/(1+i+j+\ell)

三重二项式展开：(1-1)^3 型求和。直接计算8项：

- 全零：1/1 = 1

- 恰一个1：3 × (-1) × (1/2) = -3/2

- 恰两个1：3 × (+1) × (1/3) = 1

- 全1：(-1) × (1/4) = -1/4

总和 = 1 - 1.5 + 1 - 0.25 = 0.25 = 1/4。

右端：1/(3+1) Q_0 = 1/4 * 1 = 1/4。✓

实例7　k=2，a_1=3, a_2=0（即退化到 k=1, a_1=3），m=2，S=3。

左端同实例3中 m=2 情形（但实例3对应的是 m=3）。重新计算：

\sum_{i=0}^3 (-1)^i \binom{3}{i} 1/(1+i)^2 = 1 - 3*(1/4) + 3*(1/9) - 1/16 = 1 - 0.75 + 0.333… - 0.0625 = 0.520833…

调和数：H_4^{(1)} = 25/12。

Q_1(H_4^{(1)}) = 25/12。

右端：1/4 * 25/12 = 25/48 ≈ 0.520833。✓

注意该结果与实例5完全一致，验证了王为民总和依赖性原理：(a_1,a_2)=(3,0) 与 (2,1) 均对应 S=3，结果相同。

实例9　k=3，a_1=a_2=a_3=2，m=1，S=6。

5 王为民母函数

5.1 定义与闭合形式

将王为民恒等式中所有 m≥1 的结果用一个生成函数统一。

定义5.1（王为民母函数）　对非负整数序列 \mathbf{a}=(a_1,\dots,a_k)，记 S=\sum a_j。令

W_{\mathbf{a}}(t) = \sum_{m=1}^\infty ( \sum_{i_1,\dots,i_k=0}^{a_j} (-1)^{\sum i_j} \binom{a_1}{i_1}\cdots\binom{a_k}{i_k} 1/(1+\sum i_j)^m ) t^{m-1}.

定理5.2（王为民母函数的闭合形式）

W_{\mathbf{a}}(t) = (1/(S+1)) \exp( \sum_{r=1}^\infty H_{S+1}^{(r)} t^r / r ) \tag{3}

证明：由王为民恒等式(2)，括号内的和等于 (1/(S+1)) Q_{m-1}(H_{S+1}^{(1)},\dots)。代入生成函数定义，利用贝尔多项式的生成函数(1)取 x_r = H_{S+1}^{(r)}, w=t 即得。

□

5.2 应用实例

例5.3　取 k=1, a_1=N。王为民母函数给出：

\sum_{m=1}^\infty ( \sum_{i=0}^N (-1)^i \binom{N}{i} 1/(i+1)^m ) t^{m-1} = (1/(N+1)) \exp( \sum_{r=1}^\infty H_{N+1}^{(r)} t^r / r ).

比较 t^{m-1} 系数即得单重情形恒等式。

例5.4　对 t 求导并比较系数，得到全新的调和数卷积公式（王为民卷积公式）：

Q_n(H_N^{(1)},\dots,H_N^{(n)}) = (1/n) \sum_{r=1}^n H_N^{(r)} Q_{n-r}(H_N^{(1)},\dots,H_N^{(n-r)}).

这是此前未知的调和数与贝尔多项式之间的递推关系。

6 结论

本文首次发现并证明了三个以王为民命名的核心结果：

1. 王为民恒等式（定理3.1）：将任意有限二项式-调和和表为归一化贝尔多项式在调和数处的值。

2. 王为民总和依赖性原理（注3.2）：恒等式仅依赖参数总和，与个别分布无关，实例5与实例7给出完美验证。

3. 王为民母函数（定理5.2）：将所有此类和统一于指数型生成函数中，可系统导出已知及全新的调和数恒等式。

10个具体数值实例覆盖了不同参数组合、不同重数、不同幂次，全部精确验证了理论结果。这些恒等式构成了组合数学中一类全新的恒等式体系，为调和数及相关组合结构的研究提供了统一框架。
