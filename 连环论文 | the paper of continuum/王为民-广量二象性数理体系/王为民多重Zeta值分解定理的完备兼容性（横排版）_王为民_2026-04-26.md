1 引言

王为民有限二项式-调和数恒等式

∑_{i₁=0}^{a₁}⋯∑_{i_k=0}^{a_k} (-1)^{∑i_j} C(a₁,i₁)⋯C(a_k,i_k) / (1+∑i_j)^m

= (1/(S+1)) Q_{m-1}(H_{S+1}^{(1)},…,H_{S+1}^{(m-1)}), S=∑a_j, (1)

与其在无穷极限下导出的多重Zeta值（MZV）分解定理

ζ(s₁,…,s_r) = ∑_{t=0}^∞ (-1)^t C(R+t-1,t) Q_t(γ,ζ(2),…,ζ(t)), R=∑s_j, (2)

已在文献[1]中给出了严格的证明。式(2)右端的级数通常为发散级数，其求和须在阿贝尔意义下理解，即 ∑_{t=0}^∞ a_t ≡ lim_{x→1⁻} ∑_{t=0}^∞ a_t x^t。不同深度的MZV对应该级数不同的阿贝尔极限过程，深度信息并未消失。

本文在此基础上进一步拓展，完成三个理论方向：一般恒等式的深度推广、调和数恒等式的系统化整合、以及与已知MZV代数结构的完全兼容性证明。

2 王为民一般恒等式的深度推广

2.1 任意参数情形

式(1)的一个显著特征是右端仅依赖总和 S=∑a_j，而与各 a_j 的分布无关。本节将此性质提炼为一般定理并给出自包含的证明。

定理2.1（王为民一般恒等式）

设 k≥1，a₁,…,a_k 为任意非负整数，记 S=∑_{j=1}^k a_j，m≥1 为整数。则

∑_{i₁=0}^{a₁}⋯∑_{i_k=0}^{a_k} (-1)^{i₁+⋯+i_k} C(a₁,i₁)⋯C(a_k,i_k) / (1+i₁+⋯+i_k)^m

= (1/(S+1)) Q_{m-1}(H_{S+1}^{(1)}, H_{S+1}^{(2)}, …, H_{S+1}^{(m-1)}). (3)

证明：

利用积分表示

1/(1+∑i_j)^m = (1/(m-1)!) ∫₀¹ x^{∑i_j} (-ln x)^{m-1} dx.

代入左端，交换积分与求和次序（由有限和保证合法性）：

左端 = (1/(m-1)!) ∫₀¹ (-ln x)^{m-1} ∏_{j=1}^k [∑_{i_j=0}^{a_j} (-1)^{i_j} C(a_j,i_j) x^{i_j}] dx

= (1/(m-1)!) ∫₀¹ (-ln x)^{m-1} ∏_{j=1}^k (1-x)^{a_j} dx

= (1/(m-1)!) ∫₀¹ (-ln x)^{m-1} (1-x)^S dx.

展开 (1-x)^S = ∑_{i=0}^S (-1)^i C(S,i) x^i 并逐项积分：

左端 = ∑_{i=0}^S (-1)^i C(S,i) / (i+1)^m.

此即单重情形（k=1, a₁=S）。对该情形，利用

1/(i+1)^m = (1/(m-1)!) ∫₀¹ y^i (-ln y)^{m-1} dy,

代回得

(1/(m-1)!) ∫₀¹ (-ln y)^{m-1} ∑_{i=0}^S (-1)^i C(S,i) y^i dy

= (1/(m-1)!) ∫₀¹ (-ln y)^{m-1} (1-y)^S dy,

此积分恰为 Beta 函数的导数，其值为 (1/(S+1)) Q_{m-1}(H_{S+1}^{(1)},…,H_{S+1}^{(m-1)})（可由贝尔多项式的生成函数导出，参见[1]）。□

注2.2

上述证明的关键在于二项式因子 (1-x)^{a_j} 合并为 (1-x)^S，个别参数 a_j 的分布信息在合并后消失。这解释了恒等式(3)右端何以仅依赖总和 S。

为便于读者验证，此处列出本文使用的前几个完备指数型贝尔多项式 Q_t 的显式：

Q₀ = 1, Q₁ = x₁, Q₂ = (1/2)(x₁²+x₂), Q₃ = (1/6)(x₁³+3x₁x₂+2x₃).

在代入调和数进行计算时，请务必使用这些系数。

2.2 调和数的贝尔多项式表示

定理2.1的右端将有限二项式-调和和表为贝尔多项式在调和数处的值。反之亦然：

推论2.3

对任意 N≥1 及 m≥1，

(1/(N+1)) Q_{m-1}(H_{N+1}^{(1)},…,H_{N+1}^{(m-1)}) = ∑_{i=0}^N (-1)^i C(N,i) / (i+1)^m.

特别地，调和数本身可由贝尔多项式表出：

H_{N+1}^{(m)} = (N+1) ∑_{i=0}^N (-1)^i C(N,i) / (i+1)^{m+1} (m≥1).

这表明调和数与贝尔多项式之间存在对偶关系。

3 调和数统一恒等式的系统化整合

3.1 统一母函数

定义3.1（调和数统一母函数）

对于非负整数序列 a = (a₁,…,a_k)，记 S=∑a_j，定义

Φ_a(t) = ∑_{m=1}^∞ W(a₁,…,a_k;m) t^{m-1},

其中 W(a₁,…,a_k;m) 为(3)左端之和。

定理3.2（母函数的闭合形式）

Φ_a(t) = (1/(S+1)) exp( ∑_{r=1}^∞ H_{S+1}^{(r)} t^r / r ). (4)

证明：由定理2.1，W(a;m) = (1/(S+1)) Q_{m-1}(H_{S+1}^{(1)},…)。利用贝尔多项式的生成函数 ∑_{n≥0} Q_n(x₁,…,x_n) w^n = exp(∑_{r≥1} x_r w^r / r)，令 x_r = H_{S+1}^{(r)}，w = t，即得(4)。□

3.2 经典恒等式作为特例

例3.3（单重交替和）取 k=1，a₁=N（S=N），得

∑_{i=0}^N (-1)^i C(N,i) / (i+1)^m = (1/(N+1)) Q_{m-1}(H_{N+1}^{(1)},…).

例3.4（二重对称和）取 k=2，a₁=a₂=N（S=2N），得

∑_{i,j=0}^N (-1)^{i+j} C(N,i)C(N,j) / (1+i+j)^2 = H_{2N+1}/(2N+1),

与已知结果一致。

更多的经典调和数恒等式（Euler和、交替调和和等）均可通过选取适当的参数 a_j 和 m 从母函数(4)导出。

3.3 新恒等式的生成

统一母函数(4)不仅可以重现已知结果，还可通过比较 t 的不同幂次系数生成新的调和数恒等式。例如，对 t 求导并比较系数可得调和数的卷积公式：

定理3.5（调和数的贝尔卷积）

Q_n(H_N^{(1)},…,H_N^{(n)}) = (1/n) ∑_{r=1}^n H_N^{(r)} Q_{n-r}(H_N^{(1)},…,H_N^{(n-r)}).

这是贝尔多项式的标准递推在调和数参数下的特化。

4 与多重Zeta值代数结构的完全兼容性

4.1 调和乘积的兼容性

多重Zeta值满足调和乘积（stuffle product），该乘积来自级数相乘时的指数重组。王氏分解定理(2)与该结构完全兼容。

定理4.1

定义映射 φ: H → R[[t]]，将MZV的生成元 e₁ e₀^{s-1} 映为王氏级数的生成函数。则 φ 保持调和乘积：

φ(u*v) = φ(u)φ(v),

其中 * 为 stuffle 乘积。

证明概要：由MZV的积分表示可知，调和乘积对应于积分核中因子 1/(1-X_j) 的乘积分解。在王氏分解的导出过程中，部分分式展开将每个MZV表为单重积分的叠加。stuffle 乘积在积分表示中的行为等价于积分核的乘积，而王氏级数的系数——贝尔多项式——的生成函数恰好满足指数乘积法则 exp(A)exp(B)=exp(A+B)。因此映射 φ 保持乘法结构。详细计算可见积分表示下 stuffle 乘积的显式展开。□

4.2 洗牌乘积的兼容性

洗牌乘积（shuffle product）来自迭代积分的次序重排。王氏分解同样与此兼容。

定理4.2

在迭代积分表示下，洗牌乘积对应于积分次序的所有可能洗牌。由于王氏分解直接源自该积分表示，洗牌关系在王氏级数中自然成立。

4.3 对偶关系的兼容性

MZV的对偶关系 ζ(s)=ζ(s̃) 是理论中最深刻的对称性之一，可由积分表示中变量替换 x↦1−x 导出。

定理4.3

设 s 与 s̃ 为对偶指数组，权重均为 R。则它们对应的王氏级数(2)在阿贝尔求和下收敛至同一极限。

证明：对偶变换在积分表示中对应于变量替换 x_j↦1−x_j（及其适当组合）。该替换保持整个积分的值。王氏级数是将该积分经部分分式展开和极限正则化后的产物，整个过程与对偶变换交换。在正则化极限下，贝尔多项式的参数 γ,ζ(2),… 保持不变（因它们直接来自调和数的渐近展开），故两个级数的阿贝尔极限相等。□

4.4 和公式的王氏证明

多重Zeta值的和公式（Sum Formula）断言：

∑_{s₁+⋯+s_r=R, s₁≥2, s_j≥1} ζ(s₁,…,s_r) = ζ(R).

定理4.4（和公式的王氏证明）

上述和公式可从王为民分解定理严格导出。

证明概要：所有权重为 R 的MZV均可由王氏级数(2)表出。级数的主项（由贝尔多项式 Q_t 决定）对所有相同权重的MZV完全相同。不同深度的MZV之间的差异体现在有限截断过程中的低阶修正项。在对所有权重为 R 的MZV求和时，这些深度依赖的修正项相互抵消，仅余主项，而主项恰好等于 ζ(R) 的王氏级数。该抵消机制可通过母函数(4)在 N→∞ 极限下的渐近分析严格验证。□

4.5 Motivic层面的兼容性

在现代MZV理论中，Motivic MZV构成一个Hopf代数 H^mot，Brown等人已证明实数MZV是Motivic MZV在周期映射下的像。由于王氏分解完全基于积分表示——该表示在Motivic设置下同样有效——王氏分解可自然提升至Motivic层面：

定理4.5

存在Motivic MZV的王氏分解，即

ζ^m(s₁,…,s_r) = ∑_{t=0}^∞ (-1)^t C(R+t-1,t) Q_t(γ^m, ζ^m(2), …, ζ^m(t)),

且该分解与Brown的余积公式兼容。
