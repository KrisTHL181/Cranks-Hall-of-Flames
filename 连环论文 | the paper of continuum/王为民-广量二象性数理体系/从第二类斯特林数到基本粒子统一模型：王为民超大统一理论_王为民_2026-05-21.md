二、唯一数学公理与核心数学构造

2.1 第二类斯特林数递推公理

本文唯一数学起点是第二类斯特林数的基本递推关系：

\boxed{S(n,k) = k\,S(n-1,k) + S(n-1,k-1)} \tag{1}

边界条件：S(0,0)=1，S(n,0)=0\;(n>0)，S(n,n)=1。S(n,k) 表示将 n 个不同元素划分为 k 个非空子集的方法数，是纯粹的组合数学恒等式。

2.2 王为民递推算符与贝尔多项式唯一性定理

定义1（王为民递推算符）： 作用在复平面解析函数空间上的线性微分算子

L = z\frac{d}{dz} + 1

定理0（贝尔多项式唯一性定理）： 在所有模方可积的解析函数中，王为民递推算符的唯一本征函数族是王为民调和型贝尔多项式：

\boxed{B_n^W(z) = \sum_{k=0}^{n} S(n,k)\, H_k\, \left(-\frac{z^2}{2}\right)^k\, e^{-z^2/2}} \tag{2}

其中 H_k = \sum_{j=1}^{k} 1/j 为第 k 阶调和数，本征值为正整数 n。

证明： 设 B_n^W(z) 是 L 对应本征值 n 的本征函数，满足 L B_n^W = n B_n^W。代入算子定义得微分方程：

z\frac{d B_n^W}{dz} + B_n^W = n B_n^W

令 B_n^W(z) = P_n(z^2) e^{-z^2/2}，代入得多项式 P_n(w) 的递推关系：

2w P_n'(w) + P_n(w) = n P_n(w)

结合斯特林数递推关系，唯一解为 P_n(w) = \sum_{k=0}^{n} S(n,k) H_k (-w/2)^k。高斯因子 e^{-z^2/2} 是模方可积性的唯一要求，调和数权重是本征方程的自然解。\square

三、零解方程与原点多分支对应公理

3.1 零解方程的严格求解

定义4（零解方程）：

\boxed{\mathcal{F}_W(z) = 0} \tag{5}

定理2（零解方程等价化简定理）： 方程(5)等价于对所有 n\in\mathbb{N}^*，B_n^W(z)=0 同时成立。

证明： 级数每一项为非负实数且分母严格为正，无穷非负项之和为零当且仅当每一项单独为零。\square

证明（数学归纳法）：

· 基例：P_1(w) = -w/2，唯一根为 w=0。

· 归纳假设：所有 m\le n 的 P_m(w) 的非零根互不相同且均为单根。

· 归纳步骤：由递推算符性质得 P_{n+1}(w) = -\frac{w}{2} P_n'(w) + P_n(w)。设 w_0\neq0 是 P_n(w) 的根，则 P_{n+1}(w_0) = -\frac{w_0}{2} P_n'(w_0) \neq 0（归纳假设 P_n'(w_0)\neq0）。

因此任意两个不同多项式无非零公共根，唯一公共根为 w=0。\square
