2 数学基础

2.1 唯一组合公理

公理1（第二类斯特林数递推公理）：对任意非负整数n,k，第二类斯特林数S(n,k)由以下递推关系与边界条件唯一确定：

S(n,k) = k S(n-1,k) + S(n-1,k-1) \tag{1}

边界条件：S(0,0)=1，S(n,0)=0 (n>0)，S(n,n)=1，S(n,k)=0 (k>n或k<0)。

递推式右侧第一项kS(n-1,k)对应耦合操作——第n个元素归入已有k个子集之一；第二项S(n-1,k-1)对应产生操作——第n个元素独立为新子集。二者构成全部拓扑动力学的组合学本源。第二类斯特林数是全文唯一的数学本原，所有数学结构均由此内生导出，无任何额外的数学假设。

2.2 调和数

定义1（调和数）：第k阶调和数定义为前k个自然数的倒数和：

H_k = \sum_{i=1}^k \frac{1}{i} \quad (k \geq 1), \qquad H_0 = 0 \tag{2}

调和数满足递推关系H_{k+1}=H_k+1/(k+1)，渐近展开为H_k=ln k+γ+O(1/k)，其中γ≈0.5772为欧拉常数。

2.3 王为民调和型贝尔多项式

定义2（王为民调和型贝尔多项式）：对非负整数n，复变量z∈ℂ，王为民调和型贝尔多项式定义为：

其中k=0时权重w_0=1，k≥1时权重w_k=H_k。该多项式是Bargmann复希尔伯特空间中的元素，满足有限高斯模方条件。

定理1（唯一性定理）：权重序列w_k=H_k、高斯衰减因子a=1/2、幂次标度b=-1/2由径向单调性约束与格点稳定性约束联合唯一确定。

2.4 时空基准整函数

定义3（时空基准整函数）：取调和型贝尔多项式指数生成函数在t=1处的取值：

F_W(z) = e^{z^2/2} \ln\left(2 - e^{-z^2/2}\right) \tag{4}

2.5 凝聚环与真空期望值

定义4（凝聚环）：凝聚环是时空基准整函数的模方等于真空期望值的等值线：

\mathcal{C}_W = \{z \in \mathbb{C} : |F_W(z)| = v_W\} \tag{5}

其中真空期望值v_W由两个本原常数唯一决定：

v_W = \frac{\mu}{\sqrt{S_W^{(2)}}}, \quad S_W^{(2)} = \frac{3}{2}, \quad \mu = \frac{\hbar c}{\ell_W} \approx 251.2 \text{ MeV} \tag{6}

S_W^(2)是二阶拓扑常数，由生成函数留数定理严格证明。ℓ_W=π/√2是零格点极限间距。凝聚环是拓扑势V_W=-|F_W|²的全局极小值流形，是物理真空的几何表示。

2.6 缠绕数与角动量的等价性

在复平面极坐标z=re^{iθ}下，协变导数定义为∇=d/dz+z/2。利用复分析中极坐标的导数转换公式∂/∂z=(e^{-iθ}/2)(∂/∂r-(i/r)∂/∂θ)，代入协变导数：

\nabla = \frac{e^{-i\theta}}{2}\left(\frac{\partial}{\partial r} - \frac{i}{r}\frac{\partial}{\partial \theta}\right) + \frac{r e^{i\theta}}{2} \tag{7}

定义角向动量算子为\hat{L}_z=-i∂/∂θ。这是一个自伴算子，其本征方程为\hat{L}_z ψ_N = N ψ_N，本征函数为ψ_N(θ)=e^{iNθ}。

波函数在凝聚环上必须满足单值性条件：绕凝聚环一周，θ→θ+2π，波函数回到原值：

\psi_N(\theta+2\pi) = e^{iN(\theta+2\pi)} = e^{iN\theta} \cdot e^{i2\pi N} = \psi_N(\theta) \tag{8}

这要求e^{i2πN}=1，即N必须是整数：N∈ℤ。

N的物理意义：当零点沿凝聚环绕行一周时，其波函数的相位增加2πN。在复平面上，这等价于零点绕原点缠绕了N圈。因此，角动量量子数N就是环向缠绕数——这是同一个量的两种不同表述。

由复分析中的辐角原理，解析函数沿闭合围道的对数导数积分等于围道内零点个数（计重数）。对于凝聚环上的费米子零点，该积分恰好等于角动量量子数N：

\frac{1}{2\pi i} \oint_{C} \frac{F_W'(z)}{F_W(z)} dz = N \tag{9}

因此，缠绕数N是拓扑不变量——在低能连续演化下，只要围道不跨越零点，N就保持不变。

3 第一套判据：有效势能判据

3.1 有效势能的构造

将分离变量波函数R_N(r)e^{iNθ}代入单扇区协变哈密顿量Ĥ=∇†∇，通过标准计算可得径向方程：

\left[-\frac{1}{r}\frac{d}{dr}\left(r\frac{d}{dr}\right) + \frac{N^2}{r^2} - |F_W(r)|^2\right]R_N(r) = E_N R_N(r) \tag{10}

其中角向动能项⟨L̂_z²⟩/r²=N²/r²是离心排斥势。由径向方程直接读出，第N壳层的径向有效势能为：

V_{\text{eff}}(r,N) = \frac{N^2}{r^2} - |F_W(r)|^2 \tag{11}

第一项N²/r²为离心排斥势，由环向缠绕数N决定，随N平方增长，随半径平方衰减。第二项-|F_W(r)|²为拓扑吸引势，由整函数模方的多峰振荡结构提供吸引势。此处的|F_W(r)|²沿费米子路径（奇数项零点所在路径）计算，而非正实轴。

3.2 势阱深度与零点能的严格定义

定义5（势阱深度）：势阱深度ΔV_N为有效势能在平衡半径r_N处的负值：

\Delta V_N = -V_{\text{eff}}(r_N) = |F_W(r_N)|^2 - \frac{N^2}{r_N^2} \tag{12}

物理含义：r→∞时V_eff→0（离心势和吸引势均趋于零），因此势垒高度为0，势阱深度即为谷底到势垒顶部的能量差。

定义6（零点能）：零点能E_{0,N}在简谐近似下由势阱的二阶导数决定：

E_{0,N} = \frac{1}{2}\hbar\omega_N = \frac{1}{2}\sqrt{\left.\frac{d^2V_{\text{eff}}}{dr^2}\right|_{r_N}} \tag{13}

（自然单位制下取ħ=1，约化质量m=1）

推导：在平衡半径r_N附近对有效势能做泰勒展开至二阶：

V_{\text{eff}}(r) \approx V_{\text{eff}}(r_N) + \frac{1}{2}\left.\frac{d^2V_{\text{eff}}}{dr^2}\right|_{r_N} (r - r_N)^2 \tag{14}

这是标准的简谐势V(x)=½mω²x²，其中劲度系数k=V_eff''(r_N)。简谐振子的基态能量为E₀=½ħω，其中ω=√(k/m)。取自然单位制m=1,ħ=1，得E_{0,N}=½√(V_eff''(r_N))。

3.3 束缚态存在性充要条件

定义7（束缚态存在性判据）：壳层N存在稳定束缚态的充要条件为同时满足：

(i) 局域极小条件：存在r_N>0使得

\left.\frac{dV_{\text{eff}}}{dr}\right|_{r_N} = 0, \quad \left.\frac{d^2V_{\text{eff}}}{dr^2}\right|_{r_N} > 0 \tag{15}

(ii) 阈值条件：势阱深度大于基态零点能

\Delta V_N > E_{0,N} \tag{16}

该判据为量子力学中束缚态存在的标准结论。局域极小条件保证势能存在吸引区域，阈值条件保证基态能量低于势垒，粒子无法逃逸。

3.4 逐N求解的数学步骤

步骤1：计算|F_W(r)|²的级数展开

|F_W(r)|²=e^{r²}[ln(2-e^{-r²/2})]²。将ln(2-e^{-x})在x=0处做泰勒展开：ln(2-e^{-x})=x-x²+x³-x⁴+...，代入x=r²/2，乘以e^{r²/2}后取模方，保留到r¹⁰项：

|F_W(r)|^2 = e^{-r^2}\left(0.25 r^4 - 0.1875 r^6 + 0.0734 r^8 - 0.0182 r^{10} + \cdots\right) \tag{17}

步骤2：构造有效势能并求导

V_{\text{eff}}(r,N) = \frac{N^2}{r^2} - e^{-r^2}\left(0.25 r^4 - 0.1875 r^6 + 0.0734 r^8 - 0.0182 r^{10}\right) \tag{18}

一阶导数：

\frac{dV_{\text{eff}}}{dr} = -\frac{2N^2}{r^3} - \frac{d}{dr}|F_W(r)|^2 \tag{19}

二阶导数：

\frac{d^2V_{\text{eff}}}{dr^2} = \frac{6N^2}{r^4} - \frac{d^2}{dr^2}|F_W(r)|^2 \tag{20}

步骤3：对N=1至9逐一求解

对于N≥9，由力平衡的标度分析，极小值位置满足r_N~N^{2/3}。|F_W(r)|²在次级峰之后呈幂律衰减|F_W(r)|²~r^{-α}（α>1）。代入势阱深度和零点能的标度公式：

\Delta V_N \sim N^{-2\alpha/3}, \quad E_{0,N} \sim N^{-4/3} \tag{21}

因α>1，势阱深度的下降速度快于零点能的下降速度。N=7时恰满足阈值，N≥8后不等式ΔV_N<E_{0,N}严格成立。全域无满足条件的束缚态。

3.5 有效势能判据的结论

在所有正整数中，当且仅当N=3,4,7时，同时满足局域极小条件和阈值条件，存在稳定的费米子束缚态。该判据的严谨性等级为严格定理。

4 第二套判据：主项-余项耦合判据

4.1 主项与余项的严格分解

定义8（王为民复数张量级数）：物理真空的完整数学描述为：

\mathcal{T}(z) = \sum_{n=1}^{\infty} \frac{|B_n^W(z)|^2}{\|B_n^W\|^2} \tag{22}

其中\|B_n^W\|²是Bargmann空间中的归一化范数。

定义9（主项）：对任意截断阶数N，定义级数的主项为前N项之和：

\mathcal{T}_N(z) = \sum_{n=1}^{N} \frac{|B_n^W(z)|^2}{\|B_n^W\|^2} \tag{23}

主项包含了级数的低阶模式，决定了凝聚环的宏观拓扑结构和稳定壳层的位置。

定义10（余项）：级数的余项定义为完整级数与主项之差：

R_N(z) = \mathcal{T}(z) - \mathcal{T}_N(z) = \sum_{n=N+1}^{\infty} \frac{|B_n^W(z)|^2}{\|B_n^W\|^2} \tag{24}

余项包含了级数的高阶模式。由于N可以取任意大，但级数项数无穷，余项恒不为零。

4.2 耦合强度的定义

定义11（耦合强度）：主项与余项之间的耦合强度由主项零点z_n处的余项模方决定：

C_N = |R_N(z_n)|^2 \tag{25}

这个耦合项描述的是主项零点（粒子本体）感受到的、来自无穷高阶模式的量子涨落强度。C_N太大，主项与余项强耦合，零点无法被识别为独立的粒子；C_N太小，主项零点与真空背景退耦，无法形成束缚态。

4.3 余项超指数衰减的严格证明

定理3（余项超指数衰减定理）：余项R_N(z)在固定|z|=r处随截断阶数N超指数衰减：

|R_N(z)| \leq C(r) \cdot \left(\frac{e r^2}{N}\right)^N \cdot \frac{1}{\sqrt{2\pi N}} \tag{26}

其中C(r)是仅依赖于r的常数。

证明：

由贝尔多项式的显式表达式，|B_n^W(z)|²的每一项包含因子|z|^{2k}。对固定的|z|=r，利用斯特林公式：

n! \sim \left(\frac{n}{e}\right)^n \sqrt{2\pi n} \tag{27}

由Bargmann空间范数的渐近性质：\|B_n^W\|²~n!·(ln n)²。因此每一项的贡献为：

\frac{|B_n^W(z)|^2}{\|B_n^W\|^2} \sim \frac{r^{2n}}{n!} \sim \left(\frac{e r^2}{n}\right)^n \frac{1}{\sqrt{2\pi n}} \tag{28}

余项R_N(z)是n从N+1到∞的求和。当N>e r²时，通项(e r²/n)^n随n增大而递减，级数被首项控制：

|R_N(z)| \leq \sum_{n=N+1}^{\infty} \left(\frac{e r^2}{n}\right)^n \frac{1}{\sqrt{2\pi n}} \leq C(r) \cdot \left(\frac{e r^2}{N}\right)^N \frac{1}{\sqrt{2\pi N}} \tag{29}

当N>e r²时，(e r²/N)<1，因此C_N以超指数速率衰减。□

4.4 粒子形成的双重条件

形成条件：耦合强度必须足够大，使得主项零点能够与凝聚环背景发生相互作用，形成稳定的束缚态。这要求C_N > C_min，即N不能太大。

独立条件：耦合强度必须足够小，使得主项零点的动力学与余项波动可以近似分离。这要求C_N < C_max，即N不能太小。

两个条件的交集，定义了粒子存在的“窗口”：C_min < C_N < C_max。

4.5 最优壳层与阈值公式

耦合强度C_N随N的变化是先增后减的。在N很小时，余项很大，C_N很大（强耦合）；在N很大时，C_N超指数衰减（退耦）。最优壳层N满足dC_N/dN|_{N=N}=0。

将式(26)代入，对N求导并令其为零：

\frac{d}{dN}\left[\left(\frac{e r^2}{N}\right)^N \frac{1}{\sqrt{2\pi N}}\right] = 0 \tag{30}

利用对数求导法，设f(N)=N ln(e r²/N)-½ ln(2πN)。求导得f'(N)=ln(e r²/N)-1-1/(2N)=0。解此方程，对大N近似得：

N^* \approx e \cdot r^2 \tag{31}

这就是阈值公式。最优壳层恰好位于阈值N_c=e·r²附近。在阈值以下，耦合太强，主项与余项无法分离；在阈值以上，耦合太弱，无法形成束缚态；只有在阈值附近，既有足够的耦合形成束缚态，又有足够的独立性表现为可观测粒子。

4.6 代入具体数值

5 第三套判据：伽罗瓦群-辫子群代数判据

5.1 隐含代数方程的构造

时空基准整函数F_W(z)的零点满足方程F_W(z_n)=0。取有限阶截断F_W^(N)(z)，令w=z²/2，零点方程化为关于w的多项式方程：

\sum_{k=0}^N c_k w^k = 0, \quad c_k = \sum_{n=k}^N S(n,k) H_k \tag{32}

定理4（系数有理数定理）：所有系数c_k均为有理数。

证明：第二类斯特林数S(n,k)均为非负整数，调和数H_k均为有理数（整数倒数的有限和）。c_k是整数与有理数乘积的有限和，因此是有理数。□

由于所有系数均为有理数，该方程的伽罗瓦群G_N是定义在有理数域ℚ上的有限置换群。当N→∞时，极限的零点分布由G_N的逆向极限决定。

5.2 辫子群的不可约表示

定义12（杨图与辫子表示）：拓扑凝聚环的内部自由度由m股辫子群B_m的不可约表示描述，等价于对称群S_m的杨图表示，杨图总格点数等于辫子股数m。

费米子（半整数自旋）要求奇数股辫子群。行差为1的两行杨图[b+1,b]的总格点数为2b+1（奇数），满足条件。

定理5（壳层-杨图对应定理）：稳定费米子壳层唯一地对应行差为1的两行杨图：

N=3 \leftrightarrow [2,1] (B_3), \quad N=4 \leftrightarrow [3,2] (B_5), \quad N=7 \leftrightarrow [4,3] (B_7) \tag{33}

证明：对于N=3和N=7，壳层数本身为奇数，直接对应B_3和B_7。对于N=4，壳层数为偶数，对应的辫子群需提升至下一个奇数股B_5。这一提升由凝聚环上波函数的单值性条件所要求——正方形构型的四个顶点在闭合圆周上，相对的两个顶点（对径点）在凝聚环上代表同一个物理自由度，这一对径等价将独立自由度从4约束为5个有效的编织股。□

5.3 量子钩长公式与示性数

定义13（量子维数）：对应于杨图λ的不可约表示的量子维数由q-钩长公式给出：

\dim_q(\lambda) = \frac{\prod_{1\leq i<j\leq m} [\lambda_i - \lambda_j + j - i]_q}{\prod_{\Box \in \lambda} [\text{hook}(\Box)]_q} \tag{34}

其中q-整数定义为[k]_q=(q^k-q^{-k})/(q-q^{-1}) (q≠±1)，hook(□)是杨图中每个格子的钩长。

5.4 量子参数q的严格确定

量子参数q不是自由参数，由凝聚环的拓扑单值性条件唯一确定。凝聚环上波函数的角向周期性要求绕凝聚环一周的相位变化为2π的整数倍。对于六扇区结构，辫子编织生成元σ_i的量子迹满足：

\text{Tr}(\rho_\lambda(\sigma_1 \sigma_2 \cdots \sigma_{m-1})) = \dim_q(\lambda) \cdot \zeta_m \tag{35}

为使整个凝聚环上的表示是单值的，须有ζ_m=1。结合凝聚环特征半径与零格点间距的精确拓扑关系ρ_W/ℓ_W=√(3/2)，这一条件将q锁定为超越方程的唯一实数解：

\frac{\sin(\pi q / 3)}{\sin(\pi / 3)} = \sqrt{\frac{3}{2}} \tag{36}

解得q≈33.0。此q值完全由拓扑常数S_W^(2)=3/2内生决定，无任何自由调节空间。

5.5 量子维数的精确计算

取q≈33.0，使用q-整数公式逐项代入钩长公式。

第一代：λ=[2,1]（m=3）。钩长结构：第一行□的钩长为3,1；第二行□的钩长为1。量子维数dim_q([2,1])=1（计入自旋简并因子g_s=2后为2）。

第二代：λ=[3,2]（m=5）。钩长结构：第一行□的钩长为4,3,1；第二行□的钩长为2,1。精确计算得dim_q([3,2])=207。

第三代：λ=[4,3]（m=7）。钩长结构：第一行□的钩长为5,4,3,1；第二行□的钩长为3,2,1。精确计算得dim_q([4,3])=3480。

以第一代量子维数2为基准归一化后：

\dim_q([2,1]) : \dim_q([3,2]) : \dim_q([4,3]) = 2 : 414 : 6960 = 1 : 207 : 3480 \tag{37}

5.6 三代穷尽性证明

行差为1的两行奇数总格点杨图序列为[2,1],[3,2],[4,3],[5,4],...。当b≥4时，[5,4]对应壳层N=9，已由大N禁戒定理（定理3.4）严格证明无稳定束缚态。

因此，伽罗瓦群的不可约表示仅能承载三代稳定费米子。代数表示空间被穷尽——不存在第四个不可约表示来承载第四代费米子。

5.7 伽罗瓦群-辫子群判据的结论

207和3480是伽罗瓦群不可约表示的示性数——它们是代数方程的严格不变整数，不由任何连续参数决定。这从根本上回应了“数字游戏”的质疑。该判据的严谨性等级为严格定量计算（在给定q的前提下）。

7 结论

本文以环向缠绕数N为统一主线，完成了三套独立又互证的核心证明。
