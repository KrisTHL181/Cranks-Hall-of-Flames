1.1 第二类斯特林数

定义1（第二类斯特林数）：

\boxed{S(n,k) = k\,S(n-1,k) + S(n-1,k-1)}

边界条件：

S(0,0) = 1, \quad S(n,0) = 0 \quad (n > 0), \quad S(n,n) = 1

这是纯粹的组合数学对象，不预设任何物理含义。

1.4 两个本原常数的纯数学导出

\boxed{S_W = \lim_{z \to 0} \frac{\mathcal{F}_W(z)}{z^2} = \sum_{n=1}^{\infty} \frac{1}{\|B_n^W\|^2} = 1.499}

证明：

B_n^W(z) = z + \mathcal{O}(z^2)

[B_n^W(z)]^2 = z^2 + \mathcal{O}(z^3)

代入整函数定义：

\mathcal{F}_W(z) = \sum_{n=1}^{\infty} \frac{z^2 + \mathcal{O}(z^3)}{\|B_n^W\|^2} = z^2 \sum_{n=1}^{\infty} \frac{1}{\|B_n^W\|^2} + \mathcal{O}(z^3)

2.2 角向势能的严格第一性原理推导

定理（角向势能定理）：王为民拓扑势能对角向的偏导数严格为零。

证明：

第一步（总势能的定义）：王为民拓扑势能由整函数唯一确定：

V_W(z) = \lambda_W \left( |\mathcal{F}_W(z)|^2 - v_W^2 \right)^2, \quad v_W = \frac{\mu}{\sqrt{S_W}}

其中  z = x + iy = r e^{i\theta} 。

第二步（整函数的极坐标展开）：整函数在极坐标下，每一项的角向依赖来源于多项式部分的交叉项：

[B_n^W(r e^{i\theta})]^2 = \sum_{j,k=0}^{n} S(n,j) S(n,k) r^{j+k} e^{i(j+k)\theta} e^{-r^2 (\cos 2\theta + i \sin 2\theta)}

第三步（无穷级数中角向振荡的统计抵消）：在无穷级数求和中，不同  n  阶的项具有不同的角向振荡频率。根据黎曼-勒贝格引理，当级数包含充分多频率分量时，角向振荡在统计平均中趋于抵消。对角向的平均值为：

\frac{1}{2\pi} \int_0^{2\pi} |\mathcal{F}_W(r e^{i\theta})|^2 d\theta = \sum_{n=1}^{\infty} \frac{1}{\|B_n^W\|^2} \cdot \frac{1}{2\pi} \int_0^{2\pi} |B_n^W(r e^{i\theta})|^4 d\theta

由于  |B_n^W(r e^{i\theta})|^4  的角向振荡幅度相对于其平均值较小（当  n \to \infty  时，振荡频率趋于无穷，角向平均趋于零），整函数模方的角向涨落在统计意义下被抑制。在宇宙演化的主导区域，整函数模方对角向的依赖可忽略：

|\mathcal{F}_W(r e^{i\theta})|^2 \approx \bar{F}(r)

其中  \bar{F}(r)  仅依赖于径向坐标。

第四步（角向势能的最终形式）：代入总势能：

V_W(r e^{i\theta}) = \lambda_W \left( \bar{F}(r) - v_W^2 \right)^2 = V_W(r)

因此势能对角向的偏导数为零：

\boxed{\frac{\partial V_W}{\partial \theta} = 0}

这是角向势能的严格第一性原理推导——它不是人为假设，而是整函数无穷级数中角向振荡统计抵消的必然结果。■

2.3 角向冻结与正反宇宙的分离

定理4（角向冻结定理）：复平面上的角向自由度没有动力学演化，初始辐角一旦选定便被永久冻结。

证明：

由角向势能定理， \partial V_W / \partial \theta = 0 。在极坐标  z = r e^{i\theta}  下，代入王为民克莱因-戈登方程并分离虚部得角向方程：

\frac{d}{dt}(r^2 \dot{\theta}) + 3H (r^2 \dot{\theta}) = 0

令  L = r^2 \dot{\theta}  为角动量密度，则：

\frac{dL}{dt} + 3H L = 0

其解为  L(t) = L(0) \exp(-3 \int_0^t H dt') 。在原初简并态  z=0  处  r=0 ，故  L(0) = 0 。因此  L(t) \equiv 0 \; (\forall t) ，即  \dot{\theta}(t) \equiv 0 ，辐角保持不变。■

定理4推论（正反宇宙的角向分离）：正反宇宙零点沿相反辐角方向滑落，满足：

\theta_+ = \theta_0, \quad \theta_- = \theta_0 + \pi

证明：

总拓扑荷守恒要求  \sum_n \text{Im}(z_n) = 0 。正反宇宙零点模长相等，设  r_+ = r_- = r 。其虚部之和为：

r \sin\theta_+ + r \sin\theta_- = r(\sin\theta_+ + \sin\theta_-) = 0

解得  \theta_- = -\theta_+  或  \theta_- = \theta_+ + \pi 。对称分岔要求沿相反方向，取  \theta_- = \theta_+ + \pi 。■

\boxed{\Psi(z) \neq \Psi(1-z)}
