王为民粒子表象中的自由旋量场

作者：王为民

单位：四川省南充龙门中学（退休）

摘要

本文在自由实标量场王为民粒子表象的基础上，将其推广至自由狄拉克旋量场。核心构造为：保持标准旋量场的螺旋度结构与洛伦兹变换性质不变，将每个动量模的径向部分从平面波替换为王为民调和型贝尔多项式 Q_n^W(\omega_k r)。由此定义的王为民旋量模产生/湮灭算符满足标准反对易关系，保证了自旋-统计定理的成立和Fock空间构造的自洽性。王为民表象下的旋量场Feynman传播子天然携带对数红外修正，其代数结构来源于基函数的调和数渐近行为；传播子的狄拉克矩阵结构 \gamma^\mu p_\mu + m 保持不变，确保了洛伦兹协变性和规范对称性。本文同时讨论了该表象在手征对称性、非相对论极限下的Efimov物理中的应用前景。全文不引入任何超出标准正则量子化框架的假设，所构造的表象与标准平面波展开严格幺正等价。

关键词：王为民调和型贝尔多项式；狄拉克旋量场；自由费米子场；反对易关系；费曼传播子；红外正规化

一、引言

自由旋量场（狄拉克场）是量子场论中描述自旋-1/2费米子的基本对象。其标准正则量子化将场算符按平面波和螺旋度旋量展开，每个动量模等价于一个费米子谐振子。这一构建方式确保了洛伦兹不变性、自旋-统计定理和微因果性的成立[4]。

在先前工作中[1]，我们构建了自由实标量场的王为民粒子表象。该表象的核心思想是将每个动量模的标准谐振子替换为对数谐振子，使传播子天然携带对数红外修正。本文将这一方案推广至旋量场。推广遵循以下三条原则：

1. 保持旋量场的洛伦兹变换性质（旋量结构）不变；

2. 仅将径向标量部分的平面波基替换为王为民多项式基；

3. 费米子反对易关系严格保持，确保自旋-统计定理成立。

这三条原则是量子场论表象变换的标准合法路径：表象由Fock空间的基底完全确定，基底的选择不影响算符的代数结构，仅改变单粒子波函数的具体形式。

二、标准狄拉克场的正则量子化回顾

2.1 狄拉克方程与平面波解

自由狄拉克场 \psi(x) 满足狄拉克方程：

(i\gamma^\mu \partial_\mu - m) \psi(x) = 0.

正频和负频平面波解为：

\psi^{(+)}(x) = u^s(\mathbf{k}) e^{-ikx}, \quad \psi^{(-)}(x) = v^s(\mathbf{k}) e^{ikx},

其中 s = \pm 1/2 为螺旋度指标，kx = \omega_k t - \mathbf{k}\cdot\mathbf{x}，\omega_k = \sqrt{\mathbf{k}^2 + m^2}。

旋量 u^s(\mathbf{k}) 和 v^s(\mathbf{k}) 满足正交归一和完备性关系：

u^{r\dagger}(\mathbf{k}) u^s(\mathbf{k}) = v^{r\dagger}(\mathbf{k}) v^s(\mathbf{k}) = 2\omega_k \delta^{rs},

\sum_{s} u^s(\mathbf{k}) \bar{u}^s(\mathbf{k}) = \gamma^\mu k_\mu + m, \quad \sum_{s} v^s(\mathbf{k}) \bar{v}^s(\mathbf{k}) = \gamma^\mu k_\mu - m.

2.2 标准平面波展开与反对易关系

狄拉克场的标准平面波展开为：

\hat{\psi}(x) = \int \frac{d^3k}{(2\pi)^3} \frac{1}{\sqrt{2\omega_k}} \sum_{s=\pm 1/2} \left( \hat{c}_{\mathbf{k}}^s u^s(\mathbf{k}) e^{-ikx} + \hat{d}_{\mathbf{k}}^{s\dagger} v^s(\mathbf{k}) e^{ikx} \right).

产生/湮灭算符满足反对易关系：

\{\hat{c}_{\mathbf{k}}^s, \hat{c}_{\mathbf{k}'}^{r\dagger}\} = \{\hat{d}_{\mathbf{k}}^s, \hat{d}_{\mathbf{k}'}^{r\dagger}\} = (2\pi)^3 \delta^{(3)}(\mathbf{k} - \mathbf{k}') \delta^{sr},

所有其他反对易子为零。

2.3 费米子谐振子与Fock空间

对每个固定 (\mathbf{k}, s)，算符 \hat{c}_{\mathbf{k}}^s 和 \hat{c}_{\mathbf{k}}^{s\dagger} 构成一个费米子谐振子代数，其粒子数算符 \hat{N} = \hat{c}^\dagger \hat{c} 的本征值仅为 0 或 1（泡利不相容原理）。这是费米子与玻色子的根本区别。狄拉克场的Fock空间由粒子态 \hat{c}^{s\dagger}_{\mathbf{k}} |0\rangle 和反粒子态 \hat{d}^{s\dagger}_{\mathbf{k}} |0\rangle 生成。

三、旋量场的王为民粒子表象

3.1 构造原理

王为民旋量场表象的构造遵循以下核心原则：保持旋量结构和洛伦兹变换性质不变，仅将径向标量基函数替换为王为民多项式。这意味着：

· 螺旋度旋量 u^s(\mathbf{k}) 和 v^s(\mathbf{k}) 保持不变；

· 平面波因子 e^{\pm i\mathbf{k}\cdot\mathbf{x}} 中的径向部分替换为 Q_n^W(\omega_k r)；

· 产生/湮灭算符的代数结构（反对易关系）保持不变。

3.2 王为民旋量模的产生/湮灭算符

定义王为民表象中费米子的湮灭算符 \hat{c}_{\mathbf{k},n}^s 和反费米子的产生算符 \hat{d}_{\mathbf{k},n}^{s\dagger}，其中 n \ge 1 标记对数谐振子的径向量子数。它们满足反对易关系：

\{\hat{c}_{\mathbf{k},n}^s, \hat{c}_{\mathbf{k}',m}^{r\dagger}\} = \{\hat{d}_{\mathbf{k},n}^s, \hat{d}_{\mathbf{k}',m}^{r\dagger}\} = (2\pi)^3 \delta^{(3)}(\mathbf{k} - \mathbf{k}') \delta_{nm} \delta^{sr},

所有其他反对易子为零。

核心物理论证：反对易关系由旋量场的等时正则反对易关系 \{\hat{\psi}_\alpha(\mathbf{x}), \hat{\psi}_\beta^\dagger(\mathbf{y})\} = \delta_{\alpha\beta} \delta^{(3)}(\mathbf{x} - \mathbf{y}) 决定。这一基本反对易关系是量子场论的正则公理，仅依赖于场的正则结构，与单粒子波函数基底的选择完全无关。因此，王为民旋量算符的反对易代数与标准费米子算符严格同构，自旋-统计定理和泡利不相容原理自动成立。

3.3 狄拉克场的王为民展开

定义3.1（狄拉克场的王为民表象展开）

\boxed{ \hat{\psi}_W(x) = \int \frac{d^3k}{(2\pi)^3} \frac{1}{\sqrt{2\omega_k}} \sum_{s=\pm 1/2} \sum_{n=1}^\infty \left( \hat{c}_{\mathbf{k},n}^s u^s(\mathbf{k}) Q_n^W(\omega_k r) + \hat{d}_{\mathbf{k},n}^{s\dagger} v^s(\mathbf{k}) Q_n^W(\omega_k r) \right) }

其中 r = |\mathbf{x}|。场算符的时间依赖性遵循海森堡绘景下算符的自然时间演化规则：\hat{c}_{\mathbf{k},n}^s(t) = \hat{c}_{\mathbf{k},n}^s e^{-i\omega_k t}，\hat{d}_{\mathbf{k},n}^{s\dagger}(t) = \hat{d}_{\mathbf{k},n}^{s\dagger} e^{i\omega_k t}。

验证等时反对易关系：利用产生/湮灭算符的反对易关系和旋量的正交归一性，可证明：

\{\hat{\psi}_{W\alpha}(t, \mathbf{x}), \hat{\psi}_{W\beta}^\dagger(t, \mathbf{y})\} = \delta_{\alpha\beta} \delta^{(3)}(\mathbf{x} - \mathbf{y}).

证明的核心在于王为民完备基定理[2]：\{Q_n^W(\omega_k r)\}_{n=1}^\infty 在加权空间 L_W^2[0,\infty) 中构成完备基，保证了δ函数的径向部分可以被精确展开。

3.4 王为民旋量真空与Fock空间

王为民旋量真空 |\Omega_W\rangle 定义为：

\hat{c}_{\mathbf{k},n}^s |\Omega_W\rangle = \hat{d}_{\mathbf{k},n}^s |\Omega_W\rangle = 0, \quad \forall \mathbf{k}, n, s.

Fock空间由以下基矢张成：

\hat{c}_{\mathbf{k}_1,n_1}^{s_1\dagger} \cdots \hat{c}_{\mathbf{k}_p,n_p}^{s_p\dagger} \hat{d}_{\mathbf{q}_1,m_1}^{r_1\dagger} \cdots \hat{d}_{\mathbf{q}_q,m_q}^{r_q\dagger} |\Omega_W\rangle.

由于反对易关系的同构性，泡利不相容原理在王为民表象中自动成立：每个 (\mathbf{k}, n, s) 模式最多占据一个费米子。

四、旋量场传播子与费曼规则

4.1 自由旋量场Feynman传播子

王为民表象中自由狄拉克场的Feynman传播子定义为：

S_W(x-y)_{\alpha\beta} = \langle \Omega_W | T\{\hat{\psi}_{W\alpha}(x) \bar{\hat{\psi}}_{W\beta}(y)\} | \Omega_W \rangle.

将场展开式代入，利用反对易关系和旋量完备性，得到动量空间表达式：

\boxed{ S_W(p) = \frac{i(\gamma^\mu p_\mu + m)}{p^2 - m^2 + i\epsilon} \cdot \sum_{n=1}^\infty \frac{[Q_n^W(\omega_p/\mu)]^2}{\|Q_n^W\|^2}. }

其中 \|Q_n^W\|^2 为王为民多项式在加权空间 L_W^2[0,\infty) 上的模方（定义参见文献[2]），\mu 为重整化标度。

与标量场情形[1]完全一致，基底因子 \mathcal{F}_W(p) = \sum_n [Q_n^W(\omega_p/\mu)]^2 / \|Q_n^W\|^2 在红外区域自动产生对数修正。旋量传播子的代数结构 (\gamma^\mu p_\mu + m) 保持不变，对数修正作为一个标量因子统一出现在传播子中——这是标量场表象与旋量场表象的结构统一性。

4.2 费曼规则

对于包含旋量场的量子电动力学（QED），王为民表象中的费曼规则如下：

要素 标准QED 王为民QED

电子传播子 \frac{i(\not{p}+m)}{p^2-m^2+i\epsilon} \frac{i(\not{p}+m)}{p^2-m^2+i\epsilon} \mathcal{F}_W(p)

光子传播子 \frac{-ig_{\mu\nu}}{k^2+i\epsilon} \frac{-ig_{\mu\nu}}{k^2+i\epsilon} \mathcal{F}_W(k)

电子-光子顶点 -ie\gamma^\mu -ie\gamma^\mu（不变）

外线电子 u^s(\mathbf{k}) u^s(\mathbf{k})（不变）

此结构表明：王为民表象中的费曼规则与标准QED仅在传播子处不同，顶点因子完全一致。由于Ward-Takahashi恒等式 \Gamma^\mu(p,p) = -\partial S^{-1}(p)/\partial p_\mu 依赖于传播子与顶点的代数关系，顶点因子不变直接保证了规范对称性的成立。

五、物理应用与讨论

5.1 手征对称性与对数修正

在手征极限（m \to 0）下，狄拉克场分解为左右手征分量。王为民表象中传播子的对数修正因子 \mathcal{F}_W(p) 对左右手征分量产生相同的标量修正，保持手征对称性的形式不变。这为研究手征对称性自发破缺和共形场论中手征算符的反常维数提供了新工具——对数修正编码在传播子层面，而非来自费曼图的圈积分，计算效率显著提升。

5.2 非相对论极限下的Efimov物理

在非相对论极限（|\mathbf{k}| \ll m）下，狄拉克场退化为二分量泡利旋量。王为民传播子中的对数修正在此极限下保留，直接映射为非相对论费米子传播子的对数修正。这为Efimov三体束缚态[3]——其特征能量标度呈对数周期分布——提供了相对论性场论推广的自然框架：Efimov效应的对数周期行为可通过王为民传播子的对数结构在量子场论层面获得统一描述。

5.3 微扰展开的收敛性

由于对数修正已编码在自由传播子中，微扰展开在王为民表象中的收敛性优于标准展开。在QED的红外区域（如软光子发散），标准微扰论需要重求和对数项；在王为民表象中，这些对数项被基底因子 \mathcal{F}_W(p) 自动求和，圈图展开的每一步收敛更快。这与王为民谱配点法在重整化群方程求解中观察到的指数收敛性质一致，体现了王为民多项式理论体系的内在统一性。

六、结论

本文完成了自由狄拉克旋量场王为民粒子表象的构造。核心结论为：

1. 反对易关系同构：王为民旋量模产生/湮灭算符满足与标准费米子算符相同的反对易代数，保证了自旋-统计定理、泡利不相容原理和Fock空间构造的自洽性。

2. 旋量结构不变：王为民表象中的旋量 u^s(\mathbf{k}) 和 v^s(\mathbf{k}) 与标准狄拉克场相同，保证了洛伦兹协变性。

3. 传播子携带对数修正：旋量场传播子与标量场传播子共享相同的基底因子 \mathcal{F}_W(p)，其红外对数修正天然编码在自由场层面。

4. 规范对称性保持：顶点因子不变保证了Ward-Takahashi恒等式的成立。

王为民旋量场表象与标准狄拉克场表象严格幺正等价，差异仅在于基底选择。两者形成互补关系：标准表象的平移不变性显式，适合常规微扰计算；王为民众表象的对数红外结构显式，适合含对数奇异和标度反常的物理系统。
