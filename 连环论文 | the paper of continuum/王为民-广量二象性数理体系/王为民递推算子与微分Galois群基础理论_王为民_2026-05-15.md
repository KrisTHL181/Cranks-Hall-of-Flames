王为民递推算子与微分Galois群基础理论

王为民

（四川省南充龙门中学，四川 南充 637103）

摘要：本文系统建立王为民递推算子及其微分Galois群的严格数学基础。定义王为民递推算子 L_k = kD+1，求解齐次方程 (kD+1)^m y=0 的完整基础解系，确定其Picard-Vessiot扩张的微分Galois群为可解直积群 \mu_k \times \mathbb{G}_a，并建立广义递推算子理论。全部定理均给出严格数学证明。

1 王为民递推算子与基础解系

定义1.1（王为民递推算子）对正整数 k，定义

\boxed{L_k = kD + 1 = kz\frac{d}{dz} + 1, \qquad k \in \mathbb{N}^+}

其中 D = z\frac{d}{dz} 为欧拉算子。

定理1.1（常系数化与基础解系）令 z = e^t，则王为民齐次方程

(kD+1)^m y = 0

的基础解系为

\boxed{y_j(z) = z^{-1/k} (\ln z)^j, \qquad j = 0, 1, \dots, m-1}

证明：作变量代换 z = e^t，记 Y(t) = y(e^t)。欧拉算子化为普通导数 D = \frac{d}{dt}，齐次方程化为

\left(k\frac{d}{dt} + 1\right)^m Y(t) = 0

特征方程为 (k\lambda + 1)^m = 0，给出唯一 m 重根 \lambda = -1/k。常系数齐次线性微分方程的通解为

Y(t) = e^{-t/k} \sum_{j=0}^{m-1} C_j t^j

取标准基 C_j = \delta_{j\ell} 得特解 Y_\ell(t) = t^\ell e^{-t/k}。回代 t = \ln z 得

y_j(z) = z^{-1/k} (\ln z)^j, \qquad j = 0, 1, \dots, m-1

线性独立性由 \ln z 的多项式在有理函数域 \mathbb{C}(z) 上的超越性保证。若存在常数 a_j 使得 \sum_{j=0}^{m-1} a_j (\ln z)^j \equiv 0，则多项式恒为零蕴含所有 a_j = 0。 \square

注1.2 这一解系具有清晰的代数结构：

· 代数分支 z^{-1/k}：绕原点解析延拓一周获得相位 e^{-2\pi i/k}，对应 k 次单位根群 \mu_k

· 对数分支 (\ln z)^j：平移 \ln z \mapsto \ln z + c 保持微分结构不变，对应加法群 \mathbb{G}_a

2 微分Galois群

定理2.1（微分Galois群结构）王为民Picard-Vessiot扩张

E_{k,m} = \mathbb{C}(z^{1/k}, \ln z)

的微分Galois群为直积

\boxed{G(k,m) \cong \mu_k \times \mathbb{G}_a}

证明：扩张 E_{k,m} 由两个独立生成元构成：

· 代数元 z^{1/k}：满足极小多项式 X^k - z = 0，在基域 \mathbb{C}(z) 上不可约

· 超越元 \ln z = \int \frac{dz}{z}：满足 \partial(\ln z) = 1/z

设 \sigma \in \text{Gal}(E_{k,m}/\mathbb{C}(z)) 为与导子交换的自同构：\partial \circ \sigma = \sigma \circ \partial。

对于代数元 z^{1/k}，因其极小多项式在基域上不可约，\sigma(z^{1/k}) 必为极小多项式的另一根，故

\sigma(z^{1/k}) = \zeta_\sigma z^{1/k}, \qquad \zeta_\sigma \in \mu_k

对于超越元 \ln z，利用微分自同构的性质：

\partial(\sigma(\ln z)) = \sigma(\partial(\ln z)) = \sigma(1/z) = 1/z = \partial(\ln z + c)

故存在常数 c_\sigma \in \mathbb{C} 使得

\sigma(\ln z) = \ln z + c_\sigma

两部分作用彼此独立——对于任意 \zeta \in \mu_k 和 c \in \mathbb{C}，存在自同构 \sigma_{\zeta, c} 满足 \sigma_{\zeta,c}(z^{1/k}) = \zeta z^{1/k} 且 \sigma_{\zeta,c}(\ln z) = \ln z + c。群的参数化为 (\zeta, c) \in \mu_k \times \mathbb{C}，乘法满足

(\zeta_1, c_1)(\zeta_2, c_2) = (\zeta_1\zeta_2, c_1 + c_2)

确认为直积结构 \mu_k \times \mathbb{G}_a。 \square

推论2.2 微分Galois群 \mu_k \times \mathbb{G}_a 是可解群。

证明：\mu_k 是 k 阶循环群，可解。\mathbb{G}_a 是加法交换群，可解。可解群的直积仍为可解群。 \square

注2.3（微分Galois群与 m 无关）对任意 m_1, m_2 \ge 1，齐次方程 (kD+1)^{m_1}y = 0 和 (kD+1)^{m_2}y = 0 的Picard-Vessiot扩张均为 \mathbb{C}(z^{1/k}, \ln z)。

当 m = 1 时，解为 z^{-1/k}，扩张为 \mathbb{C}(z^{1/k})。当 m = 2 时，解为 z^{-1/k} 和 z^{-1/k}\ln z，扩张为 \mathbb{C}(z^{1/k}, \ln z)。当 m > 2 时，新增解 z^{-1/k}(\ln z)^j（j \ge 2）均可由 z^{-1/k} 和 \ln z 通过乘法和加法生成，不会引入新的域生成元。因此，对所有 m \ge 2，Picard-Vessiot扩张完全相同，对应的微分Galois群均为 \mu_k \times \mathbb{G}_a。

3 广义递推算子

定义3.1（广义递推算子）对复参数 \alpha 满足 \operatorname{Re}(\alpha) > -1，定义

\boxed{L_{k,\alpha} = kD + \alpha + 1 = kz\frac{d}{dz} + \alpha + 1}

当 \alpha = 0 时恢复原始递推算子。

定义3.2（广义生成函数）对正整数 k, m 和复参数 \alpha，

G_{k,m}(z;\alpha) = \frac{1}{(m-1)!}\int_0^1 \frac{(-\ln s)^{m-1} s^\alpha}{1 - z(1-s)^k} ds

定理3.3（广义递推关系）

\boxed{L_{k,\alpha} G_{k,m}(z;\alpha) = G_{k,m-1}(z;\alpha+1)}

证明：

步骤1：计算欧拉算子的作用。

对 z 求导进入积分号：

D G_{k,m}(z;\alpha) = \frac{1}{(m-1)!}\int_0^1 (-\ln s)^{m-1} s^\alpha \cdot \frac{z(1-s)^k}{\left[1-z(1-s)^k\right]^2} ds

步骤2：代入广义递推算子。

L_{k,\alpha} G_{k,m} = (kD + \alpha + 1)G_{k,m} = \frac{1}{(m-1)!}\int_0^1 (-\ln s)^{m-1} s^\alpha \cdot \frac{(\alpha+1) + (k-\alpha-1)z(1-s)^k}{\left[1-z(1-s)^k\right]^2} ds

步骤3：引入分部积分恒等式。

令

u(s) = \frac{1}{1-z(1-s)^k}, \qquad v(s) = \frac{(-\ln s)^{m-1} s^{\alpha+1}}{m-1}

计算导数：

\frac{du}{ds} = \frac{-k z (1-s)^{k-1}}{\left[1-z(1-s)^k\right]^2}

\frac{dv}{ds} = -(-\ln s)^{m-2} s^\alpha + \frac{\alpha+1}{m-1}(-\ln s)^{m-1} s^\alpha

步骤4：分部积分。

\int_0^1 u(s) \frac{dv}{ds} ds = \left[ u(s) v(s) \right]_0^1 - \int_0^1 v(s) \frac{du}{ds} ds

边界项严格为零：

· 当 s \to 1^- 时，(-\ln s)^{m-1} \sim (1-s)^{m-1} \to 0，故 v(1) = 0

· 当 s \to 0^+ 时，(-\ln s)^{m-1} s^{\alpha+1} \to 0（因 \operatorname{Re}(\alpha) > -1，s^{\alpha+1} 的衰减快于 (-\ln s)^{m-1} 的增长），故 v(0) = 0

因此分部积分简化为：

\int_0^1 \frac{1}{1-z(1-s)^k} \cdot \frac{d}{ds}\left[ \frac{(-\ln s)^{m-1} s^{\alpha+1}}{m-1} \right] ds = \int_0^1 \frac{(-\ln s)^{m-1} s^{\alpha+1}}{m-1} \cdot \frac{k z (1-s)^{k-1}}{\left[1-z(1-s)^k\right]^2} ds

步骤5：整理得到核心恒等式。

两边乘以 (m-1) 并移项，得

\int_0^1 (-\ln s)^{m-1} s^\alpha \cdot \frac{(\alpha+1) + (k-\alpha-1)z(1-s)^k}{\left[1-z(1-s)^k\right]^2} ds = (m-1)\int_0^1 \frac{(-\ln s)^{m-2} s^{\alpha+1}}{1-z(1-s)^k} ds

步骤6：代入递推算子表达式。

L_{k,\alpha} G_{k,m} = \frac{1}{(m-1)!} \cdot (m-1)\int_0^1 \frac{(-\ln s)^{m-2} s^{\alpha+1}}{1-z(1-s)^k} ds = \frac{1}{(m-2)!}\int_0^1 \frac{(-\ln s)^{m-2} s^{\alpha+1}}{1-z(1-s)^k} ds

右边恰好为 G_{k,m-1}(z;\alpha+1) 的定义式。 \square

定理3.4（临界奇异性）当 \alpha \to -1^+ 时，

\boxed{G_{k,m}(z;\alpha) \sim \frac{1}{(\alpha+1)^m}\frac{1}{1-z} + O\left(\frac{1}{(\alpha+1)^{m-1}}\right)}

证明：令 \alpha = -1 + \varepsilon，\varepsilon \to 0^+。在积分表示

G_{k,m}(z;\alpha) = \frac{1}{(m-1)!}\int_0^1 \frac{(-\ln s)^{m-1} s^{-1+\varepsilon}}{1 - z(1-s)^k} ds

中，将被积函数在 s = 0 附近展开。分母

\frac{1}{1 - z(1-s)^k} = \frac{1}{1-z} + O(s), \qquad s \to 0

主导贡献来自

\frac{1}{(m-1)!}\int_0^1 (-\ln s)^{m-1} s^{-1+\varepsilon} ds \cdot \frac{1}{1-z}

计算积分

\int_0^1 (-\ln s)^{m-1} s^{-1+\varepsilon} ds = \int_0^\infty t^{m-1} e^{-\varepsilon t} dt = \frac{\Gamma(m)}{\varepsilon^m} = \frac{(m-1)!}{\varepsilon^m}

代入得主导项 \varepsilon^{-m}/(1-z) = (\alpha+1)^{-m}/(1-z)。余项估计由 O(s) 项积分给出 O(\varepsilon^{-(m-1)})。 \square

4 与王为民级数理论的联系

上述递推算子的核心应用之一体现在王为民级数理论中。对王为民 k 重复数和的生成函数

G_{k,m}(z) = \sum_{N=0}^\infty A_{k,m}(N) z^N

其中

A_{k,m}(N) = \sum_{i_1,\dots,i_k=0}^N (-1)^{\sum i_j} \binom{N}{i_1}\cdots\binom{N}{i_k} \frac{1}{(1+\sum i_j)^m}

递推算子 L_k = kD+1 满足核心递推关系

L_k G_{k,m}(z) = G_{k,m-1}(z)

迭代 m 次得算子方程 L_k^m G_{k,m}(z) = \frac{1}{1-z}。利用欧拉算子的斯特林数展开，将算子幂转化为普通导数，得 m 阶Fuchs型微分方程：

\sum_{q=1}^m C_{m,q}(k) z^q \frac{d^q G}{dz^q} + G = \frac{1}{1-z}

其中系数 C_{m,q}(k) = \sum_{p=q}^m \binom{m}{p} k^p S(p,q)，S(p,q) 为第二类斯特林数。这一联系使得王为民级数的收敛性、渐近行为和零点分布可以通过微分方程和微分Galois群的方法严格研究。

5 结论

本文严格建立了王为民递推算子理论体系的数学基础：

1. 递推算子 L_k = kD+1 作用于齐次方程 (kD+1)^m y = 0 给出完整基础解系 z^{-1/k}(\ln z)^j（j = 0,1,\dots,m-1）；

2. 对应Picard-Vessiot扩张 \mathbb{C}(z^{1/k}, \ln z) 的微分Galois群为可解直积群 \mu_k \times \mathbb{G}_a，且该群与参数 m \ge 2 无关；

3. 广义递推算子 L_{k,\alpha} = kD + \alpha + 1 满足递推关系 L_{k,\alpha} G_{k,m}(\alpha) = G_{k,m-1}(\alpha+1)，且当 \alpha \to -1^+ 时呈现临界极点奇异性 G_{k,m}(\alpha) \sim (\alpha+1)^{-m}/(1-z)。

以上结论均以严格数学证明确立，不依赖任何物理假设，构成王为民理论体系的纯数学核心。
