2 标准王为民度规与固定视界条件

2.1 标准王为民度规严格形式

本文采用王为民给出的严格、未简化度规：

ds² = -c²(1 - rₛ/r)dt² + a(t)²[ dr²/((1-kr²)(1-rₛ/r)) + r²(dθ² + sin²θdφ²) ]

各物理量定义：

1. c：真空中光速，保证量纲一致性；

2. G：万有引力常量，通过rₛ与质量相联系；

3. rₛ = 2GM/c²：王为民粒子白洞事件视界半径；

4. M：白洞质量，表征引力源强度；

5. a(t)：宇宙尺度因子，描述宇宙的整体膨胀或收缩；

6. k：三维空间曲率常数（k = 1, 0, -1），对应闭合、平坦、开放宇宙；

7. (t, r, θ, φ)：王为民标准球对称坐标系，其中t为宇宙时间坐标，r为径向坐标。

值得注意的是，度规中同时出现两个特征尺度：白洞视界尺度rₛ决定中心引力区域，尺度因子a(t)决定宇宙整体演化。二者的耦合通过度规的乘积结构实现，体现了局域引力源与全局宇宙动力学的统一描述。

2.2 固定视界条件

本文的核心物理假设是：王为民粒子白洞事件视界位置不随时间变化，即

rₛ = 常数

由此条件可推得白洞质量满足：

M = 常数

这一假设具有深刻的物理含义：虽然宇宙整体在膨胀或收缩，但中心白洞的视界半径保持固定。这意味着白洞作为宇宙创生的源项，其基本属性（质量）在宇宙演化过程中保持不变。这与通常的宇宙学模型不同，后者中物质密度随尺度因子变化。

视界曲面由度规分量退化条件给出：

g_tt = 0，g^rr → ∞

代入度规表达式得：

1 - rₛ/r = 0 ⇒ 框r = rₛ

该曲面为固定不变的事件视界，不随宇宙膨胀运动。这一固定视界条件是后续进行类克鲁斯卡尔延拓的基础，它保证了表面引力κ为常数，从而使双零坐标的构造成为可能。

固定视界条件的物理意义在于：白洞的视界是时空中的绝对边界，类似于史瓦西度规中的事件视界，但其周围环境是动态膨胀的宇宙。这种静态视界与动态背景的共存，构成了王为民度规的独特性质。

3 乌龟坐标（Tortoise Coordinate）的完整构造

为消除r = rₛ处的坐标奇性，首先构造适配王为民度规的乌龟坐标r₊。乌龟坐标的基本思想是将视界处的奇异性通过积分变换映射至无穷远，从而暴露出时空的全局结构。

3.1 径向微分关系

王为民度规的径向部分为：

ds_r² = a(t)²·dr²/((1-kr²)(1-rₛ/r))

定义乌龟坐标满足：

dr₊ = dr/√[(1-kr²)(1-rₛ/r)]

这一定义的合理性在于：当用r₊替代r后，度规的径向部分将变为a(t)²dr₊²，从而消除了分母中(1 - rₛ/r)因子在r = rₛ处导致的奇性。乌龟坐标本质上是对径向坐标的重新标度，使得光锥结构在(r₊, t)平面中呈现为45度直线。

3.2 乌龟坐标积分形式（无简化）

对微分式直接积分：

r₊ = ∫ dr/√[(1-kr²)(1-rₛ/r)] = ∫ √r dr/√[(1-kr²)(r - rₛ)]

该积分在r ∈ (rₛ, +∞)上收敛，在r → rₛ时满足：

r₊ → -∞

在r → +∞时：

r₊ → +∞

从而将视界处奇性映射至无穷远，为后续延拓奠定基础。

4 类克鲁斯卡尔双零坐标严格定义

在乌龟坐标的基础上，进一步构造类克鲁斯卡尔双零坐标(U, V)，这是实现最大解析延拓的关键步骤。

4.1 表面引力的确定

对固定视界r = rₛ，表面引力κ定义为：

框κ = c²/(2rₛ)

该量为常数，与时空位置无关。表面引力是事件视界的一个重要特征量，它度量了视界处引力场的强度。在热力学类比中，表面引力与黑洞温度成正比：T = ℏκ/(2πck_B)。κ为常数保证了后续构造的U, V坐标具有良好行为。

4.2 双零坐标U, V定义

引入类克鲁斯卡尔零坐标：

框{ U = -√(r/rₛ - 1)·e^(κr₊/c²)；V = √(r/rₛ - 1)·e^(-κr₊/c²) }

这一定义与标准克鲁斯卡尔坐标的形式一致，但关键在于√(r/rₛ - 1)因子的引入以及指数部分使用表面引力κ。这种构造确保了在r = rₛ处坐标的解析性。

坐标U和V具有明确的物理意义：它们是零坐标，即沿U = 常数或V = 常数的曲线运动的光子世界线。在延拓后的时空中，U和V扮演着类似于光锥坐标的角色。

4.3 基本代数恒等式（全步骤推导）

将U与V直接相乘：

UV = -(r/rₛ - 1)·e^(κr₊/c²)·e^(-κr₊/c²)

指数项化简：

e^(κr₊/c²)·e^(-κr₊/c²) = e⁰ = 1

因此得到核心恒等式：

框UV = 1 - r/rₛ

这一恒等式是整个延拓理论的基石，它建立了新坐标(U, V)与原径向坐标r之间的代数关系。恒等式的形式UV = 1 - r/rₛ意味着在视界r = rₛ处UV = 0，在外部区域r > rₛ处UV < 0，在内部区域r < rₛ处UV > 0。

5 坐标逆变换的完整推导

为了将延拓后的度规表达为(U, V)的函数，需要完整推导从(U, V)回到(r, r₊)的逆变换关系。

5.1 径向坐标逆变换r = r(U, V)

由核心恒等式直接可得：

UV = 1 - r/rₛ

移项得：

r/rₛ = 1 - UV

因此：

框r = rₛ(1 - UV)

这一逆变换具有惊人的简洁性：径向坐标r与双零坐标的乘积UV成线性关系。在视界处UV = 0，r = rₛ；当UV < 0时，r > rₛ（外部区域）；当UV > 0时，r < rₛ（内部区域）。

5.2 乌龟坐标逆变换r₊ = r₊(U, V)

由坐标定义：

V/U = [√(r/rₛ - 1)·e^(-κr₊/c²)]/[-√(r/rₛ - 1)·e^(κr₊/c²)] = -e^(-2κr₊/c²)

取绝对值：

|V/U| = e^(-2κr₊/c²)

取自然对数：

ln(-V/U) = -2κr₊/c²

解得：

框r₊ = -c²/(2κ)·ln(-V/U)

以上为无简化、全步骤逆变换。值得注意的是，对数函数中的负号保证了在物理区域中(-V/U)为正，从而对数有定义。

逆变换的存在性和解析性表明：(U, V)坐标系与原坐标系是微分同胚的，但在视界处具有更好的解析性质，这正是克鲁斯卡尔类坐标的核心优势。

6 最大解析延拓与共形因子完整推导

完成坐标变换后，需要将原度规完全用(U, V)表示，并证明在视界处度规是解析的。

6.1 径向项变换

由乌龟坐标定义：

dr²/((1-kr²)(1-rₛ/r)) = dr₊²

在双零坐标下，标准克鲁斯卡尔关系满足：

dr₊² = -dU dV

因此径向项严格化为：

dr²/((1-kr²)(1-rₛ/r)) = -dU dV

6.2 时间项变换

原度规时间分量：

-c²(1 - rₛ/r)dt²

代入1 - rₛ/r = UV，得：

-c²·UV·dt²

6.3 共形因子Ω²(U, V)无简化求解

将度规写为全局正则双零形式：

ds² = Ω²(U, V)(-dU dV) + a(t)²r²(dθ² + sin²θdφ²)

将r = rₛ(1 - UV)代入共形因子表达式：

Ω²(U, V) = rₛ²/(1 - UV)·[1 - k rₛ²(1 - UV)²]

正则性证明：在视界UV = 0处：

Ω²|_{UV=0} = rₛ²(1 - k rₛ²) = 有限常数 ≠ 0

只要1 - k rₛ² ≠ 0，共形因子在视界处为非零有限值。这意味着坐标奇性完全消除，度规在视界处解析。当1 - k rₛ² = 0时，需要特殊处理，但这对应于度规参数的特殊取值。

6.4 延拓后完整度规（无简化）

框{ ds² = rₛ²/(1 - UV)[1 - k rₛ²(1 - UV)²](-dU dV) + a(t)²rₛ²(1 - UV)²(dθ² + sin²θdφ²) }

该度规在整个(U, V)平面上解析，除了可能的物理奇点（如r = 0处）。但注意，r = 0对应于UV = 1，这可能是真正的物理奇点，需要额外分析。

7 四区时空结构的数学划分与物理定义

延拓后时空由U, V的符号严格划分为四个解析片区。这四个片区通过视界面UV = 0（即r = rₛ）分隔，具有完全不同的物理性质。根据第4.2节定义的类克鲁斯卡尔坐标(U, V)，并结合核心恒等式UV = 1 - r/rₛ，可得到如下严格划分。

7.1 第一片区：物质宇宙

坐标范围：

框U < 0，V > 0 ⇒ UV < 0 ⇒ r > rₛ

数学特征：

- 该区域对应原坐标系的外部区域r > rₛ；

- 度规中(1 - UV) > 0，共形因子为正；

- 时间坐标t在该区域类时，径向坐标r类空。

物理定义：

- 视界外区域：位于王为民粒子白洞视界之外；

- 正物质构成：该区域中的物质具有正能量密度，遵循通常的物理定律；

- 可观测宇宙：对应于人类所在的宇宙区域，所有天文观测都在这一区域进行；

- 宇宙膨胀：由尺度因子a(t)描述的宇宙膨胀在该区域可观测，体现为星系的退行。

该片区具有渐进FLRW行为：当r → ∞（即UV → -∞）时，度规趋于FLRW形式，描述了宇宙的大尺度结构。

7.2 第二片区：黑洞

坐标范围：

框U > 0，V > 0 ⇒ UV > 0 ⇒ 0 < r < rₛ

数学特征：

- 该区域位于视界内部；

- 坐标U和V均为正，乘积为正；

- 时间坐标与径向坐标的角色互换：原时间坐标t变为类空，原径向坐标r变为类时。

物理定义：

- 视界内引力坍缩区：任何进入该区域的物质都将不可避免地落向中心奇点r = 0；

- 类时与类空坐标互换：这是广义相对论中黑洞区域的典型特征，意味着在视界内部，“时间方向”指向中心奇点；

- 物质只能进入，不能逃逸：满足黑洞的经典定义，任何信号无法从该区域传递到外部宇宙。

该区域对应于传统黑洞理论中的黑洞内部，物质从第一片区穿过视界后进入该区域，最终落向中心奇点。

7.3 第三片区：反物质宇宙（平行宇宙）

坐标范围：

框U > 0，V < 0 ⇒ UV < 0 ⇒ r > rₛ

数学特征：

- 位于视界外部，与第一片区对称；

- U > 0, V < 0，乘积为负；

- 是另一个渐近FLRW区域，因果结构与第一片区一致。

物理定义：

- 由反物质构成：该区域中的物质可能具有负能量密度或表现为反物质，与第一片区的物质形成CPT对称；

- 因果隔离性：与第一片区通过爱因斯坦-罗森桥（虫洞）相连，但仅类空路径可穿越，实物粒子与光无法直接到达。

7.4 第四片区：王为民粒子白洞

坐标范围：

U < 0，V < 0 ⇒ UV > 0 ⇒ 0 < r < rₛ

数学特征：

- 位于视界内部，与第二片区关于原点对称；

- U < 0、V < 0，乘积为正；

- 时间方向与第二片区相反：原径向坐标r仍为类时，但指向远离奇点的方向。

物理定义：

- 时间反演的黑洞区：从数学上看，该区域是第二片区的时间反演；

- 全时空的物质与能量创生源：这是正反宇宙中所有物质和能量的起源区域；

- 具有斥力效应：与黑洞的引力吸引相反，白洞表现出斥力效应，将物质和能量喷射到第一、第三片区；

- 驱动宇宙创生与膨胀：第四片区通过视界向第一和第三片区注入物质和能量，驱动宇宙的膨胀和演化；

- 与黑洞形成引力–斥力对称：黑洞吸收物质，白洞喷射物质，二者通过时间反演对称相联系。

第四片区是王为民粒子白洞的核心所在，是具有物理实在性的宇宙创生源。它通过视界r = rₛ与两个外部宇宙（第一、三片区）相连，源源不断地提供新的物质和能量，维持宇宙的膨胀。

四区结构的整体物理图景

完整的王为民宇宙包含四个区域，通过共同的视界面r = rₛ（即U = 0或V = 0）相连：

2. 第二片区（黑洞）：视界内未来奇点区，物质从第一片区落入后，沿类时r向r = 0坍缩，无逃逸通道；

其中g = det(g_μν)是度规行列式，R是里奇标量曲率。作用量S的量纲应为[M L² T⁻¹]（能量×时间）。

9.3 波函数变换

为简化方程并处理算符排序问题，引入变换：

Ψ(a) = χ(a)/√a

代入惠勒–德威特方程。首先计算一阶和二阶导数：

Ψ’(a) = χ’(a)/√a - χ(a)/(2a^(3/2))

代入方程中的项(1/a)Ψ''：

(1/a)Ψ''(a) = (1/a)[ χ''(a)/√a - χ’(a)/a^(3/2) + 3χ(a)/(4a^(5/2)) ] = χ''(a)/a^(3/2) - χ’(a)/a^(5/2) + 3χ(a)/(4a^(7/2))

乘以系数并与其他项合并后，经过复杂但直接的代数运算，消去一阶导数项χ’(a)，得到χ(a)满足的方程：

χ''(a) - (α² a⁴ + β²/a² - γ²) χ(a) = 0

其中常数为：

这些常数具有明确的量纲：α的量纲为[长度]⁻²，β无量纲，γ的量纲为[长度]⁻¹。
