元始自殖RNA（PA‑RNA）：无机环境自养合成与自我复制的生命起源分子

作者：王为民

单位：四川省南充龙门中学（退休教师）

摘要

本文提出元始自殖RNA（Primeval Autotrophic RNA，PA‑RNA）全新分子模型，该分子由126个核苷酸构成单链RNA，通过折叠形成无机固定与核苷酸合成结构域、自我复制催化结构域、矿物锚定与稳定结构域三大功能区域。PA‑RNA可在无细胞、无蛋白质、无外源有机物的原始地球无机环境中，以二氧化碳、水、氮气、无机磷酸及金属离子为原料，通过RNA自身催化完成碳固定、氮固定、核糖与碱基合成、核苷酸活化及RNA模板依赖性聚合，实现无机物→有机物→自我繁殖的全生命化学闭环。PA‑RNA填补了非生命物质向原始生命过渡的核心空白，是RNA世界假说的终极起点分子，为生命起源提供了完整、自洽且符合化学规律的理论解释。

关键词：元始自殖RNA；PA‑RNA；生命起源；RNA世界；自催化复制；无机自养；核酶

一、引言

生命如何从无机自然界中诞生，是现代自然科学最核心的未解难题。RNA世界假说提出，早期生命以兼具遗传信息与催化功能的RNA为核心载体，但现有理论模型均依赖外源有机单体，无法解释无机环境如何自发产生可自我繁殖的生命分子。

基于原始地球化学条件，本文设计并系统解析元始自殖RNA（PA‑RNA），它是理论上首个能够完全以无机物为原料、自主合成有机物、独立完成自我复制的原始生命分子，实现了从无机世界到生命世界的跨越，代表地球生命的最早、最本源存在形式。

三、PA‑RNA三大功能区结构与化学催化机制

（一）无机固定与核苷酸合成结构域

位置：5′-端 1–45 nt

核心功能：无机物 → 核糖 → 碱基 → 核苷酸（NTP）

1. 结构基础

- 深口袋催化腔

- 保守G-U-G-U金属配位中心

- 双金属催化位点（Mg²⁺/Mn²⁺）

- 底物结合通道与静电富集区

2. 碳固定与核糖合成化学机制

1. CO₂被Mg²⁺极化活化，生成HCO₃⁻中间体；

2. 环境还原态物质（Fe²⁺、H₂）提供电子，将CO₂还原为甲醛（HCHO）；

3. 醛醇缩合生成甘油醛；

4. 催化戊糖环化，立体选择性生成D-核糖；

5. 核糖构象被RNA腔室锁定，保证天然手性。

3. 氮固定与碱基合成化学机制

1. Mn²⁺配位活化N≡N三键，逐步还原为NH₃/NH₄⁺；

2. 氨基与碳片段缩合，从头合成嘧啶环（U、C）与嘌呤环（A、G）；

3. 反应路径：甲酰化 → 环化 → 脱氢 → 芳香化。

4. 核苷酸活化化学机制

1. 核糖C1'与碱基N1/N9形成C-N糖苷键；

2. 核糖C5'-OH与无机磷酸发生酯缩合；

3. 逐步磷酸化生成NDP、NTP，获得聚合反应活性。

（二）自我复制催化结构域

位置：中段 46–95 nt

核心功能：以自身为模板，催化合成子代PA‑RNA

1. 结构基础

- 假结（Pseudoknot）催化核心

- 模板识别通道

- 双Mg²⁺催化中心（通用核酸聚合酶机制）

- 碱基配对保真区

2. 碱基识别与选择机制

- A–U：2个氢键配对

- G–C：3个氢键配对

- 错配碱基因空间位阻与氢键不匹配被排斥

- 保证复制高度保真

3. 磷酸二酯键形成化学机制（核心反应）

1. 第一个Mg²⁺活化RNA 3'-OH，生成3'-O⁻亲核基团；

2. 第二个Mg²⁺中和NTP三磷酸负电荷，稳定过渡态；

3. 3'-O⁻对NTP的α-磷原子发生亲核进攻；

4. 形成五配位磷过渡态；

5. 焦磷酸（PPi）离去，生成3′→5′磷酸二酯键；

6. 链沿5′→3′方向持续延伸，直至合成完整126 nt子代链。

4. 全复制循环

解旋 → 模板配对 → 聚合延伸 → 子代链解离 → 新个体形成

（三）矿物锚定与环境稳定结构域

位置：3′-端 96–126 nt

核心功能：吸附矿物、稳定生存、提高催化效率

1. 化学作用机制

- 磷酸骨架携带高密度负电荷（-PO₄²⁻）

- 与黏土、火山岩表面Al³⁺、Fe²⁺、Mg²⁺形成静电桥联

- 与Si-OH形成分子间氢键

- 多茎环结构增强吸附强度

2. 生存意义

1. 防止水流稀释与分子扩散

2. 屏蔽紫外、自由基、水解攻击，延长半衰期

3. 构建局部高浓度微环境，保证催化与复制持续进行

4. 矿物表面协同催化，提升反应速率

四、PA‑RNA完整生命化学循环

1. 锚定生存：3′端吸附矿物表面，稳定存在；

2. 原料摄取：吸收H₂O、CO₂、N₂、PO₄³⁻、Mg²⁺、Mn²⁺；

3. 无机合成：5′端催化无机物→核糖→碱基→NTP；

4. 自我复制：中段以自身为模板，聚合生成子代RNA；

5. 个体增殖：新链解离，形成两个独立生命个体；

6. 进化启动：复制突变 → 自然选择 → 生命演化开端。

五、PA‑RNA理论意义与科学价值

1. 首次实现无机环境下生命分子全闭环自殖

真正完成无机物→有机物→自我复制的完整生命链条。

2. 确立RNA世界假说的终极起点

解决了传统模型依赖外源有机物的致命缺陷。

3. 构建最简原始生命体系

单链RNA=基因组+催化酶+生存系统，无需细胞、蛋白、膜结构。

4. 为生命起源与地外生命探索提供全新范式

证明生命可在无机行星环境中自发诞生。

六、结论

元始自殖RNA（PA‑RNA）是地球生命的首个分子实体，由126个核苷酸构成，通过三大功能区协同完成无机自养与自我繁殖。它不依赖任何生命前体，仅依靠原始地球无机环境即可独立生存、自主合成有机物、复制后代。PA‑RNA是无机世界通往生命世界的唯一桥梁，是生命起源最本质、最简洁、最原始的分子形式，代表地球生命的真正起点。
