王为民度规严格回归史瓦西度规、FLRW度规

数学条件 + 物理条件（论文级清晰版）

标准王为民度规

ds² = −c²(1 − rs/r) dt²

+ a(t)² [ dr² / ((1−kr²)(1−rs/r)) + r²(dθ² + sin²θ dφ²) ]

其中：

rs = 2GM/c² 为王为民白洞视界

a(t) 为宇宙尺度因子

k 为空间曲率

M(t) = c³ t/(2G) 为正反宇宙总质量

一、王为民度规 → 回归史瓦西度规

1. 数学严格条件

1. 尺度因子固定为 1

a(t) ≡ 1

2. 空间曲率为 0

k = 0

3. 中心质量为常数

M = 常数，质量不随时间变化

4. 无量子创生、无辐射功率

W = 0

2. 物理条件

1. 无宇宙膨胀，时空静态

2. 无正反宇宙创生，无白洞辐射

3. 只有孤立球对称引力源

4. 时空渐近平坦

3. 回归后的史瓦西度规

ds² = −c²(1 − rs/r) dt²

+ dr²/(1 − rs/r)

+ r²(dθ² + sin²θ dφ²)

二、王为民度规 → 回归FLRW度规

1. 数学严格条件

1. 中心引力源质量为 0

rs → 0，M → 0

2. 保留尺度因子 a(t) 动力学

3. 保留空间曲率 k = +1, 0, −1

4. 去掉局域引力项 (1−rs/r)

2. 物理条件

1. 宇宙均匀、各向同性，无中心引力源

2. 无王为民白洞，无局域强引力

3. 只有宇宙整体膨胀

4. 标准宇宙学背景时空

3. 回归后的FLRW度规

ds² = −c² dt²

+ a(t)² [ dr²/(1−kr²) + r²(dθ² + sin²θ dφ²) ]

三、核心物理总结

1. 王为民度规 = 史瓦西局域引力 + FLRW宇宙膨胀 + 量子白洞创生

2. 回归史瓦西：a(t)=1，k=0，M=常数

3. 回归FLRW：rs=0，M=0

4. 度规本身不分裂正反宇宙，正反宇宙来自白洞量子辐射

四、极简条件表

回归目标 数学条件 物理条件

FLRW度规 rs=0，M=0 均匀各向同性、无中心白洞、纯膨胀
