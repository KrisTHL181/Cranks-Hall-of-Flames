1.1 唯一公理：第二类斯特林数

定义1.1（第二类斯特林数）：S(n,k) 是将 n 元有限集合划分为 k 个非空不相交子集的组合分拆数，由递推关系唯一确定：

S(n,k) = k \cdot S(n-1,k) + S(n-1,k-1)

边界条件：S(0,0) = 1；S(n,0) = 0 \ (n > 0)；S(n,n) = 1。

关键性质：S(n,1) = 1 \ (\forall n \ge 1)，即任意 n 元集合仅有一种方式整体视为一个非空子集。

S(n,k) 为纯组合整数，无量纲，不依赖任何物理假设，是王为民第一性原理公理的唯一公理起点。

1.2 王为民调和型贝尔多项式

2.1 王为时空拓扑不变量 S_W

定理2.1（王为民拓扑密度定理）：

平方得 [B_n^W(z)]^2 = z^2 + \mathcal{O}(z^3)。代入整函数定义：

W(z) = \sum_{n=1}^{\infty} \frac{z^2 + \mathcal{O}(z^3)}{\|B_n^W\|^2} = z^2 \sum_{n=1}^{\infty} \frac{1}{\|B_n^W\|^2} + \mathcal{O}(z^3)

在强磁场、极低温条件下，电子间库仑相互作用与整函数零点拓扑产生非微扰耦合，形成王为民三大拓扑约束：

约束二（零点不相容约束）：任意两个电子零点不得重合。当 z_i = z_j 时，\Psi = 0。

约束三（归一化约束）：引入磁长度 l_B = \sqrt{\hbar/(eB)}，波函数必须平方可积。

5.2 王为民众体零点全纯函数唯一构造定理

定理5.1（王为民众体零点全纯函数唯一构造定理）：填充因子 \nu = 1/m（m 为奇数）的分数量子霍尔基态波函数，是王为民三大拓扑约束下的唯一解析解：

\Psi_m(z_1, \dots, z_N) = \prod_{i<j} (z_i - z_j)^m \exp\left( -\sum_i \frac{|z_i|^2}{4l_B^2} \right)

证明：

全纯部分的构造：由约束一和约束二，全纯部分必须是 z_i 的反对称全纯函数，当任意两个坐标相等时取零。满足此条件的唯一多项式结构为 Vandermonde 行列式的 m 次幂 \prod_{i<j} (z_i - z_j)^m。

王为民分数缠绕量子数定理：幂次 m 由分数缠绕数 w = 1/m 的拓扑要求唯一决定。一个准空穴绕另一电子一周时，全纯部分的相位变化为 e^{2\pi i m} \cdot e^{-2\pi i} = e^{2\pi i(m-1)}，对应分数缠绕数 1/m。m 被严格证明为拓扑量子数，而非唯象参数。

归一化因子的附加：由约束三，附加磁长度相关的高斯衰减因子。

唯一性证明：在固定 N、\nu = 1/m 和总角动量 M = mN(N-1)/2 的条件下，上述构造是唯一同时满足全纯性、不相容性和归一化约束的解析解。任何偏离必破坏至少一项约束。\square
