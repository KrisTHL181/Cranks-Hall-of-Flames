王为民粒子表象中的杨-米尔斯场

——非阿贝尔规范场的量子化与费曼规则

作者：王为民

单位：四川省南充龙门中学（退休）

摘要

本文将王为民粒子表象从阿贝尔规范场推广至非阿贝尔 SU(N) 杨-米尔斯场，完成该框架向粒子物理标准模型核心的关键跨越。核心构造与阿贝尔情形完全一致：保持规范群的代数结构、顶点因子和BRST对称性不变，仅将每个动量模的径向部分从平面波替换为王为民调和型贝尔多项式 Q_n^W(\omega_k r)。通过Faddeev-Popov方法引入鬼场，严格证明BRST幂零性在王为民表象中保持不破缺。胶子传播子、鬼传播子统一携带基底因子 \mathcal{F}_W(k)，三胶子和四胶子顶点、胶子-鬼顶点与标准杨-米尔斯理论完全相同。本文给出了完整的王为民杨-米尔斯费曼规则，证明了非阿贝尔规范场论在王为民表象中的自洽性，并讨论了该框架在渐近自由和色禁闭红外问题中的潜在应用。

关键词：王为民粒子表象；杨-米尔斯场；非阿贝尔规范场；BRST对称性；Faddeev-Popov量子化；渐近自由

一、引言

杨-米尔斯理论是非阿贝尔规范场论的核心，构成了量子色动力学（QCD）和电弱统一理论的基础[4]。与阿贝尔规范场不同，杨-米尔斯场的规范玻色子携带规范荷，存在三胶子和四胶子自相互作用，其量子化需要引入Faddeev-Popov鬼场和BRST对称性来保持协变性[5]。

在前八篇文章中，我们系统构建了王为民粒子表象的完整理论框架：自由标量场[1]、旋量场[2]、矢量场[3]的表象构造，相互作用场论的微扰展开与费曼规则[4]，QED单圈电子自能与顶角修正的计算[5]，重整化群方程的推导[6]，以及LSZ约化与外线修正[7]。这一框架的核心特征是：传播子天然携带基底因子 \mathcal{F}_W(k)，顶点因子不变，红外对数修正内置在传播子中。

然而，上述所有工作均局限于阿贝尔规范场（QED）。杨-米尔斯场包含规范玻色子的自相互作用，其顶点结构远比QED复杂。核心问题是：该表象的顶点因子不变性在自相互作用下是否依然成立，BRST对称性能否保持，基底因子 \mathcal{F}_W(k) 在规范场自耦合中是否产生额外效应。

二、经典 SU(N) 杨-米尔斯场

2.1 场强张量与拉格朗日量

SU(N) 杨-米尔斯场的场强张量为：

F_{\mu\nu}^a = \partial_\mu A_\nu^a - \partial_\nu A_\mu^a + g f^{abc} A_\mu^b A_\nu^c,

其中 a = 1, \dots, N^2-1 为伴随表示色指标，f^{abc} 为 SU(N) 的结构常数。经典拉格朗日量：

\mathcal{L}_{\text{YM}} = -\frac{1}{4} F_{\mu\nu}^a F^{a\mu\nu}.

展开后包含胶子的动能项、三胶子顶点和四胶子顶点。

2.2 规范固定与Faddeev-Popov量子化

在Feynman规范（\partial^\mu A_\mu^a = 0）下，引入反对易的标量鬼场 c^a 和反鬼场 \bar{c}^a：

\mathcal{L}_{\text{GF}} = -\frac{1}{2} (\partial^\mu A_\mu^a)^2, \qquad

\mathcal{L}_{\text{FP}} = \bar{c}^a (-\partial^\mu D_\mu^{ab}) c^b,

其中 D_\mu^{ab} = \delta^{ab} \partial_\mu - g f^{abc} A_\mu^c 为伴随表示协变导数。

2.3 标准平面波展开

标准胶子场（Feynman规范）的平面波展开为：

\hat{A}_\mu^a(x) = \int \frac{d^3k}{(2\pi)^3} \frac{1}{\sqrt{2\omega_k}} \sum_{\lambda=1}^2 \left( \hat{a}_{\mathbf{k}}^{a(\lambda)} \epsilon_\mu^{(\lambda)}(\mathbf{k}) e^{-ikx} + \text{h.c.} \right).

鬼场展开形式类似，产生/湮灭算符满足标准对易（胶子）和反对易（鬼）关系。

三、王为民表象中的杨-米尔斯场量子化

3.1 构造原则

推广遵循与先前完全统一的四条原则：

1. 保持规范群 SU(N) 的代数结构（结构常数 f^{abc}）不变；

2. 保持所有顶点因子的形式不变；

3. 仅将径向基函数从平面波替换为王为民多项式 Q_n^W(\omega_k r)；

4. 产生/湮灭算符的代数和反对易关系保持不变。

3.2 胶子场的王为民展开

自由胶子场在Feynman规范下的王为民展开为：

\boxed{ \hat{A}_{W\mu}^a(x) = \int \frac{d^3k}{(2\pi)^3} \frac{1}{\sqrt{2\omega_k}} \sum_{\lambda=1}^2 \sum_{n=1}^\infty \left( \hat{a}_{\mathbf{k},n}^{a(\lambda)} \epsilon_\mu^{(\lambda)}(\mathbf{k}) Q_n^W(\omega_k r) + \text{h.c.} \right) }.

产生/湮灭算符满足标准对易关系，额外增加色指标和径向量子数的克罗内克δ：

[\hat{a}_{\mathbf{k},n}^{a(\lambda)}, \hat{a}_{\mathbf{k}',m}^{b(\lambda')\dagger}] = (2\pi)^3 \delta^{(3)}(\mathbf{k} - \mathbf{k}') \delta_{nm} \delta^{ab} \delta^{\lambda\lambda'}.

3.3 鬼场的王为民展开

鬼场和反鬼场是反对易的标量场，其王为民展开为：

\hat{c}_W^a(x) = \int \frac{d^3k}{(2\pi)^3} \frac{1}{\sqrt{2\omega_k}} \sum_{n=1}^\infty \left( \hat{\gamma}_{\mathbf{k},n}^a Q_n^W(\omega_k r) + \text{h.c.} \right),

\bar{\hat{c}}_W^a(x) = \int \frac{d^3k}{(2\pi)^3} \frac{1}{\sqrt{2\omega_k}} \sum_{n=1}^\infty \left( \bar{\hat{\gamma}}_{\mathbf{k},n}^a Q_n^W(\omega_k r) + \text{h.c.} \right).

鬼场算符满足反对易关系：

\{\hat{\gamma}_{\mathbf{k},n}^a, \bar{\hat{\gamma}}_{\mathbf{k}',m}^{b\dagger}\} = (2\pi)^3 \delta^{(3)}(\mathbf{k} - \mathbf{k}') \delta_{nm} \delta^{ab}.

3.4 BRST对称性的保持

标准杨-米尔斯理论的BRST变换为：

s A_\mu^a = D_\mu^{ab} c^b, \quad s c^a = -\frac{g}{2} f^{abc} c^b c^c, \quad s \bar{c}^a = B^a, \quad s B^a = 0,

其中 B^a 为辅助场，BRST算符满足幂零性 s^2 = 0。

核心论证：BRST变换是场算符的纯代数关系，不依赖于时空坐标的具体函数形式。在王为民表象中，场算符通过幺正变换 U 与标准场算符联系：\hat{\phi}_W = U \hat{\phi} U^\dagger。BRST变换在幺正变换下保持形式不变：

s_W \hat{\phi}_W = U (s \hat{\phi}) U^\dagger.

因此，BRST幂零性 s_W^2 = 0 在王为民表象中严格成立。这是非阿贝尔王为民场论自洽性的核心保证。

四、费曼规则

王为民 SU(N) 杨-米尔斯理论的费曼规则（Feynman规范）：

要素 标准QCD 王为民QCD

胶子传播子 \frac{-i\delta^{ab} g_{\mu\nu}}{k^2+i\epsilon} \frac{-i\delta^{ab} g_{\mu\nu}}{k^2+i\epsilon} \mathcal{F}_W(k)

鬼传播子 \frac{i\delta^{ab}}{k^2+i\epsilon} \frac{i\delta^{ab}}{k^2+i\epsilon} \mathcal{F}_W(k)

三胶子顶点 -g f^{abc} [g_{\mu\nu}(k_1-k_2)_\rho + \text{cyclic}] 不变

四胶子顶点 -ig^2 f^{abe}f^{cde} [g_{\mu\rho}g_{\nu\sigma} - \cdots] 不变

胶子-鬼顶点 -g f^{abc} p_\mu 不变

圈积分 \int \frac{d^4k}{(2\pi)^4} \int \frac{d^4k}{(2\pi)^4} \sum_{n=1}^\infty

顶点因子全部不变，因为规范群 SU(N) 的代数结构在表象变换下不改变。基底因子 \mathcal{F}_W(k) 与所有先前构建的场完全一致——无论自旋和规范量子数如何，传播子的标量修正函数统一，红外极限下 \mathcal{F}_W(k) \sim \ln(\mu/|k|) 为所有规范场传播子提供内在对数衰减。

五、物理应用与讨论

5.1 渐近自由与紫外重整化

基底因子 \mathcal{F}_W(k) 在紫外区域（|k| \to \infty）趋于常数，不改变胶子自相互作用的紫外发散结构。三胶子和四胶子顶点的反屏蔽效应依然导致β函数为负，渐近自由性质严格保持。

5.2 红外正规化与色禁闭

在红外区域（|k| \to 0），传播子的基底因子 \mathcal{F}_W(k) \sim \ln(\mu/|k|) 提供对数衰减，与QED光子传播子[5]的机制完全相同。由于胶子携带色荷，其传播子的红外修正可能为色禁闭研究提供新的内在正规化视角。

5.3 幺正等价性

王为民QCD与标准QCD通过幺正变换联系，所有物理可观测量的预言保持一致。费曼规则中唯一的差异——传播子中的基底因子——在物理振幅中通过LSZ约化公式的外线修正因子[7]和其他重整化常数的协调而保持幺正等价性。

六、结论

本文完成了 SU(N) 杨-米尔斯场的王为民粒子表象构造。核心结论为：

1. 顶点因子全部不变：三胶子、四胶子、胶子-鬼顶点与标准杨-米尔斯理论完全相同——规范群代数结构在幺正变换下保持的必然结果。

2. BRST对称性保持：幂零性 s_W^2 = 0 严格成立，保证了非阿贝尔场论的自洽性。

3. 传播子基底因子普适：胶子和鬼传播子均携带 \mathcal{F}_W(k)，与所有先前构建的场完全一致。

4. 渐近自由保持不变：紫外区域标准重整化程序完全适用，QCD的核心物理性质得以保持。

至此，王为民粒子表象已覆盖自旋0（标量场）、自旋1/2（旋量场）和自旋1（矢量场，包括阿贝尔QED和非阿贝尔QCD）的全部基本量子场，构成了一套能够描述粒子物理标准模型规范部分的完整场论框架。
