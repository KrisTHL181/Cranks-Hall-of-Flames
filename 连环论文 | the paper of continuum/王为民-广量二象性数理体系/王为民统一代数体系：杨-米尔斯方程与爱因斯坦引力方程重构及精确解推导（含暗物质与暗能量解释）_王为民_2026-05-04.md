定义1.1（欧拉标度微分算子）

定义欧拉标度微分算子（标度变换无穷小生成元）：

D = z \frac{d}{dz}.

\tag{1.1}

该算子满足线性性（对任意常数 a, b，D(af+bg) = a Df + b Dg）和莱布尼茨法则（D(fg) = f Dg + g Df），是连接平直空间与弯曲空间、实空间与动量空间的核心标度算子。

1.3 王为民完备解系

命题1.1（王为民完备解系）【本文原创命名】

针对高阶齐次微分方程

\left(kz\frac{d}{dz} + 1\right)^m y(z) = 0,

\tag{1.3}

可求得完备、线性无关、正交归一的基础解析解系——王为民完备解系。

推导：

第一步：变量代换消去变系数。 令 z = e^w，则 w = \ln z。由 \frac{d}{dz} = \frac{dw}{dz} \frac{d}{dw} = \frac{1}{z} \frac{d}{dw}，得 z\frac{d}{dz} = \frac{d}{dw}。算子简化为：

kz\frac{d}{dz} + 1 = k\frac{d}{dw} + 1.

原方程变为常系数齐次线性微分方程：

\left(k\frac{d}{dw} + 1\right)^m y(e^w) = 0.

\tag{1.4}

第二步：特征方程与特征根。 令 y(e^w) = e^{\lambda w}，代入 (1.4)：

\left(k\lambda + 1\right)^m e^{\lambda w} = 0 \quad \Rightarrow \quad (k\lambda + 1)^m = 0.

特征根为重根：\lambda = -\frac{1}{k}（重数 m）。

第三步：重根情形下的完备解系。 对于重数为 m 的特征根 \lambda = -1/k，常系数线性微分方程的通解为：

y(e^w) = e^{-w/k} \sum_{j=0}^{m-1} C_j w^j.

还原变量 w = \ln z：

y(z) = z^{-1/k} \sum_{j=0}^{m-1} C_j (\ln z)^j.

取基函数即得：

\boxed{y_j^{(k)}(z) = z^{-1/k} (\ln z)^j, \quad j = 0, 1, 2, \dots, m-1}.

\tag{1.5}

第四步：完备性与线性无关性证明。 设 \sum_{j=0}^{m-1} \alpha_j y_j^{(k)}(z) = 0 对所有 z > 0 成立。提取公因子 z^{-1/k}，得 \sum_{j=0}^{m-1} \alpha_j (\ln z)^j = 0 对所有 z > 0。令 t = \ln z，则 \sum_{j=0}^{m-1} \alpha_j t^j = 0 对所有 t \in \mathbb{R} 成立。由多项式恒等定理，\alpha_j = 0 对所有 j 成立，故基函数线性无关。解空间维度为 m，恰好等于基函数个数，因此解系完备。证毕。

物理意义：该解系张成的线性空间可作为全域场论展开的完备基底，适用于四维量子规范场与黎曼弯曲时空全域。

\sigma(y_0)^k = \sigma(y_0^k) = \sigma(z^{-1}) = z^{-1} = y_0^k,

故 \sigma(y_0) = \zeta \cdot y_0，其中 \zeta^k = 1。这给出 \mu_k 的作用。

考虑 y_1(z) = z^{-1/k} \ln z。其导数满足：

\frac{d}{dz} y_1 = \frac{d}{dz}(z^{-1/k} \ln z) = -\frac{1}{k} z^{-1/k-1} \ln z + z^{-1/k-1} = \frac{1}{z} y_0 - \frac{1}{kz} y_1.

设 \sigma(y_1) = \zeta y_1 + c。由导数的微分自同构性质 \sigma(\frac{dy_1}{dz}) = \frac{d}{dz} \sigma(y_1) 可验证 c \in \mathbb{C}，且对不同 j 的 y_j，\sigma 的作用为幂零平移。这给出 \mathbb{G}_a 的表示。

第三步：半直积结构的确定。 \mu_k 通过乘法作用于 \mathbb{G}_a 上：若 \sigma_\zeta(y_1) = \zeta y_1（\zeta \in \mu_k），\tau_c(y_1) = y_1 + c（c \in \mathbb{G}_a），则

\sigma_\zeta \circ \tau_c \circ \sigma_\zeta^{-1} (y_1) = \sigma_\zeta(y_1 + c) = \zeta y_1 + \zeta c = \tau_{\zeta c}(y_1).

这表明 \mu_k 通过乘法作用于 \mathbb{G}_a 上，两者不交换。因此总群为半直积 \mu_k \rtimes \mathbb{G}_a。证毕。
